빠른 시작 - Friendly Eats

Firebase를 사용하여 간단한 레스토랑 추천 서비스를 빌드하는 방법을 보여주는 샘플 앱입니다. 복합 쿼리, 클라이언트 측 트랜잭션, 하위 컬렉션, 오프라인 지속성 등의 기능을 시연합니다.

자세한 내용은 샘플 페이지 [https://firebaseopensource.com/projects/firebase/friendlyeats-web/]를 참조하세요.
