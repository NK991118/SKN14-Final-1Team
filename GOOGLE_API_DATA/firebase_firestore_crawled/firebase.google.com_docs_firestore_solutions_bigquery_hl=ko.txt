BigQuery와 통합하기

컬렉션을 사용해 정리하기

내 환경설정을 기준으로 콘텐츠를 저장하고 분류하세요.

BigQuery [https://cloud.google.com/bigquery/docs?hl=ko]는 방대한 양의 데이터를 분석할 수 있는 페타바이트 규모의 경제적인 완전 관리형 분석 데이터 웨어하우스입니다. 다음 Firebase Extensions를 사용하여 Cloud Firestore를 BigQuery에 연결할 수 있습니다.

BigQuery로 Cloud Firestore 데이터 스트리밍

Cloud Firestore 컬렉션에서 BigQuery로 실시간 증분 업데이트를 전송합니다.
자세히 알아보기 [https://extensions.dev/extensions/firebase/firestore-bigquery-export]

BigQuery 쿼리 결과를 Cloud Firestore로 내보내기

실시간 전송을 위해 Cloud Firestore에서 BigQuery 쿼리 결과를 예약하고 내보냅니다. 자세히 알아보기 [https://extensions.dev/extensions/googlecloud/bigquery-firestore-export]
