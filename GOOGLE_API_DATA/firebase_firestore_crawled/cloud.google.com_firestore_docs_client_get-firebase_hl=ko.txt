Firebase 기초 알아보기

Firebase 프로젝트를 시작하고 앱에 Firebase를 추가합니다. 프로젝트 관리, 사용자 개인 정보 보호, 앱 출시를 위한 주요 개념 및 권장사항을 살펴보세요.

앱에 Firebase 추가

Firebase 프로젝트를 만들고, 여기에 앱을 등록하고, 제품 및 플랫폼(iOS, [https://firebase.google.com/docs/ios/setup?hl=ko]Android [https://firebase.google.com/docs/android/setup?hl=ko], 웹 [https://firebase.google.com/docs/web/setup?hl=ko])에 대해 Firebase SDK를 통합하는 방법을 알아봅니다.

Firebase 프로젝트 관리

개념 개요, 프로젝트 권한 및 프로젝트 관리 도구와 같은 주제에 대한 심층 분석, 앱 실행 준비를 포함하여 Firebase 프로젝트에 대해 자세히 알아봅니다. Firebase 제품 이해 [https://firebase.google.com/docs/projects/learn-more?hl=ko]와 실행 체크리스트 검토 [https://firebase.google.com/support/guides/launch-checklist?hl=ko]부터 시작하세요.

사용자 개인 정보 보호

사용자 개인 정보 보호. 개인 정보 보호 권장사항을 알아보세요. Firebase로 자신 및 사용자를 보호하는 방법을 알아보세요. Firebase의 개인 정보 보호 및 보안 [https://firebase.google.com/support/privacy?hl=ko]을 참조하세요.
