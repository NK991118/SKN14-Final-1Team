Method: projects.databases.documents.rollback

Stay organized with collections

Save and categorize content based on your preferences.

Rolls back a transaction.

HTTP request
POST https://firestore.googleapis.com/v1beta1/{database=projects/*/databases/*}/documents:rollbackThe URL uses gRPC Transcoding [https://google.aip.dev/127] syntax.

Path parameters
| Parameters |
| --- |
| database | string Required. The database name. In the format: projects/{projectId}/databases/{databaseId}. |

Request body
The request body contains data with the following structure:

| JSON representation |
| --- |
| {   "transaction": string } |

| Fields |
| --- |
| transaction | string (bytes format) Required. The transaction to roll back. A base64-encoded string. |

Response body
If successful, the response body is an empty JSON object.

Authorization scopes
Requires one of the following OAuth scopes:

https://www.googleapis.com/auth/datastore
https://www.googleapis.com/auth/cloud-platform
For more information, see the Authentication Overview [https://cloud.google.com/docs/authentication#authorization-gcp].
