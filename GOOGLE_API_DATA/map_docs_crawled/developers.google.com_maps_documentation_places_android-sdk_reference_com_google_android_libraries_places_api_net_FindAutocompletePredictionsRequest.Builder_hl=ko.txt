setLocationBias

public abstract FindAutocompletePredictionsRequest.Builder [https://developers.google.com/maps/documentation/places/android-sdk/reference/com/google/android/libraries/places/api/net/FindAutocompletePredictionsRequest.Builder] setLocationBias [https://developers.google.com/maps/documentation/places/android-sdk/reference/com/google/android/libraries/places/api/net/FindAutocompletePredictionsRequest.Builder#setLocationBias(com.google.android.libraries.places.api.model.LocationBias)](@Nullable [https://developer.android.com/reference/androidx/annotation/Nullable.html] LocationBias [https://developers.google.com/maps/documentation/places/android-sdk/reference/com/google/android/libraries/places/api/model/LocationBias] locationBias)
Sets the location bias applied for autocomplete predictions. If null, location biases will not be applied.
The autocomplete predictions will not necessarily be within the location specified. To enforce a restriction, use setLocationRestriction [https://developers.google.com/maps/documentation/places/android-sdk/reference/com/google/android/libraries/places/api/net/FindAutocompletePredictionsRequest.Builder#setLocationRestriction(com.google.android.libraries.places.api.model.LocationRestriction)]. You may not set both LocationBias [https://developers.google.com/maps/documentation/places/android-sdk/reference/com/google/android/libraries/places/api/model/LocationBias] and LocationRestriction [https://developers.google.com/maps/documentation/places/android-sdk/reference/com/google/android/libraries/places/api/model/LocationRestriction] simultaneously.
The supported LocationBias [https://developers.google.com/maps/documentation/places/android-sdk/reference/com/google/android/libraries/places/api/model/LocationBias] type is based on which API version you are using. With Places API (New) enabled, both RectangularBounds [https://developers.google.com/maps/documentation/places/android-sdk/reference/com/google/android/libraries/places/api/model/RectangularBounds] and CircularBounds [https://developers.google.com/maps/documentation/places/android-sdk/reference/com/google/android/libraries/places/api/model/CircularBounds] are supported. While if you enable Places API, only RectangularBounds [https://developers.google.com/maps/documentation/places/android-sdk/reference/com/google/android/libraries/places/api/model/RectangularBounds] is supported.

The supported LocationRestriction [https://developers.google.com/maps/documentation/places/android-sdk/reference/com/google/android/libraries/places/api/model/LocationRestriction] type is based on which API version you are using.
If you enable Places API (New), both RectangularBounds [https://developers.google.com/maps/documentation/places/android-sdk/reference/com/google/android/libraries/places/api/model/RectangularBounds] and CircularBounds [https://developers.google.com/maps/documentation/places/android-sdk/reference/com/google/android/libraries/places/api/model/CircularBounds] are supported.
