GMSMapCapabilityFlags

enum GMSMapCapabilityFlags : NSUInteger {}

Flags that represent conditionally-available map capabilities (ones that require a mapID or some
other map setting) that can be used to indicate availability.

GMSMapCapabilityFlagsNone [https://developers.google.com/maps/documentation/ios-sdk/reference/objc/Enums/GMSMapCapabilityFlags#/c:@E@GMSMapCapabilityFlags@GMSMapCapabilityFlagsNone]

No conditional capabilities are enabled on the GMSMapView [https://developers.google.com/maps/documentation/ios-sdk/reference/objc/Classes/GMSMapView.html].

Declaration

Objective-C
GMSMapCapabilityFlagsNone = 0
