Title: Fleet Engine 출시 노트

Fleet Engine 출시 노트

컬렉션을 사용해 정리하기

내 환경설정을 기준으로 콘텐츠를 저장하고 분류하세요.

2025년 5월

변경됨

SearchVehicles를 제외한 모든 메서드에 실험적 기능 [https://developers.google.com/maps/launch-stages?hl=ko#experimental]으로 Fleet Engine에 장소 ID 지원이 추가되었습니다. 사용자는 TerminalLocation [https://developers.google.com/maps/documentation/mobility/fleet-engine/reference/trips/rest/v1/TerminalLocation.html?hl=ko] 또는 LocationInfo [https://developers.google.com/maps/documentation/mobility/fleet-engine/reference/tasks/rest/v1/LocationInfo.html?hl=ko]에서 장소를 지정할 수 있습니다.
여행에서 TripAttributes [https://developers.google.com/maps/documentation/mobility/fleet-engine/reference/trips/rest/v1/providers.trips?hl=ko#TripAttribute]을 설정하는 지원이 추가되었습니다.

2025년 2월

변경됨

Fleet Engine 삭제 메서드 DeleteTrip [https://developers.google.com/maps/documentation/mobility/fleet-engine/journeys/trips/delete-trip?hl=ko], DeleteVehicle [https://developers.google.com/maps/documentation/mobility/fleet-engine/essentials/vehicles/on-demand-delete-vehicle?hl=ko], DeleteTask [https://developers.google.com/maps/documentation/mobility/fleet-engine/journeys/tasks/delete-task?hl=ko], DeleteDeliveryVehicle [https://developers.google.com/maps/documentation/mobility/fleet-engine/essentials/vehicles/scheduled-tasks-delete-vehicle?hl=ko]을 추가했습니다.

2024년 5월

변경됨

Fleet Engine UpdateVehicleLocation, SearchFuzzedVehicles, SearchTasks 메서드가 지원 중단되었습니다.

2024년 3월

변경됨

로그가 250KB를 초과하면 Cloud 로그를 여러 항목으로 분할합니다. 이렇게 하면 전체 로그 데이터에 액세스할 수 있습니다. 로그 분할은 BatchCreateTasksLog [https://developers.google.com/maps/documentation/transportation-logistics/last-mile-fleet-solution/reference/fleet-engine/logging/rest/Shared.Types/BatchCreateTasksLog?hl=ko], ListDeliveryVehiclesLog [https://developers.google.com/maps/documentation/transportation-logistics/last-mile-fleet-solution/reference/fleet-engine/logging/rest/Shared.Types/ListDeliveryVehiclesLog?hl=ko], ListTasksLog [https://developers.google.com/maps/documentation/transportation-logistics/last-mile-fleet-solution/reference/fleet-engine/logging/rest/Shared.Types/ListTasksLog?hl=ko], SearchVehiclesLog [https://developers.google.com/maps/documentation/transportation-logistics/on-demand-rides-deliveries-solution/reference/fleet-engine/logging/rest/Shared.Types/SearchVehiclesLog?hl=ko]에 적용됩니다.
문서는 Last Mile Fleet 솔루션 [https://developers.google.com/maps/documentation/transportation-logistics/last-mile-fleet-solution/shipment-tracking/fleet-engine/logging?hl=ko#split_cloud_logs] 및 주문형 차량 공유 및 배송 [https://developers.google.com/maps/documentation/transportation-logistics/on-demand-rides-deliveries-solution/trip-order-progress/fleet-engine/logging?hl=ko#split_cloud_logs]에서 확인할 수 있습니다.

2024년 2월

공지사항

Fleet Engine UpdateVehicleLocation [https://developers.google.com/maps/documentation/transportation-logistics/on-demand-rides-deliveries-solution/reference/fleet-engine/rpc/maps.fleetengine.v1?hl=ko#maps.fleetengine.v1.VehicleService.UpdateVehicleLocation], SearchFuzzedVehicles [https://developers.google.com/maps/documentation/transportation-logistics/on-demand-rides-deliveries-solution/reference/fleet-engine/rpc/maps.fleetengine.v1?hl=ko#maps.fleetengine.v1.VehicleService.SearchFuzzedVehicles], SearchTasks [https://developers.google.com/maps/documentation/transportation-logistics/last-mile-fleet-solution/reference/fleet-engine/rpc/maps.fleetengine.delivery.v1?hl=ko#maps.fleetengine.delivery.v1.DeliveryService.SearchTasks] 메서드는 2024년 2월 1일부터 지원 중단됩니다. 사용 중단은 2024년 5월 1일에 예정되어 있습니다. 이러한 메서드를 사용하는 경우 문서에 정의된 대체 메서드로 마이그레이션하세요.

주문형 차량 공유 및 배송 API [https://developers.google.com/maps/documentation/transportation-logistics/on-demand-rides-deliveries-solution/reference/fleet-engine/rpc?hl=ko#maps.fleetengine.v1.vehicleservice]
Last Mile Fleet Solution Delivery API [https://developers.google.com/maps/documentation/transportation-logistics/last-mile-fleet-solution/reference/fleet-engine/rpc?hl=ko#maps.fleetengine.delivery.v1.deliveryservice]
