June 30, 2025
Weather API is generally available

Announcing the General Availability (GA) release of the Weather API. See the Release Notes [https://developers.google.com/maps/documentation/weather/release-notes] for information about this release and for all other releases.

If you are a new user, see Set up your Google Cloud project [https://developers.google.com/maps/documentation/weather/cloud-setup] to start the installation process.
April 08, 2025
Weather API in Preview

The Weather API is now available in Preview. With the Weather API, you can request information about current weather conditions, daily weather forecasts, hourly weather forecasts, and hourly weather history for a given location. For more information, see Weather API Overview [https://developers.google.com/maps/documentation/weather/overview].
