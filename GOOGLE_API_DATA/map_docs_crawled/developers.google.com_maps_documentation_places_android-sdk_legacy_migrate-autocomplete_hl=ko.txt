유럽 경제 지역 (EEA) 개발자

청구서 수신 주소가 유럽 경제 지역에 있는 경우 2025년 7월 8일부터
Google Maps Platform EEA 서비스 약관 [https://cloud.google.com/terms/maps-platform/eea?hl=ko]
이 서비스 이용에 적용됩니다. 기능은 리전에 따라 다릅니다.
자세히 알아보기 [https://developers.google.com/maps/comms/eea/faq?hl=ko]

이 기능을 사용하려면 차세대 Places API인 Places API (New)를 사용 설정해야 합니다. 자세한 내용은 API 사용 설정 [https://developers.google.com/maps/documentation/places/android-sdk/legacy/cloud-setup?hl=ko#enabling-apis]을 참고하세요.

Android용 Places SDK는 Place Autocomplete (레거시) [https://developers.google.com/maps/documentation/places/android-sdk/autocomplete?hl=ko]를 지원합니다.
Place Autocomplete (기존)에 익숙하다면 Place Autocomplete (신규) [https://developers.google.com/maps/documentation/places/android-sdk/place-autocomplete?hl=ko]에서 다음과 같은 변경사항이 적용됩니다.

새로운 가격 책정 모델을 사용합니다. 모든 API의 가격 정보는 Android용 Places SDK(신규) [https://developers.google.com/maps/documentation/places/android-sdk/usage-and-billing?hl=ko#pricing-new]를 참고하세요.
Places.initializeWithNewPlacesApiEnabled() [https://developers.google.com/maps/documentation/places/android-sdk/reference/com/google/android/libraries/places/api/Places?hl=ko#initializeWithNewPlacesApiEnabled(android.content.Context,%20java.lang.String)] 메서드를 호출하여 앱을 초기화해야 합니다. Places API 서비스 선택에 대한 자세한 내용은 Google Cloud 프로젝트 설정 [https://developers.google.com/maps/documentation/places/android-sdk/legacy/cloud-setup?hl=ko]을 참고하세요.
FindAutocompletePredictionsRequest [https://developers.google.com/maps/documentation/places/android-sdk/reference/com/google/android/libraries/places/api/net/FindAutocompletePredictionsRequest?hl=ko] 클래스에는 다음 작업을 할 수 있는 새로운 메서드가 포함되어 있습니다.

결과의 형식을 결정하는 데 사용되는 지역 코드를 설정합니다.
예측 오프셋을 설정합니다. 이는 쿼리의 0 기반 유니코드 문자 오프셋입니다.

자세한 내용은 자동 완성 (신규) [https://developers.google.com/maps/documentation/places/android-sdk/place-autocomplete?hl=ko]을 참고하세요.
이제 세션이 Place Details (신규) [https://developers.google.com/maps/documentation/places/ios-sdk/details-place?hl=ko] 또는 Address Validation [https://developers.google.com/maps/documentation/address-validation/overview?hl=ko] 호출로 종료됩니다. 자세한 내용은 자동 완성 (신규) 및 세션 가격 책정 [https://developers.google.com/maps/documentation/places/ios-sdk/place-session-pricing?hl=ko]을 참고하세요.
