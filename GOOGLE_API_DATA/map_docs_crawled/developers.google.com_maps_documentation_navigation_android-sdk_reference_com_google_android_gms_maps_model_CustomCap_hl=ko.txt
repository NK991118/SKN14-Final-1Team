public  final   class
CustomCap
extends Cap [https://developers.google.com/maps/documentation/navigation/android-sdk/reference/com/google/android/gms/maps/model/Cap]

Bitmap overlay centered at the start or end vertex of a Polyline [https://developers.google.com/maps/documentation/navigation/android-sdk/reference/com/google/android/gms/maps/model/Polyline], orientated according to
the direction of the line's first or last edge and scaled with respect to the line's stroke
width. CustomCap can be applied to Polyline with any stroke pattern.
