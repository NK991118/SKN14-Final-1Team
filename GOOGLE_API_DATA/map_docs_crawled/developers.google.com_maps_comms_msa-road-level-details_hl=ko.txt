횡단보도 및 인도 세부정보

차선 표시 및 도로 폭

2025년 2월에서 2025년 8월 사이에 Google Maps Platform에서 이 문서에 나열된 API와 SDK의 기본 지도 스타일에 횡단보도, 인도, 차선 표시, 도로 폭과 같은 도로 수준 세부정보를 추가합니다. 현재 사용 중인 지도 환경과의 호환성을 확인하기 위해, 사용하는 제품에 따라 8월 이전에 도로 수준 세부정보를 사용할 수 있습니다.

2025년 8월 중순 이후에는 API 및 SDK의 모든 버전에서 도로 수준 세부정보가 항상 기본적으로 사용 설정됩니다.

이러한 세부정보는 확대/축소 수준 17 이상에서 표시되며 시간이 지남에 따라 달라질 수 있습니다. 차선 표시를 지원하는 지역에서도 일부 도로에는 차선 표시와 같은 세부정보가 표시되지 않을 수 있습니다.

도로 수준 세부정보를 사용 설정한 상태에서 확대/축소 수준을 17 이상으로 높이면 고속도로 및 간선 도로와 같은 로드맵 기능 유형이 사라지고 '표면'이라는 새로운 도로 세부정보 지도 기능으로 대체됩니다. 도로 지도 지형지물에서 지표면 지도 지형지물로 스타일을 쉽게 전환하려면 클라우드 기반 지도 스타일 지정을 사용하여 확대/축소 수준 스타일을 맞춤설정하면 됩니다. 자세한 내용은 도로 수준 세부정보 미리보기 [https://developers.google.com/maps/documentation/javascript/cloud-customization/zoom-levels-leg?hl=ko#road-detail-preview]를 참고하세요.

제품별 기능 제공 여부

Google Maps Platform에서는 다음 표의 일정에 따라 각 API 및 SDK에 대해 기본적으로 도로 수준 세부정보가 사용 설정된 새 버전을 출시할 예정입니다. 도로 수준 세부정보를 사용하려면 나열된 버전 (또는 이후 버전)으로 업데이트하세요.

선택 해제

표에 나열된 API 또는 SDK 버전을 채택하지 않아 선택 해제합니다.
2025년 8월 중순 이후 Google Maps Platform 제품의 모든 버전에 도로 수준 세부정보가 표시되며 선택 해제 옵션은 더 이상 사용할 수 없습니다.
