Title: Routes Preferred API

Routes Preferred API

컬렉션을 사용해 정리하기

내 환경설정을 기준으로 콘텐츠를 저장하고 분류하세요.

차량 공유 및 배송 솔루션에서 라우팅에는 예상 도착 시간 (ETA)이 포함된 도로 네트워크를 따라 경로를 계산하는 프로세스가 포함됩니다.
이 정보의 품질과 적시 제공은 운전자 생산성을 개선하고 양질의 소비자 경험을 제공하는 데 필수적입니다.

이 요구사항을 해결하고 미션 크리티컬한 차량 공유 및 배송 사용 사례를 지원하기 위해 Routes Preferred API는 성능에 최적화된 Directions API 버전을 제공합니다.
성능 최적화에는 다음이 포함됩니다.

도착예정시간 정확도 개선
대답 지연 시간 감소
대답에 포함된 세부정보를 맞춤설정하는 기능
Google의 서비스 대기열에서 우선순위가 높은 API 요청입니다.
