장기 실행 작업에 대한 비동기 취소를 시작합니다. 서버는 작업이 취소되도록 최선을 다하지만 성공한다고 보장되지 않습니다. 서버에서 이 메서드를 지원하지 않으면 google.rpc.Code.UNIMPLEMENTED가 반환됩니다. 클라이언트는 Operations.GetOperation [https://developers.google.com/maps/documentation/route-optimization/reference/rest/v1/projects.locations.operations/get?hl=ko#google.longrunning.Operations.GetOperation] 또는 다른 메서드를 사용하여 취소가 성공했는지 또는 취소에도 불구하고 작업이 완료되었는지 여부를 확인할 수 있습니다. 취소가 성공하면 작업은 삭제되지 않고 대신 Operation.error [https://developers.google.com/maps/documentation/route-optimization/reference/rest/v1/projects.locations.operations?hl=ko#Operation.FIELDS.error] 값을 갖는 작업이 되며 google.rpc.Status.code [https://developers.google.com/maps/documentation/route-optimization/reference/rest/v1/projects.locations.operations?hl=ko#Status.FIELDS.code]가 1로 설정됩니다. 이 값은 Code.CANCELLED에 해당합니다.

HTTP 요청
POST https://routeoptimization.googleapis.com/v1/{name=operations/**}:cancelURL은 gRPC 트랜스코딩 [https://google.aip.dev/127] 문법을 사용합니다.
