Data-driven styling features are not available to the Navigation SDK. They exists for compile
time compatibility with com.google.android.gms:play-services-maps only.
Represents a map layer containing features of a specific FeatureType whose style can
be overridden on the client.

Returns whether this FeatureLayer [https://developers.google.com/maps/documentation/navigation/android-sdk/reference/com/google/android/gms/maps/model/FeatureLayer] is available, which requires the data-driven styling

the map style associated with this map's ID. If this is false, styling on this FeatureLayer [https://developers.google.com/maps/documentation/navigation/android-sdk/reference/com/google/android/gms/maps/model/FeatureLayer] returns to default and the click listeners are not called.
