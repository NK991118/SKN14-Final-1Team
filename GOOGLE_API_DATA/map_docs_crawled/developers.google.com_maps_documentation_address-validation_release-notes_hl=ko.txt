June 18, 2025
The Address Validation API now supports the following features:

possibleNextAction field in the verdict to help you determine what action to take after receiving the API response.
hasSpellCorrectedComponents field in the verdict.

See Build your validation logic [https://developers.google.com/maps/documentation/address-validation/build-validation-logic] and the API reference docs [https://developers.google.com/maps/documentation/address-validation/reference/rest/v1/TopLevel/validateAddress#possibleNextAction] for more details.
