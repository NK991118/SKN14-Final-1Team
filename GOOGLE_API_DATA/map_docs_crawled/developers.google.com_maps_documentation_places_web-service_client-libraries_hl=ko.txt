Places API (신규) 클라이언트 라이브러리 시작하기

컬렉션을 사용해 정리하기

내 환경설정을 기준으로 콘텐츠를 저장하고 분류하세요.

이 페이지에서는 Places API (New)용 클라이언트 라이브러리를 시작하는 방법을 보여줍니다.

클라이언트 라이브러리에 대한 자세한 내용은 클라이언트 라이브러리 설명 [https://cloud.google.com/apis/docs/client-libraries-explained?hl=ko]을 참고하세요.

클라이언트 라이브러리 설치

Maps JavaScript API자세한 내용은 Places 라이브러리 로드 [https://developers.google.com/maps/documentation/javascript/place-get-started?hl=ko#load-places-library]를 참고하세요.
자바자세한 내용은 Java 개발 환경 설정 [https://cloud.google.com/java/docs/setup?hl=ko] 및 Cloud 클라이언트 라이브러리 사용 [https://cloud.google.com/java/docs/bom?hl=ko]을 참고하세요.

전체 설치 안내는 Java용 Google Places API 클라이언트 [https://github.com/googleapis/google-cloud-java/tree/main/java-maps-places]를 참고하세요.
Gogo get cloud.google.com/go/maps

자세한 내용은 Go 개발 환경 설정 [https://cloud.google.com/go/docs/setup?hl=ko]을 참고하세요.

전체 설치 안내는 Google Places API Client for Go [https://github.com/googleapis/google-cloud-go/tree/main/maps/places/apiv1/placespb]를 참고하세요.
NodeJSnpm install @googlemaps/places

자세한 내용은 Node.js 개발 환경 설정 [https://cloud.google.com/nodejs/docs/setup?hl=ko]을 참고하세요.

전체 설치 안내는 Google Places API Client for NodeJS [https://github.com/googleapis/google-cloud-node/tree/main/packages/google-maps-places]를 참고하세요.
Pythonpip install --upgrade google-maps-places

자세한 내용은 Python 개발 환경 설정 [https://cloud.google.com/python/docs/setup?hl=ko]을 참고하세요.

전체 설치 안내는 Google Places용 Python 클라이언트 [https://github.com/googleapis/google-cloud-python/tree/main/packages/google-maps-places]를 참고하세요.
.NETInstall-Package Google.Maps.Places.V1 -Pre

자세한 내용은 .NET 개발 환경 설정 [https://cloud.google.com/dotnet/docs/setup?hl=ko] 및 Google Places용.NET [https://github.com/googleapis/google-cloud-dotnet/tree/main/apis/Google.Maps.Places.V1]을 참고하세요.

인증 설정

클라이언트 라이브러리를 사용할 때는 애플리케이션 기본 사용자 인증 정보 (ADC) [https://cloud.google.com/docs/authentication/application-default-credentials?hl=ko]를 사용하여 인증합니다. ADC 설정은 애플리케이션 기본 사용자 인증 정보에 대한 사용자 인증 정보 제공 [https://cloud.google.com/docs/authentication/provide-credentials-adc?hl=ko]을 참조하세요.
클라이언트 라이브러리에서 ADC를 사용하는 방법은 클라이언트 라이브러리를 사용하여 인증 [https://cloud.google.com/docs/authentication/client-libraries?hl=ko]을 참조하세요.

클라이언트 라이브러리 사용

사용 정보는 클라이언트 라이브러리 예 [https://developers.google.com/maps/documentation/places/web-service/client-library-examples?hl=ko]를 참고하세요.
