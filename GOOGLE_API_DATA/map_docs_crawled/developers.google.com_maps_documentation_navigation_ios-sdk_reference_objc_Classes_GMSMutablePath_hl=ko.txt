GMSMutablePath

@interface GMSMutablePath : GMSPath [https://developers.google.com/maps/documentation/navigation/ios-sdk/reference/objc/Classes/GMSPath.html]

GMSMutablePath is a dynamic (resizable) array of CLLocationCoordinate2D. All coordinates
must be valid. GMSMutablePath is the mutable counterpart to the immutable GMSPath [https://developers.google.com/maps/documentation/navigation/ios-sdk/reference/objc/Classes/GMSPath.html].
