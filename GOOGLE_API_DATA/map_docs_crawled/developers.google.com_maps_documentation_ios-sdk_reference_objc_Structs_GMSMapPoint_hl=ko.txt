GMSMapPoint

struct GMSMapPoint {}

A point on the map. May represent a projected coordinate.

x is in [-1, 1]. The axis direction is normal: y grows towards North, x grows towards East. (0,
0) is the center of the map.

See GMSProject() and GMSUnproject().
