AerialView

Aerial View API의 서비스 정의입니다.

| LookupVideo |
| --- |
| rpc LookupVideo(LookupVideoRequest) returns (Video) 주소 또는 videoId가 지정된 동영상을 가져옵니다. 응답은 활성 동영상에 대한 재생 URI 집합이 포함된 동영상이거나 보류 중인 동영상에 대한 PROCESSING 상태 또는 동영상이 없는 경우 404 오류입니다. 동영상 수신은 청구 가능한 이벤트이므로 이 메서드의 호출자는 요청 시 반환된 URI를 사용할 준비가 되어 있어야 합니다. 승인 범위 다음 OAuth 범위가 필요합니다. https://www.googleapis.com/auth/cloud-platform |

| LookupVideoMetadata |
| --- |
| rpc LookupVideoMetadata(LookupVideoMetadataRequest) returns (VideoMetadata) 특정 주소 또는 videoId가 지정된 동영상의 메타데이터를 가져옵니다. 이 메서드는 LookupVideo의 무료 호출 대체 메서드이며 재생 URI를 제외한 사용 가능한 모든 동영상 데이터를 반환합니다. 승인 범위 다음 OAuth 범위가 필요합니다. https://www.googleapis.com/auth/cloud-platform |

| RenderVideo |
| --- |
| rpc RenderVideo(RenderVideoRequest) returns (RenderVideoResponse) 동영상이 아직 렌더링되지 않은 경우 렌더기의 대기열에 주소를 추가합니다. 그렇지 않으면 동영상에 대한 메타데이터를 반환합니다. 승인 범위 다음 OAuth 범위가 필요합니다. https://www.googleapis.com/auth/cloud-platform |

| video_id | string RenderVideo에서 반환된 ID |
| address | string 미국 우편 주소. |

MediaFormat

API에서 지원하는 다양한 미디어 형식을 열거합니다.

| 열거형 |
| --- |
| MEDIA_FORMAT_UNSPECIFIED | 기본값 |
| IMAGE | 썸네일 이미지입니다. |
| MP4_HIGH | 고화질 MP4 동영상 |
| MP4_MEDIUM | 중간 품질의 MP4 동영상입니다. |
| MP4_LOW | 저화질 MP4 동영상 |
| DASH | 대부분의 기기에서 가변 비트 전송률 스트리밍을 위한 MPEG-DASH입니다. |
| HLS | Apple 기기에서 가변 비트 전송률 스트리밍을 위한 HLS입니다. |

RenderVideoRequest

AerialView.RenderVideo에 대한 요청 메시지입니다.

| 필드 |
| --- |
| address | string 필수 항목입니다. 동영상에서 렌더링할 위치의 미국 우편 주소입니다. |

RenderVideoResponse

AerialView.RenderVideo의 응답 메시지입니다.

| 필드 |
| --- |
| state | State 렌더링 요청의 현재 상태입니다. |
| metadata | VideoMetadata 동영상의 메타데이터를 포함하며, 상태가 ACTIVE인 경우에만 설정됩니다. |

URI

지정된 동영상 형식의 모든 URI를 포함합니다.

| 필드 |
| --- |
| landscape_uri | string 가로 방향 미디어의 서명된 단기 URI입니다. |
| portrait_uri | string 세로 방향 미디어의 서명된 단기 URI입니다. |

동영상

동영상에 대한 모든 데이터를 캡슐화하는 객체입니다.

| 필드 |
| --- |
| uris | map<string, Uris> 미디어 유형을 URI에 매핑합니다. 이 필드는 ACTIVE 동영상에만 포함됩니다. 키는 MediaFormat의 열거형 값입니다. |
| state | State 렌더링 요청의 현재 상태입니다. |
| metadata | VideoMetadata 동영상의 메타데이터를 포함하며, 상태가 ACTIVE인 경우에만 설정됩니다. |

상태

동영상의 다양한 상태입니다.

| 열거형 |
| --- |
| STATE_UNSPECIFIED | 기본값 이 값은 사용되지 않습니다. |
| PROCESSING | 현재 동영상을 처리 중입니다. |
| ACTIVE | 동영상 렌더링이 완료되었으며 LookupVideo를 통해 볼 수 있습니다. |
| FAILED | 동영상을 렌더링할 수 없습니다. |

VideoMetadata

동영상 ID, 길이 등 동영상에 관한 메타데이터를 포함합니다.

| 필드 |
| --- |
| video_id | string 출력 전용. 동영상 ID 및 동영상 검색 시 권장되는 방법입니다. |
| capture_date | Date 출력 전용. 동영상에 사용된 이미지가 캡처된 날짜입니다. 이 방식은 월 단위로 설정할 수 있습니다. |
| duration | Duration 출력 전용. 동영상의 길이입니다. |
| state | State 출력 전용. 렌더링 요청의 현재 상태입니다. |
