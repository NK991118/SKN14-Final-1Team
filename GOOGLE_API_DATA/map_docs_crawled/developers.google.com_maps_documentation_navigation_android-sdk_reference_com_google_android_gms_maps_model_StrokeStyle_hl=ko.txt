getStamp
()

Gets stamp style of a stroke.
The stamp style's image is repeated over the stroke. If the image has transparent or
semi-transparent portions, the underlying stroke color will show through in those places.
Otherwise, the stamped texture/image will completely cover the base stroke.

Returns
the stamp style of a stroke.
