설정 및 데모
플랫폼 선택: Android [https://developers.google.com/maps/documentation/android-sdk/utility/setup?hl=ko] iOS [https://developers.google.com/maps/documentation/ios-sdk/utility/setup?hl=ko]
이 페이지에서는 iOS용 Maps SDK의 유틸리티 라이브러리 [https://github.com/googlemaps/google-maps-ios-utils]를 설정하는 방법을 설명합니다.
이러한 유틸리티를 사용해 보려면 다음 중 하나를 수행하세요.
기존 Xcode 프로젝트에 유틸리티 라이브러리를 설치합니다 [https://developers.google.com/maps/documentation/ios-sdk/utility/setup?hl=ko#get-utility]. 이러한 유틸리티를 자체 프로젝트에 추가합니다.
데모 앱 설치 및 실행 [https://developers.google.com/maps/documentation/ios-sdk/utility/setup?hl=ko#get-demo]. 데모 앱은 유틸리티를 사용해 보려면 필요한 모든 코드가 포함된 독립형 프로젝트입니다.
Xcode 프로젝트에 iOS용 Maps SDK 유틸리티 라이브러리 추가
다음 절차에 따라 Xcode 작업 영역에 유틸리티 라이브러리를 추가합니다.
앱에 API 키 추가
API 키를 생성하고 Xcode 프로젝트에 추가하는 방법은 Xcode 프로젝트 설정 [https://developers.google.com/maps/documentation/ios-sdk/config?hl=ko]을 참고하세요.
프로젝트 빌드 및 실행
API 키를 추가한 후 프로젝트를 빌드하고 실행합니다.
앱에서 유틸리티 사용
앱에서 유틸리티를 사용하는 방법에 관한 자세한 내용은 각 유틸리티에 관한 문서를 참고하세요.
마커 클러스터링 [https://developers.google.com/maps/documentation/ios-sdk/utility/marker-clustering?hl=ko]
Quadtree [https://developers.google.com/maps/documentation/ios-sdk/utility/quadtree?hl=ko]
KML 렌더링 [https://developers.google.com/maps/documentation/ios-sdk/utility/kml?hl=ko]
GeoJSON 렌더링 [https://developers.google.com/maps/documentation/ios-sdk/utility/geojson?hl=ko]
히트맵 [https://developers.google.com/maps/documentation/ios-sdk/utility/heatmap?hl=ko]
데모 앱 설치 및 실행
유틸리티 라이브러리에는 각 유틸리티를 설치하고 사용해 볼 수 있는 데모 앱이 포함되어 있습니다.
데모 앱을 설치합니다.

데모 앱에 API 키 추가
API 키 가져오기 [https://developers.google.com/maps/documentation/ios-sdk/get-api-key?hl=ko]에서 앱의 API 키를 생성합니다. 이제 이 키를 데모 앱의 Xcode 프로젝트에 추가합니다.
참고: API 키에서 애플리케이션 제한사항을 사용하는 경우 키에 앱의 번들 식별자를 추가해야 합니다. 예를 들어 데모 앱을 실행하는 경우 번들 식별자는 com.google.gmsutils.DemoApp입니다. 키 제한에 관한 자세한 내용은 API 키 가져오기 [https://developers.google.com/maps/documentation/ios-sdk/get-api-key?hl=ko]를 참고하세요.
데모 앱 빌드 및 실행
API 키를 추가한 후 프로젝트를 빌드하고 실행합니다.
데모의 Swift 버전을 실행하는 경우 앱에서 볼 유틸리티를 선택할 수 있습니다.
Objective-C 버전의 데모를 실행하면 마커 클러스터링 유틸리티의 데모가 표시됩니다. 그런 다음 데모의 Objective-C 버전을 수정하여 각 유틸리티를 사용해 볼 수 있습니다. 자세한 내용은 아래의 앱에 유틸리티 추가 [https://developers.google.com/maps/documentation/ios-sdk/utility/setup?hl=ko#add_utility] 섹션을 참고하세요.
