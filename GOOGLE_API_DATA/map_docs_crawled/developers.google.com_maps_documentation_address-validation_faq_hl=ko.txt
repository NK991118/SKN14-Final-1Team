Address Validation API 자주 묻는 질문(FAQ)

API에 비율 제한이 있나요?

Address Validation API의 기본 비율 제한은 분당 쿼리 수 (QPM) 6,000개입니다. 자세한 내용은 사용량 한도 및 결제 [https://developers.google.com/maps/faq?hl=ko#usage-limits]를 참고하세요.

Address Validation API는 CASS Certified™인가요?

미국 우편 서비스 (USPS®)1 [https://developers.google.com/maps/documentation/address-validation/faq?hl=ko#fn1]는 주소 확인 제공업체를 지원하고 인증하기 위해 코딩 정확도 지원 시스템 (CASS™) [https://postalpro.usps.com/certifications/cass]을 유지합니다. Address Validation API는 주소 확인 프로세스의 일부로 선택적으로 사용할 수 있는 CASS Certified™ 서비스입니다.

자세한 내용은 주소 유효성 검사 [https://developers.google.com/maps/documentation/address-validation/requests-validate-address?hl=ko]를 참고하세요.

Google Maps Platform은 미국 우정청®의 비독점 라이선스 사용자입니다. 다음 상표는 미국 우정청®의 소유이며 허가를 받아 사용됩니다. United States Postal Service®, CASS™, CASS Certified™ ↩ [https://developers.google.com/maps/documentation/address-validation/faq?hl=ko#fnref1]
