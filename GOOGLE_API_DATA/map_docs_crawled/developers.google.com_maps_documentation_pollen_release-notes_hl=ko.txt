September 10, 2024
Expanded plant coverage in Japan

The Pollen API now includes information about the Japanese cypress and Japanese cedar tree species. For more information, see Country and region coverage [https://developers.google.com/maps/documentation/pollen/coverage].
October 26, 2023
Pollen API

Announcing the General Availability (GA) release of the Pollen API. See the Release Notes [https://developers.google.com/maps/documentation/pollen/release-notes] for information about this release and for all other releases.

If you are a new user, see Set Up in the Google Cloud Console [https://developers.google.com/maps/documentation/pollen/cloud-setup] to start the installation process.
