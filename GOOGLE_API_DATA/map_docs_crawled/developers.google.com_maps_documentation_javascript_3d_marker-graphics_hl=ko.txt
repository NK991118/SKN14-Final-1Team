3D 마커 API를 사용하면 기본 마커 아이콘을 맞춤 그래픽 이미지로 바꿀 수 있습니다. 글리프를 이미지 파일로 바꿀 수도 있습니다. 이 페이지에서는 다음 방법을 보여줍니다.

맞춤 그래픽 파일 사용 [https://developers.google.com/maps/documentation/javascript/3d/marker-graphics?hl=ko#custom_graphic]
맞춤 이미지 파일 사용 [https://developers.google.com/maps/documentation/javascript/3d/marker-graphics?hl=ko#png]
인라인 SVG 사용 [https://developers.google.com/maps/documentation/javascript/3d/marker-graphics?hl=ko#inline_svg]
장소 아이콘 이미지를 글리프로 사용 [https://developers.google.com/maps/documentation/javascript/3d/marker-graphics?hl=ko#place_icon]

3D 마커 API는 PNG, JPEG, GIF, SVG, TIFF 등 모든 일반적인 이미지 파일 형식 [https://developer.mozilla.org/docs/Web/Media/Formats/Image_types#common_image_file_types]을 지원합니다.

지도에 마커를 추가하려면 먼저 marker 라이브러리를 로드 [https://developers.google.com/maps/documentation/javascript/3d/marker-add?hl=ko]해야 합니다. 이 라이브러리는 Marker3DElement 및 PinElement 클래스를 제공합니다.

맞춤 그래픽 파일 사용

그래픽 파일 리소스 (예: SVG)를 참조하는 img 요소를 Marker3DElement.content 옵션에 전달하여 맞춤 이미지를 사용하여 마커를 만들 수 있습니다.

이미지 파일 사용

PNG를 사용하여 맞춤 이미지가 있는 마커를 만듭니다.
beachFlagMarker.append(templateForImg);
map.append(beachFlagMarker);

인라인 SVG 사용

인라인 SVG를 사용해야 하는 경우도 있습니다. SVG XML 요소를 Marker3DElement.content 옵션에 전달하여 인라인 SVG 이미지를 사용하여 마커를 만들 수 있습니다.

markerWithCustomSvg.append(templateForSvg);
