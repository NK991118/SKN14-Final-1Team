개요
플랫폼 선택: Android [https://developers.google.com/maps/documentation/android-sdk/advanced-markers/overview?hl=ko] iOS [https://developers.google.com/maps/documentation/ios-sdk/advanced-markers/overview?hl=ko] JavaScript [https://developers.google.com/maps/documentation/javascript/advanced-markers/overview?hl=ko]
고급 마커를 사용하면 기본 마커의 배경, 테두리, 글리프 텍스트, 충돌 동작을 맞춤설정하고 기본 마커 아이콘을 맞춤 그래픽 이미지로 대체할 수 있습니다.
중요: 고급 마커 기능은 iOS 14 이상용 Google 지도 SDK 8.0.0에서 출시되었습니다. iOS 13 이하 버전은 Google 지도 SDK 8.0.0에서 지원되지 않습니다.
시작하기
고급 마커에는 유효한 mapID [https://developers.google.com/maps/documentation/get-map-id?hl=ko]가 필요합니다. 지도 ID는 Google 지도의 단일 인스턴스를 나타내는 고유 식별자입니다. Google Cloud 콘솔 [https://developers.google.com/maps/documentation/ios-sdk/cloud-setup?hl=ko]에서 언제든지 지도 ID를 만들고 지도 ID와 연결된 스타일을 업데이트할 수 있습니다.
중요: 테스트의 경우 앱 코드에서 mapId: DEMO_MAP_ID [https://developers.google.com/maps/documentation/ios-sdk/reference/objc/Classes/GMSMapID?hl=ko]를 사용하면 지도 ID [https://developers.google.com/maps/documentation/get-map-id?hl=ko]를 만들고 구성하는 단계를 건너뛸 수 있습니다. DEMO_MAP_ID는 테스트 용도로만 사용됩니다. 프로덕션 환경에서는 DEMO_MAP_ID를 사용하지 마세요.
