경로 데모
경로 데모를 사용하면 출발지와 목적지를 위도/경도 좌표 쌍 또는 장소 ID로 지정할 수 있습니다. 위도/경도 좌표를 복사하려면 지도에서 위치를 찾아 클릭한 다음 위치를 양식에 붙여넣습니다.
경로 가져오기를 선택하면 데모에서 computeRoutes [https://developers.google.com/maps/documentation/javascript/reference/route?hl=ko#Route.computeRoutes] 메서드의 응답을 지도에 경로로 표시합니다.
