Title: TaskOutcome

TaskOutcome

컬렉션을 사용해 정리하기

내 환경설정을 기준으로 콘텐츠를 저장하고 분류하세요.

태스크 실행 시도의 결과입니다. TaskState가 닫히면 TaskOutcome는 성공적으로 완료되었는지 여부를 나타냅니다.

| 열거형 |
| --- |
| TASK_OUTCOME_UNSPECIFIED | 값이 설정되기 전에 작업 결과가 나옵니다. |
| SUCCEEDED | 태스크가 완료되었습니다. |
| FAILED | 태스크를 완료할 수 없거나 태스크가 취소되었습니다. |
