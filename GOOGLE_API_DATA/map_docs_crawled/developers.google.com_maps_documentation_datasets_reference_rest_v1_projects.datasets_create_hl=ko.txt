지정된 프로젝트의 새 데이터 세트를 만듭니다.

HTTP 요청
POST https://mapsplatformdatasets.googleapis.com/v1/{parent=projects/*}/datasetsURL은 gRPC 트랜스코딩 [https://google.aip.dev/127] 문법을 사용합니다.

경로 매개변수
| 매개변수 |
| --- |
| parent | string 필수 항목입니다. 데이터 세트를 소유할 상위 프로젝트입니다. 형식: projects/{project} |

요청 본문
요청 본문에 Dataset [https://developers.google.com/maps/documentation/datasets/reference/rest/v1/projects.datasets?hl=ko#Dataset]의 인스턴스가 포함됩니다.

응답 본문
성공한 경우 응답 본문에 새로 생성된 Dataset [https://developers.google.com/maps/documentation/datasets/reference/rest/v1/projects.datasets?hl=ko#Dataset]의 인스턴스가 포함됩니다.

승인 범위
다음 OAuth 범위가 필요합니다.

https://www.googleapis.com/auth/cloud-platform

IAM 권한
parent 리소스에 대해서는 다음 IAM [https://cloud.google.com/iam/docs?hl=ko] 권한이 필요합니다.

mapsplatformdatasets.datasets.create
자세한 내용은 IAM 문서 [https://cloud.google.com/iam/docs?hl=ko]를 참조하세요.
