Title: Fleet Engine API for scheduled tasks

Fleet Engine API for scheduled tasks

컬렉션을 사용해 정리하기

내 환경설정을 기준으로 콘텐츠를 저장하고 분류하세요.

예약된 작업을 위한 서비스입니다.

서비스: fleetengine.googleapis.com
RPC 클라이언트 스터브를 만들려면 서비스 이름 fleetengine.googleapis.com이 필요합니다.

maps.fleetengine.delivery.v1.DeliveryService [https://developers.google.com/maps/documentation/mobility/fleet-engine/reference/tasks/rpc/maps.fleetengine.delivery.v1?hl=ko#maps.fleetengine.delivery.v1.DeliveryService]

| 메서드 |
| --- |
| BatchCreateTasks | 새 Task 객체를 일괄적으로 만들고 반환합니다. |
| CreateDeliveryVehicle | 새 DeliveryVehicle를 만들고 반환합니다. |
| CreateTask | 새 Task 객체를 만들고 반환합니다. |
| DeleteDeliveryVehicle | Fleet Engine에서 DeliveryVehicle을 삭제합니다. |
| DeleteTask | 단일 할 일을 삭제합니다. |
| GetDeliveryVehicle | 지정된 DeliveryVehicle 인스턴스를 반환합니다. |
| GetTask | Task에 관한 정보를 가져옵니다. |
| GetTaskTrackingInfo | 지정된 TaskTrackingInfo 인스턴스를 반환합니다. |
| ListDeliveryVehicles | 지정된 필터링 기준을 충족하는 모든 DeliveryVehicle를 가져옵니다. |
| ListTasks | 지정된 필터링 기준을 충족하는 모든 Task를 가져옵니다. |
| UpdateDeliveryVehicle | 업데이트된 DeliveryVehicle 데이터를 Fleet Engine에 쓰고 DeliveryVehicle에 Tasks를 할당합니다. |
| UpdateTask | Task 데이터를 업데이트합니다. |
