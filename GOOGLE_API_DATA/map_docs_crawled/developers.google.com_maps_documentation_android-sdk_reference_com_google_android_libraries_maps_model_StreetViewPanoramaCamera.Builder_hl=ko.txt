확대/축소
(플로팅 확대/축소)

카메라의 확대/축소 수준을 설정합니다. 원래 확대/축소 수준은 0으로 설정됩니다. 확대/축소 1은
배율을 2배로 늘립니다. 확대/축소는 0과 최대 확대/축소 수준 사이로 고정됩니다. 최대 확대/축소 수준은 파노라마에 따라 다를 수 있습니다. 고정은 이 범위를 벗어나는 모든 값이 범위 내의 가장 가까운 최댓값으로 설정된다는 것을 의미합니다. 예를 들어 -1 값은 0으로 설정됩니다. 또 다른 예: 파노라마의 최대 확대/축소 값이 19이고 값이 20으로 지정된 경우 19로 설정됩니다. 카메라 확대/축소는 정수 값이 아니어도 됩니다.
