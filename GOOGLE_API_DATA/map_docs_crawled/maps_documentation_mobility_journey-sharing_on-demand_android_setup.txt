Android Consumer SDK 다운로드
플랫폼 선택: Android [https://developers.google.com/maps/documentation/mobility/journey-sharing/on-demand/android/setup?hl=ko] iOS [https://developers.google.com/maps/documentation/mobility/journey-sharing/on-demand/ios/setup?hl=ko] JavaScript [https://developers.google.com/maps/documentation/mobility/journey-sharing/on-demand/javascript/setup?hl=ko]
Android 소비자 앱에서 주문형 이동의 여정을 공유하려면 소비자 SDK를 사용하세요. Consumer SDK를 사용하여 여정을 추적하는 방법에 관한 자세한 내용은 Android에서 여정 추적하기 [https://developers.google.com/maps/documentation/mobility/journey-sharing/on-demand?hl=ko]를 참고하세요.
예를 보려면 GitHub의 Android 샘플 앱 [https://github.com/googlemaps/android-on-demand-rides-deliveries-samples] 또는 결합된 샘플 [https://github.com/googlemaps/on-demand-rides-deliveries-samples]을 참고하세요.
Android 소비자 앱용 Consumer SDK를 설정하려면 다음 단계를 따르세요.
최소 시스템 요구사항 확인하기 [https://developers.google.com/maps/documentation/mobility/journey-sharing/on-demand/android/setup?hl=ko#check-min]
Consumer SDK 가져오기 [https://developers.google.com/maps/documentation/mobility/journey-sharing/on-demand/android/setup?hl=ko#maven]
Google Cloud 콘솔 프로젝트 구성 [https://developers.google.com/maps/documentation/mobility/journey-sharing/on-demand/android/configure-project?hl=ko]
인증 토큰 가져오기 [https://developers.google.com/maps/documentation/mobility/journey-sharing/on-demand/android/auth-sdk?hl=ko]
소비자 SDK 초기화 [https://developers.google.com/maps/documentation/mobility/journey-sharing/on-demand/android/init-sdk?hl=ko]
최소 시스템 요구사항 확인하기
사용 중인 Consumer SDK 버전의 최소 요구사항은 출시 노트를 참고하세요. 출시 노트 [https://developers.google.com/maps/documentation/mobility/services/resources/relnotes_consumer_sdk_android?hl=ko]를 참고하세요.
Consumer SDK 사용: 앱이 minSdkVersion 23 이상을 타겟팅해야 합니다.
소비자 SDK를 사용하는 앱을 실행하려면 Android 모바일 기기가 다음 최소 요구사항을 충족해야 합니다.
Android 6.0 (API 수준 23) 이상을 실행합니다.
Google Play 서비스 [https://developers.google.com/android/guides/overview?hl=ko]가 설치되어 있어야 합니다.
Google Maven 저장소에서 소비자 SDK 가져오기
Google Maven 저장소에서 Consumer SDK 버전 2.1.0 이상을 가져옵니다. 이전에 사용된 비공개 저장소 채널은 지원이 중단되었습니다.
