소개: Fleet Engine 설정
이 문서에서는 Fleet Engine 서비스를 설정하고 차량 운영자, 운전자, 소비자 사용자에게 필요한 관련 도구를 사용 설정하는 데 필요한 모든 주제를 다룹니다.
Fleet Engine의 보안 이 가이드에서는 서비스 계정의 기본 흐름과 Fleet Engine 인증 및 승인 모델을 다룹니다. 앱과 Fleet Engine 사용자를 위한 올바른 액세스 메커니즘으로 프로젝트를 설정하는 방법을 가장 잘 이해하는 데 도움이 되는 중요한 필수 정보입니다.
Fleet Engine은 승인 토큰을 사용하여 데이터 액세스를 제한하고 시스템에 추가 보안을 제공합니다. 서비스 계정에서 이러한 보안 토큰을 발급합니다. Cloud 프로젝트를 설정하기 전에 이 문서를 읽고 Fleet Engine 역할과 보안 토큰을 더 잘 이해하세요.
Fleet Engine 프로젝트 설정 Google Cloud 콘솔에서 Fleet Engine을 설정하고, 명령줄 인터페이스를 사용하여 설정을 완료하고, 설정을 확인하고 문제를 해결하는 간단한 안내입니다.
