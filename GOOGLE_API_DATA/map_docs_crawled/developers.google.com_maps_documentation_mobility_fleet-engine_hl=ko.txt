컬렉션을 사용해 정리하기

[https://developers.google.com/maps/documentation/mobility/fleet-engine/essentials?hl=ko]

규모에 맞게 Fleet 관리 [https://developers.google.com/maps/documentation/mobility/fleet-engine/essentials?hl=ko]

Fleet Engine은 차량 Fleet의 상태를 관리하여 Fleet 운영자 전반에 일관된 가시성을 확보할 수 있도록 설계된 백엔드 시스템입니다. 또한 차량 및 운전자 활동에 대한 데이터를 제공하고 수신하는 데 사용하는 다양한 API를 제공합니다.

Fleet Engine은 다양한 SDK와도 호환되므로 운전자와 소비자가 비즈니스에서 제공하는 배송, 탑승, 기타 서비스의 상태를 관리하고 추적할 수 있습니다.

가이드 살펴보기 [https://developers.google.com/maps/documentation/mobility/fleet-engine/essentials?hl=ko]

시작하기

워크플로에 Fleet Engine을 통합하려면 다음 기본 단계를 따르세요.

[https://developers.google.com/maps/documentation/mobility/fleet-engine/essentials/set-up-fleet?hl=ko]

Fleet Engine 설정 [https://developers.google.com/maps/documentation/mobility/fleet-engine/essentials/set-up-fleet?hl=ko]

Fleet Engine 서비스를 설정하고 Fleet 운영자, 운전자, 일반 사용자에게 필요한 관련 도구를 사용 설정합니다.

local_shipping [https://developers.google.com/maps/documentation/mobility/fleet-engine/essentials/vehicles?hl=ko]

차량 만들기 [https://developers.google.com/maps/documentation/mobility/fleet-engine/essentials/vehicles?hl=ko]

Fleet Engine에서 차량을 만들고 관리하는 방법입니다.

devices [https://developers.google.com/maps/documentation/mobility/driver-sdk?hl=ko]

앱에 드라이버 SDK 통합 [https://developers.google.com/maps/documentation/mobility/driver-sdk?hl=ko]

Fleet Engine과 호환되도록 Driver SDK를 드라이버 앱에 통합합니다.

rebase_edit [https://developers.google.com/maps/documentation/mobility/fleet-engine/journeys/trips?hl=ko]

여정 만들기 및 관리 [https://developers.google.com/maps/documentation/mobility/fleet-engine/journeys/trips?hl=ko]

Google 지도 모빌리티 서비스를 사용하여 주문형 이동이나 예약된 작업을 만들고 작업해 보세요. 자세한 내용은 주문형 이동 [https://developers.google.com/maps/documentation/mobility/fleet-engine/journeys/trips?hl=ko] 또는 예약된 작업 [https://developers.google.com/maps/documentation/mobility/fleet-engine/journeys/tasks?hl=ko]을 참고하세요.
