iOS용 Places SDK 코드 샘플
플랫폼 선택: Android [https://developers.google.com/maps/documentation/places/android-sdk/examples?hl=ko] iOS [https://developers.google.com/maps/documentation/places/ios-sdk/examples?hl=ko] JavaScript [https://developers.google.com/maps/documentation/javascript/examples?hl=ko]
iOS용 Places SDK와 함께 제공되는 GitHub [https://github.com/googlemaps-samples/maps-sdk-for-ios-samples/tree/main/GoogleMaps]의 iOS용 Places SDK 데모 앱은 장소 자동 완성, 장소 사진 등 다양한 기능을 보여줍니다. 또한 이 개발자 가이드의 각 페이지에 코드 스니펫이 나와 있습니다. 앱을 가져와 빌드하고, API 키를 추가하고, 데모를 보고, 앱의 시작점으로 제공된 샘플 코드를 사용하세요.
샘플 앱을 실행하면 기기에서 실행할 수 있는 샘플 목록이 표시됩니다. 옵션 중 하나를 선택합니다.
문제 해결: 샘플 앱이 실행되었지만 지도가 표시되지 않으면 API 키 사용 [https://developers.google.com/maps/documentation/places/ios-sdk/get-api-key?hl=ko]에 설명된 대로 API 키를 앱의 매니페스트 파일에 추가했는지 확인하세요.
로컬에서 전체 샘플 앱 실행
iOS용 Places SDK 샘플 앱은 GitHub [https://github.com/googlemaps-samples/maps-sdk-for-ios-samples/tree/main/GoogleMaps]에서 다운로드 보관 파일 [https://github.com/googlemaps-samples/maps-sdk-for-ios-samples/archive/main.zip]로 제공됩니다. 다음 단계에 따라 iOS용 Places SDK 샘플 앱을 설치하고 사용해 보세요.
GitHub [https://github.com/googlemaps-samples/maps-sdk-for-ios-samples/tree/master/GooglePlaces]에서 코드 샘플 보관 파일을 다운로드 [https://github.com/googlemaps-samples/maps-sdk-for-ios-samples/archive/master.zip]하고 보관 파일을 압축해제합니다.
터미널 창을 열고 샘플 파일을 펼친 디렉터리로 이동한 다음 GooglePlaces 디렉터리로 드릴다운합니다.
Google Cloud 콘솔 프로젝트에 iOS용 Places SDK [https://console.cloud.google.com/flows/enableapi?apiid=placesios%2Cmaps_ios_backend&%3BkeyType=CLIENT_SIDE_IOS&%3Breusekey=true&hl=ko]를 사용 설정합니다.
Places API 또는 Places API (신규)가 사용 설정된 [https://developers.google.com/maps/documentation/places/ios-sdk/cloud-setup?hl=ko#enabling-apis] 프로젝트에서 API 키를 가져옵니다 [https://developers.google.com/maps/documentation/places/ios-sdk/get-api-key?hl=ko].
SDKDemoAPIKey 파일을 수정하고 API 키를 적절한 상수에 붙여넣습니다. 예를 들면 다음과 같습니다.
수정하기 위해 SDKDemoAPIKey 파일을 잠금 해제하라는 Xcode 메시지가 표시되면 잠금 해제를 선택합니다.
있는 경우 다음 줄을 삭제합니다. 이 줄은 문제를 등록하는 데 사용됩니다.
앱을 빌드합니다 [https://developer.apple.com/documentation/xcode/building-and-running-an-app].
빌드가 실패하거나 앱을 처음 실행할 때 API 키에 관한 오류와 함께 비정상 종료되는 경우 SDKDemoAPIKey 파일에 필요한 키를 제공했는지 확인합니다.
GooglePlaces 샘플을 실행하면 iOS 시뮬레이터 창에 장소 데모 목록이 표시됩니다.
GooglePlacesDemos에서 내 위치에 액세스하도록 허용하라는 메시지가 표시되면 허용을 선택합니다.
이제 선택한 샘플을 실행할 준비가 되었습니다.
