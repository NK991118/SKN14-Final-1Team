주소에 부속 건물 추가 - 예시 (미국만 해당)

컬렉션을 사용해 정리하기

내 환경설정을 기준으로 콘텐츠를 저장하고 분류하세요.

이 문서에서는 주소 유효성 검사 API가 시스템에서 하위 건물 추가 동작을 보장하는 응답 신호를 제공하는 여러 실제 시나리오를 설명합니다. 이러한 신호는 미국 주소에만 사용할 수 있습니다.
컨텍스트는 유효성 검사 로직 빌드의 워크플로 예시 [https://developers.google.com/maps/documentation/address-validation/build-validation-logic?hl=ko#example-workflows]를 참고하세요.
참고: 여기에 나온 예는 설명용이며 모든 시나리오를 다루지는 않습니다.
일반적인 예: 하위 건물 추가

이 시나리오에서는 시스템에서 고객에게 주소에 호수를 추가하라는 메시지를 표시할 수 있는 주소를 보여줍니다.
