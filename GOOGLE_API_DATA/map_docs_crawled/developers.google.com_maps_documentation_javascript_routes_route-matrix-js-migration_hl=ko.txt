RouteMatrix 클래스는 Distance Matrix Service, Maps JavaScript API (레거시) [https://developers.google.com/maps/documentation/javascript/legacy/distancematrix?hl=ko]를 대체합니다.
이 페이지에서는 기존 거리 행렬 서비스와 새로운 JavaScript 라이브러리의 차이점을 설명하고 비교를 위한 코드를 제공합니다.

Distance Matrix API (기존)와 Route Matrix 클래스 (베타) 비교

다음 표에서는 기존 Distance Matrix API [https://developers.google.com/maps/documentation/javascript/reference/next/distance-matrix?hl=ko]와 RouteMatrix [https://developers.google.com/maps/documentation/javascript/reference/route-matrix?hl=ko] 클래스의 요청 매개변수를 비교합니다.

| 거리 행렬 서비스 (기존) | RouteMatrix (베타) |
| --- | --- |
| 필수 매개변수 |
| origins | origins |
| destinations | destinations |
| 선택적 매개변수 |
| travelMode | travelMode |
| transitOptions | transitPreference |
| arrivalTime | arrivalTime |
| drivingOptions | departureTime, trafficModel |
| unitSystem | units |
| avoidHighways, avoidTolls | RouteModifiers |
