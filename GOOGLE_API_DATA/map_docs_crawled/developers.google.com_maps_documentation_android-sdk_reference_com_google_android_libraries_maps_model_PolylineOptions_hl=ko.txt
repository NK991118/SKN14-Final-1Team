다중선의 시작 꼭짓점에 캡을 설정합니다. 기본 시작 한도는 ButtCap [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/model/ButtCap?hl=ko]입니다.

매개변수
| startCap |  |
| --- | --- |

반환 값
이 PolylineOptions [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/model/PolylineOptions?hl=ko] 객체에는 새로운 시작 한도가 설정되어 있습니다.

공개
PolylineOptions [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/model/PolylineOptions?hl=ko]

표시
(boolean visible)

다중선의 공개 상태를 지정합니다. 기본 공개 상태는 true입니다.

매개변수
| 표시 |  |
| --- | --- |

반환 값
이 PolylineOptions [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/model/PolylineOptions?hl=ko] 객체에 새로운 공개 상태 설정을 적용합니다.

공개
PolylineOptions [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/model/PolylineOptions?hl=ko]

너비
(부동 소수점 너비)

다중선의 너비를 화면 픽셀로 설정합니다. 기본값은 10입니다.

매개변수
| 너비 |  |
| --- | --- |

반환 값
이 PolylineOptions [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/model/PolylineOptions?hl=ko] 객체에는 새로운 너비가 설정되어 있습니다.

공개
PolylineOptions [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/model/PolylineOptions?hl=ko]

zIndex
(부동 소수점 수 zIndex)

폴리라인의 zIndex(즉, 폴리라인을 그리는 순서)를 지정합니다. zIndex에 대한 자세한 내용은 이 클래스 상단에 있는 문서를 참조하세요.

매개변수
| zIndex |  |
| --- | --- |

반환 값
새로운 zIndex가 설정된 PolylineOptions [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/model/PolylineOptions?hl=ko] 객체를 반환합니다.
