결제를 사용 설정해야 하나요?

Aerial View API를 사용하려면 유효한 결제 계정이 필요합니다. 결제 사용 설정 [https://developers.google.com/maps/documentation/aerial-view/cloud-setup?hl=ko#billing]에서 결제 계정으로 프로젝트를 설정하는 방법을 알아보세요.

API는 어디에서 지원되나요?

Aerial View API는 미국 우편 주소에서만 사용할 수 있습니다.
