GMSNavigationDirectionsListController

@interface GMSNavigationDirectionsListController : NSObject

A controller that manages and displays a directions list. The controller maintains its
own states internally based on the associated navigator. The directionsListView is updated
automatically based on navigator states.

directionsListView [https://developers.google.com/maps/documentation/navigation/ios-sdk/reference/objc/Classes/GMSNavigationDirectionsListController#/c:objc(cs)GMSNavigationDirectionsListController(py)directionsListView]

The view that displays the directions list. The view is lazy loaded when it is accessed for the
first time.
