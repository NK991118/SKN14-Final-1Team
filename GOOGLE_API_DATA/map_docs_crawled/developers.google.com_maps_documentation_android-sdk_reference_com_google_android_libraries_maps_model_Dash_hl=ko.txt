공개 최종 클래스 Dash가 PatternItem [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/model/PatternItem?hl=ko]을 확장함

Polyline [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/model/Polyline?hl=ko]의 획 패턴이나 Polygon [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/model/Polygon?hl=ko] 또는 Circle [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/model/Circle?hl=ko]의 윤곽선에 사용된 대시를 나타내는 변경할 수 없는 클래스입니다.
