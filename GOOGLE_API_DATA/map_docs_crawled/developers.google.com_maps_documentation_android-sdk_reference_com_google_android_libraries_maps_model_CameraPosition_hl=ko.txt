위치, 확대/축소 수준, 기울기 각도, 방위와 같은 모든 카메라 위치 매개변수를 집계하는 변경할 수 없는 클래스입니다. CameraPosition.Builder [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/model/CameraPosition.Builder?hl=ko]를 사용하여 CameraPosition [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/model/CameraPosition?hl=ko] 인스턴스를 구성한 후 CameraUpdateFactory [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/CameraUpdateFactory?hl=ko]와 함께 사용할 수 있습니다. 자세한 내용은 카메라 이동하기 [https://developers.google.com/maps/documentation/android-sdk/views?hl=ko#moving_the_camera]를 참고하세요.

매개변수

| --- | --- |
| 줌 | 타겟의 확대/축소 수준입니다. 제한사항에 대한 자세한 내용은 zoom(float)를 참조하세요. |
| 기울기 | 바닥에서부터 아래 방향으로의 카메라 각도입니다 (단위: 도). 제한사항에 대한 자세한 내용은 tilt(float)를 참조하세요. |
| 방위 | 카메라가 가리키는 방향(북쪽에서 시계 방향)입니다(단위: 도). 이 값은 0도 이내에서 360도 미포함으로 정규화됩니다. |

| --- | --- |
| IllegalArgumentException | tilt가 0도에서 90도 사이의 범위를 벗어나는 경우입니다. |

fromLatLngZoom
(LatLng [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/model/LatLng?hl=ko] 타겟, 플로팅 확대/축소)

특정 타겟 및 줌레벨에 대해 가리키는 CameraPosition을 생성합니다. 결과 방위는 북쪽이고 시야각은 지표면과 직각입니다. 즉, 지면을 정면으로 향하고 화면 상단이 북쪽을 향합니다.

| target | 화면 중앙에 정렬할 타겟 위치입니다. |
| --- | --- |
