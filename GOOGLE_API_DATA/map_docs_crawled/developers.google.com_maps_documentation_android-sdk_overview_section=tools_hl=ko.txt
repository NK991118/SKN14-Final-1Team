Android용 Maps SDK를 사용해
Android 앱 [https://developer.android.com/?hl=ko](Google 지도 데이터,
지도 표시, 지도 동작 응답을 사용하는
Wear OS [https://developer.android.com/wear?hl=ko] 앱 포함)에 지도를 추가해 보세요. 또한 마커, 다각형, 오버레이를
지도에 추가하여 지도 위치에 대한 정보를 추가로 제공하고
사용자 상호작용을 지원할 수도 있습니다.

SDK는 Kotlin [https://developer.android.com/kotlin?hl=ko]과
Java 프로그래밍 언어를 모두 지원할
뿐만 아니라 고급 [https://developers.google.com/maps/documentation/android-sdk/overview?section=tools&hl=ko] 기능과 프로그래밍 기법을 사용할 수 있도록 라이브러리와
확장 프로그램을 추가로 제공합니다.

SDK를 사용하는 샘플 앱 코드 및 코드 스니펫을 작성합니다.
Android 유틸리티 라이브러리 [https://developers.google.com/maps/documentation/android-sdk/utility?hl=ko]
다중 레이어, 마커 클러스터, 거리 계산 등 지도에서 사용할 수 있는 고급 기능 라이브러리입니다.
KTX Kotlin 확장 프로그램 [https://developers.google.com/maps/documentation/android-sdk/ktx?hl=ko]
Android용 Maps SDK를 사용할 때
앱에서 더욱 효율적으로 Kotlin 코드를 작성할 수 있도록 지원하는 확장
프로그램(Maps Android KTX)의 모음입니다.
