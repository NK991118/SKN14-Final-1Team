public  final   class
BitmapDescriptorFactory
extends Object

Used to create a definition of a Bitmap image, used for marker icons and ground overlays.
Prior to using any methods from this class, you must do one of the following to ensure that
this class is initialized:

Wait for a GoogleMap [https://developers.google.com/maps/documentation/navigation/android-sdk/reference/com/google/android/gms/maps/GoogleMap] to become available from
a MapFragment [https://developers.google.com/maps/documentation/navigation/android-sdk/reference/com/google/android/gms/maps/MapFragment] or MapView [https://developers.google.com/maps/documentation/navigation/android-sdk/reference/com/google/android/gms/maps/MapView] that you have added to your application. You
can obtain the GoogleMap [https://developers.google.com/maps/documentation/navigation/android-sdk/reference/com/google/android/gms/maps/GoogleMap] instance by calling
getMapAsync() and waiting for the onMapReady(GoogleMap map) callback.
Call MapsInitializer.initialize(Context) [https://developers.google.com/maps/documentation/navigation/android-sdk/reference/com/google/android/gms/maps/MapsInitializer#initialize(android.content.Context)]. As long as a GooglePlayServicesNotAvailableException [https://developers.google.com/android/reference/com/google/android/gms/common/GooglePlayServicesNotAvailableException] isn't thrown, this class will be correctly
initialized.

Creates a BitmapDescriptor [https://developers.google.com/maps/documentation/navigation/android-sdk/reference/com/google/android/gms/maps/model/BitmapDescriptor] that refers to a colorization of the default marker image.
For convenience, there is a predefined set of hue values. See example HUE_YELLOW [https://developers.google.com/maps/documentation/navigation/android-sdk/reference/com/google/android/gms/maps/model/BitmapDescriptorFactory#HUE_YELLOW].

Parameters
| hue | The hue of the marker. Value must be greater or equal to 0 and less than 360. |
