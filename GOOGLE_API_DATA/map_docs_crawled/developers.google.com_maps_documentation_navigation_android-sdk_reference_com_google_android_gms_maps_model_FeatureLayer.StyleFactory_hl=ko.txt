public static    interface
FeatureLayer.StyleFactory

The style factory function applied to every polygon in the affected feature layer. This
function must return a FeatureStyle [https://developers.google.com/maps/documentation/navigation/android-sdk/reference/com/google/android/gms/maps/model/FeatureStyle] object that specifies how to style the polygon.
