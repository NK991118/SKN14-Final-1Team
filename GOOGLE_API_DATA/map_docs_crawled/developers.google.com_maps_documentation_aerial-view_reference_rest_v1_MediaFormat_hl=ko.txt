MediaFormat

컬렉션을 사용해 정리하기

내 환경설정을 기준으로 콘텐츠를 저장하고 분류하세요.

API에서 지원하는 다양한 미디어 형식을 열거합니다.

| 열거형 |
| --- |
| MEDIA_FORMAT_UNSPECIFIED | 기본값 |
| IMAGE | 썸네일 이미지입니다. |
| MP4_HIGH | 고화질 MP4 동영상 |
| MP4_MEDIUM | 중간 품질의 MP4 동영상입니다. |
| MP4_LOW | 저화질 MP4 동영상입니다. |
| DASH | 대부분의 기기에서 적응형 비트 전송률 스트리밍을 위한 MPEG-DASH |
| HLS | Apple 기기에서 적응형 비트 전송률 스트리밍을 위한 HLS |
