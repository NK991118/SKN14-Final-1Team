유럽 경제 지역 (EEA) 개발자

청구서 수신 주소가 유럽 경제 지역에 있는 경우 2025년 7월 8일부터
Google Maps Platform EEA 서비스 약관 [https://cloud.google.com/terms/maps-platform/eea?hl=ko]이 서비스 이용에 적용됩니다. 자세히 알아보기 [https://developers.google.com/maps/comms/eea/faq?hl=ko]
또한 Map Tiles API의 특정 콘텐츠가 더 이상 반환되지 않습니다.
자세히 알아보기 [https://developers.google.com/maps/comms/eea/map-tiles?hl=ko]

포토리얼리스틱 3D 타일은 고해상도 이미지로 텍스처 처리된 3D 메시입니다. 이러한 회사는 세계의 많은 인구 밀집 지역에서 고해상도 3D 지도를 제공합니다. 이를 통해 차세대 몰입형 3D 시각화 환경을 지원하여 사용자가 다음 작업을 할 수 있습니다.

지역 이해하기
자신 있게 위치로 이동
새로운 장소를 평가하여 결정 내리기
실제 세계의 변화 이해하기

시작하기 전에

실사 3D 타일 API를 사용하기 전에 서비스 약관 [https://cloud.google.com/maps-platform/terms/?hl=ko] 및 Map Tiles API 정책 [https://developers.google.com/maps/documentation/tile/policies?hl=ko]을 검토해야 합니다. 그런 다음 결제 계정으로 프로젝트를 시작하고 Map Tiles API를 사용 설정할 수 있습니다. 자세한 내용은 Cloud 콘솔에서 설정하기 [https://developers.google.com/maps/documentation/tile/cloud-setup?hl=ko]를 참고하세요.

타일 가져오기

API 키 [https://developers.google.com/maps/documentation/tile/get-api-key?hl=ko]를 획득한 후 원하는 3D 타일 렌더러에 루트 타일 세트 URL을 지정하여 사실적인 타일에 액세스할 수 있습니다. 그 후 최종 사용자가 지도를 탐색하면 렌더러가 포토리얼리스틱 타일에 대한 모든 후속 호출을 자동으로 실행합니다.

렌더링은 단일 루트 타일 세트 요청에서 최소 3시간의 타일 요청을 할 수 있습니다. 이 한도에 도달한 후에는 다른 루트 타일 세트 요청을 해야 합니다.

예를 들어 다음 JavaScript 코드를 사용하여 Cesium JS [https://cesium.com/platform/cesiumjs/]로 3D 타일을 탐색할 수 있습니다.
<head>
<meta charset="utf-8">
<script src="https://ajax.googleapis.com/ajax/libs/cesiumjs/1.105/Build/Cesium/Cesium.js"></script>
<link href="https://ajax.googleapis.com/ajax/libs/cesiumjs/1.105/Build/Cesium/Widgets/widgets.css" rel="stylesheet">
</head>

<body>
<div id="cesiumContainer"></div>
<script>
</script>
</body>
참고: Google의 포토리얼리스틱 타일을 렌더링하려면 저작권 출처 표시를 지원하는 3D 타일 렌더러를 사용해야 합니다. CesiumJS 버전 1.91 이상 또는 Cesium for Unreal 버전 1.12 이상을 사용할 수 있습니다.
