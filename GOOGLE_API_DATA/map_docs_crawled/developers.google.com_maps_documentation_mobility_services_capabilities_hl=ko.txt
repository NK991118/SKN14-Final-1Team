개요

Google Maps Platform Mobility Services는 비즈니스를 위한 운송 및 물류 운영을 구축하기 위한 개발 툴킷입니다.

이 툴킷은 지도, 경로, 장소의 기능을 래핑하는 웹 서비스와 API 세트를 제공합니다. 이러한 서비스는 다음과 같은 두 가지 기본적인 업계 사용 사례를 지원합니다.

차량 공유, 음식 배달과 같은 주문형 운전기사 서비스
배달 및 트럭 운송 서비스와 같은 예약된 운전기사 서비스

Google 지도는 모빌리티 고객의 요구사항과 운송 분야의 선두 기업과의 긴밀한 협력을 통해 얻은 직접적인 경험을 바탕으로 모빌리티 서비스를 설계했습니다. 모빌리티 서비스를 사용하면 예측 가능한 비용으로 사용자 환경과 운영을 확장하고 최적화할 수 있습니다.
참고: 샘플 앱 [https://github.com/googlemaps/last-mile-fleet-solution-samples]을 사용하여 툴킷을 실험해 볼 수 있습니다.

구체적인 모빌리티 제품은 무엇인가요?

포함된 서비스 및 약관에 대한 자세한 내용은 계약을 참고하세요. 자세한 내용은 영업팀에 문의 [https://cloud.google.com/contact-maps?hl=ko]하세요.

Google Maps Platform 모빌리티 서비스는 세 가지 상업용 제품으로 구성되어 있으며, 각 제품은 비즈니스 요구사항에 부합하고 운송 기술의 성숙도에 기반한 특정 기능을 제공합니다.

Mobility Accelerate

이 패키지는 모든 고객이 사용할 수 있지만 자체 차량을 관리하고 연간 5백만 개 이상의 상품을 배송하는 소매업체 및 물류 회사와 연간 1백만 건의 차량 공유 또는 배송을 수행하는 온디맨드 차량 공유 또는 배송 회사를 대상으로 합니다. 각 패키지에는 서명된 계약과 결제 통합이 필요합니다.

이동성 활성화

Mobility Activate에는 Navigation SDK를 비롯한 다양한 Google Maps Platform 독립형 API가 포함되어 있습니다. 위치 컨텍스트, 선호 경로 - 맞춤 경로, 제한 속도, 모든 Fleet Engine 기반 제품은 제외됩니다.

운송 관리 시스템이 상당히 성숙하고 차량을 소유하지 않은 경우 이 패키지를 사용하면 운송 및 물류 솔루션의 번들링을 활용할 수 있습니다. Mobility Activate를 사용하여 효율적인 경로 지정과 개선된 탐색 기능, 별도의 API 호출 가격 책정 대신 작업별 가격 책정을 통해 더 나은 엔드 투 엔드 이동성 환경을 빌드하세요.

사용 가능한 API의 전체 목록을 확인하고 시작하려면 영업팀에 문의 [https://cloud.google.com/contact-maps?hl=ko]하세요.

이동성 최적화

Mobility Optimize에는 Mobility Activate의 모든 기능과 위치 컨텍스트, 선호 경로 - 맞춤 경로, 제한 속도가 포함됩니다. Fleet Engine 기반 제품은 모두 제외됩니다.

차량 관리 초기 단계에 있는 경우 이 서비스를 사용하여 개선된 경로 및 예상 도착 시간으로 배송 기간을 단축하세요. 최적화를 사용하여 운전자 및 소비자 만족도를 더욱 개선하고 전반적인 운송 운영을 개선하세요.

사용 가능한 API의 전체 목록을 확인하고 시작하려면 영업팀에 문의 [https://cloud.google.com/contact-maps?hl=ko]하세요.

Mobility Accelerate

Mobility Accelerate에는 Optimize의 모든 기능과 Fleet Engine 기반 제품(Fleet Engine, Nearby Drivers, Vehicle Search, Driver SDK, Consumer SDK, Shipment Tracking, Fleet Tracking)이 포함됩니다.

Accelerate를 사용하면 사용자 대상 앱을 Fleet Engine에 연결할 수 있습니다. Fleet Engine은 운전자, 소비자, 운영팀 전반에서 차량과 여정 또는 작업 오케스트레이션을 지원하는 Google 백엔드 서비스입니다. 사용 사례에 따라 주문형 이동에는 Fleet Engine을 사용하고 예약된 작업에는 Fleet Engine을 사용합니다. 각 사용 사례 유형의 Fleet Engine에 대해 자세히 알아보려면 Fleet Engine 서비스란 무엇인가요? [https://developers.google.com/maps/documentation/mobility/fleet-engine/essentials?hl=ko]를 참고하세요.

대규모 기기 관리가 비즈니스의 핵심 부분인 경우 이 패키지가 요구사항에 가장 적합합니다. Accelerate를 사용하여 실시간 가시성, 향상된 운전자 환경, 차량 인텔리전스로 차량 운영을 개선하세요. Accelerate를 사용하면 작업별 가격 책정을 사용하여 이 작업을 수행할 수 있습니다.

사용 가능한 API의 전체 목록을 확인하고 시작하려면 영업팀에 문의 [https://cloud.google.com/contact-maps?hl=ko]하세요.

패키지 간 마이그레이션

하나의 모빌리티 결제 계정에는 하나의 패키지만 할당할 수 있습니다. 기존 계정을 다른 패키지로 이전하려면 새 패키지에 계정을 다시 할당해야 합니다. 서로 다른 사용 사례에 여러 패키지를 사용하려면 별도의 결제 계정이 필요합니다.
