사용 가능한 경로 옵션

컬렉션을 사용해 정리하기

내 환경설정을 기준으로 콘텐츠를 저장하고 분류하세요.

유럽 경제 지역 (EEA) 개발자

청구서 수신 주소가 유럽 경제 지역에 있는 경우 2025년 7월 8일부터
Google Maps Platform EEA 서비스 약관 [https://cloud.google.com/terms/maps-platform/eea?hl=ko]
이 서비스 이용에 적용됩니다. 기능은 리전에 따라 다릅니다.
자세히 알아보기 [https://developers.google.com/maps/comms/eea/faq?hl=ko]

다음 표에는 요청에서 지정할 수 있는 기타 경로 옵션이 나와 있습니다.

| 옵션 | 설명 |
| --- | --- |
| 여행 모드 | 이동 수단입니다(예: 자동차, 대중교통, 자전거, 도보, 이륜차). 참고: 도보, 자전거, 이륜차 경로는 베타 버전이며 명확한 인도, 보행자 전용 도로 또는 자전거 도로가 누락될 수 있습니다. 앱에 표시되는 모든 도보, 자전거, 이륜차 경로에 대해 사용자에게 이 경고를 표시해야 합니다. |
| 통행료 계산 | 경로의 통행료를 추정합니다. |
| 피해야 할 경로 기능 지정 | 유료도로 피하기, 고속도로 피하기 및 기타 옵션을 지정합니다. |
| 경로 다중선 요청 | 선택적으로 교통 정보를 포함하는 경로를 나타내는 좌표 집합을 요청합니다. |
| 현지화된 값 요청 | 여행 시간, 거리, 단위 시스템 (미터법 또는 영국식)에 대해 현지화된 텍스트를 제공하는 추가 응답 필드를 요청합니다. |

추가 옵션

이 표의 링크에서 추가 옵션을 확인하세요.

| 옵션 더보기 | 설명 |
| --- | --- |
| 경로에 사용 가능한 차량 유형 | 경로를 이동하는 차량 유형을 선택하는 추가 옵션입니다. |
| 경유지 유형 및 옵션 | 경로를 설정하고 구성하기 위한 추가 옵션입니다. |
| 경로에 사용할 수 있는 교통 옵션 | 경로를 계산할 때 교통정보를 포함하거나 제외하는 추가 옵션입니다. |
| 추가 옵션 | 출발 시간, 반환된 응답의 언어 코드 (경로 계산만 해당), 응답의 측정 단위 (경로 계산만 해당)와 같은 추가 옵션을 지정합니다. 요청 본문을 참고하세요. |
