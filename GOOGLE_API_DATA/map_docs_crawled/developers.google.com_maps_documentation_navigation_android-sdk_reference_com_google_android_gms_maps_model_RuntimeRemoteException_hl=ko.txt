public  final   class
RuntimeRemoteException
extends RuntimeException

A RuntimeException wrapper for RemoteException. Thrown when normally there is something seriously
wrong and there is no way to recover.
RuntimeRemoteException does not apply to the Navigation SDK. It exists for compile time
compatibility with com.google.android.gms:play-services-maps only.
