장소 데이터 필드 (신규)
플랫폼 선택: Android [https://developers.google.com/maps/documentation/places/android-sdk/data-fields?hl=ko] iOS [https://developers.google.com/maps/documentation/places/ios-sdk/data-fields?hl=ko] JavaScript [https://developers.google.com/maps/documentation/javascript/place-class-data-fields?hl=ko] 웹 서비스 [https://developers.google.com/maps/documentation/places/web-service/data-fields?hl=ko]
유럽 경제 지역 (EEA) 개발자
장소 데이터 필드는 장소 세부정보 (신규) [https://developers.google.com/maps/documentation/places/ios-sdk/details-place?hl=ko], 텍스트 검색 (신규) [https://developers.google.com/maps/documentation/places/ios-sdk/text-search?hl=ko], 주변 지역 검색 (신규) [https://developers.google.com/maps/documentation/places/ios-sdk/nearby-search?hl=ko]의 응답 GMSPlace [https://developers.google.com/maps/documentation/places/ios-sdk/reference/objc/Classes/GMSPlace?hl=ko] 객체에서 반환할 장소 데이터를 정의합니다. 요청에서 GMSPlaceProperty [https://developers.google.com/maps/documentation/places/ios-sdk/reference/objc/Constants?hl=ko]를 사용하여 반환할 필드의 목록(필드 마스크라고 함)을 지정합니다. 필드 마스크에 필드를 하나 이상 지정하지 않거나 필드 마스크를 생략하면 호출에서 오류가 반환됩니다.
이 페이지에는 가격 등급 (SKU)별로 모든 장소 데이터 필드가 나와 있습니다. 요청에 대한 요금 청구 방식에 대한 자세한 내용은 사용량 및 결제 [https://developers.google.com/maps/documentation/places/ios-sdk/usage-and-billing?hl=ko]를 참고하세요.
중요: 장소 데이터 결과는 비워둘 수 없으므로 데이터가 있는 장소 결과만 반환됩니다. 예를 들어 요청된 장소에 사진이 없으면 결과에 photos 필드가 표시되지 않습니다.
