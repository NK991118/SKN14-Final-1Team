경계 다각형 스타일 지정

컬렉션을 사용해 정리하기

내 환경설정을 기준으로 콘텐츠를 저장하고 분류하세요.

플랫폼 선택:

Android [https://developers.google.com/maps/documentation/android-sdk/dds-boundaries/style-polygon?hl=ko]

iOS [https://developers.google.com/maps/documentation/ios-sdk/dds-boundaries/style-polygon?hl=ko]

JavaScript [https://developers.google.com/maps/documentation/javascript/dds-boundaries/style-polygon?hl=ko]

경계 다각형의 채우기 및 획의 스타일을 지정하려면 GMSPlaceFeature [https://developers.google.com/maps/documentation/ios-sdk/reference/objc/Classes/GMSPlaceFeature?hl=ko]을 허용하고 GMSFeatureStyle [https://developers.google.com/maps/documentation/ios-sdk/reference/objc/Classes/GMSFeatureStyle?hl=ko]을 반환하는 스타일 지정 클로저를 사용하여 스타일 속성을 정의하세요. 그런 다음 스타일 속성을 스타일 지정 클로저로 설정합니다. 스타일 지정 클로저에는 스타일 지정 로직이 포함됩니다.

중요: 모든 기능에 필요한 것은 아니지만 장소 데이터 스타일 지정에는 iOS용 Places SDK [https://developers.google.com/maps/documentation/places/ios-sdk?hl=ko]와의 통합이 필요할 수 있습니다.

Swift
Objective-C
GMSMapView *mapView = [GMSMapView mapWithFrame:CGRectZero mapID:[GMSMapID mapIDWithIdentifier:@"MAP_ID"] camera:[GMSCameraPosition cameraWithLatitude: 20.773 longitude: -156.01 zoom:12]];

GMSFeatureLayer<GMSPlaceFeature *> *layer = [mapView featureLayerOfFeatureType:GMSFeatureTypeLocality];

GMSFeatureStyle *style = [GMSFeatureStyle styleWithFillColor:[[UIColor purpleColor] colorWithAlphaComponent:0.5] strokeColor:[UIColor purpleColor] strokeWidth:3.0];
