HTTP 요청
GET https://fleetengine.googleapis.com/v1/{name=providers/*/tasks/*}URL은 gRPC 트랜스코딩 [https://google.aip.dev/127] 문법을 사용합니다.

경로 매개변수
| 매개변수 |
| --- |
| name | string 필수 항목입니다. providers/{provider}/tasks/{task} 형식이어야 합니다. provider는 Google Cloud 프로젝트 ID여야 합니다. sample-cloud-project를 예로 들 수 있습니다. |

쿼리 매개변수
| 매개변수 |
| --- |
| header | object (DeliveryRequestHeader) 선택사항. 표준 게재 API 요청 헤더입니다. |
