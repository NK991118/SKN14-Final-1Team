Weather API 개요
Weather API를 사용하면 전 세계 위치의 실시간 초지역 날씨 데이터를 요청할 수 있습니다. 날씨 정보에는 온도, 강수량, 습도 등이 포함됩니다.
특정 위도와 경도의 위치에 대해 API는 다음을 쿼리할 수 있는 엔드포인트를 제공합니다.
현재 날씨: 현재 기상 상황입니다.
시간별 예보: 모든 요소에 대해 최대 240시간의 예상 상태를 제공합니다.
일일 예측: 모든 요소에 대해 최대 10일간의 예상 조건입니다.
시간별 기록: 모든 요소의 캐시된 과거 상태를 최대 24시간까지 표시합니다.
Weather API의 기능
날씨 API 데이터에는 다음이 포함됩니다.
날씨 상황 및 아이콘: 현재 날씨 상황을 나타내는 설명과 해당 아이콘입니다 (예: '흐림' 또는 '눈 소나기').
온도: 최저, 최고, 체감 온도입니다.
강수량: 총 강수량 및 강수 유형 (눈, 비, 얼음, 혼합)
바람: 평균 체감온도, 풍향, 최대 풍속, 돌풍
얼음 두께: 얼음의 두께입니다.
습도: 이슬점, 체감 온도, 습구 온도, 상대 습도
자외선 지수: 자외선 (UV) 지수입니다.
뇌우 확률: 뇌우가 발생할 확률(%)입니다.
가시거리: 물체가 명확하게 보이는 거리입니다.
기압: 평균 해수면 기압입니다.
태양 및 달 이벤트: 일출, 일몰, 월출, 월몰 시간
구름 덮임: 하늘이 구름으로 덮인 비율입니다.
날씨 API 사용 방법
1 설정하기 Google Cloud 프로젝트 설정하기 [https://developers.google.com/maps/documentation/weather/cloud-setup?hl=ko]부터 시작하여 다음 안내를 완료합니다.
2 현재 날씨 가져오기 현재 조건 가져오기 [https://developers.google.com/maps/documentation/weather/current-conditions?hl=ko]를 참고하세요.
3 시간별 예보 확인하기 시간별 예보 가져오기 [https://developers.google.com/maps/documentation/weather/hourly-forecast?hl=ko]를 참고하세요.
4 일일 예보 가져오기 일일 일기예보 가져오기 [https://developers.google.com/maps/documentation/weather/daily-forecast?hl=ko]를 참고하세요.
5 시간별 기록 가져오기 시간별 기록 가져오기 [https://developers.google.com/maps/documentation/weather/hourly-history?hl=ko]를 참고하세요.
