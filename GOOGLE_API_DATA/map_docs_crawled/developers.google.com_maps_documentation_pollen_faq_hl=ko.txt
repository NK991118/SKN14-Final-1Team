Pollen API FAQ

컬렉션을 사용해 정리하기

내 환경설정을 기준으로 콘텐츠를 저장하고 분류하세요.

API에 비율 제한이 있나요?

Pollen API의 경우 기본 비율 제한은 분당 6,000개 쿼리입니다.

결제를 사용 설정해야 하나요?

Pollen API를 사용하려면 유효한 결제 계정이 필요합니다. 결제 사용 설정 [https://developers.google.com/maps/documentation/pollen/cloud-setup?hl=ko#billing]을 참고하여 결제 계정으로 프로젝트를 설정하세요.

지원되지 않는 위치를 쿼리하면 어떻게 되나요?

위도와 경도가 지원되는 국가 목록 [https://developers.google.com/maps/documentation/pollen/coverage?hl=ko]에 포함되지 않으면 '이 위치에 대한 정보를 사용할 수 없습니다. 다른 위치에서 시도해 보세요.'

식물에 꽃가루 데이터가 없지만 해당 국가에서 지원되는 경우 어떻게 되나요?

선택한 위치와 날짜에 꽃가루 데이터가 없는 경우, 응답에는 요청된 위치에 포함된 모든 식물의 식물 코드와 표시 이름이 다른 데이터 필드 없이 포함됩니다. 예를 들면 다음과 같습니다.

"dailyInfo": [
"date": {
"year": 2023,
"month": 6,
"day": 22
