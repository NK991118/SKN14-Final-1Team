kGMSTileLayerNoTile

extern UIImage *const _Nonnull kGMSTileLayerNoTile

Stub tile that is used to indicate that no tile exists for a specific tile coordinate. May be
returned by -tileForX:y:zoom: on GMSTileOverlay.
