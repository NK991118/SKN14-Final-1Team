내 환경설정을 기준으로 콘텐츠를 저장하고 분류하세요.

이 페이지의 내용개발자 가이드 [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/MapView?hl=ko#developer-guide]상속받은 상수 요약 [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/MapView?hl=ko#inherited-constant-summary]상속받은 필드 요약 [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/MapView?hl=ko#inherited-field-summary]공개 생성자 요약 [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/MapView?hl=ko#public-constructor-summary]공개 메서드 요약 [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/MapView?hl=ko#public-method-summary]상속받은 메서드 요약 [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/MapView?hl=ko#inherited-method-summary]공개 생성자 [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/MapView?hl=ko#public-constructors]공개 메서드 [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/MapView?hl=ko#public-methods]

공개 클래스 MapView가 FrameLayout을 확장함

Google 지도 서비스에서 가져온 데이터로 지도를 표시하는 뷰입니다. 포커스가 있을 때
키 누름 및 터치 동작을 캡처하여 지도를 이동합니다.
이 클래스의 사용자는 이 뷰가 포함된 Activity [https://developer.android.com/reference/android/app/Activity.html?hl=ko] 또는 Fragment [https://developer.android.com/reference/android/app/Fragment.html?hl=ko]의 모든 수명 주기 메서드를 이 클래스의 상응하는 메서드에 전달해야 합니다. 특히 다음 메서드를 전달해야 합니다.

onCreate(Bundle) [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/MapView?hl=ko#onCreate(android.os.Bundle)]
onStart() [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/MapView?hl=ko#onStart()]
onResume() [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/MapView?hl=ko#onResume()]
onPause() [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/MapView?hl=ko#onPause()]
onStop() [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/MapView?hl=ko#onStop()]
onDestroy() [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/MapView?hl=ko#onDestroy()]
onSaveInstanceState(Bundle) [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/MapView?hl=ko#onSaveInstanceState(android.os.Bundle)]
onLowMemory() [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/MapView?hl=ko#onLowMemory()]

GoogleMap [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/GoogleMap?hl=ko]는 getMapAsync(OnMapReadyCallback) [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/MapView?hl=ko#getMapAsync(com.google.android.libraries.maps.OnMapReadyCallback)]를 사용하여 획득해야 합니다. MapView [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/MapView?hl=ko]는 지도 시스템과 뷰를 자동으로 초기화합니다.
이전 플랫폼을 타겟팅하려면 MapFragment [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/MapFragment?hl=ko] (또는 SupportMapFragment [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/SupportMapFragment?hl=ko])를 사용하면 지도를 간단하게 표시할 수 있습니다.
참고: 이 보기에는 어린이를 추가하지 않는 것이 좋습니다.

개발자 가이드
자세한 내용은 Android용 Google Maps SDK [https://developers.google.com/maps/documentation/android-sdk/?hl=ko] 개발자 가이드를 참고하세요.

getMapAsync
(OnMapReadyCallback [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/OnMapReadyCallback?hl=ko] 콜백)

바로 사용할 수 있는 GoogleMap [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/GoogleMap?hl=ko]의 null이 아닌 인스턴스를 반환합니다.
참고:

이 메서드는 기본 스레드에서 호출해야 합니다.
콜백은 기본 스레드에서 실행됩니다.
사용자의 기기에 Google Play 서비스가 설치되어 있지 않은 경우 사용자가 앱을 설치할 때까지 콜백이 트리거되지 않습니다.
콜백에서 제공하는 GoogleMap [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/GoogleMap?hl=ko] 객체는 null이 아닙니다.
