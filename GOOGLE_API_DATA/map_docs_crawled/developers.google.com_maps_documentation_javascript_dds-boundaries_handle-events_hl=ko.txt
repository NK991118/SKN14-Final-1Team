개요

지형지물 레이어가 mousemove 및 click 이벤트에 응답하도록 하고 이를 사용하여 클릭된 경계에 관한 정보를 반환합니다. 이 정보에는 ID, 표시 이름, 기능 유형이 포함됩니다. 다음 예시 지도에는 행정 구역 수준 2의 경계가 표시되어 있으며 사용자 상호작용에 따라 다각형의 스타일을 지정하는 이벤트 처리 코드가 있습니다 (mousemove는 테두리 두께를 변경하고 click는 배경색을 음영 처리하고 정보 창이 표시되도록 함).

지형지물 레이어 이벤트 사용 설정

다음 단계에 따라 지형지물 레이어에서 이벤트를 사용 설정합니다.

등록하려는 각 이벤트의 지형지물 레이어에서 addListener() 함수를 호출하여 이벤트 알림에 사용할 지형지물 레이어를 등록합니다. 이 예시에서는 지도에도 리스너가 추가됩니다.
