StampStyle [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/model/StampStyle?hl=ko]

getStamp
()

획의 스탬프 스타일을 가져옵니다.
스탬프 스타일의 이미지가 획 위에 반복됩니다. 이미지에 투명하거나
반투명한 부분이 있는 경우 해당 위치에 기본 획 색상이 표시됩니다.
그렇지 않으면 스탬프 처리된 텍스처/이미지가 기본 획을 완전히 덮게 됩니다.

반환 값
획의 스탬프 스타일입니다.
