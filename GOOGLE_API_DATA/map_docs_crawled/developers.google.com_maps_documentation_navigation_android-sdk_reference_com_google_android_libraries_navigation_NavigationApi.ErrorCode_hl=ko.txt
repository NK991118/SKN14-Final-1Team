An ErrorCode is a status code returned by NavigationApi.getNavigator(Activity, NavigatorListener) [https://developers.google.com/maps/documentation/navigation/android-sdk/reference/com/google/android/libraries/navigation/NavigationApi#getNavigator(android.app.Activity,%20com.google.android.libraries.navigation.NavigationApi.NavigatorListener)]
and NavigationApi.getNavigator(Application, NavigatorListener) [https://developers.google.com/maps/documentation/navigation/android-sdk/reference/com/google/android/libraries/navigation/NavigationApi#getNavigator(android.app.Application,%20com.google.android.libraries.navigation.NavigationApi.NavigatorListener)] when a Navigator [https://developers.google.com/maps/documentation/navigation/android-sdk/reference/com/google/android/libraries/navigation/Navigator] could not be
successfully obtained.
