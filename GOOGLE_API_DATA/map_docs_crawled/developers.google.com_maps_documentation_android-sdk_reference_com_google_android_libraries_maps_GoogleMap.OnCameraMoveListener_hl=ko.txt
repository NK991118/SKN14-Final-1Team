공개 정적 인터페이스
GoogleMap.OnCameraMoveListener

카메라의 위치가 변경되는 경우의 콜백 인터페이스입니다.

공개 메서드 요약
| 추상 void | onCameraMove() onCameraMoveStarted 호출 후 카메라가 계속 움직이면 반복적으로 호출됩니다. |
| --- | --- |

공개 메서드

공개 추상 void

onCameraMove
()

onCameraMoveStarted 호출 후 카메라가 계속 움직이면 반복적으로 호출됩니다. 이 메서드는 프레임마다 한 번씩 호출할 수 있으며, 비용이 많이 드는 작업을 실행하면 안 됩니다.
이 메서드는 Android UI 스레드에서 호출됩니다.
