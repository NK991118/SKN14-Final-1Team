공개 최종 클래스
IndoorLevel
Object 확장

건물의 층을 나타냅니다.
IndoorLevel 객체는 ID로만 동일합니다. 다른 콘텐츠가 있을 수도 있습니다.
층은 일반적으로 단일 건물로 둘러싸여 있지만, 층은 여러 건물로 둘러싸여 있을 수 있습니다 (예: 주차장 층은 여러 건물에 걸쳐 있을 수 있음).
