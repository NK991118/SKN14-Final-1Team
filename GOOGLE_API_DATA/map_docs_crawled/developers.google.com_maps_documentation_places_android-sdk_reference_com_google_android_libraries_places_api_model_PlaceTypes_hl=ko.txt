PlaceTypes

public final class PlaceTypes [https://developers.google.com/maps/documentation/places/android-sdk/reference/com/google/android/libraries/places/api/model/PlaceTypes]

Helper class that contains most of the supported place types for autocomplete place predictions.
The full list can be found here [https://developers.google.com/maps/documentation/places/android-sdk/supported_types], and should be considered as the source of truth.
