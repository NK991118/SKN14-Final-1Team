실험용: science 과학 아이콘으로 표시된 기능은 실험용입니다.

이 문서에서는 JSON을 사용하여 지도 스타일을 만들고, 수정하고, 내보내는 방법을 설명합니다.
JSON을 사용하여 스타일을 지정할 수 있는 항목에 관한 자세한 내용은 클라우드 기반 지도 스타일 지정의 JSON 참조 [https://developers.google.com/maps/documentation/javascript/cloud-customization/json-reference?hl=ko]를 참고하세요.

JSON 지도 스타일 가져오기

JSON을 사용하여 지도 스타일을 만들려면 다음 단계를 따르세요.

Google Cloud 콘솔에서 지도 스타일 [https://console.cloud.google.com/google/maps-apis/studio/styles?hl=ko]로 이동하고, 필요한 경우 프로젝트를 선택합니다.
지도 스타일에서 스타일 만들기를 클릭합니다.
대화상자에서 JSON 탭을 선택합니다.
JSON 스타일을 가져오는 방법은 두 가지입니다.

유효한 JSON 스타일 지정 코드를 필드에 붙여넣거나
scienceJSON 파일 업로드를 클릭하고 사용할 로컬 JSON 파일을 선택합니다. JSON 스타일 지정 코드가 필드에 표시됩니다.

참고: 기존 JSON을 가져오면 경고가 표시되고 스타일이 최신 JSON 형식으로 변환됩니다. 변환은 대략적인 것이지만 더 풍부하고 맞춤설정 가능한 환경을 설정할 수 있습니다. 기존 JSON만 사용하려면 삽입된 JSON 스타일 선언 사용 [https://developers.google.com/maps/documentation/javascript/json-styling-overview?hl=ko]을 참고하세요.

JSON에 오류가 있는지 확인합니다.

JSON에 오류가 있으면 'JSON에 오류가 N개 있습니다'라는 경고가 표시되며, 경고를 펼쳐 자세히 알아볼 수 있습니다.
JSON이 유효하면 지도 미리보기에 맞춤 스타일이 표시되고 파란색 맞춤설정 버튼을 사용할 수 있습니다.

맞춤설정을 클릭하여 새 지도 스타일을 만듭니다.
새 지도 스타일에서 지도 지형지물을 선택하고 필요에 따라 스타일을 검토하거나 변경합니다. 맞춤 스타일이 있는 지도 지형지물 옆에는 파란색 단색 점이 표시됩니다.
참고: 모든 스타일이 모든 확대/축소 수준에서 나타나는 것은 아닙니다.
저장을 클릭합니다.
지도 스타일 저장 및 게시 상자에 스타일 이름과 원하는 경우 설명을 입력하고 저장을 클릭합니다.

스타일이 자동으로 게시되고 새 스타일의
기본 페이지가 표시됩니다.

지도 ID에 지도 스타일을 추가하려면 지도 ID에 스타일 연결하기 [https://developers.google.com/maps/documentation/javascript/cloud-customization/map-styles?hl=ko#associate-style-with-map-id]를 참고하세요.

JSON으로 지도 스타일 수정하기
실험용: 이 기능은 실험용입니다.
스타일 편집기의 JSON 탭을 사용하여 지도 스타일을 수정할 수 있습니다.

JSON 편집기에서 변경한 내용은 스타일 패널에 반영됩니다. 또한 스타일 패널에서 변경하면 JSON 편집기에서 해당 변경사항을 확인할 수 있습니다. 두 방법 모두 지도 미리보기도 업데이트합니다.

JSON을 사용하여 지도 스타일을 수정하려면 다음 단계를 참고하세요.

Google Cloud 콘솔에서 지도 스타일 [https://console.cloud.google.com/google/maps-apis/studio/styles?hl=ko]로 이동하고, 필요한 경우 프로젝트를 선택합니다.
지도 스타일의 스타일 목록에서 기존 스타일을 선택합니다.
스타일의 기본 페이지에서 맞춤설정을 선택합니다.
JSON 탭을 선택하여 JSON 스타일 편집기를 엽니다.
편집기에서 변경합니다. JSON 파일을 업로드할 수도 있습니다.
커밋을 클릭하여 지도 미리보기를 업데이트합니다.
모든 변경사항을 적용한 후 저장을 클릭합니다. 새 스타일을 만드는 경우 변경사항이 자동으로 적용됩니다. 기존 스타일을 변경하는 경우 변경사항이 초안으로 저장되며 해당 지도 스타일을 사용하는 지도 ID에 변경사항을 적용하려면 게시를 클릭해야 합니다.

지도 스타일을 JSON으로 내보내기
실험용: 이 기능은 실험용입니다.
지도 스타일을 JSON 파일로 내보내려면 다음 단계를 따르세요.

Google Cloud 콘솔에서 지도 스타일 [https://console.cloud.google.com/google/maps-apis/studio/styles?hl=ko]로 이동하고, 필요한 경우 프로젝트를 선택합니다.
지도 스타일의 스타일 목록에서 기존 스타일을 선택합니다.
스타일의 기본 페이지에서 맞춤설정을 선택합니다.
JSON 탭을 선택하여 JSON 스타일 편집기를 엽니다.
내보내기를 선택합니다.
