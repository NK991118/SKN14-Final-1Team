업데이트된 DeliveryVehicle 데이터를 Fleet Engine에 쓰고 DeliveryVehicle에 Tasks를 할당합니다. DeliveryVehicle의 이름은 업데이트할 수 없습니다. remainingVehicleJourneySegments를 업데이트할 수 있지만 DeliveryVehicle에 유지될 모든 VehicleJourneySegment를 포함해야 합니다. taskId는 remainingVehicleJourneySegments에서 가져오며 상응하는 Tasks가 DeliveryVehicle에 할당됩니다(아직 할당되지 않은 경우).

HTTP 요청
PATCH https://fleetengine.googleapis.com/v1/{deliveryVehicle.name=providers/*/deliveryVehicles/*}URL은 gRPC 트랜스코딩 [https://google.aip.dev/127] 문법을 사용합니다.

경로 매개변수
| 매개변수 |
| --- |
| deliveryVehicle.name | string 이 배송 차량의 고유한 이름입니다. 형식은 providers/{provider}/deliveryVehicles/{vehicle}입니다. |

쿼리 매개변수
| 매개변수 |
| --- |
| header | object (DeliveryRequestHeader) 선택사항. 표준 게재 API 요청 헤더입니다. |
| updateMask | string (FieldMask format) 필수 항목입니다. 업데이트할 DeliveryVehicle 필드를 나타내는 필드 마스크입니다. updateMask에는 필드가 하나 이상 포함되어야 합니다. 정규화된 필드 이름의 쉼표로 구분된 목록입니다. 예: "remainingVehicleJourneySegments" |
