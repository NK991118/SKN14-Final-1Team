WordPress 사이트용 Google Maps Platform API 키 생성

WordPress 사이트에서 Google Maps Platform 기능을 사용하려면 Google Maps Platform API 키를 생성한 후 플러그인에 제공해야 합니다.
이 페이지는 다음 작업에 필요한 모든 단계를 안내합니다.

Google Cloud Platform 프로젝트가 없으면 새 프로젝트를 만듭니다.
프로젝트에 Google Maps Platform API 및 SDK를 사용 설정합니다.
프로젝트에 대한 API 키를 생성합니다.

API 키가 생성되면 WordPress 플러그인에 키를 제공하여 플러그인이 Google Maps Platform 기능에
액세스할 수 있도록 합니다.

요구사항 Google Maps Platform을 사용하고 API 키를 생성하려면 결제가 사용 설정된 Google Cloud Platform
계정이 필요합니다.

사용량 크레딧 모든 신규 Google Cloud Platform 계정에는 Google Maps Platform을 비롯하여 모든 Google Cloud 제품에
사용할 수 있는 300달러 상당의 일회성 크레딧이 제공됩니다.
각 Google Maps Platform SKU는 가격표 [https://developers.google.com/maps/billing-and-pricing/pricing?hl=ko]에 표시되고 Google Maps Platform 결제 [https://developers.google.com/maps/billing-and-pricing/billing?hl=ko]에 설명된 특정 금액의 무료 월 사용량을 제공합니다. [https://developers.google.com/maps/billing-and-pricing/billing?hl=ko]

API 키 생성하기
아래 버튼을 클릭하여 Google Maps Platform 빠른 시작 위젯을 실행하고 API 키를 생성하세요.
결제가 사용 설정된 Google Cloud 계정이 없는 경우 등록 및 결제 계정 생성 과정을 거치게 되며, 그러면 새 API 키가 자동으로 생성됩니다.
현재 결제가 사용 설정된 Google Cloud 계정이 있다면 Google Cloud 콘솔의 '사용자 인증 정보' 페이지로 이동하여 프로젝트의 API 키를 생성할 수 있습니다.
빠른 시작 위젯에 관한 자세한 내용은 빠른 시작 위젯 정보 [https://developers.google.com/maps/third-party-platforms/quick-start-widget-users?hl=ko]를 참고하세요.
