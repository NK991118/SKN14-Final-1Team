Route Optimization API 클라이언트 라이브러리

컬렉션을 사용해 정리하기

내 환경설정을 기준으로 콘텐츠를 저장하고 분류하세요.

이 페이지에서는 Route Optimization API용 클라이언트 라이브러리를 시작하는 방법을 설명합니다.

클라이언트 라이브러리에 대한 자세한 내용은 클라이언트 라이브러리 설명 [https://cloud.google.com/apis/docs/client-libraries-explained?hl=ko]을 참고하세요.

클라이언트 라이브러리 설치

자바자세한 내용은 자바 개발 환경 설정 [https://cloud.google.com/java/docs/setup?hl=ko]을 참조하세요.

설치 안내는 Java용 Google Route Optimization API 클라이언트 [https://github.com/googleapis/google-cloud-java/tree/main/java-maps-routeoptimization]를 참고하세요.
Gogo get cloud.google.com/go/maps

자세한 내용은 Go 개발 환경 설정 [https://cloud.google.com/go/docs/setup?hl=ko]을 참조하세요.

설치 안내는 Go용 Google 경로 최적화 API 클라이언트 [https://pkg.go.dev/cloud.google.com/go/maps/routeoptimization/apiv1]를 참고하세요.
Node.jsnpm install @googlemaps/routeoptimization

자세한 내용은 Node.js 개발 환경 설정 [https://cloud.google.com/nodejs/docs/setup?hl=ko]을 참조하세요.

전체 설치 안내는 Node.js용 Google Routes API 클라이언트 [https://github.com/googleapis/google-cloud-node/tree/main/packages/google-maps-routeoptimization]를 참고하세요.
Pythonpip install google-maps-routeoptimization

자세한 내용은 Python 개발 환경 설정 [https://cloud.google.com/python/docs/setup?hl=ko]을 참조하세요.

전체 설치 안내는 Python용 Google 경로 최적화 API 클라이언트 [https://github.com/googleapis/google-cloud-python/tree/main/packages/google-maps-routeoptimization]를 참고하세요.
.NET자세한 내용은 .Net 개발 환경 설정 [https://cloud.google.com/dotnet/docs/setup?hl=ko]을 참고하세요.

전체 설치 안내는 .Net용 Google 경로 최적화 API 클라이언트 [https://googleapis.dev/dotnet/Google.Maps.RouteOptimization.V1/latest/]를 참고하세요.
PHPcomposer require google/maps-routeoptimization

자세한 내용은 PHP 시작하기 [https://cloud.google.com/php/getting-started?hl=ko]를 참고하세요.

전체 설치 안내는 PHP용 Google Maps 경로 최적화 클라이언트 [https://packagist.org/packages/google/maps-routeoptimization]를 참고하세요.

인증 설정

클라이언트 라이브러리를 사용할 때는 애플리케이션 기본 사용자 인증 정보 (ADC) [https://cloud.google.com/docs/authentication/application-default-credentials?hl=ko]를 사용하여 인증합니다. ADC 설정에 대한 자세한 내용은 애플리케이션 기본 사용자 인증 정보에 대한 사용자 인증 정보 제공 [https://cloud.google.com/docs/authentication/provide-credentials-adc?hl=ko]을 참고하세요. 클라이언트 라이브러리에서 ADC를 사용하는 방법은 클라이언트 라이브러리를 사용하여 인증 [https://cloud.google.com/docs/authentication/client-libraries?hl=ko]을 참조하세요.
