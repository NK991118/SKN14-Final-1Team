동영상이 아직 렌더링되지 않은 경우 렌더기의 대기열에 주소를 추가합니다. 그렇지 않으면 동영상에 대한 메타데이터를 반환합니다.

HTTP 요청
POST https://aerialview.googleapis.com/v1/videos:renderVideoURL은 gRPC 트랜스코딩 [https://google.aip.dev/127] 구문을 사용합니다.

요청 본문
요청 본문에는 다음과 같은 구조의 데이터가 포함됩니다.

| JSON 표현 |
| --- |
| {   "address": string } |

| 필드 |
| --- |
| address | string 필수 항목입니다. 동영상에서 렌더링할 위치의 미국 우편 주소입니다. |

응답 본문

AerialView.RenderVideo의 응답 메시지입니다.
성공할 경우 응답 본문에 다음 구조의 데이터가 포함됩니다.

| JSON 표현 |
| --- |
| {   "state": enum (State),   "metadata": {     object (VideoMetadata)   } } |

| 필드 |
| --- |
| state | enum (State) 렌더링 요청의 현재 상태입니다. |
| metadata | object (VideoMetadata) 동영상의 메타데이터를 포함합니다. 상태가 ACTIVE인 경우에만 설정됩니다. |
