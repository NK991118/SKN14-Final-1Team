특정 주소 또는 videoId가 지정된 동영상의 메타데이터를 가져옵니다. 이 메서드는 videos.lookupVideo의 무료 호출 대체 메서드이며 재생 URI를 제외한 사용 가능한 모든 동영상 데이터를 반환합니다.

HTTP 요청
GET https://aerialview.googleapis.com/v1/videos:lookupVideoMetadataURL은 gRPC 트랜스코딩 [https://google.aip.dev/127] 구문을 사용합니다.

쿼리 매개변수
| 매개변수 |
| --- |
| 통합 매개변수 key입니다. 필수 항목입니다. key은 다음 중 하나여야 합니다. |
| videoId | string videos.renderVideo에서 반환된 ID |
| address | string 미국 우편 주소. |

요청 본문
요청 본문은 비어 있어야 합니다.

응답 본문
성공한 경우 응답 본문에 VideoMetadata [https://developers.google.com/maps/documentation/aerial-view/reference/rest/v1/VideoMetadata?hl=ko]의 인스턴스가 포함됩니다.
