GMSSyncTileLayer

@interface GMSSyncTileLayer : GMSTileLayer [https://developers.google.com/maps/documentation/navigation/ios-sdk/reference/objc/Classes/GMSTileLayer.html]

GMSSyncTileLayer is an abstract subclass of GMSTileLayer [https://developers.google.com/maps/documentation/navigation/ios-sdk/reference/objc/Classes/GMSTileLayer.html] that provides a sync interface to
generate image tile data.

-tileForX:y:zoom: [https://developers.google.com/maps/documentation/navigation/ios-sdk/reference/objc/Classes/GMSSyncTileLayer#/c:objc(cs)GMSSyncTileLayer(im)tileForX:y:zoom:]

As per -requestTileForX:y:zoom:receiver: on GMSTileLayer [https://developers.google.com/maps/documentation/navigation/ios-sdk/reference/objc/Classes/GMSTileLayer.html], but provides a synchronous
interface to return tiles. This method may block or otherwise perform work, and is not called on
the main thread.

Calls to this method may also be made from multiple threads so implementations must be
threadsafe.
