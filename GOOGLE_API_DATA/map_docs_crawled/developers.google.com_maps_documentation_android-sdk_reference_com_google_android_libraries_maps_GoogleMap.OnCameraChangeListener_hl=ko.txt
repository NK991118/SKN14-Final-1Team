공개 정적 인터페이스
GoogleMap.OnCameraChangeListener

이 인터페이스는 지원 중단되었습니다.가 GoogleMap.OnCameraMoveStartedListener [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/GoogleMap.OnCameraMoveStartedListener?hl=ko], GoogleMap.OnCameraMoveListener [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/GoogleMap.OnCameraMoveListener?hl=ko], GoogleMap.OnCameraIdleListener [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/GoogleMap.OnCameraIdleListener?hl=ko]로 대체되었습니다. 새 카메라 변경 리스너의 메서드와 관련하여 지원 중단된 onCameraChange 메서드가 호출되는 순서는 정의되어 있지 않습니다.

카메라의 위치가 변경되는 경우의 콜백 인터페이스입니다.

공개 메서드 요약
| 추상 void | onCameraChange(CameraPosition 위치) 카메라 위치가 변경된 후 호출됩니다. |
| --- | --- |

공개 메서드

공개
추상
void

onCameraChange
(CameraPosition [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/model/CameraPosition?hl=ko] 위치)

카메라 위치가 변경된 후 호출됩니다. 애니메이션 실행 중에 이 리스너에 중간 카메라 위치에 관한 알림이 전송되지 않을 수도 있습니다. 이 메서드는 항상 애니메이션의 마지막 위치를 위해 호출됩니다.
이 메서드는 Android UI 스레드에서 호출됩니다.

매개변수
| position | 마지막 카메라 변경 끝의 CameraPosition입니다. |
