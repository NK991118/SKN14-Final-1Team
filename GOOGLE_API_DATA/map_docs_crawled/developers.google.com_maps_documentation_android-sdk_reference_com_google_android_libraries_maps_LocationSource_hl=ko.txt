공개 인터페이스 LocationSource

일반적으로 GoogleMap [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/GoogleMap?hl=ko] 객체에 위치 데이터를 제공하기 위한 인터페이스를 정의합니다.
GoogleMap [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/GoogleMap?hl=ko] 객체에는 내 위치 레이어를 위한 위치 정보 제공자가 내장되어 있지만 이 인터페이스를 구현하는 다른 항목과 함께 replaced [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/GoogleMap?hl=ko#setLocationSource(com.google.android.libraries.maps.LocationSource)]일 수 있습니다.
GoogleMap [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/GoogleMap?hl=ko] 객체는 activate(OnLocationChangedListener) [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/LocationSource?hl=ko#activate(com.google.android.libraries.maps.LocationSource.OnLocationChangedListener)]를 사용하여 위치 제공자를 활성화합니다. 활성 상태 (activate(OnLocationChangedListener) [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/LocationSource?hl=ko#activate(com.google.android.libraries.maps.LocationSource.OnLocationChangedListener)]과 deactivate() [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/LocationSource?hl=ko#deactivate()] 사이)인 경우 위치 정보 제공자는 activate(OnLocationChangedListener) [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/LocationSource?hl=ko#activate(com.google.android.libraries.maps.LocationSource.OnLocationChangedListener)]에 등록된 리스너에 주기적인 위치 업데이트를 푸시해야 합니다. 지도의 수명 주기 상태에 따라 위치 서비스를
현명하게 사용하는 것은 제공업체의 책임입니다. 예를 들어 배터리를 많이 사용하는 서비스 (예: GPS)는 가끔만 사용하거나 활동이 포그라운드에 있는 동안에만 사용해야 합니다.
