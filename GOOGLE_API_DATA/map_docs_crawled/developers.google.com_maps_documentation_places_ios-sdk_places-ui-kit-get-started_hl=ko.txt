장소 UI 키트 시작하기

장소 UI 키트를 사용하기 전에 다음을 실행해야 합니다.

결제 계정으로 Cloud 프로젝트를 만듭니다. [https://developers.google.com/maps/documentation/places/ios-sdk/cloud-setup?hl=ko]

프로젝트에 Places UI Kit API를 사용 설정 [https://console.cloud.google.com/apis/library/placewidgets.googleapis.com?hl=ko]합니다.
API 키를 가져옵니다 [https://console.cloud.google.com/project/_/google/maps-apis/credentials?hl=ko].
