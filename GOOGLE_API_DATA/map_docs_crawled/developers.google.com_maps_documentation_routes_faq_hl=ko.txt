API에 비율 제한이 있나요?

경로 계산의 경우 기본 비율 제한은 3,000 QPM(분당 요청수)입니다.

경로 매트릭스를 계산하는 경우 비율 제한은 3,000EPM(분당 요소 수, 출발지 수 x 목적지 수로 계산)입니다.

자세한 내용은 사용량 한도 및 결제 [https://developers.google.com/maps/faq?hl=ko#usage-limits]를 참고하세요.
