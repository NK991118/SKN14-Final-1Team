Maps Datasets API 정책
이 문서에는 API 또는 Cloud 콘솔을 사용하여 Maps Datasets API를 사용하는 데 필요한 요구사항이 나와 있습니다. Maps Datasets API 사용에는 Google과의 계약이 적용됩니다.
업로드된 데이터
'Google 지도 플랫폼 계약'에 설명된 대로 (i) 데이터를 Google에 공유 및 제공하고 (ii) Google 제품 또는 서비스와 관련하여 데이터를 사용할 권리를 Google에 부여하는 데 필요한 모든 권리를 보유하고 보유할 경우에만 지도 데이터 세트 API를 통해 Google에 데이터를 업로드할 수 있습니다.
제품 한도
Maps Datasets API를 사용하여 Google에 업로드할 수 있는 파일당 용량은 500MB를 초과할 수 없으며, 집계 데이터는 10GB를 초과할 수 없습니다.
