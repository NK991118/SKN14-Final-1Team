strokeWidth [https://developers.google.com/maps/documentation/ios-sdk/reference/objc/Classes/GMSCircle#/c:objc(cs)GMSCircle(py)strokeWidth]

The width of the circle’s outline in screen points. Defaults to 1. As per GMSPolygon [https://developers.google.com/maps/documentation/ios-sdk/reference/objc/Classes/GMSPolygon.html], the width
does not scale when the map is zoomed.

Setting strokeWidth to 0 results in no stroke.

Objective-C
