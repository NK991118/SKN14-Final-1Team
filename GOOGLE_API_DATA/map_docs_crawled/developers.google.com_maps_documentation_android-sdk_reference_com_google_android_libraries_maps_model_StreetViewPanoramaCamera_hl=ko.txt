매개변수
| 줌 | 파노라마에 대한 카메라의 확대/축소 수준입니다. 제한사항에 대한 자세한 내용은 zoom를 참조하세요. |
| --- | --- |
| 기울기 | 파노라마의 수평선으로부터의 카메라 각도입니다(단위: 도). 제한사항에 대한 자세한 내용은 tilt를 참조하세요. |
| 방위 | 카메라가 가리키는 방향(북쪽에서 시계 방향)입니다(단위: 도). 이 값은 0도 이내에서 360도 미포함으로 정규화됩니다. |

생성 값
| IllegalArgumentException | tilt이 -90도 이상 90도 범위를 벗어나는 경우입니다. |
