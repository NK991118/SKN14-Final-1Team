공개 정적 인터페이스
StreetViewPanorama.OnStreetViewPanoramaLongClickListener

사용자가 파노라마를 길게 누를 때의 콜백 인터페이스입니다.
리스너는 기본 스레드에서 호출됩니다.

공개 메서드 요약
| 추상 void | onStreetViewPanoramaLongClick(StreetViewPanoramaOrientation 방향) 사용자가 파노라마에서 길게 누르기 동작을 할 때 호출되지만 파노라마 오버레이 중 동작을 처리하지 않는 경우에만 호출됩니다. |
| --- | --- |

공개 메서드

public
추상
void

onStreetViewPanoramaLongClick
(StreetViewPanoramaOrientation [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/model/StreetViewPanoramaOrientation?hl=ko] 방향)

사용자가 파노라마에서 길게 누르기 동작을 할 때 호출되지만 파노라마 오버레이 중 동작을 처리하지 않는 경우에만 호출됩니다. 이 메서드의 구현은 항상 기본 스레드에서 호출됩니다.

매개변수
| 방향 | 화면에서 사용자가 길게 누른 지점에 해당하는 기울기 및 방위 값입니다. 이러한 값은 특정 파노라마 내 절댓값을 가지며 카메라의 현재 방향과는 관계가 없습니다. |
