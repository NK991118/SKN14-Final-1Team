지정된 프로젝트의 모든 데이터 세트를 나열합니다.

HTTP 요청
GET https://mapsplatformdatasets.googleapis.com/v1/{parent=projects/*}/datasetsURL은 gRPC 트랜스코딩 [https://google.aip.dev/127] 문법을 사용합니다.

경로 매개변수
| 매개변수 |
| --- |
| parent | string 필수 항목입니다. 모든 데이터 세트를 나열할 프로젝트의 이름입니다. 형식: projects/{project} |

쿼리 매개변수
| 매개변수 |
| --- |
| pageSize | integer 페이지당 반환할 최대 데이터 세트 수입니다. 지정하지 않거나 0이면 모든 데이터 세트가 반환됩니다. |
| pageToken | string 이전dataset.list 호출에서 수신된 페이지 토큰입니다. 후속 페이지를 검색하려면 이를 입력합니다. |
| tag | string 각 데이터 세트에 대해 원하는 버전을 지정하는 태그입니다. 페이지로 나누기도 지정되면 페이지로 나누기 이후에 일부 필터링이 발생할 수 있으며, 이로 인해 마지막 페이지가 아니더라도 응답에 페이지 크기보다 적은 데이터 세트가 포함될 수 있습니다. 'active' 태그: 응답의 각 데이터 세트에는 마지막으로 완료된 버전의 정보가 포함되며 데이터 세트가 없으면 데이터 세트를 건너뜁니다. |

요청 본문
요청 본문은 비어 있어야 합니다.

응답 본문

dataset.list의 응답 객체입니다.
성공할 경우 응답 본문에 다음 구조의 데이터가 포함됩니다.

| JSON 표현 |
| --- |
| {   "datasets": [     {       object (Dataset)     }   ],   "nextPageToken": string } |

| 필드 |
| --- |
| datasets[] | object (Dataset) 프로젝트의 모든 데이터 세트입니다. |
| nextPageToken | string 다음 페이지를 검색하기 위해 pageToken으로 전송할 수 있는 토큰입니다. 이 필드를 생략하면 후속 페이지가 표시되지 않습니다. |
