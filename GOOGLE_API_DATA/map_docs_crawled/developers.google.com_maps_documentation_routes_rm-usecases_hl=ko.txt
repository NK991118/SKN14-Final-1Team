유럽 경제 지역 (EEA) 개발자

청구서 수신 주소가 유럽 경제 지역에 있는 경우 2025년 7월 8일부터 서비스 사용 시 Google Maps Platform EEA 서비스 약관 [https://cloud.google.com/terms/maps-platform/eea?hl=ko]이 적용됩니다. 기능은 리전에 따라 다릅니다.
자세히 알아보기 [https://developers.google.com/maps/comms/eea/faq?hl=ko]

경로 행렬에는 거리 및 예상 도착 시간을 비롯하여 여러 출발지와 도착지 간의 추천 경로에 관한 정보 행렬이 포함됩니다.

경로 매트릭스를 사용하면 다음과 같은 사용 사례와 같이 비즈니스 요구사항에 따라 출발지와 도착지의 최적 조합을 선택할 수 있습니다.

배송할 최적의 창고를 찾습니다.
가장 가까운 적합한 차량을 찾아 픽업 또는 하차 요청을 합니다.

Routes API는 경로 유형과 경로 매트릭스에 제공된 정보를 맞춤설정하는 여러 옵션을 제공합니다. 다음 가이드 주제를 참고하여 시작해 보세요.

| 주제 | 설명 |
| --- | --- |
| 경로 매트릭스 가져오기 | 경로 매트릭스를 만드는 방법과 포함해야 할 정보를 알아보세요. |
| 사용 가능한 경로 매트릭스 차량 유형 | 경로를 확인할 차량 유형을 선택합니다. |
| 사용 가능한 경로 매트릭스 옵션 | 경로 매트릭스에 포함할 정보 유형을 선택합니다. |
| 경로 매트릭스 응답 검토 | 애플리케이션에서 응답을 처리하는 방법을 알 수 있도록 Routes API에서 경로 매트릭스가 반환되는 방식을 검토합니다. |
