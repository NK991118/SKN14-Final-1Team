jointType
(int jointType)

Sets the joint type for all vertices of the polyline except the start and end vertices.
See JointType [https://developers.google.com/maps/documentation/navigation/android-sdk/reference/com/google/android/gms/maps/model/JointType] for allowed values. The default value JointType.DEFAULT [https://developers.google.com/maps/documentation/navigation/android-sdk/reference/com/google/android/gms/maps/model/JointType#DEFAULT] will
be used if joint type is undefined or is not one of the allowed values.
