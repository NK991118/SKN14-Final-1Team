Google Maps Platform용 새로운 지도 스타일

컬렉션을 사용해 정리하기

내 환경설정을 기준으로 콘텐츠를 저장하고 분류하세요.

2024년 5월에서 2025년 3월 25일 사이에 Google Maps Platform에서 다음 API와 SDK의 기본 지도 스타일이 업데이트되었습니다.

2024년 8월부터 Google Maps Platform은 가독성, 일관성, 접근성을 개선하기 위해 모든 핀을 현대화된 스타일로 업데이트했습니다.

지도 ID를 사용하는 앱의 경우

Android에서 클라우드 기반 지도 스타일 지정을 사용하려면 Android용 Maps SDK를 위한 최신 렌더기를 사용해야 합니다.
다음 안내를 참고하세요 [https://developers.google.com/maps/documentation/android-sdk/renderer?hl=ko].

2024년 3월 21일부터 앱에서 지도 ID를 사용한 경우 클라우드 기반 지도 스타일 지정을 사용하여 특정 요구사항에 맞게 최신 지도 스타일을 미리 보고 맞춤설정할 수 있습니다. 자세한 내용은 클라우드 기반 지도 스타일 지정 [https://developers.google.com/maps/documentation/javascript/cloud-customization?hl=ko]을 참고하세요.

2025년 3월 25일, 기본 지도 스타일이 최신 버전으로 업데이트되어 새로운 색상 팔레트와 모던화된 핀이 제공되고 지도 환경이 개선되며 사용성도 개선됩니다. 이제 맞춤 스타일이 기본 지도 스타일 위에 표시됩니다.

새 지도 스타일과 모던화된 핀은 roadmap 지도 유형에서만 표시됩니다.
다른 지도 유형의 모던화된 핀을 표시하려면 지도 ID 없이 Maps JavaScript API를 사용하세요.

지도 ID를 사용하지 않는 앱의 경우

Google Maps Platform에서는 다음 표의 일정에 따라 각 API 및 SDK에 대해 업데이트된 기본 지도 스타일과 모던화된 핀* 이 포함된 새 버전을 출시했습니다. 앱에서 지도 ID를 사용하지 않는 경우, 이 새 버전 (또는 이후 버전)을 선택하여 새 기본 지도 스타일을 사용할 수 있습니다.
참고: 2024년 8월 이전에 새로운 기본 지도 스타일로 업데이트된 제품에는 이제 API 및 SDK 버전에 최신 스타일의 핀이 포함됩니다.
| 제품 | 업데이트 세부정보 | 사용 가능한 날짜* |
| --- | --- | --- |
| Maps JavaScript API | v3.58  참고: 버전 번호 또는 채널을 직접 지정하지 않으면 나중에 애플리케이션에 영향을 미칠 수도 있습니다.  자세한 내용은 Maps JavaScript API의 버전 관리를 참고하세요. | 베타 채널: 2024년 3월  주간 채널: 2024년 8월  분기별 채널: 2024년 11월 |
| Android용 Maps SDK | 새 지도 렌더기가 적용된 v19.0.0  자세한 내용은 새 지도 렌더기를 참고하세요. 버전 업데이트에 관한 자세한 내용은 Android용 Maps SDK 버전을 참고하세요. | 2024년 6월 |
| iOS용 Maps SDK | v9.0.0  버전 업데이트에 관한 자세한 내용은 iOS용 Maps SDK 버전을 참고하세요. | 2024년 5월 |
| Android용 Navigation SDK | v6.0.0  버전 업데이트에 관한 자세한 내용은 Navigation SDK 설정을 참고하세요. | 2024년 9월 |
| iOS용 Navigation SDK | v9.0.0  버전 업데이트에 관한 자세한 내용은 Navigation SDK 설치를 참고하세요. | 2024년 5월 |
| Maps Static API | 사용 가능한 경우 업데이트된 지도 스타일이 적용됩니다. | 2024년 8월 |
| Maps Embed API | 사용 가능한 경우 업데이트된 지도 스타일이 적용됩니다. | 2024년 8월 |
| Map Tiles API - 2D 로드맵 | 사용 가능한 경우 업데이트된 지도 스타일이 적용됩니다. | 2025년 3월 |

녹색 체크표시 ()는 해당 제품에 새로운 지도 스타일과 최신식 핀* 을 사용할 수 있음을 나타냅니다.

*최신 스타일의 핀은 2024년 8월에 도입되었습니다.

선택 해제

2025년 3월 25일 이후에는 최신 지도 스타일을 선택 해제할 수 없습니다. 하지만 개발자는 현재 기본 지도 스타일과 유사한 맞춤 지도 스타일을 만들 수 있습니다.
