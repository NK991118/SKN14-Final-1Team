Maps Platform Datasets API

컬렉션을 사용해 정리하기

내 환경설정을 기준으로 콘텐츠를 저장하고 분류하세요.

mapsplatformdatasets.googleapis.com API를 포함하는 방법을 볼 수 있습니다.

서비스: mapsplatformdatasets.googleapis.com
RPC 클라이언트 스터브를 만들려면 서비스 이름 mapsplatformdatasets.googleapis.com이 필요합니다.

google.maps.mapsplatformdatasets.v1.MapsPlatformDatasets [https://developers.google.com/maps/documentation/datasets/reference/rpc/google.maps.mapsplatformdatasets.v1?hl=ko#google.maps.mapsplatformdatasets.v1.MapsPlatformDatasets]

| 메서드 |
| --- |
| CreateDataset | 지정된 프로젝트의 새 데이터 세트를 만듭니다. |
| DeleteDataset | 지정된 데이터 세트를 삭제합니다. |
| DownloadDataset | 데이터 세트를 다운로드합니다. |
| FetchDatasetErrors | 데이터 세트의 모든 오류를 가져옵니다. |
| GetDataset | 데이터 세트를 가져옵니다. |
| ImportDataset | 데이터 세트에 대한 데이터의 새 버전을 가져옵니다. |
| ListDatasets | 지정된 프로젝트의 모든 데이터 세트를 나열합니다. |
| UpdateDatasetMetadata | 데이터 세트의 메타데이터를 업데이트합니다. |
