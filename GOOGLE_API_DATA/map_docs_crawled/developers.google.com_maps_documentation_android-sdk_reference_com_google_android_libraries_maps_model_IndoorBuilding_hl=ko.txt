IndoorBuilding
Object 확장

건물을 나타냅니다.
두 실내 건물이 나타내는 실제 건물이 동일한 경우 두 개의 실내 건물은 .equal()입니다. 그러나
Google의 건물 모델 업데이트로 인해 건물의 구조 모델이 변경되면 이전 실내 건물 객체와 새 실내 건물 객체는 .equal()이 되지만 콘텐츠는
다를 수 있습니다.
