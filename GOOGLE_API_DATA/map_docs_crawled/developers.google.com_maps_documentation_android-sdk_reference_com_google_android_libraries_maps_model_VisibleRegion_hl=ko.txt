공개 최종 클래스 VisibleRegion이 객체를 확장함구현 Parcelable

지도의 카메라에 표시되는 4면 다각형을 정의하는 4개의 점을 포함합니다. 카메라에 기울기가 있을 수 있으므로 이 다각형은 직사각형이 아닌 사다리꼴이 될 수 있습니다. 카메라가
카메라 중앙 바로 위에 있으면 모양은 직사각형이지만, 카메라를 기울일 경우에는
가장 작은 쪽이 시점과 가장 가까운 사다리꼴 모양으로 보입니다.

이 클래스에 정의된 표시 영역을 포함하는 가장 작은 경계 상자입니다.
이 상자가 180° 자오선 (북쪽에서 남쪽으로 향하는 수직선)을 가로지르면 farRight [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/model/VisibleRegion?hl=ko#farRight]의 경도는 음수가 되고 farLeft [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/model/VisibleRegion?hl=ko#farLeft]의 경도는 양수가 됩니다. 이 규칙은 nearRight [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/model/VisibleRegion?hl=ko#nearRight] 및 nearLeft [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/model/VisibleRegion?hl=ko#nearLeft]에도 적용됩니다.

카메라의 네 모서리를 고려하여 새 VisibleRegion을 만듭니다. LatLng 매개변수는
볼록 도형을 정의해야 합니다. 결과로 만들어진 도형의 가장자리가 교차되면 안 됩니다. 런타임에는 경계 검사가 실행되지 않습니다.

매개변수
| nearLeft | 지역 왼쪽 근처 모서리의 위도와 경도가 포함된 LatLng 객체입니다. |
| --- | --- |
| nearRight | 지역 오른쪽 근처 모서리의 위도와 경도가 포함된 LatLng 객체입니다. |
| farLeft | 지역의 맨 왼쪽 모서리의 위도와 경도를 포함하는 LatLng 객체입니다. |
| farRight | 지역의 맨 오른쪽 모서리의 위도와 경도를 포함하는 LatLng 객체입니다. |
| latLngBounds | 이 클래스에 정의된 표시 영역을 포함하는 가장 작은 경계 상자입니다. 이 상자가 180° 자오선 (북쪽에서 남쪽으로의 수직선)을 가로지르면 farRight의 경도는 음수가 되고 farLeft의 경도는 양수가 됩니다. nearRight 및 nearLeft에도 동일하게 적용됩니다. |
