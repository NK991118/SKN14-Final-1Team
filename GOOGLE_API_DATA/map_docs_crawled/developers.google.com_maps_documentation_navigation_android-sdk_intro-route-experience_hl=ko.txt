경로 환경

컬렉션을 사용해 정리하기

내 환경설정을 기준으로 콘텐츠를 저장하고 분류하세요.

경로 환경은 탐색 환경에서 사용되는 경로를 맞춤설정할 수 있는 Navigation SDK 기능을 의미합니다. 경로 환경 기능은 Google 탐색 환경 [https://developers.google.com/maps/documentation/navigation/android-sdk/intro-google-nav?hl=ko]과 맞춤 탐색 환경 [https://developers.google.com/maps/documentation/navigation/android-sdk/intro-custom-nav?hl=ko] 모두에 적용됩니다.

예를 들어 다음과 같은 작업을 할 수 있습니다.

라우팅 환경설정 조정하기 [https://developers.google.com/maps/documentation/navigation/android-sdk/adjust-route-prefs?hl=ko]
경유지 관리 [https://developers.google.com/maps/documentation/navigation/android-sdk/manage-waypoints?hl=ko]
여러 목적지 경로 탐색하기 [https://developers.google.com/maps/documentation/navigation/android-sdk/multi-destination?hl=ko]
경로 계획하기 [https://developers.google.com/maps/documentation/navigation/android-sdk/customize-route?hl=ko]
