public     interface
RouteSegment

A RouteSegment is an immutable class which represents a single "leg" of a journey, either from
the device's current position to the destination, or from one destination to a subsequent
destination, along with NavigationTrafficData [https://developers.google.com/maps/documentation/navigation/android-sdk/reference/com/google/android/libraries/navigation/NavigationTrafficData] along the segment.
