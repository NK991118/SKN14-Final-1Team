GMSNavigationMutableSpeedometerUIOptions

@interface GMSNavigationMutableSpeedometerUIOptions
: GMSNavigationSpeedometerUIOptions [https://developers.google.com/maps/documentation/navigation/ios-sdk/reference/objc/Classes/GMSNavigationSpeedometerUIOptions.html]

The mutable class that encapsulates UI configuration for different severities of speed alerts,
represented by GMSNavigationSpeedAlertSeverity [https://developers.google.com/maps/documentation/navigation/ios-sdk/reference/objc/Enums/GMSNavigationSpeedAlertSeverity.html].

-setTextColor:forSpeedAlertSeverity:lightingMode: [https://developers.google.com/maps/documentation/navigation/ios-sdk/reference/objc/Classes/GMSNavigationMutableSpeedometerUIOptions#/c:objc(cs)GMSNavigationMutableSpeedometerUIOptions(im)setTextColor:forSpeedAlertSeverity:lightingMode:]

Sets the color of the speed alert text to use for the specified speed alert severity and lighting
mode.

@note: Attempting to set the color for GMSNavigationSpeedAlertSeverityUnknown or
GMSNavigationSpeedAlertSeverityNotSpeeding is a no-op.
