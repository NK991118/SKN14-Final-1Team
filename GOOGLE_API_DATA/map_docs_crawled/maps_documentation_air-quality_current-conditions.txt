현재 상태
currentConditions [https://developers.google.com/maps/documentation/air-quality/reference/rest/v1/currentConditions/lookup?hl=ko] 엔드포인트는 현재 시간별 공기질 정보를 제공합니다. 반환되는 AQI 및 관련 정보 (예: 오염물질 농도 수준 및 건강 권장사항)를 제어할 수 있습니다.
다음 주소로 HTTP POST 요청을 전송하여 currentConditions [https://developers.google.com/maps/documentation/air-quality/reference/rest/v1/currentConditions/lookup?hl=ko] 엔드포인트를 사용해 현재 시간별 대기 질을 요청할 수 있습니다.
https://airquality.googleapis.com/v1/currentConditions:lookup?key=YOUR_API_KEY
JSON 요청 본문 [https://developers.google.com/maps/documentation/air-quality/reference/rest/v1/currentConditions/lookup?hl=ko#request-body]에 요청 옵션을 포함합니다. 요청 본문에는 공기질 현재 상태를 가져오려는 위치와 응답에 포함할 공기질 정보를 제어하는 다양한 옵션이 포함되어 있습니다.
API 탐색기를 사용하면 API와 API 옵션을 숙지할 수 있도록 라이브 요청을 할 수 있습니다.
사용해 보기 [https://developers.google.com/maps/documentation/air-quality/current-conditions?hl=ko#try_it]
기본 요청의 예
기본 요청 본문
다음 코드는 currentConditions [https://developers.google.com/maps/documentation/air-quality/reference/rest/v1/currentConditions/lookup?hl=ko] 요청의 기본 요청 본문을 구성하는 방법을 보여줍니다. 이 예시에서는 위치만 설정합니다.
curl -X POST -d '{
  "location": {
    "latitude": 37.419734,
    "longitude": -122.0827784
  }
}' \
-H 'Content-Type: application/json' \
'https://airquality.googleapis.com/v1/currentConditions:lookup?key=YOUR_API_KEY'
기본 요청 응답
위 호출은 다음 JSON 응답 [https://developers.google.com/maps/documentation/air-quality/reference/rest/v1/currentConditions/lookup?hl=ko#response-body]을 생성합니다. 응답 데이터에 관한 자세한 내용은 응답 데이터 [https://developers.google.com/maps/documentation/air-quality/response-data?hl=ko]를 참고하세요.
참고: 기본적으로 응답에는 기본 AQI 정보가 포함된 범용 AQI가 포함됩니다. 해당 매개변수를 설정하여 AQI를 삭제하거나 다른 AQI로 대체할 수 있습니다.
{
  "dateTime": "2023-10-26T18:00:00Z",
  "regionCode": "us",
  "indexes": [
    {
      "code": "uaqi",
      "displayName": "Universal AQI",
      "aqi": 52,
      "aqiDisplay": "52",
      "color": {
        "red": 0.9490196,
        "green": 0.98039216,
        "blue": 0.019607844
      },
      "category": "Moderate air quality",
      "dominantPollutant": "no2"
    }
  ]
}
참고: color 필드 값이 0이면 해당 필드는 응답에서 생략됩니다.
매개변수가 여러 개인 요청의 예
여러 매개변수 요청
다음 코드는 currentConditions [https://developers.google.com/maps/documentation/air-quality/reference/rest/v1/currentConditions/lookup?hl=ko] 요청의 요청 본문을 구성하는 방법을 보여줍니다. 이 예에서는 위치 외에도 몇 가지 추가 계산과 응답 언어가 지정됩니다.
curl -X POST -d '{
  "universalAqi": true,
  "location": {
    "latitude": 37.419734,
    "longitude": -122.0827784
  },
  "extraComputations": [
    "HEALTH_RECOMMENDATIONS",
    "DOMINANT_POLLUTANT_CONCENTRATION",
    "POLLUTANT_CONCENTRATION",
    "LOCAL_AQI",
    "POLLUTANT_ADDITIONAL_INFO"
  ],
  "languageCode": "en"
}' \
-H 'Content-Type: application/json' \
'https://airquality.googleapis.com/v1/currentConditions:lookup?key=YOUR_API_KEY'
