An interface for a class that provides the tile images for a TileOverlay [https://developers.google.com/maps/documentation/navigation/android-sdk/reference/com/google/android/gms/maps/model/TileOverlay]. For information
about the tile coordinate system, see TileOverlay [https://developers.google.com/maps/documentation/navigation/android-sdk/reference/com/google/android/gms/maps/model/TileOverlay].
Calls to methods in this interface might be made from multiple threads so implementations of
this interface must be threadsafe.

getTile
(int x, int y, int zoom)

Returns the tile to be used for this tile coordinate.

Parameters
| x | The x coordinate of the tile. This will be in the range [0, 2zoom - 1] inclusive. |
| --- | --- |
| y | The y coordinate of the tile. This will be in the range [0, 2zoom - 1] inclusive. |
| zoom | The zoom level of the tile. This will be in the range [ GoogleMap.getMinZoomLevel, GoogleMap.getMaxZoomLevel] inclusive. |

Returns
the Tile [https://developers.google.com/maps/documentation/navigation/android-sdk/reference/com/google/android/gms/maps/model/Tile] to be used for this tile coordinate. If you do not wish to provide a
tile for this tile coordinate, return NO_TILE [https://developers.google.com/maps/documentation/navigation/android-sdk/reference/com/google/android/gms/maps/model/TileProvider#NO_TILE]. If the tile could not be found at
this point in time, return null and further requests might be made with an exponential
backoff.
