컬렉션을 사용해 정리하기

내 환경설정을 기준으로 콘텐츠를 저장하고 분류하세요.

JSON 표현 [https://developers.google.com/maps/documentation/aerial-view/reference/rest/v1/VideoMetadata?hl=ko#SCHEMA_REPRESENTATION]
날짜 [https://developers.google.com/maps/documentation/aerial-view/reference/rest/v1/VideoMetadata?hl=ko#Date]

JSON 표현 [https://developers.google.com/maps/documentation/aerial-view/reference/rest/v1/VideoMetadata?hl=ko#Date.SCHEMA_REPRESENTATION]

동영상에 대한 메타데이터(예: videoId 및 길이)를 포함합니다.

| JSON 표현 |
| --- |
| {   "videoId": string,   "captureDate": {     object (Date)   },   "duration": string,   "state": enum (State) } |

| 필드 |
| --- |
| videoId | string 출력 전용입니다. 동영상의 ID 및 권장되는 동영상 검색 방법입니다. |
| captureDate | object (Date) 출력 전용입니다. 동영상에 사용된 이미지를 캡처한 날짜입니다. 이 기간은 월별 단위로 세분화됩니다. |
| duration | string (Duration format) 출력 전용입니다. 동영상의 길이입니다. 소수점 아래가 최대 9자리까지이고 's'로 끝나는 초 단위 기간입니다. 예를 들면 "3.5s"입니다. |
| state | enum (State) 출력 전용입니다. 렌더링 요청의 현재 상태입니다. |

날짜

생일과 같은 전체 또는 부분 달력 날짜를 나타냅니다. 시간과 시간대는 아무 곳으로 지정되거나 중요하지 않습니다. 날짜는 그레고리력을 기준으로 합니다. 이는 다음 중 하나를 나타낼 수 있습니다.

연도, 월, 일 값이 0이 아닌 전체 날짜
월과 일, 연도는 0(예: 기념일)
연도만, 월과 일은 0
연도와 월, 일은 0(예: 신용카드 만료일)
관련 유형:

google.type.TimeOfDay
google.type.DateTime
google.protobuf.Timestamp [https://developers.google.com/protocol-buffers/docs/reference/google.protobuf?hl=ko#google.protobuf.Timestamp]

| JSON 표현 |
| --- |
| {   "year": integer,   "month": integer,   "day": integer } |

| 필드 |
| --- |
| year | integer 날짜의 연도입니다. 1~9999 사이여야 하며, 연도 없이 날짜를 지정하려면 0이어야 합니다. |
| month | integer 연도의 월입니다. 1~12 사이여야 하며, 월과 일 없이 연도를 지정하려면 0이어야 합니다. |
| day | integer 월의 일입니다. 1~31 사이여야 하고 해당 연도 및 월에 유효해야 합니다. 또는 연도만 지정하거나 일이 중요하지 않아 연도와 월을 지정하려면 0이어야 합니다. |
