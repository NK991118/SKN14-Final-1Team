Data-driven styling features are not available to the Navigation SDK. They exists for compile
time compatibility with com.google.android.gms:play-services-maps only.
Contains a LatLng [https://developers.google.com/maps/documentation/navigation/android-sdk/reference/com/google/android/gms/maps/model/LatLng] and a list of data-driven styling features associated with a click
event.
