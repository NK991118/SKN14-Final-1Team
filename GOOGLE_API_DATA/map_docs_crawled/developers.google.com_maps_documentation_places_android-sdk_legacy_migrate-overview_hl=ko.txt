이 서비스 이용에 적용됩니다. 기능은 리전에 따라 다릅니다.

Android용 Places SDK (신규)는 성능이 개선되고 새로운 가격 모델을 제공하므로 Android용 Places SDK (기존)를 사용하는 앱을 업데이트할 가치가 있습니다. 기능 비교에 관한 자세한 내용은 SDK 선택 [https://developers.google.com/maps/documentation/places/android-sdk/choose-sdk?hl=ko]을 참고하세요.

이 가이드를 통해 Android용 Places SDK (신규)와 Android용 Places SDK (기존)의 주요 차이점과 필요한 변경사항을 처리하는 방법을 알아보세요.

마이그레이션 결제 권장사항
warning_amber
이 안내는 API 사용량이 2단계 요금으로 전환될 만큼 많은 경우에 적용됩니다. API의 최신 버전으로 이전하면 다른 SKU에 대한 요금도 청구됩니다. 전환하는 달에 비용이 증가하지 않도록 하려면 가능한 한 달 초에 프로덕션에서 새 API로 전환하는 것이 좋습니다. 이렇게 하면 이전 달에 가장 비용 효율적인 월간 요금 등급에 도달할 수 있습니다. 가격 등급에 대한 자세한 내용은 가격 페이지 [https://developers.google.com/maps/billing-and-pricing/pricing?hl=ko] 및 가격 FAQ [https://developers.google.com/maps/billing-and-pricing/faq?hl=ko]를 참고하세요.

Android용 Places SDK (신규) 사용 설정

Android용 Places SDK (레거시)는 Places API 서비스를 사용합니다. Android용 Places SDK (신규)의 기능을 사용하려면 먼저 Google Cloud 프로젝트에서 Places API (신규)를 사용 설정해야 합니다. 자세한 내용은 Google Cloud 프로젝트 설정 [https://developers.google.com/maps/documentation/places/android-sdk/cloud-setup?hl=ko]을 참고하세요.

그런 다음 앱에서 사용하는 API 키에 Places API(New)를 추가해야 합니다. 자세한 내용은 API 키 사용 [https://developers.google.com/maps/documentation/places/android-sdk/get-api-key?hl=ko]을 참고하세요.

일반적인 변경사항

여러 API에 적용되는 일반적인 변경사항은 다음과 같습니다.

새로운 가격 책정 모델을 사용합니다. 모든 API의 가격 정보는 Android용 Places SDK(신규) [https://developers.google.com/maps/billing-and-pricing/pricing?hl=ko#places-pricing]를 참고하세요.
장소 세부정보(신규) [https://developers.google.com/maps/documentation/places/android-sdk/place-details?hl=ko], 주변 지역 검색(신규) [https://developers.google.com/maps/documentation/places/android-sdk/nearby-search?hl=ko], 텍스트 검색(신규) [https://developers.google.com/maps/documentation/places/android-sdk/text-search?hl=ko]에는 응답에서 반환할 필드를 지정하는 필드 마스크가 필요합니다.
요청을 정의하거나 응답에 새 필드를 추가하기 위해 새 클래스가 추가되었습니다.
Place [https://developers.google.com/maps/documentation/places/android-sdk/reference/com/google/android/libraries/places/api/model/Place?hl=ko] 객체에는 Review [https://developers.google.com/maps/documentation/places/android-sdk/reference/com/google/android/libraries/places/api/model/Review?hl=ko] 객체 목록을 반환하는 새로운 getReviews() [https://developers.google.com/maps/documentation/places/android-sdk/reference/com/google/android/libraries/places/api/model/Place?hl=ko#getReviews()] 메서드가 포함되어 있습니다. 앱에서 사진, 리뷰 등 Place 객체에서 가져온 정보를 표시하는 경우 필수 저작자 표시도 함께 표시해야 합니다.

자세한 내용은 저작자 표시 [https://developers.google.com/maps/documentation/places/android-sdk/policies?hl=ko#other_attribution_requirements] 문서를 참고하세요.

참고: 텍스트 검색 (신규)과 주변 검색 (신규)이 Android용 Places SDK에 추가되었으며 기존 API를 대체하지 않았으므로 이를 사용하기 위해 마이그레이션할 필요가 없습니다.
API별 변경사항

이 섹션에는 각 API에 대한 다음 이전 가이드가 포함되어 있습니다.

장소 세부정보 (신규)로 이전 [https://developers.google.com/maps/documentation/places/android-sdk/migrate-details?hl=ko]
장소 사진 (신규)으로 이전 [https://developers.google.com/maps/documentation/places/android-sdk/migrate-photo?hl=ko]
Autocomplete (신규)로 이전 [https://developers.google.com/maps/documentation/places/android-sdk/migrate-autocomplete?hl=ko]
