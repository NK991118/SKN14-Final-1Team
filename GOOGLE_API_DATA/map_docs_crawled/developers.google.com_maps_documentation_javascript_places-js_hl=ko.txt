개요
Maps JavaScript API는 Google의 장소 데이터를 앱에 통합할 수 있는 다양한 옵션을 제공합니다.

장소 클래스
장소 클래스 [https://developers.google.com/maps/documentation/javascript/place?hl=ko]는 장소 라이브러리에서 사용되는 위치의 표준 API 표현입니다. 장소 클래스를 사용하면 필요에 맞게 맞춤설정할 수 있는 지능형 검색 기능을 통해 사용자에게 자세한 위치 데이터를 제공할 수 있습니다. Places 서비스의 데이터는 실제 세계에서 가장 정확하고 최신이며 포괄적인 장소 모델 중 하나를 나타냅니다.

장소 UI 키트
장소 UI 키트 [https://developers.google.com/maps/documentation/javascript/places-ui-kit/overview?hl=ko]는 사용하기 쉽고 맞춤설정이 가능하며 저렴한 비용의 구성요소 라이브러리로, 익숙하고 신뢰할 수 있는 Google 지도 사용자 환경을 앱에 구현할 수 있습니다. 장소 UI 키트 요소는 장소 라이브러리 [https://developers.google.com/maps/documentation/javascript/places?hl=ko]와 고도 라이브러리 [https://developers.google.com/maps/documentation/javascript/elevation?hl=ko]에서 사용할 수 있습니다.

장소 서비스 (기존)
Places Library, Maps JavaScript API [https://developers.google.com/maps/documentation/javascript/places?hl=ko]에는 기존 장소 서비스의 메서드와 데이터가 포함되어 있으며, 이는 장소 클래스 [https://developers.google.com/maps/documentation/javascript/place?hl=ko]로 대체되었습니다. 앱을 이전하여 Places 라이브러리의 Place 클래스를 사용하려면 Places (신규)로 이전 [https://developers.google.com/maps/documentation/javascript/places-migration-overview?hl=ko]의 가이드를 참고하세요.

장소 클래스arrow_drop_down

장소 클래스는 장소 라이브러리를 사용하기 위한 API를 제공하며 프로미스 [https://developers.google.com/maps/documentation/javascript/promises?hl=ko]와 같은 최신 사용 패턴을 지원합니다.
Place 클래스를 사용하면 앱에서 위치 인식 기능을 만들고, 필요에 따라 맞춤설정할 수 있는 지능형 검색 기능을 통해 사용자에게 자세한 위치 데이터를 제공할 수 있습니다. Places 서비스의 데이터는 실제 세계에서 가장 정확하고 최신이며 포괄적인 장소 모델 중 하나입니다.
