경로 시작하기
Routes Library, Maps JavaScript API를 설정하려면 다음 단계를 따르세요.
API 키 가져오기 및 Maps JavaScript API 사용 설정
Routes Library, Maps JavaScript API를 사용하려면 결제 계정이 있는 Cloud 프로젝트가 있어야 하고 Maps JavaScript API와 Routes API가 모두 사용 설정되어 있어야 합니다. 자세한 내용은 Google Cloud 프로젝트 설정 [https://developers.google.com/maps/documentation/javascript/cloud-setup?hl=ko]을 참고하세요.
API 키 가져오기 [https://developers.google.com/maps/documentation/javascript/get-api-key?hl=ko]
Maps JavaScript API 사용 설정 [https://console.cloud.google.com/apis/library/maps-backend.googleapis.com?hl=ko]
Routes API 사용 설정 [https://console.cloud.google.com/apis/library/routes.googleapis.com?hl=ko]
지도 초기화 코드 업데이트
Maps JavaScript API를 로드 [https://developers.google.com/maps/documentation/javascript/load-maps-js-api?hl=ko]합니다.
필요한 경우 async 함수 내에서 Routes 라이브러리를 로드합니다.
    const { Route, RouteMatrix } = await google.maps.importLibrary('routes');
