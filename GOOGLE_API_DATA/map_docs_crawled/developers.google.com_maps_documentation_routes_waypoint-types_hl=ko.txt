유럽 경제 지역 (EEA) 개발자

청구서 수신 주소가 유럽 경제 지역에 있는 경우 2025년 7월 8일부터 서비스 사용 시 Google Maps Platform EEA 서비스 약관 [https://cloud.google.com/terms/maps-platform/eea?hl=ko]이 적용됩니다. 기능은 리전에 따라 다릅니다.
자세히 알아보기 [https://developers.google.com/maps/comms/eea/faq?hl=ko]

경유지란 경로의 시작 (출발) 위치와 끝(도착) 위치, 경로 행렬의 출발지와 목적지를 포함하여 경로의 한 지점입니다.

위치 지정 [https://developers.google.com/maps/documentation/routes/specify_location?hl=ko]에 설명된 대로 웨이포인트의 위치를 지정하는 방법에는 여러 가지가 있습니다. 시작 위치와 종료 위치를 설정하는 것 외에도 경로가 통과할 중간 경로를 추가하고 경로 동작에 관한 몇 가지 옵션을 사용할 수 있습니다.

| 경유지 옵션 | 설명 |
| --- | --- |
| 경유지 위치 지정 | 경로의 시작점과 끝점, 경로 중간 지점을 지정합니다. |
| 중간 경유지 설정 | 중간 웨이포인트를 설정하여 경로가 통과할 출발지와 목적지 사이의 위치를 지정합니다. 중간 웨이포인트는 경유 또는 통과 웨이포인트일 수 있습니다 (아래 섹션 참고). |
| 경로에 정류장 설정하기 | 중간 경유지를 경로의 정류장(예: 승하차)으로 지정합니다. |
| 경로가 통과할 지점 설정 | 경로가 통과할 중간 웨이포인트를 지정합니다. |
| 경유지의 차량 방향 지정 | 차량이 도착했을 때의 방향을 지정합니다. 도로변에서는 사용할 수 없습니다. |
| 경로의 도로 측면 지정 | 차량이 도착할 도로의 측면을 지정합니다. 제목과 함께 사용할 수 없습니다. |
| 경로의 정류장 순서 최적화 | 차량이 각 웨이포인트에 도착할 때 가고자 하는 방향을 지정합니다. |
