HTTP 요청
POST https://fleetengine.googleapis.com/v1/{parent=providers/*}/deliveryVehiclesURL은 gRPC 트랜스코딩 [https://google.aip.dev/127] 문법을 사용합니다.

경로 매개변수
| 매개변수 |
| --- |
| parent | string 필수 항목입니다. providers/{provider} 형식이어야 합니다. 제공업체는 Google Cloud 프로젝트 ID여야 합니다. 예를 들면 sample-cloud-project입니다. |

쿼리 매개변수
| 매개변수 |
| --- |
| header | object (DeliveryRequestHeader) 선택사항. 표준 게재 API 요청 헤더입니다. |
| deliveryVehicleId | string 필수 항목입니다. 배송 차량 ID는 고유해야 하며 다음 제한사항이 적용됩니다. 유효한 유니코드 문자열이어야 합니다. 최대 길이는 64자(영문 기준)로 제한됩니다. 유니코드 정규화 양식 C에 따라 정규화됩니다. '/', ':', '?', ',', '#' ASCII 문자는 포함할 수 없습니다. |

요청 본문
요청 본문에 DeliveryVehicle [https://developers.google.com/maps/documentation/mobility/fleet-engine/reference/tasks/rest/v1/providers.deliveryVehicles?hl=ko#DeliveryVehicle]의 인스턴스가 포함됩니다.
