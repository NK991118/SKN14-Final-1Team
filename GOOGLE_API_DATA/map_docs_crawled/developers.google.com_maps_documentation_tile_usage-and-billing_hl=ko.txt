알림: Map Tiles API를 사용하려면 각 프로젝트에서 결제를 사용 설정 [https://console.cloud.google.com/project/_/billing/enable?utm_source=Docs_BillingNudge&hl=ko]하고 모든 API 또는
SDK 요청에 API 키 [https://developers.google.com/maps/documentation/tile/get-api-key?hl=ko] 또는 OAuth 토큰을 포함해야 합니다.

사용량 한도

모든 타일 요청 (2D 타일, 스트리트 뷰 타일, 포토리얼리스틱 3D 타일의 루트 타일 요청)은 Map Tiles API의 프로젝트 일일 할당량 ('요금 기준액')에 반영됩니다. 세션 토큰 요청, 표시 영역 정보 요청, 스트리트 뷰 메타데이터 요청, 실사 3D 타일의 타일 요청은 일일 할당량에 영향을 미치지 않습니다.
참고: Map Tiles API 기본 할당량은 변경될 수 있습니다. 할당량 상향이 부여된 경우 프로젝트의 할당량이 이 페이지에 명시된 기본값과 다를 수 있습니다. 할당량을 늘려야 하는 경우 지원팀 [https://developers.google.com/maps/support?hl=ko#contact-maps-support]에 문의하세요.
2D 타일 및 스트리트 뷰 타일

동일한 프로젝트의 사용자 인증 정보를 사용하는 모든 애플리케이션의 모든 요청의 합계로 계산되는 프로젝트별 분당 최대 2D 타일, 스트리트 뷰 타일, 썸네일 또는 스트리트 뷰 파노라마 ID 쿼리 6,000개
동일한 프로젝트의 사용자 인증 정보를 사용하는 모든 애플리케이션의 모든 요청의 합계로 계산되는 프로젝트당 분당 최대 6,000개의 세션 토큰 요청, 뷰포트 정보 요청 또는 스트리트 뷰 메타데이터 요청
프로젝트별 일일 최대 2D 타일, 스트리트 뷰 타일, 썸네일 또는 스트리트 뷰 파노라마 ID 쿼리 15,000개

포토리얼리스틱 3D 타일

일일 최대 10,000개의 루트 타일셋 쿼리 이는 동일한 프로젝트의 사용자 인증 정보를 사용하는 모든 애플리케이션의 모든 요청의 합계로 계산됩니다.
시간 제한 세션 토큰을 사용하면 단일 루트 타일 세트 요청에서 최대 3시간의 렌더러 타일 요청이 허용됩니다.
일일 렌더러 시작 타일 요청 무제한 이는 동일한 프로젝트의 사용자 인증 정보를 사용하는 모든 애플리케이션의 모든 요청의 합계로 계산됩니다.
타일 렌더러의 비율 제한은 분당 12,000개 쿼리입니다.

할당량 조정

할당량 제한은 특정 기간 내에 특정 API 또는 서비스에 허용되는 최대 요청 수를 정의합니다. 프로젝트의 요청 수가 할당량 한도에 도달하면 서비스가 요청에 대한 응답을 중지합니다.

중요: 할당량 증가가 청구액에 미치는 영향을 알아보려면 가격 계산기 [https://mapsplatform.google.com/pricing/?hl=ko#pricing-calculator]를 확인하세요.

API의 할당량 값을 수정하려면 다음 단계를 따르세요.

Cloud 콘솔에서 Google Maps Platform > 할당량으로 이동합니다.
할당량을 수정할 API를 선택합니다.
변경하려는 할당량 값을 식별하고 체크박스를 사용하여 선택합니다.
수정을 클릭하고 새 할당량 값을 입력한 다음 요청 제출을 클릭합니다.

할당량 상향 요청 보기
중요: 할당량 상향 요청을 볼 수 있는 적절한 IAM 권한 [https://cloud.google.com/docs/quotas/permissions?hl=ko#permissions_increase]이 있어야 합니다.

이전 요청과 대기 중인 요청을 포함한 모든 할당량 상향 요청을 보려면 다음 단계를 따르세요.

Cloud 콘솔에서 Google Maps Platform > 할당량으로 이동합니다.
할당량 증가 요청을 보려는 API를 선택합니다.
상향 요청을 클릭합니다.

도움말: 서비스 중단을 방지하려면 할당량 사용량을 정기적으로 모니터링하세요. 할당량 및 할당량 알림에 대해 자세히 알아보기 [https://developers.google.com/maps/billing-and-pricing/manage-costs?hl=ko#quotas]
