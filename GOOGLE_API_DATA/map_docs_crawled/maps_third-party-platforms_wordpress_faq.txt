WordPress 사용자 관련
자주 묻는 질문(FAQ)
API란 무엇인가요?
API는 '애플리케이션 프로그래밍 인터페이스(Application Programing Interface)'의 줄임말입니다. WordPress 플러그인은 Google Maps Platform API를 사용하여 WordPress 사이트에 사용할 Google Maps Platform의 데이터와 기능을 요청합니다.
API 키란 무엇이며 왜 필요한가요?
API 키는 WordPress 플러그인에서 사용자를 대신하여 Google Maps Platform에 요청을 전송하는 데 사용되는 고유한 사용자 인증 정보입니다. API 키가 생성되면 WordPress 플러그인에 키를 제공하여 플러그인이 Google Maps Platform에 액세스하도록 할 수 있습니다.
어떤 Google Maps Platform API를 사용해야 하나요?
Maps JavaScript API, Places API, Directions API는 WordPress 플러그인에서 가장 일반적으로 사용하는 Google Maps Platform API입니다. 이러한 API를 사용 설정하면 WordPress 플러그인이 사용자를 대신하여 지도를 로드하고, 장소 데이터를 검색하고, 경로를 생성할 수 있습니다.
Google의 API 키 제작 도구 [https://developers.google.com/maps/third-party-platforms/wordpress/generate-api-key?hl=ko]를 사용하여 WordPress 플러그인에 사용할 새 API 키를 생성한 경우, 이 세 가지 API가 기본으로 자동 사용 설정됩니다.
사용 설정/중지한 API를 어떻게 알 수 있나요?
Google Cloud 콘솔의 Google Maps Platform API 페이지 [https://console.cloud.google.com/google/maps-apis/api-list?hl=ko]로 이동하여 Google Cloud 프로젝트에 사용 설정된 API를 확인할 수 있습니다.
현재 사용 설정된 API는 '사용 설정된 API' 아래에 표시됩니다.
현재 사용 중지된 API는 '추가 API' 아래에 표시됩니다.
사용 설정/중지되는 API를 변경하려면 어떻게 해야 하나요?
Google Cloud 콘솔의 Google Maps Platform API 페이지 [https://console.cloud.google.com/google/maps-apis/api-list?hl=ko]로 이동하여 언제든지 Google Maps Platform API를 사용 설정 또는 중지할 수 있습니다.
API를 사용 중지하려면 '사용 설정된 API' 목록에서 사용 중지하려는 API의 이름을 클릭한 후 페이지 상단에서 '사용 중지' 버튼을 클릭합니다.
API를 사용 설정하려면 '추가 API' 목록에서 사용 설정하려는 API의 이름을 클릭한 다음 '사용 설정' 버튼을 클릭합니다.
Google Maps Platform을 사용하는 데 비용이 얼마나 드나요?
Google Maps Platform에서는 Google Cloud 프로젝트에 연결된 결제 계정에 매월 200달러 상당의 무료 사용량을 자동으로 제공합니다.
지도 로드, 경로, 장소 정보 요청과 같이 Google Maps Platform API에 대해 이루어지는 요청별로 사용량에 대한 요금이 부과됩니다. 요청당 비용은 요청 유형과 월간 총 요청 수에 따라 다릅니다. 자세한 내용은 요금 계산기를 참고하세요.
요금 계산기로 이동 [https://mapsplatform.google.com/pricing/?hl=ko]
