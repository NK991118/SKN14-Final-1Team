Places 통계 청구 방식

참고: 장소 통계는 GA 이전 프리뷰 단계에 있으며 프리뷰 기간에는 사용 요금이 청구되지 않습니다. Cloud 콘솔의 결제 페이지에 통화가 표시되지만 청구 금액은 0달러입니다.

미리보기 기간에는 장소 통계 데이터에 액세스하는 데 비용이 청구되지 않지만 BigQuery 사용과 관련된 비용은 사용자가 부담해야 합니다. 자세한 내용은 BigQuery 가격 [https://cloud.google.com/bigquery/pricing?hl=ko]을 참조하세요.

이용약관 제한

이용약관에 대한 자세한 내용은 장소 통계 정책 [https://developers.google.com/maps/documentation/placesinsights/policies?hl=ko] 및 Google Maps Platform 서비스 약관의 라이선스 제한 섹션 [https://cloud.google.com/maps-platform/terms/?hl=ko#3.-license.]을 참고하세요.
