TileProvider [https://developers.google.com/maps/documentation/navigation/android-sdk/reference/com/google/android/gms/maps/model/TileProvider]

A partial implementation of TileProvider [https://developers.google.com/maps/documentation/navigation/android-sdk/reference/com/google/android/gms/maps/model/TileProvider] that only requires a URL that points to an image
to be provided.
Note that this class requires that all the images have the same dimensions.
