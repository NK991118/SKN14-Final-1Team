유럽 경제 지역 (EEA) 개발자

청구서 수신 주소가 유럽 경제 지역에 있는 경우 2025년 7월 8일부터 서비스 사용 시 Google Maps Platform EEA 서비스 약관 [https://cloud.google.com/terms/maps-platform/eea?hl=ko]이 적용됩니다. 기능은 리전에 따라 다릅니다.
자세히 알아보기 [https://developers.google.com/maps/comms/eea/faq?hl=ko]

다양한 유형의 차량에 대한 경로 매트릭스를 가져올 수 있습니다. 차량 유형을 지정하지 않으면 Routes API는 기본값인 DRIVE를 사용합니다.

| 차량 유형 | 설명 |
| --- | --- |
| 여행 모드 | 이동 수단을 설정합니다. 운전, 이륜차, 대중교통, 도보 중에서 선택할 수 있습니다. 경로 매트릭스에는 자전거가 지원되지 않습니다. 참고: 이륜차, 도보, 자전거 경로는 베타 버전이며 명확한 인도, 보행자 전용 도로 또는 자전거 전용 도로가 누락될 수 있습니다. 앱에 표시하는 모든 이륜차, 걷기 또는 자전거 경로에 대해 이 경고를 사용자에게 표시해야 합니다. |
| 이륜차 | Travel_mode 매개변수를 사용하여 경로의 경로 매트릭스를 지정하는 것과 동일한 방식으로 이륜차의 경로 매트릭스를 지정합니다. |
| 대중교통 | 해당 지역에서 이용 가능한 대중교통 옵션을 사용하는 대중교통 경로 매트릭스를 가져옵니다. 대중교통 옵션에는 버스, 지하철, 기차 등이 포함될 수 있습니다. |
