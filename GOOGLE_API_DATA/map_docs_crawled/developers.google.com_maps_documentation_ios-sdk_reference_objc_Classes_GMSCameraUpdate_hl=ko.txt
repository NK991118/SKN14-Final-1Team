GMSCameraUpdate

@interface GMSCameraUpdate : NSObject

GMSCameraUpdate represents an update that may be applied to a GMSMapView [https://developers.google.com/maps/documentation/ios-sdk/reference/objc/Classes/GMSMapView.html].

It encapsulates some logic for modifying the current camera.

It should only be constructed using the factory helper methods below.

+fitBounds: [https://developers.google.com/maps/documentation/ios-sdk/reference/objc/Classes/GMSCameraUpdate#/c:objc(cs)GMSCameraUpdate(cm)fitBounds:]

Returns a GMSCameraUpdate that transforms the camera such that the specified bounds are
centered on screen at the greatest possible zoom level. The bounds will have a default padding of
64 points.

The returned camera update will set the camera’s bearing and tilt to their default zero values
(i.e., facing north and looking directly at the Earth).

+fitBounds:withPadding: [https://developers.google.com/maps/documentation/ios-sdk/reference/objc/Classes/GMSCameraUpdate#/c:objc(cs)GMSCameraUpdate(cm)fitBounds:withPadding:]

This is similar to +fitBounds: [https://developers.google.com/maps/documentation/ios-sdk/reference/objc/Classes/GMSCameraUpdate.html#/c:objc(cs)GMSCameraUpdate(cm)fitBounds:] but allows specifying the padding (in points) in order to inset
the bounding box from the view’s edges.

If the requested padding is larger than the view size in either the vertical or horizontal
direction the map will be maximally zoomed out.
