공개 인터페이스 OnStreetViewPanoramaReadyCallback

스트리트 뷰 파노라마를 사용할 준비가 된 경우에 대한 콜백 인터페이스입니다.
이 인터페이스의 인스턴스가 StreetViewPanoramaFragment [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/StreetViewPanoramaFragment?hl=ko] 또는 StreetViewPanoramaView [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/StreetViewPanoramaView?hl=ko] 객체에 설정되면 파노라마를 사용할 준비가 되면 onStreetViewPanoramaReady(StreetViewPanorama) [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/OnStreetViewPanoramaReadyCallback?hl=ko#onStreetViewPanoramaReady(com.google.android.libraries.maps.StreetViewPanorama)] 메서드가 트리거되고 null이 아닌 StreetViewPanorama [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/StreetViewPanorama?hl=ko] 인스턴스를 제공합니다.
Google Play 서비스가 기기에 설치되어 있지 않으면 사용자에게 설치하라는 메시지가 표시되며 onStreetViewPanoramaReady(StreetViewPanorama) [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/OnStreetViewPanoramaReadyCallback?hl=ko#onStreetViewPanoramaReady(com.google.android.libraries.maps.StreetViewPanorama)] 메서드는 사용자가 서비스를 설치하고 앱으로 돌아올 때만 트리거됩니다.

공개 메서드 요약
| 추상 void | onStreetViewPanoramaReady(StreetViewPanorama 파노라마) 스트리트 뷰 파노라마를 사용할 준비가 되면 호출됩니다. |
