플랫폼별 Google Maps Platform API

컬렉션을 사용해 정리하기

내 환경설정을 기준으로 콘텐츠를 저장하고 분류하세요.

어떤 API가 필요한지 잘 모르시나요?
API 선택기를 사용해 보세요 [https://developers.google.com/maps/documentation/api-picker?hl=ko].

Android

Android용 Maps SDK [https://developers.google.com/maps/documentation/android-sdk?hl=ko].
Android 앱용 지도

Android용 Places SDK [https://developers.google.com/maps/documentation/places/android-sdk?hl=ko].
사용자를 수백만 곳의 장소 정보와 연결합니다.

Android용 Navigation SDK [https://developers.google.com/maps/documentation/navigation/android-sdk?hl=ko]
Android 앱을 위한 동적 탐색 환경을 빌드하세요.

iOS

iOS용 Maps SDK [https://developers.google.com/maps/documentation/ios-sdk?hl=ko].
iOS 앱용 지도

iOS용 Places SDK [https://developers.google.com/maps/documentation/places/ios-sdk?hl=ko].
사용자를 수백만 곳의 장소 정보와 연결합니다.

iOS용 Navigation SDK [https://developers.google.com/maps/documentation/navigation/ios-sdk?hl=ko].
iOS 앱을 위한 동적 탐색 환경을 빌드하세요.

Web API

Maps Embed API [https://developers.google.com/maps/documentation/embed/guide?hl=ko].
코드 또는 할당량 제한 없이 Google 지도를 사이트에 추가합니다.

Maps JavaScript API [https://developers.google.com/maps/documentation/javascript/tutorial?hl=ko].
자신만의 콘텐츠와 이미지를 사용하여 지도를 맞춤설정하세요.

Maps JavaScript API의 장소 라이브러리 [https://developers.google.com/maps/documentation/javascript/place?hl=ko].
수백만 곳의 장소에 대한 최신 정보

Maps Static API [https://developers.google.com/maps/documentation/maps-static?hl=ko].
최소한의 코드로 삽입 가능한 간단한 지도 이미지입니다.

Street View Static API [https://developers.google.com/maps/documentation/streetview?hl=ko].
실제 영상 및 파노라마 영상

Web Service API

Address Validation API [https://developers.google.com/maps/documentation/address-validation?hl=ko]
주소 및 구성요소를 확인합니다.

Aerial View API [https://developers.google.com/maps/documentation/aerial-view?hl=ko]
Google의 3D 지리 공간 이미지를 사용하여 렌더링된 항공 뷰 동영상을 만들고 표시합니다.

Air Quality API [https://developers.google.com/maps/documentation/air-quality?hl=ko]
특정 위치의 공기질 데이터를 요청합니다.

Elevation API [https://developers.google.com/maps/documentation/elevation?hl=ko].
전 세계 모든 지점에 대한 고도 데이터

Geocoding API [https://developers.google.com/maps/documentation/geocoding?hl=ko].
주소와 지리 좌표를 서로 변환합니다.

Geolocation API [https://developers.google.com/maps/documentation/geolocation?hl=ko].
휴대폰 기지국 및 Wi-Fi 노드의 위치 데이터

Maps Datasets API [https://developers.google.com/maps/documentation/datasets?hl=ko].
Google Cloud 콘솔에 지리 공간 데이터를 업로드하고 관리하여 데이터 기반 스타일 지정에 사용하세요.

Map Tiles API [https://developers.google.com/maps/documentation/tile?hl=ko]
생생한 맞춤 지도 시각화를 빌드하기 위한 고해상도 실사 3D 타일, 2D 타일, 스트리트 뷰 타일을 제공합니다.

Places API [https://developers.google.com/maps/documentation/places/web-service?hl=ko].
수백만 곳의 장소에 대한 최신 정보

Places Aggregate API [https://developers.google.com/maps/documentation/places-aggregate?hl=ko]
특정 지역의 장소 밀도와 분포를 파악합니다.

Pollen API [https://developers.google.com/maps/documentation/pollen?hl=ko]
특정 위치의 유형, 식물, 지수를 포함한 꽃가루 정보를 가져옵니다.

Roads API [https://developers.google.com/maps/documentation/roads?hl=ko].
도로에 스냅하기(Snap to Road) 기능을 통해 GPS를 통한 이동 경로를 정확하게 추적합니다.

Routes API [https://developers.google.com/maps/documentation/routes?hl=ko].
차세대 성능 최적화 버전의 Directions API (기존) 및 Distance Matrix API (기존)입니다.

Route Optimization API [https://developers.google.com/maps/documentation/route-optimization?hl=ko].
하나 이상의 차량에 대해 여러 경유지를 포함하는 경로를 유연하고 지능적으로 최적화하세요.

Solar API [https://developers.google.com/maps/documentation/solar?hl=ko]
수억 개의 건물의 태양광 잠재력을 확인할 수 있습니다.

Time Zone API [https://developers.google.com/maps/documentation/timezone?hl=ko].
전 세계 모든 지점에 대한 시간대 데이터

Weather API [https://developers.google.com/maps/documentation/weather?hl=ko].
전 세계 여러 지역의 현재 날씨, 시간별 및 일별 일기예보, 시간별 날씨 기록을 확인할 수 있습니다.

교차 플랫폼

지도 URL [https://developers.google.com/maps/documentation/urls/guide?hl=ko].
Google 지도를 실행하고 일반적인 교차 플랫폼 URL 구성표를 사용하여 검색이나 경로 같은 작업을 시작합니다.
