GMSGeometryLength

extern CLLocationDistance GMSGeometryLength(GMSPath [https://developers.google.com/maps/documentation/ios-sdk/reference/objc/Classes/GMSPath.html] *_Nonnull path)

Returns the great circle length of path, in meters, on Earth.

This is the sum of GMSGeometryDistance() over the path segments.

All the coordinates of the path must be valid.
