GMSPanoramaCallback

typedef void (^GMSPanoramaCallback)(GMSPanorama [https://developers.google.com/maps/documentation/ios-sdk/reference/objc/Classes/GMSPanorama.html] *_Nullable, NSError *_Nullable)

Callback for when a panorama metadata becomes available.
If an error occurred, panorama is nil and error is not nil.
Otherwise, panorama is not nil and error is nil.

@related GMSPanoramaService [https://developers.google.com/maps/documentation/ios-sdk/reference/objc/Classes/GMSPanoramaService.html]
