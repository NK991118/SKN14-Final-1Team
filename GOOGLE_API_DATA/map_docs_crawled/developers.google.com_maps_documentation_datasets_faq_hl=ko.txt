API에 비율 제한이 있나요?

Maps Datasets API의 기본 비율 한도는 다음과 같습니다.

모든 읽기 API에 분당 3,000개의 쿼리 수 (QPM)
모든 쓰기 API에서 분당 600개의 쿼리 (QPM)

결제를 사용 설정해야 하나요?

Maps Datasets API를 사용하려면 유효한 결제 계정이 필요하지만 요금이 청구되지는 않습니다. 결제 사용 설정 [https://developers.google.com/maps/documentation/datasets/cloud-setup?hl=ko#billing]을 참고하여 결제 계정으로 프로젝트를 설정하세요.
