공개 최종 클래스
Polyline
객체를 확장합니다.

다중선은 점의 목록으로, 연속된 점 사이에 선분이 그려집니다. 가
다중선에는 다음과 같은 속성이 있습니다.

포인트
선의 꼭짓점입니다. 선분은 연속되는 점 사이에 그려집니다. 다중선은
않습니다. 닫힌 폴리라인을 만들려면 시작점과 끝점이
사용할 수 있습니다.
너비
선 세그먼트 너비(화면 픽셀)입니다. 너비는 일정하며 카메라의 너비와 무관합니다.
확대/축소 수준입니다. 기본값은 10입니다.
색상
Color [https://developer.android.com/reference/android/graphics/Color.html?hl=ko]에서 사용하는 것과 동일한 ARGB 형식의 선분 색상입니다.
기본값은 검은색 (0xff000000)입니다.
를 통해 개인정보처리방침을 정의할 수 있습니다.시작/끝 모양
다중선의 시작이나 끝에 사용할 도형을 정의합니다. 지원되는 한도 유형: ButtCap [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/model/ButtCap?hl=ko], SquareCap [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/model/SquareCap?hl=ko], RoundCap [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/model/RoundCap?hl=ko] (실선 패턴에 적용 가능)
CustomCap [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/model/CustomCap?hl=ko] (모든 획 패턴에 적용 가능). 시작 및 끝 모두 기본값:
ButtCap [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/model/ButtCap?hl=ko]
연결부 유형
연결점 유형은 인접한 선분을 결합할 때 사용할 도형을 정의합니다.
시작 및 끝 꼭짓점을 제외한 폴리라인의 꼭짓점입니다. JointType [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/model/JointType?hl=ko]에서 다음을 확인하세요.
지원되는 연결 유형 기본값은 DEFAULT [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/model/JointType?hl=ko#DEFAULT]입니다.
획 패턴
솔리드 (기본값, null로 표시됨) 또는 일련의 PatternItem [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/model/PatternItem?hl=ko] 객체
선을 따라 반복될 것입니다. 사용 가능한 PatternItem 유형: Gap [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/model/Gap?hl=ko] (다음으로 정의됨)
간격 길이(픽셀), Dash [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/model/Dash?hl=ko](선 너비 및 대시 길이(픽셀)로 정의) 및
Dot [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/model/Dot?hl=ko](원형, 선 중앙, 선 너비(픽셀 단위로 정의되는 지름)).
Z-색인
다른 오버레이(
GroundOverlay [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/model/GroundOverlay?hl=ko], TileOverlay [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/model/TileOverlay?hl=ko], Circle [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/model/Circle?hl=ko], Polygon [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/model/Polygon?hl=ko](포함되지 않음)
Marker [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/model/Marker?hl=ko])을 입력합니다. Z-색인이 큰 오버레이는 작은 크기의 오버레이 위에 그려집니다.
z-색인입니다. 동일한 Z-인덱스를 가진 오버레이의 순서는 무작위입니다. 기본 zIndex:
0.
공개 상태
다중선의 표시 여부(예: 지도에 다중선을 그렸는지 여부)를 나타냅니다.
보이지 않는 다중선은 그려지지 않지만 다른 속성은 모두 유지합니다. 기본값은
true, 즉 표시.
최단 거리 상태
다중선의 선분을 다중선 선분이 아닌 최단 거리로 그려야 하는지 여부를 나타냅니다.
직선이 표시됩니다. 최단 거리는 두 지점 간의 최단 경로입니다.
점의 개수입니다. 최단 거리 곡선은 지구가
구
를 통해 개인정보처리방침을 정의할 수 있습니다.클릭 가능 여부
사용자가 다중선을 클릭할 때 발생하는 이벤트를 처리하려면 이 속성을 다음과 같이 설정합니다.
true입니다. 이 값은 언제든지 변경할 수 있습니다. 기본값은 false입니다. 만약
속성이 true로 설정되면 앱에서 setOnPolylineClickListener(GoogleMap.OnPolylineClickListener) [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/GoogleMap?hl=ko#setOnPolylineClickListener(com.google.android.libraries.maps.GoogleMap.OnPolylineClickListener)]를 통해 등록된 GoogleMap.OnPolylineClickListener [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/GoogleMap.OnPolylineClickListener?hl=ko]로 알림을 수신합니다.
태그
다중선과 연결된 Object입니다. 예를 들어 Object에는
표시할 수 있습니다. 이는 별도의 Map<Polyline, Object>를 저장하는 것보다 쉽습니다. 또 다른 예로 String ID를
데이터 세트의 ID에 해당합니다. Android용 Google Maps SDK는
이 속성을 씁니다.
스팬
이 다중선을 렌더링하는 데 사용되는 다양한 스타일로 스팬을 정의합니다. 이
속성을 사용하여 의 여러 세그먼트에 다른 StrokeStyle [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/model/StrokeStyle?hl=ko]이 있는 다중선을 만듭니다.
표시합니다.

이 클래스의 메서드는 Android UI 스레드에서 호출되어야 합니다. 그러지 않으면 런타임에 IllegalStateException [https://developer.android.com/reference/java/lang/IllegalStateException.html?hl=ko]이 발생합니다.
예
GoogleMap map;
Polyline line = map.addPolyline(new PolylineOptions()
.add(new LatLng(51.5, -0.1), new LatLng(40.7, -74.0))
.width(5)
.color(Color.RED));
