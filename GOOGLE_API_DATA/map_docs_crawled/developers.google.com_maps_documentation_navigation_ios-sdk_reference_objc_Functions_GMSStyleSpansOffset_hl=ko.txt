GMSStyleSpansOffset

extern NSArray<GMSStyleSpan [https://developers.google.com/maps/documentation/navigation/ios-sdk/reference/objc/Classes/GMSStyleSpan.html] *> *_Nonnull GMSStyleSpansOffset(
GMSPath [https://developers.google.com/maps/documentation/navigation/ios-sdk/reference/objc/Classes/GMSPath.html] *_Nonnull path, NSArray<GMSStrokeStyle [https://developers.google.com/maps/documentation/navigation/ios-sdk/reference/objc/Classes/GMSStrokeStyle.html] *> *_Nonnull styles,
NSArray<NSNumber *> *_Nonnull lengths, GMSLengthKind [https://developers.google.com/maps/documentation/navigation/ios-sdk/reference/objc/Enums/GMSLengthKind.html] lengthKind,
double lengthOffset)

Similar to GMSStyleSpans(path, styles, lengths, lengthKind) but additionally takes an initial
length offset that will be skipped over relative to the lengths array.

lengthOffset the length (e.g. in meters) that should be skipped initially from lengths.
