MapsPlatformDatasets

Maps Platform Datasets API의 서비스 정의입니다.

| --- |
| rpc CreateDataset(CreateDatasetRequest) returns (Dataset) 지정된 프로젝트의 새 데이터 세트를 만듭니다. 승인 범위 다음 OAuth 범위가 필요합니다. https://www.googleapis.com/auth/cloud-platform IAM 권한 parent 리소스에 대해서는 다음 IAM 권한이 필요합니다. mapsplatformdatasets.datasets.create 자세한 내용은 IAM 문서를 참조하세요. |

| --- |

| 필드 |
| --- |
| parent | string 필수 항목입니다. 데이터 세트를 소유할 상위 프로젝트입니다. 형식: projects/{project} |
| dataset | Dataset 필수 항목입니다. 만들 데이터 세트 버전입니다. |

데이터 세트

데이터 세트 리소스를 나타냅니다.

| 필드 |
| --- |
| name | string 리소스 이름입니다. 형식: projects/{project}/datasets/{dataset_id} |
| display_name | string 콘솔 UI에 표시되는 인간이 읽을 수 있는 이름입니다. 프로젝트 내에서 고유해야 합니다. |
| description | string 이 데이터 세트의 설명입니다. |
| version_id | string 데이터 세트의 버전 ID입니다. |
| usage[] | Usage 이 데이터 세트의 지정된 사용 사례입니다. |
| status | Status 출력 전용입니다. 이 데이터 세트 버전의 상태입니다. |
| create_time | Timestamp 출력 전용입니다. 데이터 세트가 처음 생성된 시간입니다. |
| update_time | Timestamp 출력 전용입니다. 데이터 세트 메타데이터가 마지막으로 업데이트된 시간입니다. |
| version_create_time | Timestamp 출력 전용입니다. 이 버전이 생성된 시간입니다. |
| version_description | string 출력 전용입니다. 이 버전의 데이터 세트에 대한 설명입니다. 데이터 세트로 데이터를 가져올 때 제공됩니다. |
| 통합 필드 data_source. 데이터 세트의 데이터 소스에 대한 세부정보입니다. data_source은 다음 중 하나여야 합니다. |
| local_file_source | LocalFileSource 단일 업로드를 위한 데이터 세트의 로컬 파일 소스입니다. |
| gcs_source | GcsSource 단일 업로드를 위한 데이터 세트의 Google Cloud Storage 파일 소스입니다. |

FetchDatasetErrorsRequest

데이터 세트에 속한 자세한 오류를 나열하는 요청입니다.

| 필드 |
| --- |
| dataset | string 필수 항목입니다. 모든 오류를 나열할 데이터 세트의 이름입니다. 형식: projects/{project}/datasets/{dataset_id} |
| page_size | int32 페이지당 반환할 최대 오류 수입니다. 최댓값은 500입니다. 500을 초과하는 값은 500으로 제한됩니다. 지정하지 않으면 최대 50개의 오류가 반환됩니다. |
| page_token | string 이전 ListDatasetErrors 호출에서 수신된 페이지 토큰입니다. 후속 페이지를 검색하려면 이를 입력합니다. |

| errors[] | Status 데이터 세트와 관련된 오류입니다. |

ListDatasetsRequest

프로젝트의 데이터 세트 나열 요청입니다.

| 필드 |
| --- |
| parent | string 필수 항목입니다. 모든 데이터 세트를 나열할 프로젝트의 이름입니다. 형식: projects/{project} |
| page_size | int32 페이지당 반환할 최대 데이터 세트 수입니다. 지정하지 않거나 0이면 모든 데이터 세트가 반환됩니다. |
| page_token | string 이전 ListDatasets 호출에서 수신된 페이지 토큰입니다. 후속 페이지를 검색하려면 이를 입력합니다. |
| tag | string 각 데이터 세트에 대해 원하는 버전을 지정하는 태그입니다. 페이지로 나누기도 지정되면 페이지로 나누기 이후에 일부 필터링이 발생할 수 있으며, 이로 인해 마지막 페이지가 아니더라도 응답에 페이지 크기보다 적은 데이터 세트가 포함될 수 있습니다. 'active' 태그: 응답의 각 데이터 세트에는 마지막으로 완료된 버전의 정보가 포함되며 데이터 세트가 없으면 데이터 세트를 건너뜁니다. |

주

데이터 세트의 상태 목록입니다.

| 열거형 |
| --- |
| STATE_UNSPECIFIED | 이 데이터 세트의 상태가 설정되지 않았습니다. |
| STATE_IMPORTING | 데이터 세트로 데이터를 가져오는 중입니다. |
| STATE_IMPORT_SUCCEEDED | 데이터 세트로 데이터를 가져왔습니다. |
| STATE_IMPORT_FAILED | 데이터 세트로 데이터를 가져올 수 없습니다. |
| STATE_DELETING | 데이터 세트를 삭제하는 중입니다. |
| STATE_DELETION_FAILED | 삭제 실패 상태입니다. 이 상태는 데이터 세트 삭제가 실패했음을 나타냅니다. 삭제를 다시 시도할 수 있습니다. |
| STATE_PROCESSING | 데이터를 처리하는 중입니다. |
| STATE_PROCESSING_FAILED | 처리 실패 상태입니다. 이 상태는 처리에 실패했으며 오류가 보고될 수 있음을 나타냅니다. |
| STATE_NEEDS_REVIEW | 이 상태는 현재 사용되지 않습니다. |
| STATE_PUBLISHING | 게시 상태입니다. 이 상태는 게시가 진행 중임을 나타냅니다. |
| STATE_PUBLISHING_FAILED | 게시 실패 상태입니다. 이 상태는 게시에 실패했음을 나타냅니다. 게시를 다시 시도할 수 있습니다. |
| STATE_COMPLETED | 완료된 상태입니다. 이 상태는 데이터 세트를 특정 용도로 사용할 수 있음을 나타냅니다. |

사용

용도는 데이터 처리 방법을 알리기 위해 데이터가 사용되는 위치를 지정합니다.
