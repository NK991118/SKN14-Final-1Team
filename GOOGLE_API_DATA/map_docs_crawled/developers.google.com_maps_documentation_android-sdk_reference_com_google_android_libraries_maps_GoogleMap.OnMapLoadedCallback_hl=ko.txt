공개 정적 인터페이스
GoogleMap.OnMapLoadedCallback

지도의 렌더링이 완료된 경우를 위한 콜백 인터페이스입니다. 이는 지도를 렌더링하는 데 필요한
모든 타일을 가져오고 모든 라벨 지정이 완료된 후에 발생합니다. 연결 문제로 인해 지도가 로드되지 않거나 사용자가 지도와 계속 상호작용하여 지도가 계속 변경되고 로드를 완료하지 않는 경우에는 이 이벤트가 실행되지 않습니다.

공개 메서드 요약
| 추상 void | onMapLoaded() 지도의 렌더링이 완료되면 호출됩니다. |
| --- | --- |

공개 메서드

공개 추상 void

onMapLoaded
()

지도의 렌더링이 완료되면 호출됩니다. 이 호출은 한 번만 호출됩니다. 알림을 다시 받으려면 콜백을 다시 요청해야 합니다.
이 메서드는 Android UI 스레드에서 호출됩니다.
