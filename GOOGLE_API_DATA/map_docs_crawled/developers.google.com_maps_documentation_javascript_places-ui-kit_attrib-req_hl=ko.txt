장소 UI 키트의 저작자 표시 요구사항

컬렉션을 사용해 정리하기

내 환경설정을 기준으로 콘텐츠를 저장하고 분류하세요.

이 주제에서는 장소 UI 키트에 대한 저작자 표시 요구사항을 안내합니다. Places UI Kit 약관에 관한 자세한 내용은 Google Maps Platform 서비스 약관 [https://cloud.google.com/maps-platform/terms/?hl=ko]을 참고하세요.

Google 지도 저작자 표시 포함

Google 지도 저작자 표시는 장소 UI 키트에 포함되어 있습니다.

저작자 표시가 표시되는 위치와 관계없이 포함된 저작자 표시는 삭제하지 마세요. 저작자 표시를 변경, 숨기거나 가리지 마세요. 저작자 표시는 배경에 쉽게 표시되어야 합니다.
테두리, 배경 색상, 음영 또는 충분한 여백과 같은 UI 신호를 사용하여 항상 Google Maps Platform 콘텐츠를 다른 콘텐츠와 시각적으로 구분합니다.
시각적 수정을 할 때는 Google 지도 저작자 표시 요구사항을 준수해야 합니다. 자세한 내용은 다음 섹션을 참고하세요.

장소 UI 키트 기여 분석 예시

장소 세부정보 좁게 표시 구성요소에 필수 저작자 표시가 적용되었습니다.
이 Google 이외의 지도에서는 Google 지도 저작자 표시가 명확하게 표시되고 Google Maps Platform 콘텐츠가 다른 콘텐츠와 시각적으로 구분됩니다.

Google 지도 저작자 표시 요구사항

저작자 표시는 가능하면 항상 Google 지도 로고 형식이어야 합니다. 공간이 제한된 경우에는 Google 지도 텍스트를 사용할 수 있습니다. 최종 사용자는 항상 Google 지도에서 제공하는 콘텐츠를 명확하게 파악할 수 있어야 합니다.

왼쪽: Google 지도 로고 저작자 표시, 오른쪽: Google 지도 텍스트 저작자 표시

로고 저작자 표시

앱 또는 웹사이트에서 Google 지도 로고를 사용하려면 다음 요구사항을 따르세요.

Google 지도 로고 저작자 표시에 허용되는 변형

Google 지도 로고 다운로드

공식 Google 지도 로고 파일을 사용하세요. 아래 로고를 다운로드하고 이 섹션의 가이드라인을 따르세요.

다운로드:
google_logo_ui_kit.zip [https://developers.google.com/static/maps/documentation/images/google_logo_ui_kit.zip?hl=ko]

Google 지도 로고를 사용할 때는 다음 가이드라인을 따르세요.

로고를 어떤 식으로든 수정하지 마세요.
로고의 가로세로 비율을 유지하여 왜곡을 방지합니다.
지도나 이미지와 같이 복잡한 배경에는 윤곽선이 있는 로고를 사용하세요.
단색 또는 은은한 그라데이션과 같은 일반 배경에는 윤곽선이 없는 로고를 사용하세요.

로고 크기 사양

Google 지도 로고의 크기 사양은 다음과 같습니다.

최소 로고 높이: 16dp
최대 로고 높이: 19dp
최소 로고 여백: 왼쪽, 오른쪽, 상단 10dp, 하단 5dp

dp에 관해 자세히 알아보려면 Material Design 웹사이트의 픽셀 밀도 [https://m2.material.io/design/layout/pixel-density.html#pixel-density]를 참고하세요.

최소 여백 및 허용되는 크기 범위를 보여주는 Google 지도 로고

로고 접근성

Google 지도 로고의 접근성 요구사항은 다음과 같습니다.

로고와 배경 사이에 접근 가능한 대비 [https://www.w3.org/WAI/WCAG21/Understanding/contrast-minimum.html]를 유지합니다.
Google 지도 텍스트와 함께 접근성 라벨을 포함합니다.

Google 지도 로고 저작자 표시에 허용되지 않는 변형 및 접근성 문제

텍스트 저작자 표시

인터페이스 크기가 Google 지도 로고 사용을 지원하지 않는 경우 텍스트로 Google 지도라고 표시할 수 있습니다. 다음 가이드라인을 준수하세요.

허용되는 Google 지도 텍스트 저작자 표시 변형

Google 지도 텍스트를 어떤 식으로든 수정하지 마세요.

Google 지도의 대소문자를 변경하지 않음
Google 지도를 여러 줄로 래핑하지 않음
Google 지도를 다른 언어로 현지화하지 마세요.
HTML 속성 translate="no"를 사용하여 브라우저가 Google 지도를 번역하지 못하도록 합니다.

허용되지 않는 Google 지도 텍스트 저작자 표시 변형

다음 표에 설명된 대로 Google 지도 텍스트의 스타일을 지정합니다.
| Google 지도 텍스트 스타일 지정 요구사항 |
| --- |
| 속성 | 스타일 |
| 글꼴 모음 | Roboto. 글꼴을 로드하는 것은 선택사항입니다. |
| 대체 글꼴 모음 | 제품에 이미 사용된 모든 sans-serif 본문 글꼴 또는 'Sans-Serif'(기본 시스템 글꼴 호출) |
| 글꼴 스타일 | 보통 |
| 글꼴 두께 | 400 |
| 글꼴 색상 | 흰색, 검은색 (#1F1F1F) 또는 회색 (#5E5E5E) 배경에 대해 액세스 가능한 대비(4.5:1)를 유지합니다. |
| 글꼴 크기 | 최소 글꼴 크기: 12sp 최대 글꼴 크기: 16sp sp에 관해 알아보려면 Material Design 웹사이트의 글꼴 크기 단위를 참고하세요. |
| 문자 간격 | 보통 |

예시 CSS

다음 CSS는 흰색 또는 밝은 색상의 배경에서 적절한 서체 스타일과 색상으로 Google 지도를 렌더링합니다.

@import url('https://fonts.googleapis.com/css2?family=Roboto&display=swap');

Google 지도 저작자 표시의 시각적 처리에 관한 다음 요구사항을 따르세요.

저작자 표시를 콘텐츠 상단 또는 하단 근처, 동일한 시각적 컨테이너 내에 배치합니다. 콘텐츠가 한 줄인 경우 저작자 표시는 오른쪽이나 왼쪽에 배치할 수 있습니다.
테두리, 배경 색상, 음영 또는 충분한 공백과 같은 UI 신호를 사용하여 Google Maps Platform 콘텐츠를 다른 콘텐츠와 시각적으로 구분합니다.
Google Maps Platform 이외의 콘텐츠를 사용하여 Google 지도를 허위로 표시해서는 안 됩니다.
저작자 표시는 항상 눈에 잘 띄고 읽을 수 있어야 합니다. 삭제, 숨기기, 가리기 또는 수정해서는 안 됩니다.

다음 그림은 이러한 시각적 요구사항의 예를 보여줍니다.

콘텐츠 상단, 하단, 측면에 배치된 Google 지도 저작자 표시의 예

Google 지도 콘텐츠 (장소 평점)를 다른 콘텐츠와 구분하는 세 가지 접근 방식의 예

Google 지도 저작자 표시를 가리거나 다른 출처의 콘텐츠와 혼합하지 않음
