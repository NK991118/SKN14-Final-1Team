GoogleMap.OnCameraMoveStartedListener

카메라 모션이 시작될 때의 콜백 인터페이스입니다.

| int | REASON_GESTURE | 지도에서 사용자 동작에 반응하여 시작된 카메라 움직임입니다. |

REASON_API_ANIMATION

사용자 작업에 응답하여 시작되는 비동작 애니메이션입니다. 예: 확대/축소 버튼, 내 위치 버튼, 마커 클릭

상수 값:

2

공개

REASON_DEVELOPER_ANIMATION

개발자가 시작한 애니메이션입니다.

상수 값:

3

공개
정적
최종
int

REASON_GESTURE

지도에서 사용자 동작에 반응하여 시작된 카메라 움직임입니다. 예를 들어 화면 이동, 기울이기, 손가락을 모으거나 펼쳐 확대/축소 또는 회전할 수 있습니다.

상수 값:

1

공개 메서드

공개 추상 void

onCameraMoveStarted
(int reason)

유휴 상태가 된 후 카메라가 움직이기 시작하거나 카메라 모션의 이유가 변경되었을 때 호출됩니다. 이 메서드 내에서 카메라를 업데이트하거나 애니메이션을 적용해서는 안 됩니다.
이 메서드는 Android UI 스레드에서 호출됩니다.

매개변수
| reason | 카메라를 변경한 이유입니다. 가능한 값은 다음과 같습니다. REASON_GESTURE: 지도 상의 사용자 동작입니다. REASON_API_ANIMATION: 사용자 상호작용에서 비롯된 기본 애니메이션입니다. REASON_DEVELOPER_ANIMATION: 개발자 애니메이션입니다. |
