공개 생성자

public

LatLng
(이중 위도, 이중 경도)

도 단위로 측정된 지정된 위도와 경도로 LatLng를 생성합니다.

매개변수
| latitude | 점의 위도입니다. -90도 이상 +90도 이하로 조정됩니다. |
| --- | --- |
| longitude | 점의 경도입니다. 이 값은 -180도 이하, +180도 이하로 정규화됩니다. |

공개 메서드

공개
부울

같음
(객체 o)

이 LatLng [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/model/LatLng?hl=ko]가 다른 것과 같은지 테스트합니다.
두 지점의 위도가 비트로 같고 경도가 비트가 동일한 경우에만 두 지점이 동일한 것으로 간주됩니다. 즉, 기하학적 거리 측면에서 매우 가까운 두 LatLng는 .equal()로 간주되지 않을 수 있습니다.
