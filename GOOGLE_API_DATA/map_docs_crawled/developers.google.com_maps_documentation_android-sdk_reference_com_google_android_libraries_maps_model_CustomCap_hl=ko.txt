공개 최종 클래스 CustomCap이 Cap [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/model/Cap?hl=ko]를 확장

Polyline [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/model/Polyline?hl=ko]의 시작 또는 끝 꼭짓점 중앙에 배치된 비트맵 오버레이로, 선의 첫 번째 또는 마지막 가장자리 방향에 따라 방향이 지정되고 선의 획 너비를 기준으로 크기가 조정됩니다. CustomCap은 획 패턴으로 Polyline에 적용할 수 있습니다.
