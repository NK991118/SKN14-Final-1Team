Overview
Pollen API를 사용하면 최대 15개의 식물 종, 꽃가루 유형 3개, 꽃가루 색인, 건강 추천 등 특정 위치의 꽃가루 데이터를 요청할 수 있습니다. 65개 이상의 국가에 대해 1km/0.6마일의 해상도를 포함합니다.
API는 다음을 쿼리할 수 있는 엔드포인트를 제공합니다.
예측
65개 이상의 국가에서 최대 1km의 화분 정보를 매일 5일간 제공합니다.
히트맵
꽃가루 유형 3가지의 색상으로 구분된 타일입니다.
