Title: 프리미엄 플랜 보고

프리미엄 플랜 보고

컬렉션을 사용해 정리하기

내 환경설정을 기준으로 콘텐츠를 저장하고 분류하세요.

중요: Google Maps Platform 프리미엄 플랜은 이제 가입이 불가능하며 신규 고객에게 제공되지 않습니다.

보고서 도구

이 페이지는 2016년 1월에 출시된 새로운 Google Maps Platform 프리미엄 플랜 버전을 보유한 고객만을 위한 페이지입니다.

Google Maps Platform 프리미엄 플랜을 사용하면 다양한 유형의 사용 데이터를 제공하는 여러 보고서 도구에 액세스할 수 있습니다.

Google Cloud 콘솔 사용량 보고서
주의: 2021년 3월에 지원이 중단된 Google Cloud 지원 포털 사용량 보고서는 더 이상 사용할 수 없습니다. 그러나 Google Cloud 프로젝트 사용량 측정항목은 Google Cloud 콘솔 [https://console.cloud.google.com/?hl=ko]에서 확인할 수 있습니다.

Cloud 콘솔의 Google Maps Platform 측정항목 [https://console.cloud.google.com/project/_/google/maps-apis/metrics?hl=ko] 페이지에서 최대 30일 분량의 사용량 보고서를 확인할 수 있습니다.

사용량은 사용자 인증 정보를 기준으로 세분화할 수 있습니다.

API 키의 경우 키 이름이 표시됩니다.
클라이언트 ID의 경우 다음 형식의 라벨이 사용됩니다. project_number:<YOUR_PROJECT_NUMBER>

이 API에 대한 30일 이상의 사용량 보고서가 필요한 경우, 프로젝트에서 결제를 사용 설정했거나 프리미엄 플랜에서 Maps Platform 라이선스로 전환했다면 Google Cloud 콘솔에서 결제 보고서 [https://developers.google.com/maps/billing-and-pricing/billing?hl=ko#view-your-billing-report]를 사용할 수 있습니다. 사용된 channel 매개변수는 이전 Google Cloud 지원 포털 사용 보고서에서와 같이 결제 보고서에 반영됩니다.

결제 보고서를 설정할 수 없는 경우 지원팀에 문의 [https://developers.google.com/maps/premium/support?hl=ko#contact-support]하여 해당 사용량 보고서를 요청하세요.
