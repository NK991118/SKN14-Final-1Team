GMSMarkerLayer

@interface GMSMarkerLayer : GMSOverlayLayer [https://developers.google.com/maps/documentation/navigation/ios-sdk/reference/objc/Classes/GMSOverlayLayer.html]

GMSMarkerLayer is a subclass of GMSOverlayLayer [https://developers.google.com/maps/documentation/navigation/ios-sdk/reference/objc/Classes/GMSOverlayLayer.html], available on a per-marker basis, that allows
animation of several properties of its associated GMSMarker [https://developers.google.com/maps/documentation/navigation/ios-sdk/reference/objc/Classes/GMSMarker.html].

Note that this CALayer is never actually rendered directly, as GMSMapView [https://developers.google.com/maps/documentation/navigation/ios-sdk/reference/objc/Classes/GMSMapView.html] is provided
entirely via an OpenGL layer. As such, adjustments or animations to ‘default’ properties of
CALayer will not have any effect.
