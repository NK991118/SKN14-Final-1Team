RxJava는 관측 가능한 시퀀스를 사용하여 비동기 프로그램 및 이벤트 기반 프로그램을 작성하기 위한 반응형 프로그래밍 라이브러리입니다.

Places Rx 라이브러리 [https://github.com/googlemaps/android-maps-rx]
Android용 Maps SDK에서 비동기 이벤트의 관측 가능한 시퀀스 수신
Android용 Places SDK를 지원하므로 다양한 RxJava 기능을 활용할 수 있습니다.
