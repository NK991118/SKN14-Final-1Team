@interface GMSMapViewOptions : NSObject

backgroundColor [https://developers.google.com/maps/documentation/navigation/ios-sdk/reference/objc/Classes/GMSMapViewOptions#/c:objc(cs)GMSMapViewOptions(py)backgroundColor]

Specifies the background color of the map view, which displays whenever the map tiles are not
fully loaded. This is required because GMSMapView [https://developers.google.com/maps/documentation/navigation/ios-sdk/reference/objc/Classes/GMSMapView.html] ignores the inherited mutable
backgroundColor. The color is displayed on the background of the map. Defaults to a light grey
color.

Declaration

Swift
Objective-C
@property (nonatomic, nullable) UIColor *backgroundColor;
