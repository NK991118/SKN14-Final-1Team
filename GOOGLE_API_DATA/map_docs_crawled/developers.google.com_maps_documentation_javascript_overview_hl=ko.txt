JavaScript [https://developers.google.com/maps/documentation/javascript/overview?hl=ko]

Maps JavaScript API는 전 세계 어디든 위치를 표시하고, 자체 맞춤 그래픽, 애니메이션, 대화형 맞춤 데이터 레이어를 추가하고, Maps JavaScript API의 Places API를 사용하여 위치 인식 기능을 만들 수 있는 클라이언트 측 웹 API입니다. Maps JavaScript API를 사용하면 다음 작업을 할 수 있습니다.

2D 및 3D 지도를 만들어 전 세계의 위치를 표시합니다.
맞춤설정 가능한 마커를 추가하여 지도에서 위치를 나타냅니다.
장소 라이브러리를 사용하여 2억 개 이상의 장소에 대한 위치 데이터를 가져옵니다.
지도의 스타일과 색상을 맞춤설정합니다.
자체 대화형 맞춤 데이터 레이어를 추가합니다.
행정 경계를 표시하고 대화형으로 만듭니다.
지도에 맞춤 2D 및 3D 그래픽과 애니메이션 콘텐츠를 추가할 수 있습니다.
