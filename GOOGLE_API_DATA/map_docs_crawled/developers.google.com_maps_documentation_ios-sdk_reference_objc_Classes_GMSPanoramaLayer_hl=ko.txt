GMSPanoramaLayer

@interface GMSPanoramaLayer : GMSCALayer [https://developers.google.com/maps/documentation/ios-sdk/reference/objc/Classes/GMSCALayer.html]

GMSPanoramaLayer is a custom subclass of CALayer, provided as the layer class on
GMSPanoramaView [https://developers.google.com/maps/documentation/ios-sdk/reference/objc/Classes/GMSPanoramaView.html]. This layer should not be instantiated directly.
