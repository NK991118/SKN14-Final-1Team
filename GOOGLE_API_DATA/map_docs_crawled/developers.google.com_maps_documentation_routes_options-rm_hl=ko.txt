사용 가능한 경로 매트릭스 옵션

컬렉션을 사용해 정리하기

내 환경설정을 기준으로 콘텐츠를 저장하고 분류하세요.

유럽 경제 지역 (EEA) 개발자

청구서 수신 주소가 유럽 경제 지역에 있는 경우 2025년 7월 8일부터 서비스 사용 시 Google Maps Platform EEA 서비스 약관 [https://cloud.google.com/terms/maps-platform/eea?hl=ko]이 적용됩니다. 기능은 리전에 따라 다릅니다.
자세히 알아보기 [https://developers.google.com/maps/comms/eea/faq?hl=ko]

출발지와 도착지를 지정하는 것 외에도 경로에 사용할 수 있는 여러 옵션을 사용하여 경로 매트릭스를 맞춤설정할 수 있습니다. 이러한 옵션을 설정하는 방법에 관한 자세한 내용은 표에 링크된 주제를 참고하세요.

| 옵션 | 설명 |
| --- | --- |
| 여행 모드 | 운전, 대중교통, 도보, 이륜차와 같은 이동 수단입니다. 참고: 도보, 자전거, 이륜차 경로는 베타 버전이며 명확한 인도, 보행자 전용 도로 또는 자전거 전용 도로가 누락되는 경우가 있습니다. 앱에 표시하는 모든 도보, 자전거, 이륜차 경로에 이 경고를 사용자에게 표시해야 합니다. |
| 트래픽 데이터를 포함할지 여부 및 방법 지정 | 더 정확한 결과를 얻으려면 트래픽 데이터를 포함하고, 더 빠른 응답을 얻으려면 트래픽 데이터를 제외하세요. |
| 경로 매트릭스의 통행료 계산 | 경로 매트릭스에 경로의 예상 통행료 요금을 포함합니다. |
| 현지화된 값 요청 | 필드 마스크를 사용하여 경로와 동일한 방식으로 경로 매트릭스 응답에 대해 현지화된 텍스트를 요청합니다. |
| 사용 가능한 경로 매트릭스 차량 유형 | 경로 매트릭스의 차량 유형을 선택하는 추가 옵션입니다. |
| 추가 옵션 | 출발 시간과 같은 추가 옵션을 지정하려면 요청 본문을 참고하세요. |
