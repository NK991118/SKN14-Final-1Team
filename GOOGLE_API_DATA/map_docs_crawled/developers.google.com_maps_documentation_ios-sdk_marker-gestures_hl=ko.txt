마커 이벤트 및 동작

컬렉션을 사용해 정리하기

내 환경설정을 기준으로 콘텐츠를 저장하고 분류하세요.

특정 고급 마커 속성이 설정되면 탭, 동작 [https://developers.google.com/maps/documentation/ios-sdk/controls?hl=ko#map_gestures]과 같은 마커 이벤트를 모니터링할 수 있습니다.
마커를 탭하면 마커 제목이나 스니펫과 같은 추가 정보를 볼 수 있습니다. 길게 누르기 동작을 사용하여 드래그 가능한 마커를 이동할 수도 있습니다.

마커 이벤트를 추적하려면 view에 GMSMapViewDelegate [https://developers.google.com/maps/documentation/ios-sdk/reference/objc/Protocols/GMSMapViewDelegate?hl=ko]를 추가하세요.
마커를 드래그 가능하도록 설정하려면 GMSMarker.draggable [https://developers.google.com/maps/documentation/ios-sdk/reference/objc/Classes/GMSMarker?hl=ko#draggable] 속성을 설정합니다.
마커의 설명 텍스트를 설정하려면 GMSMarker.title [https://developers.google.com/maps/documentation/ios-sdk/reference/objc/Classes/GMSOverlay?hl=ko#title] 속성을 사용합니다.

마커 이벤트에 응답

뷰에 GMSMapViewDelegate [https://developers.google.com/maps/documentation/ios-sdk/reference/objc/Protocols/GMSMapViewDelegate?hl=ko] 프로토콜을 추가하고 해당 콜백을 구현하여 마커 이벤트 [https://developers.google.com/maps/documentation/ios-sdk/events?hl=ko]에 응답할 수 있습니다. 이 예시에서는 선택한 마커의 title 및 snippet을 식별합니다.

Objective-C

marker.draggable = true

marker.draggable = YES;
