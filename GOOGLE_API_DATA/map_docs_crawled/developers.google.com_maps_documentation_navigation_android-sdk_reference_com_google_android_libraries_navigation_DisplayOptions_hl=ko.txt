showStopSigns
(boolean showStopSigns)

Configures whether to show stop signs along the route during navigation.
Changes to this setting take effect only in subsequent route requests.
Once the setting is applied, the user interface shows the first 3 upcoming traffic lights or
stop signs at any time during navigation, provided the data is available.

Parameters
| showStopSigns | If true, shows stop signs. |
