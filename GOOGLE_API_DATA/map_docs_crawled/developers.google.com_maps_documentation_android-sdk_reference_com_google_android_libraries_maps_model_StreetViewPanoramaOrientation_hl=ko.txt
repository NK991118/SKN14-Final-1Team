(부동 소수점 기울기, 플로팅 방위)

StreetViewPanoramaOrientation을 생성합니다.

매개변수
| 기울기 | 방향의 각도입니다(단위: 도). 제한사항에 대한 자세한 내용은 tilt를 참조하세요. |
| --- | --- |
| 방위 | 북쪽에서 시계 방향의 방향입니다(단위: 도). 이 값은 0도 이내에서 360도 미포함으로 정규화됩니다. |

생성 값
| IllegalArgumentException | tilt이 -90도 이상 90도 범위를 벗어나는 경우입니다. |
