텍스트 검색의 사용 및 결제 (신규)

컬렉션을 사용해 정리하기

내 환경설정을 기준으로 콘텐츠를 저장하고 분류하세요.

이 제품 또는 기능은 미리보기(GA 이전) 버전입니다.
GA 이전 제품과 기능은 지원이 제한적일 수 있으며, GA 이전 제품과 기능에 대한 변경사항은 다른 GA 이전 버전과 호환되지 않을 수 있습니다. GA 이전 버전 제품 및 서비스에는 Google Maps Platform 서비스별 약관 [https://cloud.google.com/maps-platform/terms/maps-service-terms?hl=ko]이 적용됩니다. 자세한 내용은 출시 단계 설명 [https://developers.google.com/maps/launch-stages?hl=ko]을 참고하세요.

Text Search (신규)는 사용한 만큼만 지불하는 가격 모델 [https://developers.google.com/maps/documentation/places/web-service/usage-and-billing?hl=ko#new-payg]을 사용합니다.
Text Search (신규) 요청은 여러 SKU에 대한 호출을 생성할 수 있습니다.

Text Search (신규) 요청에 FieldMask 헤더를 사용하여 응답에 반환할 필드 목록을 지정합니다. 그러면 요청에 적용할 수 있는 가장 높은 SKU로 요금이 청구됩니다. 즉, Text Search (Basic) SKU와 Text Search (Advanced) SKU에서 모두 필드를 선택하면 Text Search (Advanced) SKU를 기준으로 요금이 청구됩니다.

참고: 텍스트 검색 (신규)은 미리보기 버전입니다.
아래 가격 정보는 정보 제공 목적으로만 게시되었으며, 미리보기 기간 동안 사용량이 제한되며 $0로 할인됩니다.

SKU: 텍스트 검색 (ID만 해당)

참고: Text Search (Basic) SKU에서 필드를 선택하면 해당 요율로 요금이 청구됩니다. Text Search (고급) SKU에서 필드를 선택하면 해당 요율로 요금이 청구됩니다. Text Search (기본) SKU에서 필드를 선택하면 해당 요율로 요금이 청구됩니다.

Text Search (신규) 요청에 FieldMask 헤더를 사용하여 지정된 필드로만 응답을 제한할 수 있습니다. 텍스트 검색 (ID 전용) SKU는 마스크에서 장소 ID (places.id) 또는 리소스 이름 (places.name)만 요청하는 경우에만 트리거됩니다.

| 월간 사용량 범위 (호출당 가격) |
| --- |
| 0~100,000회 | 100,001~500,000회 | 500,000회 이상 |
| 미화 0.0달러/회 (미화 0.00달러/1,000회) | 미화 0.0달러/회 (미화 0.00달러/1,000회) | 미화 0.0달러/회 (미화 0.00달러/1,000회) |

SKU: 텍스트 검색 (기본)

참고: Text Search (고급) SKU에서 필드를 선택하면 해당 요율로 요금이 청구됩니다. Text Search (기본) SKU에서 필드를 선택하면 해당 요율로 요금이 청구됩니다.

Text Search (신규) 요청에 FieldMask 헤더를 사용하여 지정된 필드로만 응답을 제한할 수 있습니다. 다음 필드 중 하나가 요청되면 Text Search (Basic) SKU가 트리거됩니다.
