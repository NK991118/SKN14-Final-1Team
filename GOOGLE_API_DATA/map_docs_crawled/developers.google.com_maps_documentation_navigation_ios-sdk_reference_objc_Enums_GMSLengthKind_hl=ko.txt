GMSLengthKind

enum GMSLengthKind : NSUInteger {}

GMSLengthKind indicates the type of a length value, which can be geodesic (in meters), rhumb
length (in meters) and projected length (in GMSMapPoint [https://developers.google.com/maps/documentation/navigation/ios-sdk/reference/objc/Structs/GMSMapPoint.html] units).
