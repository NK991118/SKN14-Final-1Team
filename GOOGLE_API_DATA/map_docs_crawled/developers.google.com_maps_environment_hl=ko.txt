환경 API

컬렉션을 사용해 정리하기

내 환경설정을 기준으로 콘텐츠를 저장하고 분류하세요.

Google의 Environment API 모음에는 다음이 포함됩니다.

Solar API [https://developers.google.com/maps/documentation/solar?hl=ko]: 전 세계 수억 개의 건물에 대한 태양광 잠재력 정보를 요청합니다.
Air Quality API [https://developers.google.com/maps/documentation/air-quality?hl=ko]: 특정 위치의 70개가 넘는 AQ 지수, 오염 물질, 건강 권장사항을 비롯한 공기질 데이터를 요청합니다.
Pollen API [https://developers.google.com/maps/documentation/pollen?hl=ko]: 특정 위치의 유형, 식물, 색인 등 꽃가루 정보를 요청합니다.
Weather API [https://developers.google.com/maps/documentation/weather?hl=ko]: 특정 위치의 현재 날씨, 시간별 및 일일 일기예보, 시간별 기록을 요청합니다.
