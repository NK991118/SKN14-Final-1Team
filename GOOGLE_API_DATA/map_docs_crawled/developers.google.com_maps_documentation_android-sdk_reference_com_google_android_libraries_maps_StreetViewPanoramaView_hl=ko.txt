Title: StreetViewPanoramaView

의견 보내기

StreetViewPanoramaView

컬렉션을 사용해 정리하기

내 환경설정을 기준으로 콘텐츠를 저장하고 분류하세요.

이 페이지의 내용상속받은 상수 요약 [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/StreetViewPanoramaView?hl=ko#inherited-constant-summary]상속받은 필드 요약 [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/StreetViewPanoramaView?hl=ko#inherited-field-summary]공개 생성자 요약 [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/StreetViewPanoramaView?hl=ko#public-constructor-summary]공개 메서드 요약 [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/StreetViewPanoramaView?hl=ko#public-method-summary]상속받은 메서드 요약 [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/StreetViewPanoramaView?hl=ko#inherited-method-summary]공개 생성자 [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/StreetViewPanoramaView?hl=ko#public-constructors]공개 메서드 [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/StreetViewPanoramaView?hl=ko#public-methods]

공개 클래스 StreetViewPanoramaView가 FrameLayout을 확장함

스트리트 뷰 파노라마를 표시하는 뷰 (Google 지도 서비스에서 가져온 데이터 포함).
초점을 맞추면 키 누름 및 터치 동작을 캡처하여 파노라마를 이동합니다.
이 클래스의 사용자는 이 뷰가 포함된 Activity [https://developer.android.com/reference/android/app/Activity.html?hl=ko] 또는 Fragment [https://developer.android.com/reference/android/app/Fragment.html?hl=ko]의 모든 수명 주기 메서드를 이 클래스의 상응하는 메서드에 전달해야 합니다. 특히 다음 메서드를 전달해야 합니다.

onCreate(Bundle) [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/StreetViewPanoramaView?hl=ko#onCreate(android.os.Bundle)]
onStart() [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/StreetViewPanoramaView?hl=ko#onStart()]
onResume() [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/StreetViewPanoramaView?hl=ko#onResume()]
onPause() [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/StreetViewPanoramaView?hl=ko#onPause()]
onStop() [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/StreetViewPanoramaView?hl=ko#onStop()]
onDestroy() [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/StreetViewPanoramaView?hl=ko#onDestroy()]
onSaveInstanceState(Bundle) [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/StreetViewPanoramaView?hl=ko#onSaveInstanceState(android.os.Bundle)]
onLowMemory() [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/StreetViewPanoramaView?hl=ko#onLowMemory()]

StreetViewPanorama [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/StreetViewPanorama?hl=ko]는 getStreetViewPanoramaAsync(OnStreetViewPanoramaReadyCallback) [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/StreetViewPanoramaView?hl=ko#getStreetViewPanoramaAsync(com.google.android.libraries.maps.OnStreetViewPanoramaReadyCallback)]를 사용하여 획득해야 합니다. StreetViewPanoramaView [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/StreetViewPanoramaView?hl=ko]는 스트리트 뷰 시스템과 뷰를 자동으로 초기화합니다.
이전 플랫폼을 타겟팅하려는 경우 StreetViewPanorama를 표시하는 더 간단한 방법으로 StreetViewPanoramaFragment [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/StreetViewPanoramaFragment?hl=ko](또는 SupportStreetViewPanoramaFragment [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/SupportStreetViewPanoramaFragment?hl=ko])를 사용합니다.
참고: 이 보기에는 어린이를 추가하지 않는 것이 좋습니다.
참고: Android용 Google Maps SDK는 하나의 활동에서 여러 StreetViewPanoramaView
객체를 지원하지 않습니다.
