public  final   class
ButtCap
extends Cap [https://developers.google.com/maps/documentation/navigation/android-sdk/reference/com/google/android/gms/maps/model/Cap]

Cap that is squared off exactly at the start or end vertex of a Polyline [https://developers.google.com/maps/documentation/navigation/android-sdk/reference/com/google/android/gms/maps/model/Polyline] with solid
stroke pattern, equivalent to having no additional cap beyond the start or end vertex. This is
the default cap type at start and end vertices of Polyline [https://developers.google.com/maps/documentation/navigation/android-sdk/reference/com/google/android/gms/maps/model/Polyline]s with solid stroke pattern.
