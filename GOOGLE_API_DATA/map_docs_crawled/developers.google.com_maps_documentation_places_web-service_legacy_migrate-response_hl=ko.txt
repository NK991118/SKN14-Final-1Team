유럽 경제 지역 (EEA) 개발자

청구서 수신 주소가 유럽 경제 지역에 있는 경우 2025년 7월 8일부터
Google Maps Platform EEA 서비스 약관 [https://cloud.google.com/terms/maps-platform/eea?hl=ko]
이 서비스 이용에 적용됩니다. 기능은 리전에 따라 다릅니다.
자세히 알아보기 [https://developers.google.com/maps/comms/eea/faq?hl=ko]

소개

장소 세부정보 (신규) [https://developers.google.com/maps/documentation/places/web-service/place-details?hl=ko], 주변 지역 검색 (신규) [https://developers.google.com/maps/documentation/places/web-service/nearby-search?hl=ko], 텍스트 검색 (신규) [https://developers.google.com/maps/documentation/places/web-service/text-search?hl=ko] API의 응답 형식이 기존 API의 형식에서 변경되었습니다. 주요 차이점은 응답 장소 객체 변경사항 [https://developers.google.com/maps/documentation/places/web-service/legacy/migrate-response?hl=ko#place-object]에 설명된 대로 각 메서드에서 반환되는 Place 객체의 형식에 있습니다.

또한 각 API의 응답 객체의 최상위 필드도 변경되었습니다. 다음을 참조하세요.

텍스트 검색 응답 변경사항 [https://developers.google.com/maps/documentation/places/web-service/legacy/migrate-response?hl=ko#text-response]
Nearby Search 응답 변경사항 [https://developers.google.com/maps/documentation/places/web-service/legacy/migrate-response?hl=ko#nearby-response]
장소 세부정보 응답 변경사항 [https://developers.google.com/maps/documentation/places/web-service/legacy/migrate-response?hl=ko#details-response]

응답 장소 객체 변경사항

Place 객체 [https://developers.google.com/maps/documentation/places/web-service/reference/rest/v1/places?hl=ko]와 기존
Place 객체 [https://developers.google.com/maps/documentation/places/web-service/details?hl=ko#Place]를 비교하여 변경사항을 보여줍니다.
