SearchMode

컬렉션을 사용해 정리하기

내 환경설정을 기준으로 콘텐츠를 저장하고 분류하세요.

검색 동작을 정의하고 지연 시간과 솔루션 품질 사이에서 절충하는 모드입니다. 모든 모드에서 전역 요청 기한이 적용됩니다.

| 열거형 |
| --- |
| SEARCH_MODE_UNSPECIFIED | 지정되지 않은 검색 모드입니다. RETURN_FAST와 같습니다. |
| RETURN_FAST | 첫 번째 좋은 해결 방법을 찾은 후에는 검색을 중지합니다. |
| CONSUME_ALL_AVAILABLE_TIME | 더 나은 솔루션을 찾는 데 가능한 시간을 모두 할애하세요. |
