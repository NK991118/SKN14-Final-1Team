RxJava는 관측 가능한 시퀀스를 사용하여 비동기 프로그램 및 이벤트 기반 프로그램을 작성하기 위한 반응형 프로그래밍 라이브러리입니다.

지도 Rx 라이브러리 [https://github.com/googlemaps/android-maps-rx]를 사용하면 Android용 Maps SDK 및 Android용 Places SDK에서 비동기 이벤트에 대한 관측 가능한 시퀀스를 수신하여 다양한 RxJava 기능을 활용할 수 있습니다.

설치

참고: 최신 시스템 요구사항 및 설치 안내는 GitHub의 지도 Rx 라이브러리 [https://github.com/googlemaps/android-maps-rx] 문서를 참고하세요.
