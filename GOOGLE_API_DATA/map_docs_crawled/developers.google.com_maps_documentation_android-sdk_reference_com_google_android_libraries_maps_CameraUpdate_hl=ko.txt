CameraUpdate

컬렉션을 사용해 정리하기

내 환경설정을 기준으로 콘텐츠를 저장하고 분류하세요.

공개 최종 클래스
CameraUpdate
객체를 확장합니다.

카메라 이동을 정의합니다. 이 유형의 객체는
animateCamera(CameraUpdate) [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/GoogleMap?hl=ko#animateCamera(com.google.android.libraries.maps.CameraUpdate)], animateCamera(CameraUpdate, GoogleMap.CancelableCallback) [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/GoogleMap?hl=ko#animateCamera(com.google.android.libraries.maps.CameraUpdate,%20com.google.android.libraries.maps.GoogleMap.CancelableCallback)] 또는 moveCamera(CameraUpdate) [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/GoogleMap?hl=ko#moveCamera(com.google.android.libraries.maps.CameraUpdate)]입니다.
CameraUpdate [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/CameraUpdate?hl=ko]를 가져오려면 팩토리 클래스 CameraUpdateFactory [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/CameraUpdateFactory?hl=ko]를 사용하세요.
예를 들어 지도를 확대하려면 다음 코드를 사용할 수 있습니다.
GoogleMap map = ...;
map.animateCamera(CameraUpdateFactory.zoomIn());
