Data-driven styling features are not available to the Navigation SDK. They exists for compile
time compatibility with com.google.android.gms:play-services-maps only.
The feature style to specify how the style of a Feature [https://developers.google.com/maps/documentation/navigation/android-sdk/reference/com/google/android/gms/maps/model/Feature] should be modified on a map.
