공개 클래스
SupportMapFragment
프래그먼트를 확장합니다.

앱의 지도 구성요소 이 프래그먼트는 애플리케이션에 지도를 배치하는 가장 간단한 방법입니다.
지도뷰를 둘러싸는 래퍼로, 필요한 수명 주기 요구사항을 자동으로 처리합니다.
이 구성 요소는 프래그먼트이므로 XML을 사용하여 간단히 활동의 레이아웃 파일에 추가할 수 있습니다.
참조하세요.
<fragment
class="com.google.android.libraries.maps.SupportMapFragment"
android:layout_width="match_parent"
android:layout_height="match_parent"/> 드림
GoogleMap [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/GoogleMap?hl=ko]는 getMapAsync(OnMapReadyCallback) [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/SupportMapFragment?hl=ko#getMapAsync(com.google.android.libraries.maps.OnMapReadyCallback)]를 사용하여 획득해야 합니다. 이 수업
는 지도 시스템과 뷰를 자동으로 초기화합니다.
SupportMapFragment의 onDestroyView() [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/SupportMapFragment?hl=ko#onDestroyView()] 메서드가 호출되고
useViewLifecycleInFragment(boolean) [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/GoogleMapOptions?hl=ko#useViewLifecycleInFragment(boolean)] 옵션이 설정됩니다. 이 경우
SupportMapFragment의
onCreateView(LayoutInflater, ViewGroup, Bundle) [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/SupportMapFragment?hl=ko#onCreateView(android.view.LayoutInflater,%20android.view.ViewGroup,%20android.os.Bundle)] 메서드가 호출됩니다.
GoogleMap [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/GoogleMap?hl=ko]에서 가져온 모든 객체는 뷰와 연결됩니다. 중요한 것은
다음의 객체 (예: Marker [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/model/Marker?hl=ko])를 유지하지 않습니다.
확인할 수 있습니다. 그러지 않으면 뷰를 해제할 수 없으므로 메모리 누수가 발생합니다.
이 클래스를 사용하려면 빌드 경로에 Android 지원 라이브러리를 포함해야 합니다.

콜백이 트리거되지 않습니다.
콜백에서 제공하는 GoogleMap [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/GoogleMap?hl=ko] 객체는 null이 아닙니다.
