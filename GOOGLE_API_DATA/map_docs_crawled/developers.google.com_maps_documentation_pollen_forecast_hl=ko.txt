예측 요청 정보

예보 엔드포인트를 사용하여 일일 꽃가루 정보를 요청하려면 다음 주소로 HTTP GET 요청을 전송하세요.

https://pollen.googleapis.com/v1/forecast:lookup?key=YOUR_API_KEY

URL에 매개변수 [https://developers.google.com/maps/documentation/pollen/reference/rest/v1/forecast/lookup?hl=ko#query-parameters]를 포함합니다. 요청에는 꽃가루 예보를 가져오려는 위치, 예보 일수, 응답에 포함할 꽃가루 정보가 포함됩니다.

예측 응답에 대한 정보

응답 본문 [https://developers.google.com/maps/documentation/pollen/reference/rest/v1/forecast/lookup?hl=ko#response-body]은 요청된 좌표의 지역 코드와 요청된 각 날짜의 일일 예보 정보를 제공합니다. 2일 이상의 정보를 요청하는 경우 정보가 요청된 페이지 크기에 맞지 않으면 응답 본문에 다음 페이지를 가져오는 토큰도 포함됩니다.

응답 본문에는 다음 세 가지 객체도 포함됩니다.

DayInfo [https://developers.google.com/maps/documentation/pollen/reference/rest/v1/forecast/lookup?hl=ko#dayinfo]는 꽃가루 유형 정보 및 식물 정보와 함께 표시된 예측의 날짜를 제공합니다.
PollenTypeInfo [https://developers.google.com/maps/documentation/pollen/reference/rest/v1/forecast/lookup?hl=ko#pollentypeinfo]는 사용 가능한 꽃가루 유형 (GRASS, TREE, WEED), 꽃가루 지수, 요청된 좌표의 관련 건강 권장사항의 계절성을 제공합니다.
PlantInfo [https://developers.google.com/maps/documentation/pollen/reference/rest/v1/forecast/lookup?hl=ko#plantinfo]는 요청된 좌표에서 계절성, 외관, 교차 반응을 비롯한 식물 이름과 설명을 제공합니다.

참고: 꽃가루 유형 또는 식물이 제철이 아니고 꽃가루 수가 적은 경우 응답 본문의 indexInfo 객체가 생략됩니다.

요청 예시

다음 코드는 forecast:lookup [https://developers.google.com/maps/documentation/pollen/reference/rest/v1/forecast/lookup?hl=ko] 요청의 기본 요청을 구성하는 방법을 보여줍니다. 이 예에서는 꽃가루 정보의 위치와 일수를 설정합니다.

curl -X GET "https://pollen.googleapis.com/v1/forecast:lookup?key=YOUR_API_KEY&location.longitude=35.32&location.latitude=32.32&days=1"
