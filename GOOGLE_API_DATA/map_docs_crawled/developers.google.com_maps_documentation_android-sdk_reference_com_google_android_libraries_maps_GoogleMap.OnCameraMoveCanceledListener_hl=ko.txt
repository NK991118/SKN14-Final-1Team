공개 정적 인터페이스
GoogleMap.OnCameraMoveCanceledListener

카메라의 움직임이 중지되었거나 새로운 이유로 카메라가 움직이기 시작하는 경우에 대한 콜백 인터페이스입니다.

공개 메서드 요약
| 추상 void | onCameraMoveCanceled()를 참조하세요. 개발자가 stopAnimation() 메서드를 명시적으로 호출하거나 onCameraIdle이 이전 애니메이션 후에 실행되도록 카메라 모션의 이유가 변경된 경우 호출됩니다. |
| --- | --- |

공개 메서드

공개 추상 void

onCameraMoveCanceled
()

개발자가 stopAnimation() [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/GoogleMap?hl=ko#stopAnimation()] 메서드를 명시적으로 호출하거나 onCameraIdle이 이전 애니메이션 후에 실행되도록 카메라 모션의 이유가 변경된 경우 호출됩니다. 이 메서드 내에서 카메라를 업데이트하거나 애니메이션을 적용해서는 안 됩니다.
이 메서드는 Android UI 스레드에서 호출됩니다.
