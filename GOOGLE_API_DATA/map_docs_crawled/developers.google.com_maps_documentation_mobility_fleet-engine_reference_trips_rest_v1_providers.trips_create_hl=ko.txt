Fleet Engine에서 이동을 만들고 새 이동을 반환합니다.

HTTP 요청
POST https://fleetengine.googleapis.com/v1/{parent=providers/*}/tripsURL은 gRPC 트랜스코딩 [https://google.aip.dev/127] 문법을 사용합니다.

경로 매개변수
| 매개변수 |
| --- |
| parent | string 필수 항목입니다. providers/{provider} 형식이어야 합니다. 제공업체는 이 호출을 수행하는 서비스 계정이 구성원인 Google Cloud 프로젝트의 프로젝트 ID (예: sample-cloud-project)여야 합니다. |

쿼리 매개변수
| 매개변수 |
| --- |
| header | object (RequestHeader) 표준 Fleet Engine 요청 헤더입니다. |
| tripId | string 필수 항목입니다. 고유한 이동 ID입니다. 다음과 같은 제한사항이 적용됩니다. 유효한 유니코드 문자열이어야 합니다. 최대 길이는 64자(영문 기준)로 제한됩니다. 유니코드 정규화 양식 C에 따라 정규화됩니다. '/', ':', '?', ',', '#' ASCII 문자는 포함할 수 없습니다. |

요청 본문
요청 본문에 Trip [https://developers.google.com/maps/documentation/mobility/fleet-engine/reference/trips/rest/v1/providers.trips?hl=ko#Trip]의 인스턴스가 포함됩니다.

응답 본문
성공한 경우 응답 본문에 새로 생성된 Trip [https://developers.google.com/maps/documentation/mobility/fleet-engine/reference/trips/rest/v1/providers.trips?hl=ko#Trip]의 인스턴스가 포함됩니다.
