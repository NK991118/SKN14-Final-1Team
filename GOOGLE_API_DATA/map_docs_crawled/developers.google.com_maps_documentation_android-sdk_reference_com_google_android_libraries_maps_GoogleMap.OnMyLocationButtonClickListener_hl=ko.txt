공개 정적 인터페이스
GoogleMap.OnMyLocationButtonClickListener

My Location 버튼이 클릭될 때의 콜백 인터페이스입니다.

공개 메서드 요약
| 추상 불리언 | onMyLocationButtonClick() 내 위치 버튼을 클릭하면 호출됩니다. |
| --- | --- |

공개 메서드

공개
추상
불리언

onMyLocationButtonClick
()

내 위치 버튼을 클릭하면 호출됩니다.
이 메서드는 Android UI 스레드에서 호출됩니다.

사용자의 현재 위치를 가져와야 한다면 com.google.android.gms.location.FusedLocationProviderApi를 사용하세요.

반환 값
리스너가 이벤트를 사용했다면 true이고 (즉 기본 동작이 발생해서는 안 됨) 그 외의 경우에는 false입니다 (즉, 기본 동작이 발생해야 함). 기본 동작은 카메라가 사용자 위치의 중앙에 오도록 이동하는 것입니다.
