주소 또는 videoId가 지정된 동영상을 가져옵니다. 응답은 ACTIVE 동영상에 대한 재생 URI 집합이 있는 동영상, 보류 중인 동영상에 대한 PROCESSING 상태 또는 동영상이 없는 경우 404 오류입니다. 동영상 수신은 청구 가능한 이벤트이므로 이 메서드의 호출자는 요청 시 반환된 URI를 사용할 준비가 되어 있어야 합니다.

HTTP 요청
GET https://aerialview.googleapis.com/v1/videos:lookupVideoURL은 gRPC 트랜스코딩 [https://google.aip.dev/127] 구문을 사용합니다.

쿼리 매개변수
| 매개변수 |
| --- |
| 통합 매개변수 key입니다. 필수 항목입니다. 동영상을 검색하는 데 사용되는 키입니다. key은 다음 중 하나여야 합니다. |
| videoId | string videos.renderVideo에서 반환된 ID입니다. |
| address | string 미국 우편 주소입니다. |

요청 본문
요청 본문은 비어 있어야 합니다.

응답 본문

동영상에 대한 모든 데이터를 캡슐화하는 객체입니다.
성공할 경우 응답 본문에 다음 구조의 데이터가 포함됩니다.

| JSON 표현 |
| --- |
| {   "uris": {     string: {       object (Uris)     },     ...   },   "state": enum (State),   "metadata": {     object (VideoMetadata)   } } |

| 필드 |
| --- |
| uris | map (key: string, value: object (Uris)) URI에 대한 미디어 유형 매핑 이 필드는 ACTIVE개 동영상에만 포함됩니다. 키는 MediaFormat의 열거형 값입니다. "key": value 쌍 목록을 포함하는 객체입니다. 예: { "name": "wrench", "mass": "1.3kg", "count": "3" } |
| state | enum (State) 렌더링 요청의 현재 상태입니다. |
| metadata | object (VideoMetadata) 동영상의 메타데이터를 포함합니다. 상태가 ACTIVE인 경우에만 설정됩니다. |

URI

지정된 동영상 형식의 모든 URI가 포함됩니다.

| JSON 표현 |
| --- |
| {   "landscapeUri": string,   "portraitUri": string } |

| 필드 |
| --- |
| landscapeUri | string 가로 방향 미디어의 서명된 단기 URI입니다. |
| portraitUri | string 세로 방향 미디어의 서명된 단기 URI입니다. |
