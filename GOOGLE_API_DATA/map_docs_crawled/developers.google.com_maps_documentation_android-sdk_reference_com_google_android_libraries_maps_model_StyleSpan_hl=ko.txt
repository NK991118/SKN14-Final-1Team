|  | StyleSpan(StrokeStyle 스타일, 이중 세그먼트) 지정된 style 및 세그먼트 길이로 스팬을 만듭니다. |
| --- | --- |
|  | StyleSpan(StrokeStyle 스타일) 길이가 한 세그먼트인 지정된 style로 스팬을 만듭니다. |
|  | StyleSpan(int color) 한 세그먼트 길이의 단색 범위를 만듭니다. |
|  | StyleSpan(int color, double 세그먼트) 지정된 수의 세그먼트로 단색 범위를 만듭니다. |
