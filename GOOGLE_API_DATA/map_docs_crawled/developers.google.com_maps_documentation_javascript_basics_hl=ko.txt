이 페이지의 내용지도 유형 [https://developers.google.com/maps/documentation/javascript/basics?hl=ko#map_types]지도 및 타일 좌표 [https://developers.google.com/maps/documentation/javascript/basics?hl=ko#map_and_tile_coordinates]지도 현지화 [https://developers.google.com/maps/documentation/javascript/basics?hl=ko#localizing_the_map]버전 관리 [https://developers.google.com/maps/documentation/javascript/basics?hl=ko#versioning]URL 매개변수 [https://developers.google.com/maps/documentation/javascript/basics?hl=ko#url_parameters]권장사항 [https://developers.google.com/maps/documentation/javascript/basics?hl=ko#best_practices]TypeScript 사용 [https://developers.google.com/maps/documentation/javascript/basics?hl=ko#using_typescript]프로미스 [https://developers.google.com/maps/documentation/javascript/basics?hl=ko#promises]

이 섹션에서는 Maps JavaScript API를 사용하는 데 필수적인 다양한 개념을 다룹니다.

지도 유형 [https://developers.google.com/maps/documentation/javascript/maptypes?hl=ko]

이 페이지에서는 Maps JavaScript API를 사용하여 표시할 수 있는 지도 유형을 설명합니다.

지도 및 타일 좌표 [https://developers.google.com/maps/documentation/javascript/coordinates?hl=ko]

이 페이지에서는 Maps JavaScript API에서 사용하는 좌표계를 설명합니다.

지도 현지화 [https://developers.google.com/maps/documentation/javascript/localization?hl=ko]

기본 언어 설정을 변경하고 지역 코드를 지정함으로써 Maps JavaScript API 애플리케이션을 현지화하여 지정된 국가 또는 지역을 기반으로 지도의 동작을 변경할 수 있습니다.

버전 관리 [https://developers.google.com/maps/documentation/javascript/versions?hl=ko]

Maps JavaScript API팀은 새로운 기능, 버그 수정, 성능 개선 사항을 반영하여 API를 정기적으로 업데이트합니다. Maps JavaScript API 부트스트랩 요청의 v 매개변수에 버전을 지정하여 애플리케이션 내에서 로드할 API 버전을 나타낼 수 있습니다.

URL 매개변수 [https://developers.google.com/maps/documentation/javascript/url-params?hl=ko]

이 페이지에서는 스크립트 로드 URL의 쿼리 문자열을 통해 Maps JavaScript API를 로드할 때 사용할 수 있는 옵션을 설명합니다.

권장사항 [https://developers.google.com/maps/documentation/javascript/best-practices?hl=ko]

이 페이지에서는 Maps JavaScript API를 사용할 때 CSS 및 JavaScript와 관련해 고려해야 할 권장사항을 설명합니다.

TypeScript 사용 [https://developers.google.com/maps/documentation/javascript/using-typescript?hl=ko]

이 페이지에서는 Google 지도에서 TypeScript를 사용하는 방법을 설명합니다.

프로미스 [https://developers.google.com/maps/documentation/javascript/promises?hl=ko]

이 페이지에서는 Maps JavaScript API와 함께 비동기 메서드를 사용하는 방법을 설명합니다.
