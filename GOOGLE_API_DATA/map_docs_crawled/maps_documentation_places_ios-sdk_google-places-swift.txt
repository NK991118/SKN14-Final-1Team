iOS용 Places Swift SDK
유럽 경제 지역 (EEA) 개발자
iOS용 Places Swift SDK는 iOS용 Places SDK의 Swift 우선 버전을 제공합니다. iOS용 Places Swift SDK는 사용자가 iOS용 Places SDK에서 익숙한 모든 기능을 제공하며, Swift로 직접 구현되어 많은 이점을 추가로 제공합니다. 이러한 이점은 Swift의 고유한 특성을 활용하므로 더 자연스러운 Swift 코드를 작성하도록 지원합니다.
Places Swift SDK의 주요 기능은 다음과 같습니다.
값 유형 [https://www.swift.org/documentation/articles/value-and-reference-types.html], 기본 매개변수 값 [https://docs.swift.org/swift-book/documentation/the-swift-programming-language/functions/#Default-Parameter-Values], 선택사항 [https://developer.apple.com/documentation/swift/optional]을 광범위하게 사용
Swift 동시 실행 지원
타입 안전성 향상
iOS용 Places Swift SDK는 다음 제품으로 구성된 iOS용 Places SDK (신규) API에서만 사용할 수 있습니다.
Place Autocomplete (신규) [https://developers.google.com/maps/documentation/places/ios-sdk/place-autocomplete?hl=ko]
Place Details (신규) [https://developers.google.com/maps/documentation/places/ios-sdk/details-place?hl=ko]
장소 UI 키트 [https://developers.google.com/maps/documentation/places/ios-sdk/places-ui-kit-overview?hl=ko]
Place Photos (신규) [https://developers.google.com/maps/documentation/places/ios-sdk/place-photos?hl=ko]
텍스트 검색 (신규) [https://developers.google.com/maps/documentation/places/ios-sdk/text-search?hl=ko]
주변 지역 검색 (신규) [https://developers.google.com/maps/documentation/places/ios-sdk/nearby-search?hl=ko]
iOS용 Places SDK에서 iOS용 Places Swift SDK로 이전
iOS용 Places Swift SDK 코드는 이전 구현과 함께 작동하므로 점진적으로 이전할 수 있습니다.
자세한 내용을 확인하고 새 SDK로 이전하는 방법을 알아보려면 이전 가이드를 검토 [https://developers.google.com/maps/documentation/places/ios-sdk/migrate-places-sdk?hl=ko]하세요.
참고: iOS용 Places Swift SDK에 액세스하는 URL이 변경되었습니다. 이제 https://github.com/googlemaps/ios-places-sdk [https://github.com/googlemaps/ios-places-sdk]입니다. 이전 URL(https://github.com/googlemaps/ios-places-swift-sdk)을 사용해 온 경우 Xcode 패키지 종속 항목에서 해당 GitHub URL을 삭제하고 업데이트된 URL을 입력합니다.
지원 중단된 저장소에서 GooglePlacesSwift를 추가하려고 하면 Failed to resolve dependencies Dependencies could not be resolved because no versions of 'ios-places-swift-sdk' match the requirement 0.2.0 and root depends on 'ios-places-swift-sdk' 0.2.0. 오류 메시지가 표시됩니다.
자세한 내용은 설치 페이지 [https://developers.google.com/maps/documentation/places/ios-sdk/config?hl=ko]를 참고하세요.
