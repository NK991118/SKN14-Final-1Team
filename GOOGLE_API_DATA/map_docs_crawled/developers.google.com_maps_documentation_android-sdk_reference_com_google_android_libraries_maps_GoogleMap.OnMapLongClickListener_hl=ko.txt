공개 정적 인터페이스
GoogleMap.OnMapLongClickListener

사용자가 지도를 길게 누를 때의 콜백 인터페이스입니다.
리스너는 Android UI 스레드에서 호출됩니다.

공개 메서드 요약
| 추상 void | onMapLongClick(LatLng 지점) 사용자가 지도에서 길게 누르기 동작을 할 때 호출되지만, 지도의 오버레이가 이 동작을 처리하지 않는 경우에만 호출됩니다. |
| --- | --- |

공개 메서드

public
추상
void

onMapLongClick
(LatLng [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/model/LatLng?hl=ko] 지점)

사용자가 지도에서 길게 누르기 동작을 할 때 호출되지만, 지도의 오버레이가 이 동작을 처리하지 않는 경우에만 호출됩니다. 이 메서드의 구현은 항상 Android UI 스레드에서 호출됩니다.

매개변수
| point | 누른 지면의 지점 (화면 지점에서 투영됨)입니다. |
