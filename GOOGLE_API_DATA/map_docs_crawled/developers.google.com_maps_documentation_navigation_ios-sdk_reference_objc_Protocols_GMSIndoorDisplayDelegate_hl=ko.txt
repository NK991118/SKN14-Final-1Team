GMSIndoorDisplayDelegate

@protocol GMSIndoorDisplayDelegate <NSObject>

Delegate for events on GMSIndoorDisplay [https://developers.google.com/maps/documentation/navigation/ios-sdk/reference/objc/Classes/GMSIndoorDisplay.html].

-didChangeActiveBuilding: [https://developers.google.com/maps/documentation/navigation/ios-sdk/reference/objc/Protocols/GMSIndoorDisplayDelegate#/c:objc(pl)GMSIndoorDisplayDelegate(im)didChangeActiveBuilding:]

Raised when the activeBuilding has changed.  The activeLevel will also have already been updated
for the new building, but -didChangeActiveLevel: [https://developers.google.com/maps/documentation/navigation/ios-sdk/reference/objc/Protocols/GMSIndoorDisplayDelegate.html#/c:objc(pl)GMSIndoorDisplayDelegate(im)didChangeActiveLevel:] will be raised after this method.
