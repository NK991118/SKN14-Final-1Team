GMSGroundOverlay

@interface GMSGroundOverlay : GMSOverlay [https://developers.google.com/maps/documentation/ios-sdk/reference/objc/Classes/GMSOverlay.html]

GMSGroundOverlay specifies the available options for a ground overlay that exists on the
Earthâ€™s surface. Unlike a marker, the position of a ground overlay is specified explicitly and it
does not face the camera.

position [https://developers.google.com/maps/documentation/ios-sdk/reference/objc/Classes/GMSGroundOverlay#/c:objc(cs)GMSGroundOverlay(py)position]

The position of this GMSGroundOverlay, or more specifically, the physical position of its
anchor. If this is changed, bounds [https://developers.google.com/maps/documentation/ios-sdk/reference/objc/Classes/GMSGroundOverlay.html#/c:objc(cs)GMSGroundOverlay(py)bounds] will be moved around the new position.

anchor [https://developers.google.com/maps/documentation/ios-sdk/reference/objc/Classes/GMSGroundOverlay#/c:objc(cs)GMSGroundOverlay(py)anchor]

The anchor specifies where this GMSGroundOverlay is anchored to the Earth in relation to
bounds [https://developers.google.com/maps/documentation/ios-sdk/reference/objc/Classes/GMSGroundOverlay.html#/c:objc(cs)GMSGroundOverlay(py)bounds]. If this is modified, position [https://developers.google.com/maps/documentation/ios-sdk/reference/objc/Classes/GMSGroundOverlay.html#/c:objc(cs)GMSGroundOverlay(py)position] will be set to the corresponding new position within
bounds [https://developers.google.com/maps/documentation/ios-sdk/reference/objc/Classes/GMSGroundOverlay.html#/c:objc(cs)GMSGroundOverlay(py)bounds].

icon [https://developers.google.com/maps/documentation/ios-sdk/reference/objc/Classes/GMSGroundOverlay#/c:objc(cs)GMSGroundOverlay(py)icon]

Icon to render within bounds [https://developers.google.com/maps/documentation/ios-sdk/reference/objc/Classes/GMSGroundOverlay.html#/c:objc(cs)GMSGroundOverlay(py)bounds] on the Earth. If this is nil, the overlay will not be visible
(unlike GMSMarker [https://developers.google.com/maps/documentation/ios-sdk/reference/objc/Classes/GMSMarker.html] which has a default image).
