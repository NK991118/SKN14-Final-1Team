Title: Google Cloud Console에 로그인하기

Google Cloud Console에 로그인하기

컬렉션을 사용해 정리하기

내 환경설정을 기준으로 콘텐츠를 저장하고 분류하세요.

개요

Google Maps Platform 프리미엄 플랜에는 Cloud 콘솔에서 관리하는 API의 액세스 권한이 포함됩니다. 시작하려면 Google 계정을 사용하여 Cloud 콘솔에 로그인할 수 있어야 합니다.

Google Maps Platform 프리미엄 플랜에 개인 Gmail 계정을 사용하지 않는 것이 좋습니다. 대신 아래에 설명된 대로 회사 이메일 주소를 사용하세요.

계정이 있는지 확인

이전에 회사 이메일 주소를 사용하여 G Suite, Google 애널리틱스 등 다른 Google 서비스에 로그인한 적이 있다면 이미 Google 계정이 있는 것입니다. Cloud 콘솔 [https://console.cloud.google.com/?hl=ko]에 로그인해 보세요.

Google 비밀번호가 기억나지 않을 경우 '비밀번호를 잊어버렸습니다.'를 선택하고 비밀번호 지원 페이지 [https://www.google.com/accounts/recovery/?hl=ko]에 이메일 주소를 입력하면 됩니다.

Google 계정 만들기

여기에서 회사 이메일 주소를 사용하여 새 Google 계정을 만들 수 있습니다.

Gmail 없이 Google 계정 만들기 [https://accounts.google.com/SignUpWithoutGmail?hl=ko]

중요: 계정이 생성되면 입력한 주소로 사용자가 주소의 소유자이고 해당 회사의 직원임을 확인하기 위한 이메일이 발송됩니다. 이메일에 포함된 링크를 클릭하여 이 주소의 소유자임을 확인하세요. 계정이 확인될 때까지는 프리미엄 플랜이 프로비저닝되지 않습니다.

Google 계정 확인 도움말 [https://support.google.com/accounts/answer/63950/?hl=ko]
