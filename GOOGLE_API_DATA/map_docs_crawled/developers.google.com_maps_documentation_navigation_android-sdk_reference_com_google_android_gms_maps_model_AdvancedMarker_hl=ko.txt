AdvancedMarker
extends Marker [https://developers.google.com/maps/documentation/navigation/android-sdk/reference/com/google/android/gms/maps/model/Marker]

Extends Marker [https://developers.google.com/maps/documentation/navigation/android-sdk/reference/com/google/android/gms/maps/model/Marker], and provides additional premium features.
This class does not apply to the Navigation SDK. It exists for compile time compatibility with
com.google.android.gms:play-services-maps only.

Sets the view to be used for the marker.
If iconView is set, the BitmapDescriptor [https://developers.google.com/maps/documentation/navigation/android-sdk/reference/com/google/android/gms/maps/model/BitmapDescriptor] set via Marker.setIcon(BitmapDescriptor) [https://developers.google.com/maps/documentation/navigation/android-sdk/reference/com/google/android/gms/maps/model/Marker#setIcon(com.google.android.gms.maps.model.BitmapDescriptor)] is ignored.
