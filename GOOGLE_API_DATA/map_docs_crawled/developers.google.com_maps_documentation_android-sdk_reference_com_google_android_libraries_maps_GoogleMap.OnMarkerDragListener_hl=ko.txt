공개 정적 인터페이스 GoogleMap.OnMarkerDragListener

마커의 드래그 이벤트에 대한 콜백 인터페이스입니다.
리스너는 Android UI 스레드에서 호출됩니다.

공개 메서드 요약
| 추상 void | onMarkerDrag(마커 마커) 마커를 드래그하는 동안 반복적으로 호출됩니다. |
| --- | --- |
| 추상 void | onMarkerDragEnd(마커 마커) 마커 드래그가 완료되면 호출됩니다. |
| 추상 void | onMarkerDragStart(마커 마커) 마커가 드래그되기 시작하면 호출됩니다. |

공개 메서드

공개
추상
void

onMarkerDrag
(마커 [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/model/Marker?hl=ko] 마커)

마커를 드래그하는 동안 반복적으로 호출됩니다. 마커의 위치는 getPosition() [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/model/Marker?hl=ko#getPosition()]를 통해 액세스할 수 있습니다.

매개변수
| 마커 | 드래그되는 마커입니다. |
| --- | --- |

공개
추상
void

onMarkerDragEnd
(마커 [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/model/Marker?hl=ko] 마커)

마커 드래그가 완료되면 호출됩니다. 마커의 위치는 getPosition() [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/model/Marker?hl=ko#getPosition()]를 통해 액세스할 수 있습니다.

매개변수

공개
추상
void

onMarkerDragStart
(마커 [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/model/Marker?hl=ko] 마커)

마커가 드래그되기 시작하면 호출됩니다. 마커의 위치는 getPosition() [https://developers.google.com/maps/documentation/android-sdk/reference/com/google/android/libraries/maps/model/Marker?hl=ko#getPosition()]를 통해 액세스할 수 있습니다. 마커가 터치 포인트 위에 팝업되므로 이 위치는 드래그를 시작하기 전의 위치와 다를 수 있습니다.

| 마커 | 드래그되는 마커입니다. |
