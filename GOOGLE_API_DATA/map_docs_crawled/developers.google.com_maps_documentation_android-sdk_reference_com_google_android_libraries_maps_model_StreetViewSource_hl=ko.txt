스트리트 뷰 소스

외부

스트리트 뷰 검색을 야외 컬렉션으로 제한합니다. 실내 컬렉션은 검색결과에 포함되지 않습니다. 또한 검색 시 실내에 있는지 야외에 있는지 확인할 수 있는 파노라마만 반환됩니다. 예를 들어 Photo Sphere는 실내에 있는지 야외에 있는지 알 수 없으므로 반환되지 않습니다.
