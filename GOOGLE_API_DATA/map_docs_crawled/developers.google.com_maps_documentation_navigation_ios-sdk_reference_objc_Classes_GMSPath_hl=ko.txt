GMSPath

@interface GMSPath : NSObject <NSCopying, NSMutableCopying>

GMSPath encapsulates an immutable array of CLLocationCooordinate2D. All the coordinates of a
GMSPath must be valid. The mutable counterpart is GMSMutablePath [https://developers.google.com/maps/documentation/navigation/ios-sdk/reference/objc/Classes/GMSMutablePath.html].
