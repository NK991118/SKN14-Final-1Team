지원 받기

Google 서포트 패키지 이용하기

Google Cloud 는 다양한 니즈를 충족할 수 있도록 연중무휴 지원, 전화 지원, 기술 지원 관리자 상담과 같은 다양한 서포트 패키지를 제공합니다. 자세한 내용은 Cloud Customer Care [https://cloud.google.com/support?hl=ko]를 참조하세요.

커뮤니티에서 지원받기

Stack Overflow에 질문하기

Stack Overflow [http://stackoverflow.com/questions/tagged/google-cloud-firestore]에서 Firestore에 대해 질문할 수 있습니다.
Firestore에 대한 질문에는 google-cloud-firestore 태그를 사용하세요. 이 태그를 사용하면 Stack Overflow 커뮤니티뿐만 아니라 이 태그를 모니터링하고 비공식 지원을 제공하는 Google 엔지니어의 답변을 받을 수 있습니다.
