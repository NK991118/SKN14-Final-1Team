Describes the progress of the operation. Unit of work is generic and must be interpreted based on where Progress [https://cloud.google.com/firestore/docs/reference/rest/Shared.Types/Progress] is used.

| JSON representation |
| --- |
| {   "estimatedWork": string,   "completedWork": string } |

| Fields |
| --- |
| estimatedWork | string (int64 format) The amount of work estimated. |
| completedWork | string (int64 format) The amount of work completed. |
