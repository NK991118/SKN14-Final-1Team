Title: 약정 사용 할인

약정 사용 할인

컬렉션을 사용해 정리하기

내 환경설정을 기준으로 콘텐츠를 저장하고 분류하세요.

약정 사용 할인(CUD)은 1년 또는 3년 동안 Cloud Firestore 작업(읽기/쓰기/삭제)에 특정 금액을 계속 사용하겠다는 약속 하에 대폭 할인된 가격이 제공됩니다.

CUD는 Cloud Firestore 작업을 최소한 어느 정도까지 예측할 수 있어서 이를 최소 1년 이상 약정할 수 있을 때 이상적입니다.

중요: 이 페이지에서는 2025년 7월 15일 이후에 첫 번째 CUD를 구매하는 모든 고객에게 적용되는 새롭고 개선된 약정 사용 할인(CUD) 프로그램을 설명합니다. 고객은 지금 선택하여 새로운 CUD 환경을 이용할 수 있습니다. 자세한 내용은 지출 기반 CUD 프로그램 개선사항 [https://cloud.google.com/docs/cuds-multiprice?hl=ko]을 참조하세요.

Cloud Firestore CUD 가격 책정

Cloud Firestore는 약정 기간에 따라 두 가지 수준의 할인을 제공합니다.

20% 할인: 1년 약정 시 제공됩니다. 약정 기간 동안 약정 시간당 지출 금액으로 Cloud Firestore CUD 1년 가격(소비 모델 ID 3892-BA17-92A7)을 지불합니다.
40% 할인: 3년 약정 시 제공됩니다. 약정 기간 동안 약정 시간당 지출 금액으로 Cloud Firestore CUD 3년 가격(소비 모델 ID 2FD9-44B6-D2AC)을 지불합니다.

약정을 구매할 때 1년 또는 3년 기간을 선택합니다. 또한 해당 기간 동안 시간당 읽기/쓰기/삭제 작업에 대한 예상 Cloud Firestore 지출인 약정 금액을 지정합니다. 이 약정 금액이 약정 수수료가 됩니다. 그런 다음 CUD를 수신하고 CUD 기간 동안 매월 약정 수수료가 청구됩니다.

할인은 데이터베이스의 리전에 관계없이 약정 구매에 사용된 Cloud Billing 계정의 프로젝트와 연결된 Cloud Firestore 데이터베이스의 모든 요건을 충족하는 작업 사용량에 적용됩니다.

약정을 초과하는 지출은 주문형 요율에 따라 청구됩니다. Cloud Firestore 사용량이 증가함에 따라 추가 약정을 구매하여 이전 약정에 포함되지 않은 증가한 지출에 대해 할인을 받을 수 있습니다.

Cloud Firestore 읽기/쓰기/삭제 작업의 주문형 가격이 약정을 구매한 후 변경되어도 약정 수수료는 변경되지 않고 여전히 해당 사용량에 대해 동일한 할인율을 받습니다.

운영 가능 리소스

Cloud Firestore CUD는 프로젝트 간 읽기/쓰기/삭제 작업으로 측정된 Cloud Firestore 작업 사용량에 대한 지출에 자동으로 적용됩니다.

Cloud Firestore CUD는 읽기/쓰기/삭제 작업 이외의 Cloud Firestore 리소스에 대한 지출에는 적용되지 않습니다.

적용 가능한 SKU 목록은 Cloud Firestore 요건 충족 SKU [https://cloud.google.com/skus/sku-groups/cloud-firestore-cud-eligible-skus?hl=ko]를 참조하세요.

약정 구매

약정을 구매하기 전에 약정 단위와 관련된 서비스별 약관 [https://cloud.google.com/terms/service-terms?hl=ko]을 읽어보세요.

약정을 구매하면 1시간 이내에 할인이 적용되며 이후 대상 사용량에 자동으로 적용됩니다.

약정 크기 및 기간이 Cloud Firestore 작업에 대한 이전 및 예상되는 최소 지출액에 맞게 조정되었는지 확인하세요.
주의: 약정을 구매한 후에는 CUD를 취소할 수 없습니다.
Cloud Billing 계정의 CUD를 구매하거나 관리하려면 지출 기반 약정 구매 [https://cloud.google.com/docs/cuds-spend-based?hl=ko#purchasing]의 안내를 따르세요.

Cloud Firestore CUD 시나리오 예시

이상적으로 약정은 다음 1년 또는 3년 동안 프로젝트에서 Cloud Firestore 읽기/쓰기/삭제 작업에 대한 최소 시간당 예상 지출을 나타냅니다.
참고: 이 섹션의 가격은 예시입니다. 현재 가격은 Cloud Firestore 가격 책정 [https://firebase.google.com/docs/firestore/pricing?hl=ko]을 참고하세요.
예를 들어 us-central1(아이오와) 리전에 있는 Cloud Firestore 데이터베이스가 있다고 가정해 보겠습니다.

가격 책정 페이지 [https://firebase.google.com/docs/firestore/pricing?hl=ko]에서 대략적인 시간별 1년 약정 비용을 다음과 같이 계산할 수 있습니다.

1년 월별 비용, $1.92/시간 약정: (시간당 $2.40 - 20% 할인) * 730시간 = 월별 $1,401.60
월별 총 절약액: $1,752 - $1,401.60 = $350.40

다음 연도 또는 그 이후에도 계속해서 최소 시간당 $1.92를 소비할 것으로 예상되는 경우에는 해당 금액으로 약정을 설정할 수 있습니다.
즉, CUD를 구매할 때 시간별 약정 금액으로 '$1.92'를 입력할 수 있습니다.

기존 CUD 프로그램에서는 약정 금액이 주문형 가격입니다. 기존 및 신규 지출 기반 CUD 프로그램의 차이점에 대한 자세한 내용은 지출 기반 CUD 프로그램 개선 사항 [https://cloud.google.com/docs/cuds-multiprice?hl=ko]을 참조하세요.

반면에 경우에 따라 용량이 축소될 것으로 예상되는 경우에는 낮은 금액으로 약정을 설정할 수 있습니다. 한도를 초과하는 지출은 주문형 요율로 청구됩니다.

비교 기준으로 약정 할인을 적용하지 않고 Cloud Firestore 용량의 주문형 비용을 계산합니다.

읽기 작업 비용: 시간당 문서 읽기 2백만 건 * 문서 읽기 100,000건당 $0.03 = 시간당 $0.60
쓰기 작업 비용: 시간당 문서 쓰기 2백만 건 * 문서 쓰기 100,000건당 $0.09 = 시간당 $1.80
총 지출액: $0.60 + $1.80 = 시간당 $2.40
주문형 가격 책정 기반의 월별 비용: 시간당 $2.40 * 730시간 = 월별 $1,752

여기에서는 1년간 전체 요금을 지불할 때와 비교해 20% 할인이 적용된 약정의 월별 비용 및 절감액을 계산할 수 있습니다.

1년 월별 비용, $1.92/시간 약정: (시간당 $2.40 - 20% 할인) * 730시간 = 월별 $1,401.60
월별 총 절약액: $1,752 - $1,401.60 = $350.40
1년 총 절약액, $1.92/시간 약정: 월별 $350.40 * 12개월 = $4,204.80

3년 CUD의 경우에도 주문형 요율에 40% 할인을 적용한 비슷한 수식을 사용해서 비용과 절약액을 계산할 수 있습니다.

3년 월별 비용, $1.44/시간 약정: (시간당 $2.40 - 40% 할인) * 730시간 = 월별 $1,051.20
월별 총 절약액: $1,752 - $1,051.20 = $700.80
3년 총 절약액, $1.44/시간 약정: 시간당 $700.80 * 36개월 = $25,228.80

향후 몇 년 동안 예상되는 최소 Cloud Firestore 사용량이 포함된 약정을 통해 상당한 비용 절감 효과를 얻을 수 있습니다.

약정 금액 선택 권장사항

Cloud Firestore CUD와 약정할 금액을 고려할 때는 다음 사항에 유의하세요.

리전: Cloud Firestore CUD는 소속 리전과 관계없이 프로젝트의 모든 읽기/쓰기/삭제 작업에 적용됩니다. 여러 리전에 Cloud Firestore 데이터베이스가 있는 경우 약정 구매를 결정하기 전에 프로젝트에 사용되는 모든 리전의 지출을 계산합니다.
프로젝트: 총 약정을 계산할 때 프로젝트별로 일관적인 기준 지출액을 결정합니다. 프로덕션 로드는 일반적으로 100% 실행되지만 개발 또는 스테이징 환경은 간헐적으로 실행되는 경우가 많을 수 있습니다.

약정 요금은 실제 사용량에 관계없이 약정 기간 동안 매시간 적용됩니다. 이전 Cloud Firestore 사용량과 이후 예상 사용량을 기준으로 약정 금액을 신중하게 선택하세요.
Cloud Firestore 읽기/쓰기/삭제 작업 사용량이 약정한 지출 수준을 초과하는 한 약정 기간 동안 가능한 최대 할인을 받을 수 있습니다.
