미디어의 가치 증대

컬렉션을 사용해 정리하기

내 환경설정을 기준으로 콘텐츠를 저장하고 분류하세요.

Cloud Firestore 및 Firebase Extensions를 사용하여 AI 모델로 오디오 및 이미지 리소스의 가치를 최대한 활용하세요.

이미지 분류

Cloud Vision API를 사용하여 이미지에서 라벨을 추출하고 Firestore에 저장하세요.
자세히 알아보기 [https://extensions.dev/extensions/googlecloud/storage-label-images]

오디오 파일의 음성을 텍스트로 변환

Cloud Speech-to-Text를 사용하여 Cloud Storage의 오디오 파일을 .txt 파일로 텍스트 변환하세요.
자세히 알아보기 [https://extensions.dev/extensions/googlecloud/speech-to-text]

광학 문자 인식(OCR) 수행

Cloud Vision API를 사용하여 이미지에서 텍스트를 추출하고 Firestore에 저장하세요.
자세히 알아보기 [https://extensions.dev/extensions/googlecloud/storage-extract-image-text]
