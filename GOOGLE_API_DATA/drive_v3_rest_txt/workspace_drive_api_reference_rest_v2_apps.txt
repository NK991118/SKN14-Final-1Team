리소스: App
앱 리소스는 사용자가 설치한 앱 목록을 제공하며 각 앱의 지원되는 MIME 유형, 파일 확장자, 기타 세부정보를 포함합니다.
일부 리소스 메서드 (예: apps.get)에는 appId이 필요합니다. apps.list 메서드를 사용하여 설치된 애플리케이션의 ID를 가져옵니다.
JSON 표현
{
  "primaryMimeTypes": [
    string
  ],
  "secondaryMimeTypes": [
    string
  ],
  "primaryFileExtensions": [
    string
  ],
  "secondaryFileExtensions": [
    string
  ],
  "icons": [
    {
      "size": integer,
      "category": string,
      "iconUrl": string
    }
  ],
  "name": string,
  "objectType": string,
  "supportsCreate": boolean,
  "productUrl": string,
  "id": string,
  "supportsImport": boolean,
  "installed": boolean,
  "authorized": boolean,
  "useByDefault": boolean,
  "kind": string,
  "shortDescription": string,
  "longDescription": string,
  "supportsMultiOpen": boolean,
  "productId": string,
  "openUrlTemplate": string,
  "createUrl": string,
  "createInFolderTemplate": string,
  "supportsOfflineCreate": boolean,
  "hasDriveWideScope": boolean
}
필드
primaryMimeTypes[]
string
기본 MIME 유형 목록입니다.
secondaryMimeTypes[]
string
보조 MIME 유형 목록입니다.
primaryFileExtensions[]
string
기본 파일 확장자 목록입니다.
secondaryFileExtensions[]
string
보조 파일 확장자 목록입니다.
icons[]
object
앱의 다양한 아이콘입니다.
icons[].size
integer
아이콘의 크기입니다. 너비와 높이의 최댓값으로 표현됩니다.
icons[].category
string
아이콘의 카테고리입니다. 허용되는 값은 다음과 같습니다.
application - 애플리케이션 아이콘
document - 앱과 연결된 파일의 아이콘
documentShared - 앱과 연결된 공유 파일의 아이콘
icons[].iconUrl
string
아이콘의 URL입니다.
name
string
앱 이름입니다.
objectType
string
이 앱이 만드는 객체의 유형입니다 (예: 차트). 비어 있으면 앱 이름을 대신 사용해야 합니다.
supportsCreate
boolean
이 앱이 새 객체 생성을 지원하는지 여부입니다.
productUrl
string
이 앱의 제품 등록정보 링크입니다.
id
string
앱의 ID입니다.
supportsImport
boolean
이 앱이 Docs 편집기에서 가져오기를 지원하는지 여부입니다.
installed
boolean
앱이 설치되어 있는지 여부입니다.
authorized
boolean
앱이 사용자의 Drive에 있는 데이터에 액세스할 수 있는지 여부입니다.
useByDefault
boolean
앱이 지원하는 유형의 기본 핸들러로 선택되었는지 여부입니다.
kind
string
항상 drive#app입니다.
shortDescription
string
앱에 대한 간단한 설명입니다.
longDescription
string
앱에 대한 자세한 설명입니다.
supportsMultiOpen
boolean
이 앱이 두 개 이상의 파일 열기를 지원하는지 여부입니다.
productId
string
이 앱의 제품 등록정보 ID입니다.
openUrlTemplate
string
이 앱으로 파일을 여는 템플릿 URL입니다. 템플릿에는 실제 파일 ID로 대체될 {ids} 또는 {exportIds}가 포함됩니다. 전체 문서는 파일 열기 를 참고하세요.
createUrl
string
이 앱으로 새 파일을 만드는 URL입니다.
createInFolderTemplate
string
지정된 폴더에 이 앱으로 새 파일을 만드는 템플릿 URL입니다. 템플릿에는 새 파일을 만들 폴더로 대체될 {folderId}가 포함됩니다.
supportsOfflineCreate
boolean
이 앱이 오프라인 상태에서 새 파일 만들기를 지원하는지 여부입니다.
hasDriveWideScope
boolean
앱에 드라이브 전체 범위가 있는지 여부입니다. 드라이브 전체 범위가 있는 앱은 사용자의 드라이브에 있는 모든 파일에 액세스할 수 있습니다.
메서드
get
특정 앱을 가져옵니다.
list
사용자의 설치된 앱을 나열합니다.
