필드
driveThemes[]
object
공유 드라이브에서 지원되는 테마 목록입니다.
driveThemes[].id
string
테마의 ID입니다.
driveThemes[].backgroundImageLink
string
이 테마의 배경 이미지 링크입니다.
driveThemes[].colorRgb
string
이 테마의 색상(RGB 16진수 문자열)입니다.
importFormats
map (key: string, value: value (Value [https://protobuf.dev/reference/protobuf/google.protobuf/#value] format))
지원되는 모든 가져오기의 소스 MIME 유형과 가능한 타겟의 맵입니다.
"key": value 쌍 목록을 포함하는 객체입니다. 예: { "name": "wrench", "mass": "1.3kg", "count": "3" }
exportFormats
map (key: string, value: value (Value [https://protobuf.dev/reference/protobuf/google.protobuf/#value] format))
지원되는 모든 내보내기의 소스 MIME 유형과 가능한 타겟의 맵입니다.
"key": value 쌍 목록을 포함하는 객체입니다. 예: { "name": "wrench", "mass": "1.3kg", "count": "3" }
folderColorPalette[]
string
현재 지원되는 폴더 색상(RGB 16진수 문자열)입니다.
maxImportSizes
map (key: string, value: string (int64 [https://developers.google.com/discovery/v1/type-format?hl=ko] format))
MIME 유형별 최대 가져오기 크기(바이트)의 맵입니다.
"key": value 쌍 목록을 포함하는 객체입니다. 예: { "name": "wrench", "mass": "1.3kg", "count": "3" }

kind

리소스 종류를 식별합니다. 값: 고정 문자열 "drive#about".
storageQuota
object
사용자의 스토리지 할당량 한도 및 사용량입니다. 공용 스토리지를 사용하는 조직에 속한 사용자의 경우 모든 서비스의 한도 및 사용량에 관한 정보는 개별 사용자가 아닌 조직에 적용됩니다. 모든 필드는 바이트 단위로 측정됩니다.
storageQuota.limit
string (int64 [https://developers.google.com/discovery/v1/type-format?hl=ko] format)
사용량 한도(해당하는 경우) 사용자에게 무제한 스토리지가 있는 경우 이 값이 표시되지 않습니다. 공용 스토리지를 사용하는 조직에 속한 사용자의 경우 개인 사용자가 아닌 조직의 한도입니다.
storageQuota.usageInDrive
string (int64 [https://developers.google.com/discovery/v1/type-format?hl=ko] format)
Google Drive의 모든 파일의 사용량입니다.
storageQuota.usageInDriveTrash
string (int64 [https://developers.google.com/discovery/v1/type-format?hl=ko] format)
Google Drive에서 휴지통에 있는 파일의 사용량입니다.
storageQuota.usage
string (int64 [https://developers.google.com/discovery/v1/type-format?hl=ko] format)
모든 서비스의 총 사용량입니다. 공용 스토리지를 사용하는 조직에 속한 사용자의 경우 개인 사용량이 아닌 조직의 모든 서비스에 걸친 사용량입니다.
