Drive API 설정과 함께 현재 사용자에 대한 정보를 가져옵니다.
HTTP 요청
GET https://www.googleapis.com/drive/v2/about
URL은 gRPC 트랜스코딩 구문을 사용합니다.
쿼리 매개변수
매개변수
includeSubscribed
boolean
내 드라이브 계층 구조 외부의 변경사항을 계산할지 여부입니다. false로 설정하면 애플리케이션 데이터 폴더의 파일이나 내 드라이브에 추가되지 않은 공유 파일과 같은 파일의 변경사항이 maxChangeIdCount에서 생략됩니다.
maxChangeIdCount
string (int64 format)
계산할 남은 변경 ID의 최대 개수
startChangeId
string (int64 format)
남은 변경 ID 수를 계산할 때부터 집계를 시작하도록 변경 ID

요청 본문은 비어 있어야 합니다.
응답 본문
