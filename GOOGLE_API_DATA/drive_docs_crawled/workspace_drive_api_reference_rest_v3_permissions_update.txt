승인 범위
패치 시맨틱스로 권한을 업데이트합니다.
경고: 동일한 파일에서 동시 권한 작업은 지원되지 않습니다. 마지막 업데이트만 적용됩니다.
HTTP 요청
PATCH https://www.googleapis.com/drive/v3/files/{fileId}/permissions/{permissionId}
URL은 gRPC 트랜스코딩 [https://google.aip.dev/127] 문법을 사용합니다.
경로 매개변수
매개변수
fileId
string
파일 또는 공유 드라이브의 ID입니다.
permissionId
string
권한의 ID입니다.
쿼리 매개변수
매개변수
removeExpiration
boolean
만료일을 삭제할지 여부입니다.
supportsAllDrives
boolean
요청하는 애플리케이션에서 내 드라이브와 공유 드라이브를 모두 지원하는지 여부입니다.
supportsTeamDrives
(deprecated)
boolean
지원 중단됨: 대신 supportsAllDrives를 사용하세요.
transferOwnership
boolean
지정된 사용자에게 소유권을 이전하고 현재 소유자를 작성자로 다운그레이드할지 여부입니다. 이 매개변수는 부작용에 대한 확인으로 필요합니다.
useDomainAdminAccess
boolean
도메인 관리자로 요청을 제출합니다. true로 설정하면 파일 ID 매개변수가 공유 드라이브를 참조하고 요청자가 공유 드라이브가 속한 도메인의 관리자인 경우 요청자에게 액세스 권한이 부여됩니다.
enforceExpansiveAccess
boolean
요청에 확장된 액세스 규칙을 적용해야 하는지 여부입니다.
요청 본문
요청 본문에 Permission [https://developers.google.com/workspace/drive/api/reference/rest/v3/permissions?hl=ko#Permission]의 인스턴스가 포함됩니다.
응답 본문
성공한 경우 응답 본문에 Permission [https://developers.google.com/workspace/drive/api/reference/rest/v3/permissions?hl=ko#Permission]의 인스턴스가 포함됩니다.
승인 범위
다음 OAuth 범위 중 하나가 필요합니다.
https://www.googleapis.com/auth/drive
https://www.googleapis.com/auth/drive.file
일부 범위는 제한되어 있으며 앱에서 이를 사용하려면 보안 평가가 필요합니다. 자세한 내용은 승인 가이드 [https://developers.google.com/workspace/guides/configure-oauth-consent?hl=ko]를 참고하세요.
