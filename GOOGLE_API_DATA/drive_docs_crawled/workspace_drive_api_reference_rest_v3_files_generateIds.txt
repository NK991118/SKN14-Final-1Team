생성 또는 복사 요청에 제공할 수 있는 파일 ID 집합을 생성합니다.
HTTP 요청
GET https://www.googleapis.com/drive/v3/files/generateIds
URL은 gRPC 트랜스코딩 [https://google.aip.dev/127] 구문을 사용합니다.
쿼리 매개변수
매개변수
count
integer
반환할 ID 수입니다.
space
string
ID를 사용하여 새 파일을 만들 수 있는 공간입니다. 지원되는 값은 'drive' 및 'appDataFolder'입니다. (기본값: 'drive')
type
string
ID를 사용할 수 있는 항목 유형입니다. 지원되는 값은 'files' 및 'shortcuts'입니다. '바로가기'는 drive '스페이스'에서만 지원됩니다. (기본값: 'files')
요청 본문
요청 본문은 비어 있어야 합니다.
응답 본문
생성 요청에 제공할 수 있는 생성된 파일 ID 목록입니다.
성공한 경우 응답 본문은 다음과 같은 구조의 데이터를 포함합니다.
JSON 표현
{
  "ids": [
    string
  ],
  "space": string,
  "kind": string
}
필드
ids[]
string
지정된 스페이스에서 요청하는 사용자에 대해 생성된 ID입니다.
space
string
이러한 ID로 만들 수 있는 파일 유형입니다.
kind
string
리소스 종류를 식별합니다. 값: 고정 문자열 "drive#generatedIds".
승인 범위
다음 OAuth 범위 중 하나가 필요합니다.
https://www.googleapis.com/auth/drive
https://www.googleapis.com/auth/drive.appdata
https://www.googleapis.com/auth/drive.file
일부 범위는 제한되어 있으며 앱에서 이를 사용하려면 보안 평가가 필요합니다. 자세한 내용은 승인 가이드 [https://developers.google.com/workspace/guides/configure-oauth-consent?hl=ko]를 참고하세요.
