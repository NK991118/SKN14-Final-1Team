ModifiedDateBehavior
bookmark_border

열거형
fromBody modifiedDate를 요청 본문에 제공된 값으로 설정합니다. 값이 제공되지 않은 경우 변경되지 않습니다.
fromBodyIfNeeded 업데이트의 다른 콘텐츠에 따라 modifiedDate를 요청 본문에 제공된 값으로 설정합니다.
fromBodyOrNow modifiedDate를 요청 본문에 제공된 값으로 설정하거나 값이 제공되지 않은 경우 현재 시간으로 설정합니다.
noChange 이전 modifiedDate 값을 유지합니다.
now modifiedDate를 현재 시간으로 설정합니다.
nowIfNeeded 업데이트 내용에 따라 modifiedDate를 현재 시간으로 설정합니다.
