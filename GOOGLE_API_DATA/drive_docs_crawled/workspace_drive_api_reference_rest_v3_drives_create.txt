HTTP 요청
POST https://www.googleapis.com/drive/v3/drives
URL은 gRPC 트랜스코딩 [https://google.aip.dev/127] 구문을 사용합니다.
쿼리 매개변수
매개변수
requestId
string
필수입니다. 공유 드라이브의 idempotent 생성을 위한 이 사용자의 요청을 고유하게 식별하는 ID(예: 임의 UUID)입니다. 동일한 사용자가 동일한 요청 ID로 반복 요청하면 동일한 공유 드라이브를 생성하려고 시도하여 중복이 생성되지 않습니다. 공유 드라이브가 이미 있으면 409 오류가 반환됩니다.
요청 본문
요청 본문에 Drive [https://developers.google.com/workspace/drive/api/reference/rest/v3/drives?hl=ko#Drive]의 인스턴스가 포함됩니다.
응답 본문
성공한 경우 응답 본문에 새로 생성된 Drive [https://developers.google.com/workspace/drive/api/reference/rest/v3/drives?hl=ko#Drive]의 인스턴스가 포함됩니다.
승인 범위
다음 OAuth 범위가 필요합니다.
https://www.googleapis.com/auth/drive
일부 범위는 제한되어 있으며 앱에서 이를 사용하려면 보안 평가가 필요합니다. 자세한 내용은 승인 가이드 [https://developers.google.com/workspace/guides/configure-oauth-consent?hl=ko]를 참고하세요.
