리소스: Comment
파일에 대한 댓글입니다.
일부 리소스 메서드 (예: comments.update)에는 commentId이 필요합니다. comments.list 메서드를 사용하여 파일의 댓글 ID를 가져옵니다.
JSON 표현
{
  "replies": [
    {
      object (Reply [https://developers.google.com/workspace/drive/api/reference/rest/v3/replies?hl=ko#Reply])
    }
  ],
  "id": string,
  "kind": string,
  "createdTime": string,
  "modifiedTime": string,
  "resolved": boolean,
  "anchor": string,
  "author": {
    object (User [https://developers.google.com/workspace/drive/api/reference/rest/v3/User?hl=ko])
  },
  "deleted": boolean,
  "htmlContent": string,
  "content": string,
  "quotedFileContent": {
    "mimeType": string,
    "value": string
  }
}
필드
replies[]
object (Reply [https://developers.google.com/workspace/drive/api/reference/rest/v3/replies?hl=ko#Reply])
출력 전용입니다. 댓글에 대한 답글의 전체 목록입니다(시간순).
id
string
출력 전용입니다. 댓글의 ID입니다.
kind
string
출력 전용입니다. 리소스 종류를 식별합니다. 값: 고정 문자열 "drive#comment".
createdTime
string
출력 전용입니다. 댓글이 작성된 시간입니다 (RFC 3339 날짜/시간).
modifiedTime
string
출력 전용입니다. 댓글 또는 댓글의 답글이 마지막으로 수정된 시간입니다 (RFC 3339 날짜/시간).
resolved
boolean
출력 전용입니다. 댓글이 답글 중 하나에 의해 해결되었는지 여부입니다.
anchor
string
JSON 문자열로 표현된 문서의 영역입니다. 앵커 속성 정의에 관한 자세한 내용은 댓글 및 답글 관리 [https://developers.google.com/workspace/drive/api/v3/manage-comments?hl=ko]를 참고하세요.
author
object (User [https://developers.google.com/workspace/drive/api/reference/rest/v3/User?hl=ko])
출력 전용입니다. 댓글 작성자입니다. 작성자의 이메일 주소와 권한 ID는 채워지지 않습니다.
deleted
boolean
출력 전용입니다. 댓글이 삭제되었는지 여부입니다. 삭제된 댓글에는 콘텐츠가 없습니다.
htmlContent
string
출력 전용입니다. HTML 형식이 지정된 댓글의 콘텐츠입니다.
content
string
댓글의 일반 텍스트 콘텐츠입니다. 이 필드는 콘텐츠를 설정하는 데 사용되며 htmlContent가 표시되어야 합니다.
quotedFileContent
object
댓글이 참조하는 파일 콘텐츠입니다(일반적으로 앵커 영역 내). 예를 들어 텍스트 파일의 경우 댓글 위치에 있는 텍스트가 됩니다.
quotedFileContent.mimeType
string
인용된 콘텐츠의 MIME 유형입니다.
quotedFileContent.value
string
인용된 콘텐츠 자체입니다. API를 통해 설정된 경우 일반 텍스트로 해석됩니다.
메서드
create [https://developers.google.com/workspace/drive/api/reference/rest/v3/comments/create?hl=ko]
파일에 댓글을 만듭니다.
delete [https://developers.google.com/workspace/drive/api/reference/rest/v3/comments/delete?hl=ko]
댓글을 삭제합니다.
get [https://developers.google.com/workspace/drive/api/reference/rest/v3/comments/get?hl=ko]
ID로 댓글을 가져옵니다.
list [https://developers.google.com/workspace/drive/api/reference/rest/v3/comments/list?hl=ko]
파일의 댓글을 나열합니다.
update [https://developers.google.com/workspace/drive/api/reference/rest/v3/comments/update?hl=ko]
패치 시맨틱스를 사용하여 댓글을 업데이트합니다.
