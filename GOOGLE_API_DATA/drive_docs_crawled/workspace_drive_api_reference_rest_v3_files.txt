필드
exportLinks
map (key: string, value: string)
출력 전용입니다. Docs 편집기 파일을 특정 형식으로 내보내기 위한 링크입니다.
"key": value 쌍 목록을 포함하는 객체입니다. 예: { "name": "wrench", "mass": "1.3kg", "count": "3" }
parents[]
string
파일이 포함된 상위 폴더의 ID입니다.
파일에는 상위 폴더가 하나만 있을 수 있으며, 상위 폴더를 여러 개 지정하는 것은 지원되지 않습니다.
생성 요청의 일부로 지정되지 않은 경우 파일은 사용자의 내 드라이브 폴더에 직접 배치됩니다. 복사 요청의 일부로 지정되지 않은 경우 파일은 소스 파일의 검색 가능한 상위를 상속합니다. files.update 요청은 addParents 및 removeParents 매개변수를 사용하여 상위 목록을 수정해야 합니다.
owners[]
object (User [https://developers.google.com/workspace/drive/api/reference/rest/v3/User?hl=ko])
출력 전용입니다. 이 파일의 소유자입니다. 일부 기존 파일에만 소유자가 2명 이상 있을 수 있습니다. 공유 드라이브의 항목에는 이 필드가 채워지지 않습니다.
permissions[]
object (Permission [https://developers.google.com/workspace/drive/api/reference/rest/v3/permissions?hl=ko#Permission])
출력 전용입니다. 파일의 전체 권한 목록입니다. 요청하는 사용자가 파일을 공유할 수 있는 경우에만 사용할 수 있습니다. 공유 드라이브의 항목에 대해서는 채워지지 않습니다.
spaces[]
string
출력 전용입니다. 파일이 포함된 스페이스 목록입니다. 현재 지원되는 값은 'drive', 'appDataFolder', 'photos'입니다.
properties
map (key: string, value: value (Value [https://protobuf.dev/reference/protobuf/google.protobuf/#value] format))
모든 앱에 표시되는 임의의 키-값 쌍 모음입니다.
null 값이 있는 항목은 업데이트 및 복사 요청에서 삭제됩니다.
"key": value 쌍 목록을 포함하는 객체입니다. 예: { "name": "wrench", "mass": "1.3kg", "count": "3" }
appProperties
map (key: string, value: value (Value [https://protobuf.dev/reference/protobuf/google.protobuf/#value] format))
요청 앱에 비공개인 임의 키-값 쌍의 컬렉션입니다.
null 값이 있는 항목은 업데이트 및 복사 요청에서 삭제됩니다.
이러한 속성은 인증된 요청을 사용해서만 가져올 수 있습니다. 인증된 요청은 OAuth 2 클라이언트 ID로 획득한 액세스 토큰을 사용합니다. API 키를 사용하여 비공개 속성을 가져올 수 없습니다.
"key": value 쌍 목록을 포함하는 객체입니다. 예: { "name": "wrench", "mass": "1.3kg", "count": "3" }
permissionIds[]
string
출력 전용입니다. 이 파일에 액세스할 수 있는 사용자의 권한 ID 목록입니다.
contentRestrictions[]
object (ContentRestriction [https://developers.google.com/workspace/drive/api/reference/rest/v3/files?hl=ko#File.ContentRestriction])
파일 콘텐츠 액세스 제한입니다. 이러한 제한이 있는 경우에만 채워집니다.
kind
string
출력 전용입니다. 리소스 종류를 식별합니다. 값: 고정 문자열 "drive#file".
driveId
string
출력 전용입니다. 파일이 있는 공유 드라이브의 ID입니다. 공유 드라이브의 항목에 대해서만 채워집니다.
fileExtension
string
출력 전용입니다. fullFileExtension의 마지막 구성요소입니다. Google Drive의 바이너리 콘텐츠가 있는 파일에만 사용할 수 있습니다.
copyRequiresWriterPermission
boolean
읽기 권한 사용자 및 댓글 작성자의 이 파일 복사, 인쇄, 다운로드 옵션을 사용 중지할지 여부입니다.
md5Checksum
string
출력 전용입니다. 파일 콘텐츠의 MD5 체크섬입니다. Google Drive의 바이너리 콘텐츠가 있는 파일에만 적용됩니다.
contentHints
object
파일 콘텐츠에 관한 추가 정보입니다. 이러한 필드는 응답에 채워지지 않습니다.
contentHints.indexableText
string
전체 텍스트 쿼리를 개선하기 위해 파일에 대해 색인이 생성될 텍스트입니다. 길이는 128KB로 제한되며 HTML 요소를 포함할 수 있습니다.
contentHints.thumbnail
object
파일의 썸네일입니다. 이 옵션은 Google Drive에서 표준 썸네일을 생성할 수 없는 경우에만 사용됩니다.
contentHints.thumbnail.image
string (bytes [https://developers.google.com/discovery/v1/type-format?hl=ko] format)
URL 안전 Base64 (RFC 4648 섹션 5)로 인코딩된 썸네일 데이터입니다.
base64 인코딩 문자열입니다.
contentHints.thumbnail.mimeType
string
썸네일의 MIME 유형입니다.
writersCanShare
boolean
writer 권한만 있는 사용자가 파일의 권한을 수정할 수 있는지 여부입니다. 공유 드라이브의 항목에 대해서는 채워지지 않습니다.
viewedByMe
boolean
출력 전용입니다. 이 사용자가 파일을 조회했는지 여부입니다.
mimeType
string
파일의 MIME 유형입니다.
값이 제공되지 않으면 Google Drive에서 업로드된 콘텐츠에서 적절한 값을 자동으로 감지하려고 시도합니다. 새 수정 버전을 업로드하지 않는 한 값을 변경할 수 없습니다.
Google 문서 MIME 유형으로 파일을 만든 경우 업로드된 콘텐츠가 가능하면 가져옵니다. 지원되는 가져오기 형식은 About 리소스에 게시됩니다.
thumbnailLink
string
출력 전용입니다. 사용 가능한 경우 파일의 썸네일로 연결되는 단기 링크입니다. 일반적으로 몇 시간 정도 지속됩니다. 교차 출처 리소스 공유 (CORS) [https://developer.mozilla.org/en-US/docs/Web/HTTP/CORS] 정책으로 인해 웹 애플리케이션에서 직접 사용할 수 없습니다. 프록시 서버를 사용하는 것이 좋습니다. 요청 앱이 파일의 콘텐츠에 액세스할 수 있는 경우에만 채워집니다. 파일이 공개적으로 공유되지 않은 경우 Files.thumbnailLink에서 반환된 URL은 사용자 인증 정보가 있는 요청을 사용하여 가져와야 합니다.
iconLink
string
출력 전용입니다. 파일 아이콘에 대한 정적이고 인증되지 않은 링크입니다.
shared
boolean
출력 전용입니다. 파일이 공유되었는지 여부입니다. 공유 드라이브의 항목에 대해서는 채워지지 않습니다.
lastModifyingUser
object (User [https://developers.google.com/workspace/drive/api/reference/rest/v3/User?hl=ko])
출력 전용입니다. 파일을 마지막으로 수정한 사용자입니다. 이 필드는 로그인한 사용자가 마지막 수정을 실행한 경우에만 채워집니다.
headRevisionId
string
출력 전용입니다. 파일의 최신 버전 ID입니다. 이 기능은 현재 Google Drive의 바이너리 콘텐츠가 있는 파일에만 사용할 수 있습니다.
sharingUser
object (User [https://developers.google.com/workspace/drive/api/reference/rest/v3/User?hl=ko])
출력 전용입니다. 요청 사용자에게 파일을 공유한 사용자입니다(해당하는 경우).
webViewLink
string
출력 전용입니다. 브라우저에서 관련 Google 편집기 또는 뷰어로 파일을 여는 링크입니다.
webContentLink
string
출력 전용입니다. 브라우저에서 파일의 콘텐츠를 다운로드하는 링크입니다. Google Drive의 바이너리 콘텐츠가 있는 파일에만 사용할 수 있습니다.
size
string (int64 [https://developers.google.com/discovery/v1/type-format?hl=ko] format)
출력 전용입니다. blob 및 퍼스트 파티 편집기 파일의 크기(바이트)입니다. 바로가기 및 폴더와 같이 크기가 없는 파일은 채워지지 않습니다.
viewersCanCopyContent
(deprecated)
boolean
이 항목은 지원 중단되었습니다.
지원 중단됨: 대신 copyRequiresWriterPermission를 사용하세요.
hasThumbnail
boolean
출력 전용입니다. 이 파일에 썸네일이 있는지 여부입니다. 이는 요청 앱이 썸네일에 액세스할 수 있는지 여부를 나타내지 않습니다. 액세스를 확인하려면 thumbnailLink 필드가 있는지 확인합니다.
folderColorRgb
string
폴더 또는 폴더 바로가기의 색상으로, RGB 16진수 문자열로 지정됩니다. 지원되는 색상은 About 리소스의 folderColorPalette 필드에 게시됩니다.
지원되지 않는 색상을 지정하면 팔레트에서 가장 가까운 색상이 대신 사용됩니다.
id
string
파일의 ID입니다.
name
string
파일 이름입니다. 폴더 내에서 반드시 고유하지는 않습니다. 공유 드라이브의 최상위 폴더, 내 드라이브 루트 폴더, 애플리케이션 데이터 폴더와 같은 변경 불가능한 항목의 이름은 일정합니다.
description
string
파일에 대한 간단한 설명입니다.
starred
boolean
사용자가 파일에 별표표시를 했는지 여부입니다.
trashed
boolean
파일이 명시적으로 또는 휴지통에 있는 상위 폴더에서 휴지통으로 이동되었는지 여부입니다. 소유자만 파일을 휴지통으로 이동할 수 있으며 다른 사용자는 소유자의 휴지통에 있는 파일을 볼 수 없습니다.
explicitlyTrashed
boolean
출력 전용입니다. 파일이 상위 폴더에서 재귀적으로 휴지통으로 이동된 것이 아니라 명시적으로 휴지통으로 이동되었는지 여부입니다.
createdTime
string
파일이 생성된 시간입니다 (RFC 3339 날짜-시간).
modifiedTime
string
파일이 마지막으로 수정된 시간 (RFC 3339 날짜-시간)입니다.
modifiedTime를 설정하면 사용자의 modifiedByMeTime도 업데이트됩니다.
modifiedByMeTime
string
출력 전용입니다. 사용자가 파일을 마지막으로 수정한 시간입니다 (RFC 3339 날짜-시간).
viewedByMeTime
string
사용자가 파일을 마지막으로 본 시간입니다 (RFC 3339 날짜 및 시간).
sharedWithMeTime
string
출력 전용입니다. 파일이 사용자와 공유된 시간입니다(해당하는 경우)(RFC 3339 날짜-시간).
quotaBytesUsed
string (int64 [https://developers.google.com/discovery/v1/type-format?hl=ko] format)
출력 전용입니다. 파일에서 사용된 저장용량 할당량 바이트 수입니다. 여기에는 헤드 버전과 keepForever가 사용 설정된 이전 버전이 포함됩니다.
version
string (int64 [https://developers.google.com/discovery/v1/type-format?hl=ko] format)
출력 전용입니다. 파일의 단조 증가 버전 번호입니다. 여기에는 사용자에게 표시되지 않는 서버를 포함하여 서버의 파일에 대한 모든 변경사항이 반영됩니다.

출력 전용입니다. LinkShare 관련 세부정보입니다. 클라이언트가 이 항목을 참조하는 데 사용하는 링크 URL에 관한 세부정보가 포함되어 있습니다.
linkShareMetadata.securityUpdateEligible
boolean
출력 전용입니다. 파일에 보안 업데이트가 적용될 수 있는지 여부입니다.
linkShareMetadata.securityUpdateEnabled
boolean
출력 전용입니다. 이 파일에 보안 업데이트가 사용 설정되어 있는지 여부입니다.
labelInfo
object
출력 전용입니다. 파일의 라벨 개요입니다.
labelInfo.labels[]
object (Label [https://developers.google.com/workspace/drive/api/reference/rest/v3/Label?hl=ko])
출력 전용입니다. includeLabels 매개변수의 라벨 ID에 의해 요청된 파일의 라벨 집합입니다. 기본적으로 라벨이 반환되지 않습니다.
sha1Checksum
string
출력 전용입니다. 이 파일과 연결된 SHA1 체크섬입니다(사용 가능한 경우). 이 필드는 Google Drive에 콘텐츠가 저장된 파일에만 채워지며, Docs 편집기 또는 바로가기 파일에는 채워지지 않습니다.
sha256Checksum
string
출력 전용입니다. 이 파일과 연결된 SHA256 체크섬입니다(사용 가능한 경우). 이 필드는 Google Drive에 콘텐츠가 저장된 파일에만 채워지며, Docs 편집기 또는 바로가기 파일에는 채워지지 않습니다.
inheritedPermissionsDisabled
boolean
이 파일에 상속된 권한이 사용 중지되었는지 여부입니다. 상속된 권한은 기본적으로 사용 설정되어 있습니다.
downloadRestrictions
object (DownloadRestrictionsMetadata [https://developers.google.com/workspace/drive/api/reference/rest/v3/files?hl=ko#File.DownloadRestrictionsMetadata])
파일에 적용된 다운로드 제한사항입니다.
ContentRestriction
파일 콘텐츠 액세스 제한입니다.
JSON 표현
{
  "readOnly": boolean,
  "reason": string,
  "type": string,
  "restrictingUser": {
    object (User [https://developers.google.com/workspace/drive/api/reference/rest/v3/User?hl=ko])
  },
  "restrictionTime": string,
  "ownerRestricted": boolean,
  "systemRestricted": boolean
}
필드
readOnly
boolean
파일의 콘텐츠가 읽기 전용인지 여부입니다. 파일이 읽기 전용인 경우 파일의 새 버전이 추가되지 않고, 댓글이 추가되거나 수정되지 않으며, 파일의 제목이 수정되지 않을 수 있습니다.
reason
string
파일의 콘텐츠가 제한된 이유입니다. 이는 readOnly=true도 설정하는 요청에서만 변경할 수 있습니다.
type
string
출력 전용입니다. 콘텐츠 제한의 유형입니다. 현재 가능한 유일한 값은 globalContentRestriction입니다.
restrictingUser
object (User [https://developers.google.com/workspace/drive/api/reference/rest/v3/User?hl=ko])
출력 전용입니다. 콘텐츠 제한을 설정한 사용자입니다. readOnly가 true인 경우에만 채워집니다.
restrictionTime
string
출력 전용입니다. 콘텐츠 제한이 설정된 시간입니다 (형식화된 RFC 3339 타임스탬프). readOnly가 true인 경우에만 채워집니다.
ownerRestricted
boolean
콘텐츠 제한을 파일 소유자만 수정하거나 삭제할 수 있는지 여부입니다. 공유 드라이브의 경우 organizer 기능이 있는 모든 사용자가 이 콘텐츠 제한을 수정하거나 삭제할 수 있습니다.
systemRestricted
boolean
출력 전용입니다. 콘텐츠 제한이 시스템에 의해 적용되었는지 여부입니다(예: 전자 서명으로 인해). 사용자는 시스템 제한 콘텐츠 제한을 수정하거나 삭제할 수 없습니다.
DownloadRestrictionsMetadata
파일에 적용된 다운로드 제한사항입니다.
JSON 표현
{
  "itemDownloadRestriction": {
    object (DownloadRestriction [https://developers.google.com/workspace/drive/api/reference/rest/v3/files?hl=ko#File.DownloadRestriction])
  },
  "effectiveDownloadRestrictionWithContext": {
    object (DownloadRestriction [https://developers.google.com/workspace/drive/api/reference/rest/v3/files?hl=ko#File.DownloadRestriction])
  }
}
필드
itemDownloadRestriction
object (DownloadRestriction [https://developers.google.com/workspace/drive/api/reference/rest/v3/files?hl=ko#File.DownloadRestriction])
소유자 또는 주최자가 직접 적용한 파일의 다운로드 제한입니다. 공유 드라이브 설정 또는 DLP 규칙은 고려하지 않습니다.
effectiveDownloadRestrictionWithContext
object (DownloadRestriction [https://developers.google.com/workspace/drive/api/reference/rest/v3/files?hl=ko#File.DownloadRestriction])
출력 전용입니다. 이 파일에 적용된 유효한 다운로드 제한입니다. 모든 제한 설정과 DLP 규칙이 고려됩니다.
DownloadRestriction
파일의 복사 및 다운로드에 대한 제한입니다.
JSON 표현
{
  "restrictedForReaders": boolean,
  "restrictedForWriters": boolean
}
필드
restrictedForReaders
boolean
독자의 다운로드 및 복사가 제한되는지 여부입니다.
restrictedForWriters
boolean
작성자의 다운로드 및 복사가 제한되는지 여부입니다. true인 경우 독자의 다운로드도 제한됩니다.
메서드
copy [https://developers.google.com/workspace/drive/api/reference/rest/v3/files/copy?hl=ko]
파일의 사본을 만들고 패치 의미 체계를 사용하여 요청된 업데이트를 적용합니다.
create [https://developers.google.com/workspace/drive/api/reference/rest/v3/files/create?hl=ko]
새 파일을 만듭니다.
delete [https://developers.google.com/workspace/drive/api/reference/rest/v3/files/delete?hl=ko]
사용자가 소유한 파일을 휴지통으로 이동하지 않고 영구적으로 삭제합니다.
download [https://developers.google.com/workspace/drive/api/reference/rest/v3/files/download?hl=ko]
파일의 콘텐츠를 다운로드합니다.
emptyTrash [https://developers.google.com/workspace/drive/api/reference/rest/v3/files/emptyTrash?hl=ko]
사용자의 휴지통에 있는 모든 파일을 영구적으로 삭제합니다.
export [https://developers.google.com/workspace/drive/api/reference/rest/v3/files/export?hl=ko]
Google Workspace 문서를 요청된 MIME 유형으로 내보내고 내보낸 바이트 콘텐츠를 반환합니다.
generateIds [https://developers.google.com/workspace/drive/api/reference/rest/v3/files/generateIds?hl=ko]
생성 또는 복사 요청에 제공할 수 있는 파일 ID 집합을 생성합니다.
get [https://developers.google.com/workspace/drive/api/reference/rest/v3/files/get?hl=ko]
ID로 파일의 메타데이터 또는 콘텐츠를 가져옵니다.
list [https://developers.google.com/workspace/drive/api/reference/rest/v3/files/list?hl=ko]
사용자의 파일을 나열합니다.
listLabels [https://developers.google.com/workspace/drive/api/reference/rest/v3/files/listLabels?hl=ko]
파일의 라벨을 나열합니다.
modifyLabels [https://developers.google.com/workspace/drive/api/reference/rest/v3/files/modifyLabels?hl=ko]
파일에 적용된 라벨 집합을 수정합니다.
update [https://developers.google.com/workspace/drive/api/reference/rest/v3/files/update?hl=ko]
파일의 메타데이터 또는 콘텐츠를 업데이트합니다.
watch [https://developers.google.com/workspace/drive/api/reference/rest/v3/files/watch?hl=ko]
파일의 변경사항을 구독합니다.
