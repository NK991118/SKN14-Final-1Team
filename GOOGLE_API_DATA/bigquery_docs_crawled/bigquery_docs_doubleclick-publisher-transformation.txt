Google Ad Manager 보고서 변환
bookmark_border
Google Ad Manager(이전 DoubleClick for Publishers) 데이터 전송 파일이 BigQuery로 전송될 때는 다음 BigQuery 테이블 및 뷰로 변환됩니다.
BigQuery에서 테이블과 뷰를 볼 때는 network_code 값이 Google Ad Manager 네트워크 코드에 해당합니다.
