이 튜토리얼에서는 scikit-learn [https://scikit-learn.org/stable/index.html]으로 학습된 Open Neural Network Exchange [https://onnx.ai/](ONNX) 모델을 가져오는 방법을 보여줍니다. 모델을 BigQuery 데이터 세트로 가져와 SQL 쿼리를 사용하여 예측하는 데 사용합니다.
ONNX는 모든 머신러닝(ML) 프레임워크를 나타내도록 설계된 균일한 형식을 제공합니다. ONNX에 대한 BigQuery ML 지원을 통해 다음을 수행할 수 있습니다.
원하는 프레임워크를 사용하여 모델을 학습시키기
ONNX 모델 형식으로 모델 변환
ONNX 모델을 BigQuery로 가져와 BigQuery ML을 사용하여 예측 수행
목표
scikit-learn [https://scikit-learn.org/stable/index.html]을 사용하여 모델 생성 및 학습
sklearn-onnx [https://onnx.ai/sklearn-onnx/]를 사용하여 ONNX 형식으로 모델 변환 [https://github.com/onnx/tutorials#converting-to-onnx-format]
CREATE MODEL 문을 사용하여 BigQuery로 ONNX 모델 가져오기
ML.PREDICT 함수를 사용하여 가져온 ONNX 모델로 예측 수행
