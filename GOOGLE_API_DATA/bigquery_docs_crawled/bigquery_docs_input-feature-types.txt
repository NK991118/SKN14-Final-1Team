밀집 벡터 입력
BigQuery ML은 ARRAY<numerical>를 모델 학습 중 밀집 벡터 입력으로 지원합니다. 임베딩 특성은 특수한 유형의 밀집 벡터입니다. 자세한 내용은 ML.GENERATE_EMBEDDING 함수 [https://cloud.google.com/bigquery/docs/reference/standard-sql/bigqueryml-syntax-generate-embedding?hl=ko]를 참조하세요.
희소 입력
BigQuery ML은 ARRAY<STRUCT>를 모델 학습 중 희소 입력으로 지원합니다. 각 구조체에는 해당 0 기반 색인을 나타내는 INT64 값과 해당 값을 나타내는 숫자 유형 [https://cloud.google.com/bigquery/docs/reference/standard-sql/data-types?hl=ko#numeric_types]이 포함됩니다.
다음은 정수 배열 [0,1,0,0,0,0,1]의 희소 텐서 입력 예시입니다.
ARRAY<STRUCT<k INT64, v INT64>>[(1, 1), (6, 1)] AS f1
