분류 개요
bookmark_border
머신러닝의 일반적인 사용 사례는 유사한 라벨이 지정된 데이터로 학습된 모델을 사용하여 새 데이터를 분류하는 것입니다. 예를 들어 이메일이 스팸인지 또는 고객 제품 리뷰가 긍정적인지, 부정적인지, 중립적인지 예측할 수 있습니다.
ML.PREDICT 함수 [https://cloud.google.com/bigquery/docs/reference/standard-sql/bigqueryml-syntax-predict?hl=ko]와 함께 다음 모델을 사용하여 분류를 실행할 수 있습니다.
로지스틱 회귀 모델 [https://cloud.google.com/bigquery/docs/reference/standard-sql/bigqueryml-syntax-create-glm?hl=ko]: MODEL_TYPE 옵션을 LOGISTIC_REG로 설정하여 로지스틱 회귀 [https://developers.google.com/machine-learning/crash-course/logistic-regression?hl=ko]를 사용합니다.
부스티드 트리 모델 [https://cloud.google.com/bigquery/docs/reference/standard-sql/bigqueryml-syntax-create-boosted-tree?hl=ko]: MODEL_TYPE 옵션을 BOOSTED_TREE_CLASSIFIER로 설정하여 그래디언트 부스팅 결정 트리 [https://developers.google.com/machine-learning/decision-forests/intro-to-gbdt?hl=ko]를 사용합니다.
랜덤 포레스트 모델 [https://cloud.google.com/bigquery/docs/reference/standard-sql/bigqueryml-syntax-create-random-forest?hl=ko]: MODEL_TYPE 옵션을 RANDOM_FOREST_CLASSIFIER로 설정하여 랜덤 포레스트 [https://developers.google.com/machine-learning/decision-forests/intro-to-decision-forests?hl=ko]를 사용합니다.
심층신경망(DNN) 모델 [https://cloud.google.com/bigquery/docs/reference/standard-sql/bigqueryml-syntax-create-dnn-models?hl=ko]: MODEL_TYPE 옵션을 DNN_CLASSIFIER로 설정하여 신경망 [https://developers.google.com/machine-learning/crash-course/neural-networks?hl=ko]을 사용합니다.
와이드 앤 딥 모델 [https://cloud.google.com/bigquery/docs/reference/standard-sql/bigqueryml-syntax-create-wnd-models?hl=ko]: MODEL_TYPE 옵션을 DNN_LINEAR_COMBINED_CLASSIFIER로 설정하여 와이드 앤 딥 러닝 [https://dl.acm.org/doi/10.1145/2988450.2988454]을 사용합니다.
AutoML 모델 [https://cloud.google.com/bigquery/docs/reference/standard-sql/bigqueryml-syntax-create-automl?hl=ko]: MODEL_TYPE 옵션을 AUTOML_CLASSIFIER로 설정하여 AutoML 분류 모델 [https://cloud.google.com/vertex-ai/docs/tabular-data/classification-regression/overview?hl=ko]을 사용합니다.
추천 지식
CREATE MODEL 문의 기본 설정과 ML.PREDICT 함수를 사용하면 ML 지식이 많지 않더라도 분류 모델을 만들고 사용할 수 있습니다. 하지만 ML 개발에 관한 기본 지식을 갖추면 데이터와 모델을 모두 최적화하여 더 나은 결과를 얻을 수 있습니다. 다음 리소스를 사용하여 ML 기법과 프로세스에 익숙해지는 것이 좋습니다.
