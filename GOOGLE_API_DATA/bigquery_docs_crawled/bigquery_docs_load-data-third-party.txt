타사 애플리케이션을 사용하여 데이터 로드
bookmark_border
일부 타사 애플리케이션과 서비스에서는 데이터를 BigQuery에 수집할 수 있는 커넥터를 제공합니다. 예를 들어 Informatica [https://cloud.google.com/bigquery/docs/bigquery-ready-partners?hl=ko#informatica]를 사용하면 외부 소스의 데이터를 BigQuery 스토리지에 로드할 수 있습니다.
Informatica를 사용하여 데이터 로드
Informatica Data Loader를 사용하여 데이터를 로드하려면 다음 단계를 수행합니다.
Google Cloud 콘솔에서 BigQuery 페이지로 이동합니다.
BigQuery로 이동 [https://console.cloud.google.com/bigquery?hl=ko]
탐색기 창에서 add 추가를 클릭합니다.
데이터 추가 대화상자가 열립니다.
예를 들어 Amazon S3 또는 Google Cloud AlloyDB와 같은 데이터 소스를 클릭합니다.
Informatica 데이터 로더 솔루션 카드를 클릭합니다.
You're leaving Google(Google 외부로 이동) 대화상자의 약관에 동의하면 Confirm(확인)을 클릭합니다.
Informatica 로그인 페이지에서 Informatica 계정에 로그인합니다.
BigQuery에 데이터를 로드하려면 Informatica 계정의 단계를 수행합니다.
Fivetran을 사용하여 데이터 로드
Fivetran 데이터 파이프라인을 사용하여 데이터를 로드하려면 다음 단계를 수행합니다.
Google Cloud 콘솔에서 BigQuery 페이지로 이동합니다.
BigQuery로 이동 [https://console.cloud.google.com/bigquery?hl=ko]
탐색기 창에서 add 추가를 클릭합니다.
데이터 추가 대화상자가 열립니다.
예를 들어 Amazon S3 또는 Google Cloud AlloyDB와 같은 데이터 소스를 클릭합니다.
Fivetran Data Pipelines 솔루션 카드를 클릭합니다.
You're leaving Google(Google 외부로 이동) 대화상자의 약관에 동의하면 Confirm(확인)을 클릭합니다.
Fivetran 로그인 페이지에서 Fivetran 계정에 로그인합니다.
BigQuery에 데이터를 로드하려면 Fivetran 계정의 단계를 수행합니다.
