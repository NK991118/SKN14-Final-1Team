BigQuery Data Transfer Service란?
bookmark_border
BigQuery Data Transfer Service는 정해진 일정에 따라 관리되는 방식으로 데이터를 BigQuery [https://cloud.google.com/bigquery/docs/introduction?hl=ko]로 자동 이동합니다. 그러면 분석팀에서 코드를 단 한 줄도 작성하지 않고 BigQuery 데이터 웨어하우스의 기반을 마련할 수 있습니다.
다음을 사용하여 BigQuery Data Transfer Service에 액세스할 수 있습니다.
Google Cloud console [https://cloud.google.com/bigquery/docs/bigquery-web-ui?hl=ko]
bq 명령줄 도구 [https://cloud.google.com/bigquery/docs/reference/bq-cli-reference?hl=ko]
BigQuery Data Transfer Service API [https://cloud.google.com/bigquery-transfer/docs/reference/datatransfer/rest?hl=ko]
데이터 전송을 구성하면 BigQuery Data Transfer Service가 정기적으로 데이터를 BigQuery로 자동 로드합니다. 중단이나 불일치 문제를 해결하기 위해 데이터 백필을 시작할 수도 있습니다. BigQuery Data Transfer Service를 사용하여 BigQuery에서 다른 곳으로 데이터를 전송할 수 없습니다.
BigQuery Data Transfer Service는 BigQuery에 데이터를 로드하는 것 외에도 데이터 세트 사본 [https://cloud.google.com/bigquery/docs/copying-datasets?hl=ko]과 예약된 쿼리 [https://cloud.google.com/bigquery/docs/scheduling-queries?hl=ko]의 두 가지 BigQuery 작업에 사용됩니다.
참고: BigQuery Data Transfer Service와 관련된 공지사항을 받으려면 BigQuery DTS 공지사항 그룹 [https://groups.google.com/g/bigquery-dts-announcements?hl=ko]을 구독하세요.

지원되는 리전
BigQuery와 마찬가지로 BigQuery Data Transfer Service는 여러 단일 리전을 사용할 수 있는 멀티 리전 리소스 [https://cloud.google.com/docs/geography-and-regions?hl=ko#regional_resources]입니다.
BigQuery 데이터 세트의 지역은 BigQuery Data Transfer Service로 전송되는 데이터를 저장하기 위해 대상 데이터 세트를 만들 [https://cloud.google.com/bigquery/docs/datasets?hl=ko] 때 지정됩니다. 전송을 설정할 때 전송 구성은 대상 데이터세트와 동일한 위치에 설정됩니다. BigQuery Data Transfer Service는 대상 데이터 세트와 동일한 위치에서 데이터를 처리하고 스테이징합니다.
BigQuery로 전송할 데이터에는 리전이 있을 수도 있습니다. 대부분의 경우 데이터가 저장되는 리전과 BigQuery에서 대상 데이터 세트의 위치는 관련성이 없습니다. 다른 유형의 전송에서는 데이터 세트와 소스 데이터가 동일한 리전 또는 호환되는 리전의 같은 위치에 배치되어야 합니다.
BigQuery Data Transfer Service의 전송 및 리전 호환성에 대한 자세한 내용은 데이터 세트 위치 및 전송 [https://cloud.google.com/bigquery/docs/dts-locations?hl=ko]을 참조하세요. BigQuery에서 지원되는 리전은 데이터 세트 위치 [https://cloud.google.com/bigquery/docs/locations?hl=ko#supported_locations]를 참조하세요.
가격 책정
BigQuery Data Transfer Service 가격에 대한 자세한 내용은 가격 책정 [https://cloud.google.com/bigquery/pricing?hl=ko] 페이지를 참조하세요.
데이터가 BigQuery로 전송되면 표준 BigQuery 스토리지 [https://cloud.google.com/bigquery/pricing?hl=ko#storage]와 쿼리 [https://cloud.google.com/bigquery/pricing?hl=ko#queries] 가격 책정이 적용됩니다.
할당량
BigQuery Data Transfer Service 할당량에 대한 자세한 내용은 할당량 및 한도 [https://cloud.google.com/bigquery/quotas?hl=ko] 페이지를 참조하세요.
