프리뷰
이 제품 또는 기능에는 서비스별 약관 [https://cloud.google.com/terms/service-terms?hl=ko#1]의 일반 서비스 약관 섹션에 있는 'GA 이전 제공 서비스 약관'이 적용됩니다. GA 이전 제품 및 기능은 '있는 그대로' 제공되며 지원이 제한될 수 있습니다. 자세한 내용은 출시 단계 설명 [https://cloud.google.com/products?hl=ko#product-launch-stages]을 참조하세요.
참고: 이 미리보기 기능과 관련된 의견을 제공하거나 궁금한 점이 있으면 bigquery-repositories-feedback@google.com [mailto:%20bigquery-repositories-feedback@google.com]으로 문의하세요.
이 문서는 BigQuery의 작업공간 개념을 이해하는 데 도움이 됩니다. 저장소 [https://cloud.google.com/bigquery/docs/repository-intro?hl=ko] 내의 워크스페이스를 사용하여 저장소에 저장된 코드를 수정할 수 있습니다. 저장소는 Git을 사용하여 변경사항을 기록하고 파일 버전을 관리하여 파일의 버전 제어를 실행합니다.
BigQuery 페이지에서 작업공간은 연결된 저장소 아래에 알파벳순으로 표시됩니다. 저장소가 탐색기 창의 저장소 폴더에 표시됩니다.

기타 유형의 파일
자세한 내용은 작업공간에서 파일 작업하기 [https://cloud.google.com/bigquery/docs/workspaces?hl=ko#work_with_code_assets_in_a_workspace]를 참고하세요.
Git 통합
작업공간에서 파일을 만들고 수정할 때 변경사항을 커밋하고 변경사항을 저장소에 푸시하는 등의 Git 작업을 실행할 수 있습니다. 자세한 내용은 파일에 버전 관리 사용 [https://cloud.google.com/bigquery/docs/workspaces?hl=ko#use_version_control_with_a_file]을 참고하세요.
위치
각 워크스페이스는 워크스페이스가 포함된 저장소와 동일한 위치 [https://cloud.google.com/bigquery/docs/repository-intro?hl=ko#locations]를 사용합니다.
할당량
BigQuery 워크스페이스 사용에는 Dataform 할당량 [https://cloud.google.com/dataform/docs/quotas?hl=ko#quotas]이 적용됩니다.
가격 책정
작업공간 생성, 업데이트, 삭제 또는 작업공간의 파일 저장에 대해서는 요금이 청구되지 않습니다.
BigQuery 가격에 대한 자세한 내용은 가격 [https://cloud.google.com/bigquery/pricing?hl=ko]을 참조하세요.
