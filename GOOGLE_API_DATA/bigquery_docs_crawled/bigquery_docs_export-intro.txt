데이터 내보내기 소개
bookmark_border
이 문서에서는 BigQuery에서 데이터를 내보내는 다양한 방법을 설명합니다.
데이터 통합에 관한 자세한 내용은 데이터 로드, 변환, 내보내기 소개 [https://cloud.google.com/bigquery/docs/load-transform-export-intro?hl=ko]를 참조하세요.
쿼리 결과 내보내기
쿼리 결과를 로컬 파일(CSV 또는 JSON 파일), Google Drive 또는 Google Sheets로 내보낼 수 있습니다. 자세한 내용은 쿼리 결과를 파일로 내보내기 [https://cloud.google.com/bigquery/docs/export-file?hl=ko]를 참조하세요.
테이블 내보내기
다음 데이터 형식으로 BigQuery 테이블을 내보낼 수 있습니다.
데이터 형식 지원되는 압축 유형 지원되는 내보내기 방법
CSV GZIP Cloud Storage로 내보내기 [https://cloud.google.com/bigquery/docs/exporting-data?hl=ko]
JSON GZIP Cloud Storage로 내보내기 [https://cloud.google.com/bigquery/docs/exporting-data?hl=ko]
Dataflow를 사용하여 BigQuery에서 읽기 [https://cloud.google.com/dataflow/docs/guides/read-from-bigquery?hl=ko]
Avro DEFLATE, SNAPPY Cloud Storage로 내보내기 [https://cloud.google.com/bigquery/docs/exporting-data?hl=ko]
Dataflow를 사용하여 BigQuery에서 읽기 [https://cloud.google.com/dataflow/docs/guides/read-from-bigquery?hl=ko]
Parquet GZIP, SNAPPY, ZSTD Cloud Storage로 내보내기 [https://cloud.google.com/bigquery/docs/exporting-data?hl=ko]
객체 유형 안전성이 필요한 중첩된 데이터 구조를 사용하거나 더욱 광범위한 언어 지원이 필요한 경우 BigQuery 테이블을 Protobuf 열로 내보낼 [https://cloud.google.com/bigquery/docs/protobuf-export?hl=ko] 수도 있습니다.
BigQuery 코드 애셋 내보내기
저장된 쿼리 [https://cloud.google.com/bigquery/docs/saved-queries-introduction?hl=ko] 또는 노트북 [https://cloud.google.com/bigquery/docs/notebooks-introduction?hl=ko]과 같은 BigQuery Studio [https://cloud.google.com/bigquery/docs/query-overview?hl=ko#bigquery-studio] 코드 애셋을 다운로드하여 애셋 로컬 사본을 유지할 수 있습니다. BigQuery 코드 애셋을 다운로드하는 방법에 대한 자세한 내용은 다음을 참조하세요.
저장된 쿼리 다운로드 [https://cloud.google.com/bigquery/docs/manage-saved-queries?hl=ko#download_saved_queries]
노트북 다운로드 [https://cloud.google.com/bigquery/docs/manage-notebooks?hl=ko#download_a_notebook]
역방향 ETL을 사용하여 내보내기
역방향 ETL(RETL) 워크플로(미리보기 [https://cloud.google.com/products?hl=ko#product-launch-stages])를 설정하여 BigQuery의 데이터를 다음 데이터베이스로 이동할 수 있습니다.
Bigtable로 내보내기 [https://cloud.google.com/bigquery/docs/export-to-bigtable?hl=ko]
Spanner로 내보내기 [https://cloud.google.com/bigquery/docs/export-to-spanner?hl=ko]
Pub/Sub로 내보내기 [https://cloud.google.com/bigquery/docs/export-to-pubsub?hl=ko]
