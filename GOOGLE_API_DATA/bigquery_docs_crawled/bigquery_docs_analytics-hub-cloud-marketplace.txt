Google Cloud Marketplace의 등록정보 상용화
bookmark_border
Analytics Hub 게시자는 Google Cloud Marketplace [https://cloud.google.com/marketplace?hl=ko]와의 Sharing 통합을 사용하여 Cloud Marketplace에 자체 데이터 제품을 등록하여 수익을 창출할 수 있습니다. 게시자-구독자 모델을 사용하면 모든 거래 및 구독을 관리할 필요 없이 고객과 대규모로 데이터 제품을 공유할 수 있습니다. 제공하는 데이터 유형(예: BigQuery 데이터 세트), 구독 가격(유료, 무료, 무료 체험), 기간 등 데이터 제품의 측면을 구성할 수 있습니다.
Analytics Hub 구독자는 이 통합을 사용하여 다양한 Google 및 서드 파티 데이터 제품과 상용 데이터 세트를 탐색하고 사용할 수 있습니다.
계속하기 전에 Sharing 데이터 교환 [https://cloud.google.com/bigquery/docs/analytics-hub-manage-exchanges?hl=ko]을 숙지해야 합니다.
시작하기 전에
사용자에게 이 문서의 각 작업을 수행하는 데 필요한 권한을 부여하는 Identity and Access Management(IAM) 역할을 부여 [https://cloud.google.com/bigquery/docs/analytics-hub-cloud-marketplace?hl=ko#required-roles]합니다.
Analytics Hub API를 사용 설정 [https://cloud.google.com/bigquery/docs/analytics-hub-cloud-marketplace?hl=ko#enable-api]합니다.
필요한 역할
Cloud Marketplace 통합 등록정보를 사용하는 데 필요한 권한을 얻으려면 관리자에게 다음 IAM 역할을 부여해 달라고 요청하세요.
Sharing 등록정보 만들기 및 관리:
Analytics Hub 관리자 [https://cloud.google.com/iam/docs/roles-permissions/analyticshub?hl=ko#analyticshub.admin](roles/analyticshub.admin)
BigQuery 데이터 소유자 [https://cloud.google.com/iam/docs/roles-permissions/bigquery?hl=ko#bigquery.dataOwner](roles/bigquery.dataOwner)
Service Management 관리자 [https://cloud.google.com/iam/docs/roles-permissions/servicemanagement?hl=ko#servicemanagement.admin](roles/servicemanagement.admin)
Cloud Marketplace에서 데이터 제품 등록정보를 만들고 관리하기: Commerce Producer 관리자 [https://cloud.google.com/iam/docs/roles-permissions/commerceproducer?hl=ko#commerceproducer.admin](roles/commerceproducer.admin)
Cloud Marketplace에서 유료 Sharing 등록정보 구독하기:
결제 계정 관리자 [https://cloud.google.com/iam/docs/roles-permissions/billing?hl=ko#billing.admin](roles/billing.admin)
Analytics Hub 뷰어 [https://cloud.google.com/iam/docs/roles-permissions/analyticshub?hl=ko#analyticshub.viewer](roles/analyticshub.viewer)
역할 부여에 대한 자세한 내용은 프로젝트, 폴더, 조직에 대한 액세스 관리 [https://cloud.google.com/iam/docs/granting-changing-revoking-access?hl=ko]를 참조하세요.
커스텀 역할 [https://cloud.google.com/iam/docs/creating-custom-roles?hl=ko]이나 다른 사전 정의된 역할 [https://cloud.google.com/iam/docs/roles-overview?hl=ko#predefined]을 통해 필요한 권한을 얻을 수도 있습니다.
Analytics Hub API 사용 설정
Analytics Hub API를 사용 설정하려면 다음 옵션 중 하나를 선택합니다.
--- 탭: 콘솔 [https://cloud.google.com/bigquery/docs/analytics-hub-cloud-marketplace?hl=ko#%EC%BD%98%EC%86%94] ---
Analytics Hub API 페이지로 이동하여 Google Cloud 프로젝트에 Analytics Hub API를 사용 설정합니다.

API 사용 설정하기 [https://console.cloud.google.com/apis/library/analyticshub.googleapis.com?hl=ko]

--- 탭: gcloud [https://cloud.google.com/bigquery/docs/analytics-hub-cloud-marketplace?hl=ko#gcloud] ---
gcloud services enable 명령어를 실행합니다 [https://cloud.google.com/sdk/gcloud/reference/services/enable?hl=ko].

gcloud services enable analyticshub.googleapis.com
Analytics Hub API를 사용 설정하고 나면Google Cloud 콘솔에서 Sharing(Analytics Hub) 페이지 [https://console.cloud.google.com/bigquery/analytics-hub?hl=ko]에 액세스할 수 있습니다.
제한사항
Cloud Marketplace 통합 등록정보에는 다음과 같은 제한사항이 있습니다.
모든 Sharing 제한사항 [https://cloud.google.com/bigquery/docs/analytics-hub-introduction?hl=ko#limitations]이 적용됩니다.
게시자와 구독자가 지원되는 Cloud Marketplace 대리 관할권 [https://cloud.google.com/terms/marketplace-agency-jurisdictions?hl=ko]에 있어야 합니다.
Cloud Marketplace 통합 등록정보는 Data Catalog [https://cloud.google.com/bigquery/docs/data-catalog?hl=ko](지원 중단됨) 및 Dataplex Universal Catalog [https://cloud.google.com/dataplex/docs/catalog-overview?hl=ko]에서 색인이 생성되지만 해당 리소스 유형을 구체적으로 필터링할 수는 없습니다.
Cloud Marketplace 통합 등록정보의 결제 사용량 측정항목은 제공업체 사용량 측정항목이나 INFORMATION_SCHEMA 뷰 [https://cloud.google.com/bigquery/docs/information-schema-intro?hl=ko]에 캡처되지 않습니다.
데이터 클린룸 및 Pub/Sub 주제는 Cloud Marketplace 통합에서 지원되지 않습니다.
아키텍처 및 용어
다음 다이어그램은 상용 등록정보에 대한 Cloud Marketplace와 BigQuery Sharing 간의 상호작용을 보여줍니다.
Cloud Marketplace의 데이터 제품
Cloud Marketplace 데이터 제품 등록정보를 만들려면 Sharing 등록정보를 선택하고, 가격 책정 모델을 선택하고, 검토를 위해 Cloud Marketplace에 제품을 제출합니다.
Sharing의 Cloud Marketplace 통합 등록정보
Cloud Marketplace 데이터 제품 등록정보가 승인되고 게시되면 Sharing 등록정보가 Cloud Marketplace 통합 등록정보가 되어 Sharing과 Cloud Marketplace 간의 통합 링크가 생성되고 등록정보를 구매할 수 있게 됩니다. 이 유형의 Sharing 등록정보는 공유 데이터 세트를 지원합니다.
연결된 리소스
Cloud Marketplace 통합 등록정보를 구독하면 구독자 프로젝트에 연결된 리소스가 생성됩니다. 연결된 리소스에 대한 액세스는 활성 Cloud Marketplace 주문에 의해 관리됩니다. 연결된 데이터 세트는 Cloud Marketplace 통합 등록정보에서 지원됩니다.
Cloud Marketplace 통합 등록정보 만들기
Sharing 등록정보를 만들고 Cloud Marketplace에 게시하려면 다음 안내를 따르세요.
Google Cloud 콘솔에서 Sharing(Analytics Hub) 페이지로 이동합니다.
Sharing(Analytics Hub)으로 이동 [https://console.cloud.google.com/bigquery/analytics-hub?hl=ko]
새 Sharing 데이터 교환 [https://cloud.google.com/bigquery/docs/analytics-hub-manage-exchanges?hl=ko#create-exchange]을 만듭니다. 또는 기존 데이터 교환을 선택하여 기존 구독을 유지할 수 있습니다.
데이터 교환에 등록정보 [https://cloud.google.com/bigquery/docs/analytics-hub-manage-listings?hl=ko#create_a_listing]를 만듭니다. 또는 기존 등록정보를 선택하여 기존 구독을 유지할 수 있습니다.
참고: 액세스 요청과 Cloud Marketplace 통합 흐름은 단일 Sharing 등록정보에서 모두 지원됩니다. 즉, 기존 구독을 중단하지 않고 기존 오프라인 상용 제품 등록정보에서 Cloud Marketplace 통합 제품 등록정보를 만들 수 있습니다.
데이터 교환 행에서 more_vert 작업 더보기 > Marketplace에 등록을 클릭합니다. Cloud Marketplace Producer Portal [https://console.cloud.google.com/producer-portal?hl=ko]로 리디렉션됩니다.
Cloud Marketplace Producer Portal의 안내에 따라 Sharing 등록정보를 데이터 제품 [https://cloud.google.com/marketplace/docs/partners/data?hl=ko]으로 온보딩합니다.
Sharing(Analytics Hub) 페이지로 다시 이동합니다. 데이터 교환 행에서 Marketplace 열에 게시되지 않음이라는 문구가 표시되면 데이터 제품이 생성되어 승인을 위해 제출되었음을 나타냅니다. 게시되지 않음 문구를 클릭하면 상태를 확인할 수 있는 Cloud Marketplace Producer Portal로 리디렉션됩니다.
승인되면 Marketplace 열에 게시됨이라는 단어가 표시됩니다. 게시됨이라는 단어를 클릭하면 Cloud Marketplace의 등록정보로 리디렉션됩니다.
추가 요구사항은 Google Cloud Marketplace에서 소프트웨어 제공 [https://cloud.google.com/marketplace/docs/partners/offer-products?hl=ko]을 참조하세요.
Cloud Marketplace 통합 등록정보 업데이트
Cloud Marketplace 통합 등록정보를 업데이트하는 방법은 데이터 교환에서 표준 등록정보를 업데이트 [https://cloud.google.com/bigquery/docs/analytics-hub-manage-listings?hl=ko#update_a_listing]할 때와 동일합니다. Cloud Marketplace Producer Portal [https://console.cloud.google.com/producer-portal?hl=ko]에서 데이터 제품 등록정보를 업데이트해야 할 수도 있으며, 이 경우 다시 검토 및 승인이 필요할 수 있습니다.
Cloud Marketplace 통합 등록정보의 구독 관리
Cloud Marketplace 통합 등록정보의 상용 구독은 Cloud Marketplace 주문으로 관리됩니다. 표준 등록정보와 동일한 절차 [https://cloud.google.com/bigquery/docs/analytics-hub-manage-exchanges?hl=ko#give_users_access_to_a_data_exchange]에 따라 구독자를 수동으로 추가하고 업데이트할 수는 있지만 연결된 Cloud Marketplace 거래는 이루어지지 않습니다.
표준 등록정보와 동일한 절차 [https://cloud.google.com/bigquery/docs/analytics-hub-manage-subscriptions?hl=ko#remove_a_subscription]에 따라 Marketplace 서비스 ID를 입력하고 경고 알림에 동의하면 구독을 수동으로 취소할 수도 있습니다. 하지만 상용 구독을 취소하면 고객에게 영향을 미치고 Cloud Marketplace 서비스 약관 [https://cloud.google.com/terms/marketplace/launcher?hl=ko]을 위반할 수 있습니다. 또한 구독을 취소해도 Sharing 또는 Cloud Marketplace에서 등록정보가 삭제되지 않습니다.
Cloud Marketplace 통합 등록정보 삭제
경고: Cloud Marketplace 통합 등록정보를 삭제한 후에는 실행취소할 수 없습니다. 목록을 삭제하면 고객에게 영향을 미치고 Cloud Marketplace 서비스 약관 [https://cloud.google.com/terms/marketplace/launcher?hl=ko]을 위반할 수 있습니다.
Sharing 및 Cloud Marketplace에서 등록정보를 삭제하려면 다음 안내를 따르세요.
데이터 교환과 동일한 절차 [https://cloud.google.com/bigquery/docs/analytics-hub-manage-subscriptions?hl=ko#remove_a_subscription]에 따라 Cloud Marketplace 통합 등록정보의 모든 상용 구독을 취소합니다. 활성 상용 구독이 있는 Cloud Marketplace 통합 등록정보는 삭제할 수 없습니다.
표준 절차에 따라 데이터 교환을 삭제 [https://cloud.google.com/bigquery/docs/analytics-hub-manage-exchanges?hl=ko#delete_a_data_exchange]합니다.
경고 알림이 표시되면 Marketplace 서비스 ID를 입력하여 동의한 다음 확인을 클릭합니다.
Cloud Marketplace 통합 등록정보 구독
Cloud Marketplace에서 Sharing 등록정보를 구독하려면 다음 안내를 따르세요.
Google Cloud 콘솔에서 Sharing(Analytics Hub) 페이지로 이동합니다.
Sharing(Analytics Hub)으로 이동 [https://console.cloud.google.com/bigquery/analytics-hub?hl=ko]
등록정보 검색을 클릭합니다.
구독하려는 등록정보를 검색합니다.
등록정보를 클릭합니다.
조직에서 이미 등록정보를 구매한 경우(구독 버튼 및 구매 날짜가 표시됨) 다음 안내를 따르세요.
구독을 클릭합니다.
프로젝트와 연결된 데이터 세트 이름을 지정합니다.
저장을 클릭합니다.
조직에서 등록정보를 구매하지 않은 경우(Marketplace에서 구매하기 버튼이 표시됨) 다음 안내를 따르세요.
Marketplace에서 구매하기를 클릭합니다.
Cloud Marketplace 주문 요약 페이지에서 구독 요금제, 구매 세부정보를 지정하고 약관에 동의하는 경우 약관에 동의합니다.
Sharing(Analytics Hub)에서 관리를 클릭합니다. Sharing(Analytics Hub)으로 다시 리디렉션됩니다.
몇 분 정도 기다린 후 Sharing 등록 페이지에서 구독을 클릭합니다.
프로젝트와 연결된 데이터 세트 이름을 지정합니다.
저장을 클릭합니다.
동일한 결제 계정을 사용하는 프로젝트도 등록정보를 구독할 수 있습니다.
가격 책정
표준 Sharing 가격 책정 [https://cloud.google.com/bigquery/docs/analytics-hub-introduction?hl=ko#pricing]이 적용됩니다. 또한 Cloud Marketplace 통합 등록정보에는 Cloud Marketplace 수익 공유 요구사항 [https://cloud.google.com/marketplace/docs/partners/get-started?hl=ko]이 적용됩니다. 데이터 제품 사용에 대해 게시자가 구독자에게 청구하는 방식에 관한 자세한 내용은 Cloud Marketplace 제품의 결제 관리 [https://cloud.google.com/marketplace/docs/manage-billing?hl=ko]를 참조하세요.
