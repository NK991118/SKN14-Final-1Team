bookmark_border

이전 Search Ads 360 Reporting API를 사용하는 Search Ads 360 보고서 전환을 보려면 Search Ads 360 보고서 전환(지원 중단됨) [https://cloud.google.com/bigquery/docs/sa360-transformation?hl=ko]을 참조하세요.

Search Ads 360 보고서의 열 세부정보
Search Ads 360 전송으로 생성되는 BigQuery 테이블은 다음 열(필드)로 구성됩니다.
Search Ads 360 테이블 이름: Account
Search Ads 360 API 리소스: customer [https://developers.google.com/search-ads/reporting/api/reference/fields/v0/customer?hl=ko]
Search Ads 360 필드 이름 설명
customer_account_level 고객의 계정 수준입니다(관리자, 하위 관리자, 보조 관리자 또는 서비스 계정).
customer_account_type 엔진 계정 유형입니다. 예: Google Ads, Microsoft Advertising, Yahoo Japan, Baidu, Facebook, Engine Track
customer_creation_time 이 고객이 생성된 타임스탬프입니다. 타임스탬프는 고객의 시간대로 표시되며 "yyyy-MM-dd HH:mm:ss" 형식입니다.
customer_currency_code 계정에 사용되는 통화입니다. ISO 4217 표준의 통화 코드 중 일부가 지원됩니다.
customer_descriptive_name 고객에 대해 선택적이고 고유하지 않은 설명이 포함된 이름입니다.
customer_manager_descriptive_name 관리자의 설명 이름입니다.
customer_sub_manager_descriptive_name 하위 관리자의 설명 이름입니다.
customer_associate_manager_descriptive_name 연결된 관리자의 설명 이름입니다.
customer_engine_id 외부 엔진 계정의 계정 ID입니다.
customer_id 고객의 ID입니다.
customer_manager_id 관리자의 고객 ID입니다.
customer_sub_manager_id 하위 관리자의 고객 ID입니다.
customer_associate_manager_id 연결된 관리자의 고객 ID입니다.
customer_last_modified_time 이 고객이 마지막으로 수정된 날짜/시간입니다. 날짜/시간은 고객의 시간대로 표시되며 'yyyy-MM-dd HH:mm:ss.ssssss' 형식입니다.
customer_status 고객의 상태입니다.
customer_time_zone 고객의 현지 시간대 ID입니다.
Search Ads 360 테이블 이름: AccountAssetStats
Search Ads 360 API 리소스: customer_asset [https://developers.google.com/search-ads/reporting/api/reference/fields/v0/customer_asset?hl=ko]
Search Ads 360 필드 이름 설명
customer_asset_asset 필수. 변경할 수 없습니다. 고객에 연결된 애셋입니다.
customer_id 고객의 ID입니다.
metrics_average_cpc 모든 클릭의 총 비용을 발생한 총 클릭수로 나눈 값입니다.
metrics_average_cpm 평균 CPM(1,000회 노출당비용)입니다.
metrics_clicks 클릭수입니다.
metrics_client_account_conversions 고객 계정 전환수입니다. 여기에는 include_in_client_account_conversions_metric 속성이 true로 설정된 전환 액션만 포함됩니다. 전환 기반 입찰을 사용할 경우 입찰 전략이 이러한 전환에 맞게 최적화됩니다.
metrics_client_account_conversions_value 고객 계정 전환의 가치입니다. 여기에는 include_in_client_account_conversions_metric 속성이 true로 설정된 전환 액션만 포함됩니다. 전환 기반 입찰을 사용할 경우 입찰 전략이 이러한 전환에 맞게 최적화됩니다.
metrics_cost_micros 일정 기간에 발생한 클릭당비용(CPC)과 1,000회 노출당비용(CPM)의 합계입니다.
metrics_ctr 광고에서 발생한 클릭수를 광고가 게재된 횟수(노출수)로 나눈 값입니다.
metrics_impressions Google 네트워크에서 검색 결과 페이지 또는 웹사이트에 광고가 게재된 횟수입니다.
segments_date 측정항목이 적용되는 날짜입니다(yyyy-MM-dd 형식). 예: 2024-04-17
Search Ads 360 테이블 이름: AccountConversionActionAndAssetStats
Search Ads 360 API 리소스: asset_set_asset [https://developers.google.com/search-ads/reporting/api/reference/fields/v0/asset_set_asset?hl=ko]
Search Ads 360 필드 이름 설명
asset_set_asset_asset 변경할 수 없습니다. 이 애셋 집합이 연결된 애셋입니다.
asset_set_asset_asset_set 변경할 수 없습니다. 이 애셋 집합이 연결된 애셋 집합입니다.
customer_id 고객의 ID입니다.
metrics_all_conversions 총 전환수입니다. 여기에는 include_in_conversions_metric의 값과 관계없이 모든 전환이 포함됩니다.
metrics_all_conversions_value 모든 전환의 가치입니다.
metrics_cross_device_conversions 고객이 하나의 기기에서 광고를 클릭한 후 다른 기기 또는 브라우저에서 전환할 때 발생하는 전환입니다. 교차 기기 전환은 이미 all_conversions에 포함되어 있습니다.
metrics_cross_device_conversions_value 교차 기기 전환의 가치의 합계입니다.
segments.conversion_action_name 전환 액션 이름입니다.
segments_date 측정항목이 적용되는 날짜입니다(yyyy-MM-dd 형식). 예: 2024-04-17

metrics_client_account_conversions 고객 계정 전환수입니다. 여기에는 include_in_client_account_conversions_metric 속성이 true로 설정된 전환 액션만 포함됩니다. 전환 기반 입찰을 사용할 경우 입찰 전략이 이러한 전환에 맞게 최적화됩니다.
metrics_client_account_conversions_value 고객 계정 전환의 가치입니다. 여기에는 include_in_client_account_conversions_metric 속성이 true로 설정된 전환 액션만 포함됩니다. 전환 기반 입찰을 사용할 경우 입찰 전략이 이러한 전환에 맞게 최적화됩니다.
metrics_client_account_view_through_conversions 총 조회 후 전환 수입니다. 이는 고객이 이미지 또는 리치 미디어 광고를 보고 나서 다른 광고와 상호작용(예: 클릭)하지 않고 사이트에서 전환을 완료할 때 발생합니다.
metrics_cost_micros 일정 기간에 발생한 클릭당비용(CPC)과 1,000회 노출당비용(CPM)의 합계입니다.
metrics_ctr 광고에서 발생한 클릭수를 광고가 게재된 횟수(노출수)로 나눈 값입니다.
metrics_impressions Google 네트워크에서 검색 결과 페이지 또는 웹사이트에 광고가 게재된 횟수입니다.
segments_date 측정항목이 적용되는 날짜입니다(yyyy-MM-dd 형식). 예: 2024-04-17
segments_device 측정항목이 적용되는 기기입니다.
