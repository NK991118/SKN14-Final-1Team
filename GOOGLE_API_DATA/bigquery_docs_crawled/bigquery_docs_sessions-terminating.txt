세션 종료
bookmark_border
이 문서에서는 BigQuery에서 세션 종료가 작동하는 방법을 설명합니다. 이 문서는 BigQuery 세션 [https://cloud.google.com/bigquery/docs/sessions-intro?hl=ko]을 대체적으로 이해하는 사용자를 대상으로 합니다.
이 단계를 완료하기 전에 세션을 종료하는 데 필요한 권한 [https://cloud.google.com/bigquery/docs/sessions-intro?hl=ko#roles_and_permissions]이 있는지 확인합니다.
세션 종료
세션은 수동으로 또는 자동으로 종료할 수 있습니다. 종료된 세션의 기록은 종료 후 20일 동안 확인할 수 있습니다.
세션 자동 종료
세션 [https://cloud.google.com/bigquery/docs/sessions-intro?hl=ko]은 24시간 동안 활동이 없거나 7일이 지나거나 어느 쪽이든 먼저 발생하는 시점에 자동으로 종료됩니다.
현재 세션 종료
SQL 문 또는Google Cloud 콘솔에서 세션을 만든 경우 현재 세션을 종료할 수 있습니다.
--- 탭: 콘솔 [https://cloud.google.com/bigquery/docs/sessions-terminating?hl=ko#%EC%BD%98%EC%86%94] ---
Google Cloud 콘솔에서 BigQuery 페이지로 이동합니다.

BigQuery로 이동 [https://console.cloud.google.com/bigquery?hl=ko] 
세션이 포함된 편집기 탭을 찾아 닫습니다. 세션이 종료됩니다.

--- 탭: SQL [https://cloud.google.com/bigquery/docs/sessions-terminating?hl=ko#sql] ---
세션을 종료하려면 다음 안내를 따르세요.




 Google Cloud 콘솔에서 BigQuery 페이지로 이동합니다.

BigQuery로 이동 [https://console.cloud.google.com/bigquery?hl=ko] 
쿼리 편집기에서 다음 문을 입력합니다.

CALL BQ.ABORT_SESSION();


play_circle 실행을 클릭합니다.




쿼리를 실행하는 방법에 대한 자세한 내용은 대화형 쿼리 실행 [https://cloud.google.com/bigquery/docs/running-queries?hl=ko#queries]을 참조하세요.
ID로 세션 종료
해당 ID를 사용하여 세션을 종료할 수 있습니다. 이 방법으로 세션을 종료하기 위해 세션에 있을 필요는 없습니다.
세션 ID를 가져온 [https://cloud.google.com/bigquery/docs/sessions-get-ids?hl=ko] 후 다음 문을 실행합니다.
CALL BQ.ABORT_SESSION(SESSION_ID);
SESSION_ID를 종료할 세션의 ID로 바꿉니다.
