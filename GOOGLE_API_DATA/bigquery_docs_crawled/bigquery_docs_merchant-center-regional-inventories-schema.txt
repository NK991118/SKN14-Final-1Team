Google 판매자 센터 리전별 인벤토리 표
bookmark_border
개요
리전별 인벤토리 테이블 스키마는 판매자에게 해당 제품의 리전별 재고 및 가격 책정 재정의를 보여줍니다. 리전별 인벤토리 데이터 형식은 주로 Content API의 regionalinventory [https://developers.google.com/shopping-content/reference/rest/v2.1/regionalinventory?hl=ko] 리소스의 관련 필드 형식을 따릅니다.
데이터는 개별 판매자 ID를 사용하는 경우 RegionalInventories_MERCHANT_ID, MCA 계정을 사용하는 경우 RegionalInventories_AGGREGATOR_ID라는 테이블에 기록됩니다.
스키마
RegionalInventories_ 테이블에는 다음과 같은 스키마가 있습니다.
열 BigQuery 데이터 유형 설명 예시 데이터
product_id STRING 다음과 같은 형식의 Content API의 상품 REST ID: channel:content_language:feed_label:offer_id. 이 필드는 기본 키입니다. online:en:AU:666840730
merchant_id INTEGER 판매자 계정 ID 이 필드는 기본 키입니다.
aggregator_id INTEGER 멀티 클라이언트 계정의 애그리게이터 계정 ID
region_id STRING 인벤토리의 리전 ID
price RECORD 항목의 리전별 가격
price.value NUMERIC 항목의 리전별 가격 99
price.currency STRING 항목의 리전별 가격 통화 CHF
sale_price RECORD 항목의 리전별 할인가
sale_price.value NUMERIC 항목의 리전별 할인가 49
sale_price.currency STRING 항목의 리전별 할인가 통화 CHF
sale_price_effective_start_date TIMESTAMP 해당 리전에서 항목의 할인 시작 날짜 및 시간 2021-03-30 00:00:00 UTC
sale_price_effective_end_date TIMESTAMP 해당 리전에서 항목의 할인 종료 날짜 및 시간 2021-04-14 00:00:00 UTC
availability STRING 항목의 리전별 재고 상태 재고 없음
