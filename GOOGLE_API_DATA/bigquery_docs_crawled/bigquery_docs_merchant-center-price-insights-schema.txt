Google 판매자 센터 가격 정보 테이블
bookmark_border
개요
가격 정보 보고서는 제품의 추천 할인가를 표시하고 제품 가격을 업데이트할 때 예상되는 성과를 예측합니다. 가격 정보 보고서를 사용하여 제품 가격을 더 효과적으로 책정할 수 있습니다. 이 보고서는 Google 판매자 센터의 가격 정보 보고서 [https://support.google.com/merchants/answer/11916926?hl=ko]와 비슷합니다.
개별 판매자 ID를 사용하는 경우에는 데이터가 PriceInsights_MERCHANT_ID 테이블에 기록되고, MCA 계정을 사용하는 경우에는 PriceInsights_AGGREGATOR_ID 테이블에 기록됩니다.
스키마
PriceInsights_ 테이블에는 다음과 같은 스키마가 있습니다.
열 BigQuery 데이터 유형 설명 예시 데이터
aggregator_id INTEGER 판매자가 MCA에 속하는 경우 멀티 클라이언트 계정(MCA) ID입니다. 그렇지 않으면 Null입니다. 12345
merchant_id INTEGER Google 판매자 센터 계정 ID 이 필드는 기본 키입니다. 1234
id STRING channel:content_language:feed_label:offer_id 형식으로 된 제품의 Content API REST ID입니다. 제품 테이블 스키마 [https://cloud.google.com/bigquery/docs/merchant-center-products-schema?hl=ko]에 정의된 방식과 유사합니다. 이 필드는 기본 키입니다. online:en:AU:666840730
title STRING 제품의 이름입니다. TN2351 black USB
brand STRING 제품의 브랜드입니다. 브랜드 이름
offer_id STRING 판매자가 제품의 ID [https://support.google.com/merchants/answer/6324405?hl=ko]를 제공했습니다. tddy123uk
price RECORD 제품의 현재 가격입니다. 1$ = amount_micros: 1000000
currency_code: USD
suggested_price RECORD
비즈니스의 총이익을 극대화하기 위해 Google에서 예측한 판매가 또는 할인 가격입니다. 복잡한 모델을 기반으로 지난 7일 동안 다양한 가격대에서 제품 성능을 시뮬레이션합니다.
Google은 추천 가격을 계산하기 위해 기존 가격을 동일한 제품을 판매하는 유사 여러 업체의 가격과 비교합니다. 이 모델에서는 제품 수요, 유사 제품 판매자 수, 유사 비즈니스의 예상 이익률과 같은 요소도 반영됩니다. 이러한 요인을 분석하여 추천 가격을 적용할 경우 예상되는 노출수, 클릭수, 전환수, 총 이익을 계산합니다.
권장 가격이 유용한 정보를 제공하지만 예측에서 향후 성과 결과를 보장하지는 않습니다.
1$ = amount_micros: 1000000
currency_code: USD
predicted_impressions_change_fraction FLOAT 추천 할인가를 적용할 경우 예상되는 노출수 증가 지난 7일간의 성과 데이터를 기반으로 한 모델에 의해 예측이 생성됩니다.
predicted_clicks_change_fraction FLOAT 추천 할인가를 적용할 경우 예상되는 클릭수 증가 지난 7일간의 성과 데이터를 기반으로 한 모델에 의해 예측이 생성됩니다.
predicted_conversions_change_fraction FLOAT 추천 할인가를 적용할 경우 예상되는 전환수 증가 지난 7일간의 성과 데이터를 기반으로 한 모델에 의해 예측이 생성됩니다.
product_type_l1 STRING 제품의 제품 유형 속성 [https://support.google.com/merchants/answer/6324406?hl=ko]입니다.
product_type_l2 STRING 제품의 제품 유형 속성 [https://support.google.com/merchants/answer/6324406?hl=ko]입니다.
product_type_l3 STRING 제품의 제품 유형 속성 [https://support.google.com/merchants/answer/6324406?hl=ko]입니다.
product_type_l4 STRING 제품의 제품 유형 속성 [https://support.google.com/merchants/answer/6324406?hl=ko]입니다.
product_type_l5 STRING 제품의 제품 유형 속성 [https://support.google.com/merchants/answer/6324406?hl=ko]입니다.
category_l1 STRING 제품의 Google 제품 카테고리 [https://support.google.com/merchants/answer/6324436?hl=ko]입니다. 애완동물 및 애완동물 용품
category_l2 STRING 제품의 Google 제품 카테고리 [https://support.google.com/merchants/answer/6324436?hl=ko]입니다. 반려동물 용품
category_l3 STRING 제품의 Google 제품 카테고리 [https://support.google.com/merchants/answer/6324436?hl=ko]입니다. 반려견 용품
category_l4 STRING 제품의 Google 제품 카테고리 [https://support.google.com/merchants/answer/6324436?hl=ko]입니다. 반려견 침대
category_l5 STRING 제품의 Google 제품 카테고리 [https://support.google.com/merchants/answer/6324436?hl=ko]입니다.
