Facebook Ads 보고서의 테이블 매핑
Facebook Ads 보고서를 BigQuery로 전송하면 보고서가 다음과 같은 BigQuery 테이블 및 뷰로 변환됩니다.
AdAccounts 보고서
Meta API 필드 이름 매핑된 BigQuery 필드 이름 유형 설명
id ID [KEY] 문자열 광고 계정의 ID입니다.
대상 문자열 광고 계정을 가져오는 데 사용되는 타겟입니다. 이 값은 모든 광고 계정을 가져오기 위해 `null`입니다.
account_id AccountId 문자열 Facebook에서 직접 볼 때 광고 계정의 ID입니다.
account_status AccountStatus 정수 계정 상태입니다. 1 = 활성, 2 = 사용 중지됨, 3 = 미결제, 7 = 검토 대기 중, 9 = 유예 기간, 101 = 일시적으로 사용 불가, 100 = 폐쇄 대기 중
age 연령 Double 광고 계정이 개설된 기간(일)입니다.
amount_spent AmountSpent 정수 계정에서 현재까지 지출한 총금액입니다. 이 값은 재설정할 수 있습니다.
balance 잔고 정수 미결제 청구 금액입니다.
business_city BusinessCity 문자열 비즈니스 주소의 도시입니다.
business_country_code BusinessCountryCode 문자열 비즈니스 주소의 국가 코드입니다.
business_name BusinessName 문자열 계정의 비즈니스 이름입니다.
business_state BusinessState 문자열 비즈니스 주소의 주 약어입니다.
business_street BusinessStreet 문자열 계정의 비즈니스 상세 주소의 첫 번째 줄입니다.
business_street2 BusinessStreet2 문자열 계정의 비즈니스 상세 주소의 두 번째 줄입니다.
business_zip BusinessZip 문자열 비즈니스 주소의 우편번호입니다.
capabilities 기능 문자열 이 광고 계정에 허용된 기능입니다.
created_time CreatedTime 날짜/시간 계정이 생성된 시간입니다.
currency 통화 문자열 계정 설정의 해당 값을 기반으로 계정에 사용되는 통화입니다.
min_campaign_group_spend_cap MinCampaignGroupSpendCap 문자열 최소 캠페인 그룹 지출 한도입니다.
name 이름 문자열 계정 이름입니다. 많은 계정의 이름이 지정되지 않으므로 이 필드는 비어 있을 수 있습니다.
offsite_pixels_tos_accepted OffsitePixelsTosAccepted 문자열 오프사이트 픽셀 서비스 약관 계약이 서명되었는지 여부를 나타냅니다.
owner OwnerId 문자열 광고 계정 소유자의 Facebook ID입니다.
spend_cap SpendCap 정수 이 계정에서 지출할 수 있는 최대 금액으로, 이 금액을 초과하면 캠페인이 일시중지됩니다. 값이 0이면 지출 한도가 없음을 의미합니다.
timezone_id TimezoneId 문자열 시간대의 ID입니다.
timezone_name TimezoneName 문자열 시간대 이름입니다.
timezone_offset_hours_utc TimezoneOffsetHoursUTC Double UTC와의 시간대 차이입니다.
AdInsights 보고서
Meta API 필드 이름 매핑된 BigQuery 필드 이름 유형 설명
대상 문자열 통계를 가져올 계정의 ID입니다.
date_start DateStart 날짜 통계를 가져올 시작 날짜입니다. Facebook UI에서는 '보고서 시작' 필드입니다.
date_stop DateEnd 날짜 통계를 가져올 종료일입니다. Facebook UI에서는 '보고서 종료' 필드입니다.
TimeIncrement 문자열 데이터 집계 일수입니다. 이 값은 1로 설정됩니다.
수준 문자열 결과를 나타낼 수준입니다. 이 값은 'ad'로 설정됩니다.
account_currency AccountCurrency 문자열 광고 계정에서 사용되는 통화입니다.
action_attribution_windows ActionAttributionWindows 문자열 행동의 기여도 분석 기간을 결정하는 쉼표로 구분된 목록입니다. 예를 들어 28d_click은 사용자가 광고를 클릭한 후 28일 동안 발생한 모든 액션을 API가 반환한다는 의미입니다. 이 옵션은 [1d_view,28d_click]으로 설정됩니다.
account_id AdAccountId 문자열 보고서 행과 연결된 광고 계정의 ID입니다.
account_name AdAccountName 문자열 보고서 행과 연결된 광고 계정의 이름입니다.
campaign_id CampaignId 문자열 보고서 행과 연결된 캠페인의 ID입니다.
campaign_name CampaignName 문자열 보고서 행과 연결된 캠페인의 이름입니다.
adset_id AdSetId 문자열 보고서 행과 연결된 광고 세트의 ID입니다.
adset_name AdSetName 문자열 보고서 행과 연결된 광고 세트의 이름입니다.
ad_id AdId 문자열 보고서 행과 연결된 광고의 ID입니다.
ad_name AdName 문자열 보고서 행과 연결된 광고의 이름입니다.
buying_type BuyingType 문자열 캠페인에서 타겟 광고 비용을 지불하는 방법입니다.
clicks 클릭수 Long 광고의 총 클릭수입니다. 홍보하는 항목에 따라 페이지 좋아요, 이벤트 응답 또는 앱 설치가 포함될 수 있습니다. Facebook UI에서는 '클릭수 (전체)' 필드입니다.
conversion_rate_ranking ConversionRateRanking 문자열 전환율 순위입니다.
cost_per_estimated_ad_recallers CostPerEstimatedAdRecallers 십진수 2일 이내에 질문을 받으면 광고를 본 것을 기억할 것으로 예상되는 추가 사용자 1인당 평균 비용입니다.
cost_per_inline_link_click CostPerInlineLinkClick 십진수 광고의 링크에 대한 평균 클릭당비용입니다.
cost_per_inline_post_engagement CostPerInlinePostEngagement 십진수 게시물의 참여당 평균 비용입니다.
cost_per_unique_click CostPerUniqueClick 십진수 이러한 광고의 순 클릭당 평균 비용으로, 지출액을 발생한 순 클릭수로 나누어 계산합니다.
cost_per_unique_inline_link_click CostPerUniqueInlineLinkClick 십진수 각 고유 인라인 링크 클릭에 지불한 평균 비용입니다.
cpc CPC 십진수 이러한 광고의 평균 클릭당비용으로, 지출액을 발생한 클릭수로 나누어 계산합니다.
cpm CPM 십진수 광고가 1,000회 노출되도록 지불한 평균 비용입니다.
cpp CPP 십진수 광고가 1,000명의 순 사용자에게 게재되는 데 지불한 평균 비용입니다.
ctr 클릭률 Double 발생한 클릭수를 노출수로 나눈 값입니다. Facebook UI에서는 CTR (전체) % 필드입니다.
estimated_ad_recall_rate EstimatedAdRecallRate Double 광고를 기억하는 것으로 추정되는 사용자 수를 광고가 도달한 사용자 수로 나눈 값입니다.
estimated_ad_recallers EstimatedAdRecallers Double 2일 이내에 질문을 받으면 광고를 본 것을 기억할 것으로 예상되는 추가 사용자 수입니다.
frequency 빈도 Double 각 사용자에게 광고가 게재된 평균 횟수입니다.
impressions 노출수 Long 광고가 게재된 횟수입니다. 모바일 앱에서는 광고가 처음 조회될 때 게재된 것으로 집계됩니다. 다른 모든 Facebook 인터페이스에서는 광고가 사용자의 뉴스 피드에 처음 게재될 때 또는 오른쪽 열에 게재될 때마다 게재됩니다.
inline_link_clicks InlineLinkClicks Long 광고의 링크를 클릭한 총횟수입니다.
inline_link_click_ctr InlineLinkClicksCounter Double 링크로 연결되는 인라인 클릭의 클릭률입니다.
inline_post_engagement InlinePostEngagement Long 게시물의 총 참여수입니다.
instant_experience_clicks_to_open InstantExperienceClicksToOpen Long META API의 instant_experience_clicks_to_open 필드에 해당합니다.
instant_experience_clicks_to_start InstantExperienceClicksToStart Long META API의 instant_experience_clicks_to_start 필드에 해당합니다.
instant_experience_outbound_clicks InstantExperienceOutboundClicks Long META API의 instant_experience_outbound_clicks 필드에 해당합니다.
objective 목표 문자열 캠페인에 선택한 목표입니다. 목표는 광고로 달성하고자 하는 목표를 반영합니다.
quality_ranking QualityRanking 문자열 품질 순위입니다.
reach 도달범위 Long 광고가 게재된 사용자 수입니다.
spend 지출 십진수 지금까지 지출한 총금액입니다.
UniqueClicks Long 광고를 클릭한 순 사용자의 총수입니다. 예를 들어 3명이 동일한 광고를 5번 클릭하면 순 클릭수는 3회로 집계됩니다.
UniqueCTR Double 광고를 클릭한 사용자 수를 도달한 사용자 수로 나눈 값입니다. 예를 들어 순 클릭수가 20회이고 광고가 1,000명의 순 사용자에게 게재된 경우 순 클릭률은 2%가 됩니다.
inline_link_clicks UniqueInlineLinkClicks Long 광고가 획득한 순 인라인 링크 클릭수입니다. Facebook UI에서는 '링크에 대한 고유 클릭수' 필드입니다.
UniqueInlineLinkClickCounter Double 링크로 연결되는 고유 인라인 클릭의 클릭률입니다.
UniqueLinkClicksCounter Double 링크 클릭의 고유 클릭률입니다. Facebook 외부로 연결되는 광고의 링크를 클릭한 사용자 수를 도달한 사용자 수로 나눈 값입니다. 예를 들어 링크에 대한 순 클릭수가 20회이고 광고가 1,000명의 순 사용자에게 게재되었다면 순 클릭률은 2%가 됩니다.
체크인 Int 광고에 기여한 체크인 수입니다.
EventResponses Int 광고에 기여한 이벤트 응답 수입니다.
inline_link_clicks LinkClicks Int 광고에 기여한 링크 클릭수입니다.
OfferSaves Int 광고에 기여한 수신 제안 수입니다.
outbound_clicks OutboundClicks Int 광고에 기여한 아웃바운드 클릭수입니다.
PageEngagements Int 광고로 인해 발생한 페이지 참여도 수입니다.
PageLikes Int 광고로 인해 발생한 페이지 좋아요 수입니다.
PageMentions Int 광고에 기여한 페이지 멘션 수입니다.
PagePhotoViews Int 광고로 인해 발생한 사진 조회수입니다.
PostComments Int 광고에 기여한 게시물 댓글 수입니다.
PostEngagements Int 광고로 인해 발생한 게시물 참여 수입니다.
PostShares Int 광고로 인해 발생한 게시물 공유 수입니다.
PostReactions Int 광고에 기여한 게시물 반응 수입니다.
PageTabViews Int 광고에 기여한 탭 조회수입니다.
리전 문자열 사용자가 광고를 조회한 지역입니다. 분류 필드입니다.
Video3SecondViews Int 광고에 기여한 동영상 조회수입니다. 3초 이상 또는 동영상 전체 (동영상이 3초 미만인 경우)가 재생되면 조회수로 집계됩니다.

ad_id AdId 문자열 보고서 행과 연결된 광고의 ID입니다.
ad_name AdName 문자열 보고서 행과 연결된 광고의 이름입니다.
actions.value ActionValue 정수 기본 기여 분석 기간의 측정항목 값입니다.
actions.1d_click Action1dClick 문자열 광고를 클릭한 후 1일이 지난 시점의 기여 분석 기간의 측정항목 값입니다.
