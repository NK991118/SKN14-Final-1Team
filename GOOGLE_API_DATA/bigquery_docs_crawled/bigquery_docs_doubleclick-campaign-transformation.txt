Campaign Manager 보고서 변환
bookmark_border
Campaign Manager(이전 명칭: DoubleClick Campaign Manager) 데이터 전송 파일이 BigQuery로 전송되면 파일은 다음과 같은 BigQuery 표와 뷰로 변환됩니다.
BigQuery에서 테이블과 뷰를 볼 때 campaign_manager_id 값은 Campaign Manager 네트워크, 광고주, 플러드라이트 ID입니다.
