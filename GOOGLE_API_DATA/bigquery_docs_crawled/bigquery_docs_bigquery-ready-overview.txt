Google Cloud Ready - BigQuery
bookmark_border
Google Cloud Ready - BigQuery에서는 핵심 요구사항을 충족하여 파트너 제품과 BigQuery 간의 최상의 통합을 보장하는 파트너 솔루션을 검증합니다. 검증된 파트너 제품을 제공하여 기존 고객과 잠재고객이 입증된 파트너 제품을 탐색하고 사용할 수 있습니다. 이제 BigQuery 고객이 새로운 도구를 평가하는 시간을 단축하고, 엄격한 검증 프로세스를 통해 BigQuery에서 최적의 작동을 보이는 것으로 입증된 파트너 제품을 사용하여 솔루션을 빌드하는 데 집중할 수 있습니다.
Google Cloud Ready - BigQuery는 어떻게 작동하나요?
Google의 BigQuery 검증 프로그램을 이용해 파트너는 BigQuery와의 통합을 검증하여 최상의 고객 경험을 보장할 수 있습니다. BigQuery 검증 프로그램은 다음과 같은 3단계로 진행됩니다.
평가: Google팀이 샌드박스 프로덕션 환경에서 일련의 데이터 통합 테스트를 실행하고 결과를 권장사항 및 성능 벤치마크와 비교합니다.
강화: 평가 단계의 결과를 얻고 부족한 부분이 있는 경우 파트너와 협력하여 이를 채웁니다.
지원: 파트너와 함께 공통 고객이 통합 제품을 사용해 성공적으로 솔루션을 구현하는 데 필요한 모든 것을 갖출 수 있도록 기존 문서를 수정합니다.
프로그램 범위 및 파트너 혜택
파트너가 이 이니셔티브에 가입하면 Google Cloud 팀에서 파트너의 BigQuery 통합 품질을 테스트합니다. 이후 파트너는 테스트에서 확인된 부족한 부분이나 개선 영역을 해결하기 위해 노력할 수 있습니다. 파트너 통합이 검증된 파트너는 Google Cloud Ready - BigQuery라는 명칭을 통해 역량을 인정받게 됩니다.
이 프로그램에서 BigQuery 권장사항을 따르고 개선 영역을 파악할 수 있는 포괄적인 소프트웨어 검토를 파트너에게 무료로 제공합니다.
다음은 참여 ISV에 제공되는 혜택 목록입니다.

파트너 배지 및 Google Cloud Ready - BigQuery 명칭

Google Cloud 파트너 엔지니어링 및 BigQuery 제품팀의 지원을 받아 로드맵을 함께 개발하고 분기별 독점 기능 출시 미리보기 이용
권장사항 가이드, 솔루션 요약, 공동 참조 아키텍처 작성 지원
검증 대상 파트너 선정 방식
프로그램 참여에 관심 있는 파트너는 다음 최소 조건을 충족해야 합니다.
파트너 제품에 프로덕션 품질의 BigQuery 통합 포함
BigQuery로 프로덕션에서 파트너 제품을 사용하는 고객 5명 이상
공개 고객 우수사례 1개
Google Cloud 파트너팀에서 검증 리소스의 가용성에 따라 귀사와 검증 일정을 조율합니다.
Google Cloud Ready - BigQuery 파트너 되기
자격요건을 충족하는 Google Cloud 파트너는 Google Cloud Ready - BigQuery 이니셔티브에 참여하여 파트너 제품의 분류 방식에 따른 요구사항 [https://cloud.google.com/bigquery/docs/bigquery-ready-overview?hl=ko#requirements]을 충족함으로써 솔루션을 검증받고 홍보할 수 있습니다. 카테고리는 다음과 같습니다.
BI, ML 및 고급 분석 [https://cloud.google.com/bigquery/docs/bigquery-ready-partners?hl=ko#bi-ml-analytics]
커넥터 및 개발자 도구 [https://cloud.google.com/bigquery/docs/bigquery-ready-partners?hl=ko#connectors-developer-tools]
데이터 거버넌스, 보안, MDM [https://cloud.google.com/bigquery/docs/bigquery-ready-partners?hl=ko#data-governance-mdm]
데이터 품질, 관측 가능성, FinOps [https://cloud.google.com/bigquery/docs/bigquery-ready-partners?hl=ko#data-quality-observability]
ETL 및 데이터 통합 [https://cloud.google.com/bigquery/docs/bigquery-ready-partners?hl=ko#etl-data-integration]
카테고리별 통합 및 상호 운용성 테스트를 완료해야 합니다.
이미 Google Cloud 파트너이고 이 프로그램에 참여하고자 하는 경우 파트너 관리자에게 문의하거나 파트너 어드밴티지 포털 [https://www.partneradvantage.goog/GCPPRM/s/partneradvantageportallogin]에서 직접 신청하거나 이 양식 [https://forms.gle/HecFemUBqxyiAwAU6]을 작성하세요. 아직 Google Cloud 파트너가 아니면 등록 양식 [https://partners.cloud.google.com/enrollment/?hl=ko]을 작성하여 신청 절차를 시작할 수 있습니다. 프로그램에 대해 궁금한 점이 있으면 문의 [mailto:google_cloud_ready_bq@google.com]해 주세요.
파트너 센터
Google Cloud Ready의 많은 BigQuery 파트너가 BigQuery에 표시됩니다. Google Cloud Ready 파트너 목록을 보려면 다음 단계를 따르세요.
Google Cloud 콘솔에서 파트너 센터 페이지로 이동합니다.
파트너 센터로 이동 [https://console.cloud.google.com/bigquery/partner-center?hl=ko]
파트너 목록이 해당 카테고리에 표시됩니다.
특정 파트너 목록을 찾으려면 검색 창에 이름을 입력하고 Enter 키를 누릅니다.
통합 단계 및 결제 정보를 포함한 파트너의 세부정보를 보려면 목록을 선택합니다. 파트너가 Google Cloud Marketplace에서도 사용할 수 있으면 Cloud Marketplace 태그로 표시됩니다.
참고: 서드 파티 소프트웨어가 BigQuery가 의도한 대로 작동한다고 판단되면 Cloud Customer Care에서 지원되지 않습니다.
