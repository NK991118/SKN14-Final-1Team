개요
주의: BigQuery는 2025년 9월 1일에 제품 인벤토리 테이블을 내보내는 보고서를 더 이상 지원하지 않습니다. 대신 새로운 베스트셀러 보고서 [https://cloud.google.com/bigquery/docs/merchant-center-best-sellers-schema?hl=ko]를 사용하도록 마이그레이션하는 것이 좋습니다. 새 보고서로 마이그레이션하는 방법에 관한 자세한 내용은 베스트셀러 보고서 마이그레이션 [https://cloud.google.com/bigquery/docs/merchant-center-best-sellers-migration?hl=ko]을 참조하세요.
베스트셀러 데이터는 판매자가 쇼핑 광고 및 무료 목록에서 가장 인기있는 브랜드와 제품을 파악하는 데 도움이 됩니다. 베스트셀러에 대한 상세 정보는 지원되는 보고서 [https://cloud.google.com/bigquery/docs/merchant-center-transfer?hl=ko#supported_reports]의 설명을 참조하세요.
데이터는 BestSellers_TopProducts_Inventory_MERCHANT_ID라는 테이블에 기록됩니다.
스키마
BestSellers_TopProducts_Inventory_ 테이블에는 다음과 같은 스키마가 있습니다.
열 BigQuery 데이터 유형 설명
rank_id STRING 인기 제품 [https://cloud.google.com/bigquery/docs/merchant-center-top-products-schema?hl=ko] 테이블과 조인할 순위 식별자
product_id STRING 다음과 같은 형식의 Content API의 상품 REST ID: channel:content_language:feed_label:offer_id
merchant_Id INTEGER 판매자 계정 ID
aggregator_id INTEGER 멀티 클라이언트 계정의 애그리게이터 계정 ID
