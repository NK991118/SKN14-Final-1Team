YouTube 채널 보고서 변환
bookmark_border
BigQuery로 전송된 YouTube 채널 보고서는 다음 BigQuery 테이블 및 뷰로 변환됩니다.
BigQuery에서 테이블과 뷰를 볼 때 suffix 값은 전송을 생성할 때 구성한 테이블 서픽스입니다.
