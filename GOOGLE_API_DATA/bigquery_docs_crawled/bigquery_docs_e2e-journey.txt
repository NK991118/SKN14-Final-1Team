5BigQuery ML은 부스티드 트리, 랜덤 포레스트, DNN, 와이드 앤 딥, Autoencoder, AutoML 모델의 가중치를 검색하는 함수를 지원하지 않습니다. 이러한 모델의 가중치를 보려면 BigQuery ML에서 Cloud Storage로 기존 모델을 내보내고 XGBoost 라이브러리 또는 TensorFlow 라이브러리를 사용하여 트리 모델의 트리 구조 또는 신경망의 그래프 구조를 시각화할 수 있습니다. 자세한 내용은 EXPORT MODEL 문서 [https://cloud.google.com/bigquery/docs/exporting-models?hl=ko] 및 EXPORT MODEL 튜토리얼 [https://cloud.google.com/bigquery/docs/export-model-tutorial?hl=ko]을 참조하세요.
