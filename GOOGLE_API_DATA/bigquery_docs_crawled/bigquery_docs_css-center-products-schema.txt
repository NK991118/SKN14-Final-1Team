Comparison Shopping Services Center 테이블 스키마
bookmark_border
Comparison Shopping Services(CSS) Center 보고 데이터가 BigQuery로 전송되면 제품 및 제품 문제 데이터 형식이 주로 Content API의 ProductView [https://developers.google.com/shopping-content/reference/rest/v2.1/reports/search?hl=ko#productview] 및 Productstatuses [https://developers.google.com/shopping-content/v2/reference/v2.1/productstatuses?hl=ko] 리소스의 관련 필드 형식을 따릅니다.
CSS Center 제품 스키마
다음 표에는 Products_ 테이블의 스키마가 나와 있습니다. 일부 필드는 다른 필드의 하위 집합으로 포함됩니다. 예를 들어 Price 필드에는 Value 및 Currency 필드가 모두 포함됩니다.
