이상 감지 개요
bookmark_border
이상 감지는 특정 데이터 세트의 데이터 편차를 식별하는 데 사용할 수 있는 데이터 마이닝 기법입니다. 예를 들어, 특정 제품의 반품율이 해당 제품의 기준보다 크게 증가하면 제품 결함 또는 사기 가능성을 나타낼 수 있습니다. 이상 감지를 사용하여 기술 문제와 같은 중대한 이슈 또는 소비자 행동 변화와 같은 기회를 감지할 수 있습니다.
이상 감지를 사용할 때의 한 가지 도전과제는 어떤 데이터를 이상 데이터로 간주할지를 결정하는 것입니다. 이상치를 나타내는 데이터에 라벨이 지정된 경우 다음 지도 머신러닝 모델 중 하나와 함께 ML.PREDICT 함수 [https://cloud.google.com/bigquery/docs/reference/standard-sql/bigqueryml-syntax-predict?hl=ko]를 사용하여 이상 감지를 수행할 수 있습니다.
선형 및 로지스틱 회귀 모델 [https://cloud.google.com/bigquery/docs/reference/standard-sql/bigqueryml-syntax-create-glm?hl=ko]
부스티드 트리 모델 [https://cloud.google.com/bigquery/docs/reference/standard-sql/bigqueryml-syntax-create-boosted-tree?hl=ko]
랜덤 포레스트 모델 [https://cloud.google.com/bigquery/docs/reference/standard-sql/bigqueryml-syntax-create-random-forest?hl=ko]
심층신경망(DNN) 모델 [https://cloud.google.com/bigquery/docs/reference/standard-sql/bigqueryml-syntax-create-dnn-models?hl=ko]
와이드 앤 딥 모델 [https://cloud.google.com/bigquery/docs/reference/standard-sql/bigqueryml-syntax-create-wnd-models?hl=ko]
AutoML 모델 [https://cloud.google.com/bigquery/docs/reference/standard-sql/bigqueryml-syntax-create-automl?hl=ko]
무엇을 이상 데이터로 간주해야 하는지 확실하지 않거나 모델을 학습시킬 라벨 지정 데이터가 없는 경우 비지도 머신러닝을 사용하여 이상 감지를 수행할 수 있습니다. 다음 모델 중 하나와 함께 ML.DETECT_ANOMALIES 함수 [https://cloud.google.com/bigquery/docs/reference/standard-sql/bigqueryml-syntax-detect-anomalies?hl=ko]를 사용하여 학습 데이터 또는 새 서빙 데이터에서 이상치를 감지합니다.
데이터 유형 모델 유형 ML.DETECT_ANOMALIES [https://cloud.google.com/bigquery/docs/reference/standard-sql/bigqueryml-syntax-detect-anomalies?hl=ko]의 역할
시계열 ARIMA_PLUS [https://cloud.google.com/bigquery/docs/reference/standard-sql/bigqueryml-syntax-create-time-series?hl=ko] 시계열에서 이상치를 감지합니다.
ARIMA_PLUS_XREG [https://cloud.google.com/bigquery/docs/reference/standard-sql/bigqueryml-syntax-create-multivariate-time-series?hl=ko] 외부 회귀를 사용하여 시계열에서 이상치를 감지합니다.
독립적이고 동일하게 분포된 무작위 변수(IID) [https://en.wikipedia.org/wiki/Independent_and_identically_distributed_random_variables] k-평균 [https://cloud.google.com/bigquery/docs/reference/standard-sql/bigqueryml-syntax-create-kmeans?hl=ko] 입력 데이터에서 각 클러스터 중심까지의 정규화된 거리 중에서 최단 거리를 기반으로 이상치를 감지합니다. 정규화된 거리의 정의는 ML.DETECT_ANOMALIES 함수의 k-평균 모델 출력 [https://cloud.google.com/bigquery/docs/reference/standard-sql/bigqueryml-syntax-detect-anomalies?hl=ko#k-means_model_output]을 참조하세요.
Autoencoder [https://cloud.google.com/bigquery/docs/reference/standard-sql/bigqueryml-syntax-create-autoencoder?hl=ko] 평균 제곱 오차의 측면에서 재구성 손실을 기준으로 이상치를 감지합니다. 자세한 내용은 ML.RECONSTRUCTION_LOSS [https://cloud.google.com/bigquery/docs/reference/standard-sql/bigqueryml-syntax-reconstruction-loss?hl=ko]을 참조하세요. ML.RECONSTRUCTION_LOSS 함수는 모든 유형의 재구성 손실을 검색할 수 있습니다.
PCA [https://cloud.google.com/bigquery/docs/reference/standard-sql/bigqueryml-syntax-create-pca?hl=ko] 평균 제곱 오차의 측면에서 재구성 손실을 기준으로 이상치를 감지합니다.
추천 지식
