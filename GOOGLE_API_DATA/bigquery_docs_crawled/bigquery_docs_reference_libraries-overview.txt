BigQuery API 및 라이브러리 개요
bookmark_border
이 페이지에서는 BigQuery와 관련된 다양한 API에 대해 간략하게 설명합니다. 서버에 원시 요청을 수행해서 API를 직접 사용할 수 있지만 클라이언트 라이브러리를 사용하면 원하는 언어로 코드화할 수 있고 작성해야 하는 코드 양을 크게 줄이도록 간소화할 수 있습니다. BigQuery는 C#, Go, Java, Node.js, PHP, Python, Ruby의 클라이언트 라이브러리를 지원합니다. Google Cloud의 클라이언트 라이브러리에 대한 일반적인 개요는 클라이언트 라이브러리 설명 [https://cloud.google.com/apis/docs/client-libraries-explained?hl=ko]을 참고하세요.
다양한 BigQuery 라이브러리와 API를 사용하는 예시는 BigQuery 코드 샘플 [https://cloud.google.com/bigquery/docs/samples?hl=ko]을 참조하세요.
API를 사용하려면 인증을 거쳐 클라이언트 ID를 확인해야 합니다. 애플리케이션 기본 사용자 인증 정보 [https://cloud.google.com/bigquery/docs/authentication/getting-started?hl=ko], 서비스 계정 키 파일 [https://cloud.google.com/bigquery/docs/authentication/service-account-file?hl=ko], 사용자 인증 정보 [https://cloud.google.com/bigquery/docs/authentication/end-user-installed?hl=ko]를 사용하면 됩니다. 인증에 대해 자세히 알아보려면 인증 소개 [https://cloud.google.com/bigquery/docs/authentication?hl=ko]를 참조하세요.
데이터 수집 [https://cloud.google.com/bigquery/pricing?hl=ko#data_ingestion_pricing] 및 데이터 추출 [https://cloud.google.com/bigquery/pricing?hl=ko#data_extraction_pricing] 가격 책정을 포함한 BigQuery 가격 책정에 대한 자세한 내용은 가격 책정 [https://cloud.google.com/bigquery/pricing?hl=ko]을 참조하세요.

BigQuery Sharing(이전의 Analytics Hub)
이 API는 조직 내부 및 조직 간 데이터 공유를 용이하게 합니다. 데이터 제공업체가 BigQuery 데이터 세트 및 Pub/Sub 주제를 비롯한 공유 리소스를 참조하는 목록을 게시할 수 있습니다. BigQuery Sharing을 사용하면 사용자가 액세스할 수 있는 목록을 탐색하고 검색할 수 있습니다. 구독자가 목록을 보고 구독할 수 있습니다. 목록을 구독하면 공유를 통해 프로젝트에 연결된 데이터 세트가 생성됩니다.
이 API 및 사용에 대한 자세한 내용은 Analytics Hub API [https://cloud.google.com/bigquery/docs/reference/analytics-hub/rest?hl=ko]를 참조하세요. 참고 문서 및 소스 코드의 링크를 보려면 언어를 선택하세요.

BigQuery Data Transfer Service API
이 API는 관리형 수집 파이프라인에 사용됩니다. 파이프라인 예시에는 Cloud Storage의 주기적 수집 예약, YouTube와 같은 다른 Google 서비스의 분석 데이터 자동 수집 또는 서비스와 통합되는 제3자 파트너의 데이터 전송이 있습니다.
이 API는 또한 BigQuery 내에서 예약된 쿼리를 정의하고 관리하는 데 사용됩니다.
설치 및 사용에 대한 자세한 내용은 BigQuery Data Transfer Service 클라이언트 라이브러리 [https://cloud.google.com/bigquery/docs/reference/datatransfer/libraries?hl=ko]를 참조하세요. 참고 문서 및 소스 코드의 링크를 보려면 언어를 선택하세요.
