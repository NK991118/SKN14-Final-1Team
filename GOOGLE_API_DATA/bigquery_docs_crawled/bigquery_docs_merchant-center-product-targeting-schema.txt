Google 판매자 센터 제품 타겟팅 테이블
bookmark_border
개요
제품 타겟팅 데이터는 판매자가 액세스할 수 있는 각 연결된 광고주 계정에서 쇼핑 및 실적 최대화 캠페인으로 타겟팅된 제품을 파악하는 데 도움이 됩니다.
등록정보 그룹 [https://support.google.com/google-ads/answer/11596074?hl=ko], 제품 그룹 [https://support.google.com/google-ads/answer/6275317?hl=ko], 피드 라벨 [https://support.google.com/merchants/answer/14994087?hl=ko]과 같은 모든 캠페인 타겟팅 파라미터가 고려됩니다.
활성 광고 그룹 또는 애셋 그룹이 있는 활성 캠페인만 고려됩니다.
targeting_status를 계산하기 위해 BigQuery 내보내기를 소유하는 사용자가 액세스할 수 있는 광고주 계정만 고려합니다. 자세한 내용은 Google Ads 계정 액세스 관리 [https://support.google.com/google-ads/answer/6372672?hl=ko]를 참조하세요.
표 이름
Google 판매자 센터 전송 설정 [https://cloud.google.com/bigquery/docs/merchant-center-transfer-schedule-transfers?hl=ko#set_up_a_google_merchant_center_transfer]을 수행할 때 제품 타겟팅 보고서 옵션을 선택한 경우 BigQuery가 전송 중 데이터에 대한 테이블을 만듭니다. 테이블 이름은 ProductTargeting_ 프리픽스로 시작합니다.
개별 판매자 ID로 전송을 구성한 경우 테이블 이름이 ProductTargeting_MERCHANT_ID입니다.
MCA 계정으로 전송을 구성한 경우 테이블 이름이 ProductTargeting_AGGREGATOR_ID입니다.
스키마
제품 타겟팅 테이블에는 다음 스키마가 있습니다.
이름 유형 설명 예시 데이터
product_id STRING 제품의 Content API REST ID는 channel:content_language:feed_label:offer_id 형식입니다. 이 필드는 기본 키입니다. online:en:AU:666840730
advertiser_id INTEGER 캠페인의 광고주 ID입니다. 이 필드는 기본 키입니다. 4321
targeting_status STRING 광고 캠페인에서 제품이 타겟팅되는지 여부입니다.
참고: 광고 캠페인에서 타겟팅되는 제품은 제공하지 못할 수 있습니다. 제품이 적격한지 확인하려면 제품 보고서 [https://cloud.google.com/bigquery/docs/merchant-center-products-schema?hl=ko]의 destinations.status 필드를 사용합니다.
TARGETED, NOT_TARGETED
