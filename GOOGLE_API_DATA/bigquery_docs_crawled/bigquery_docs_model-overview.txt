BigQuery ML을 사용하면 SQL을 사용하여 BigQuery의 데이터에 대해 머신러닝(ML) 모델을 빌드하고 운영할 수 있습니다.
BigQuery ML의 일반적인 모델 개발 워크플로는 다음과 유사합니다.
CREATE MODEL 문을 [https://cloud.google.com/bigquery/docs/reference/standard-sql/bigqueryml-syntax-create?hl=ko] 사용하여 모델을 만듭니다.
테이블로 구성된 데이터와 같은 기능 사전 처리를 수행합니다. 일부 사전 처리는 자동 [https://cloud.google.com/bigquery/docs/reference/standard-sql/bigqueryml-auto-preprocessing?hl=ko]으로 수행되며 TRANSFORM 절 [https://cloud.google.com/bigquery/docs/reference/standard-sql/bigqueryml-syntax-create?hl=ko#transform] 내에서 수동 사전 처리 함수 [https://cloud.google.com/bigquery/docs/reference/standard-sql/bigqueryml-preprocessing-functions?hl=ko]를 사용하여 추가 사전 처리를 수행할 수 있습니다.
학습 데이터에 모델을 맞추도록 하이퍼파라미터 미세 조정 [https://cloud.google.com/bigquery/docs/hp-tuning-overview?hl=ko]을 수행하여 모델을 세분화합니다.
모델을 평가 [https://cloud.google.com/bigquery/docs/evaluate-overview?hl=ko]하여 학습 세트 외부의 데이터에서 수행할 수 있는 방법을 평가하고 다른 모델과 비교합니다(해당하는 경우).
모델을 사용하여 데이터를 분석하는 추론을 실행 [https://cloud.google.com/bigquery/docs/inference-overview?hl=ko]합니다.
모델에 대한 설명 기능 [https://cloud.google.com/bigquery/docs/xai-overview?hl=ko]을 제공하여 특정 기능이 특정 예측 및 모델 전체에 미치는 영향을 명확하게 설명합니다.
모델 가중치 [https://cloud.google.com/bigquery/docs/weights-overview?hl=ko]를 사용하여 모델을 구성하는 구성요소에 대해 자세히 알아보세요.
BigQuery ML에는 여러 종류의 모델을 사용할 수 있기 때문에 각 모델에서 사용 가능한 함수가 다릅니다. 각 모델에서 사용할 수 있는 특정 함수는 각 모델의 엔드 투 엔드 사용자 여정 [https://cloud.google.com/bigquery/docs/e2e-journey?hl=ko]을 참고하세요.
