이 제품 또는 기능에는 서비스별 약관 [https://cloud.google.com/terms/service-terms?hl=ko#1]의 일반 서비스 약관 섹션에 있는 'GA 이전 제공 서비스 약관'이 적용됩니다. GA 이전 제품 및 기능은 '있는 그대로' 제공되며 지원이 제한될 수 있습니다. 자세한 내용은 출시 단계 설명 [https://cloud.google.com/products?hl=ko#product-launch-stages]을 참조하세요.
참고: 이 미리보기 기능과 관련된 의견을 제공하거나 궁금한 점이 있으면 bigquery-repositories-feedback@google.com [mailto:%20bigquery-repositories-feedback@google.com]으로 문의하세요.
이 문서에서는 BigQuery의 저장소 개념을 설명합니다. 저장소를 사용하여 BigQuery에서 사용하는 파일의 버전 관리를 실행할 수 있습니다. BigQuery는 Git을 사용하여 변경사항을 기록하고 파일 버전을 관리합니다.
각 BigQuery 저장소는 Git 저장소를 나타냅니다. BigQuery의 내장 Git 기능을 사용하거나 서드 파티 Git 저장소에 연결할 수 있습니다. 각 저장소 내에서 하나 이상의 작업공간 [https://cloud.google.com/bigquery/docs/workspaces-intro?hl=ko]을 만들어 저장소에 저장된 코드를 수정할 수 있습니다.
BigQuery 페이지의 탐색기 창에 있는 저장소 폴더에 저장소가 알파벳순으로 표시됩니다.
중요: BigQuery 저장소에서 쿼리, 노트북 (Apache Spark 작업이 있는 노트북 포함), BigQuery 파이프라인, Dataform 워크플로와 같은 애셋을 만드는 경우 BigQuery 저장소에서 실행되도록 예약할 수 없습니다. Dataform 파이프라인을 예약하고 실행하려면 Dataform 저장소를 사용해야 합니다. 쿼리 및 노트북을 예약하려면 BigQuery Studio를 사용하세요. 자세한 내용은 쿼리 예약 [https://cloud.google.com/bigquery/docs/scheduling-queries?hl=ko], 노트북 예약 [https://cloud.google.com/bigquery/docs/orchestrate-notebooks?hl=ko], 파이프라인 예약 [https://cloud.google.com/bigquery/docs/schedule-pipelines?hl=ko]을 참고하세요.
서드 파티 저장소
원하는 경우 BigQuery 저장소를 서드 파티 Git 저장소에 연결할 수 있습니다. 이 경우 BigQuery 대신 서드 파티 저장소에 저장소 코드가 저장됩니다. BigQuery는 서드 파티 저장소와 상호작용하여 BigQuery 작업공간에서 콘텐츠를 수정하고 실행할 수 있습니다. 선택한 저장소 유형에 따라 SSH 또는 HTTPS를 사용하여 서드 파티 저장소에 연결할 수 있습니다.
다음 표에는 지원되는 Git 제공업체와 해당 저장소에 사용할 수 있는 연결 방법이 나와 있습니다.
Git 제공업체 연결 방법
Microsoft Azure DevOps Services SSH
Bitbucket SSH
GitHub SSH 또는 HTTPS
GitLab SSH 또는 HTTPS
자세한 내용은 서드 파티 저장소에 연결 [https://cloud.google.com/bigquery/docs/repositories?hl=ko#connect-third-party]을 참고하세요.
서비스 계정
모든 BigQuery 저장소는 기본 Dataform 서비스 계정에 연결됩니다. 이 서비스 계정은 프로젝트 번호에서 파생되며 형식은 다음과 같습니다.
service-
YOUR_PROJECT_NUMBER@gcp-sa-dataform.iam.gserviceaccount.com
위치
모든 BigQuery Studio 위치 [https://cloud.google.com/bigquery/docs/locations?hl=ko#bqstudio-loc]에서 저장소를 만들 수 있습니다.
할당량
Dataform 할당량 [https://cloud.google.com/dataform/docs/quotas?hl=ko#quotas]은 BigQuery 저장소 사용에 적용됩니다.
가격 책정
저장소를 만들거나, 업데이트하거나, 삭제하는 데는 요금이 청구되지 않습니다.
BigQuery 가격에 대한 자세한 내용은 가격 [https://cloud.google.com/bigquery/pricing?hl=ko]을 참조하세요.
