Google 판매자 센터 베스트셀러 테이블
bookmark_border
개요
베스트셀러 보고서를 사용하여 Google 쇼핑과 쇼핑 광고에서 베스트셀러 제품과 브랜드를 볼 수 있습니다. 이 보고서의 정보를 사용하여 Google에서 실적이 좋은 제품과 이 제품을 보유하고 있는지 파악할 수 있습니다.
전송에 베스트셀러 보고서를 선택하면 5개의 테이블이 생성됩니다. 특정 판매자 ID의 경우 보고서에서 다음 테이블을 만듭니다.
BestSellersBrandWeekly_MERCHANT_ID
BestSellersBrandMonthly_MERCHANT_ID
BestSellersProductClusterWeekly_MERCHANT_ID
BestSellersProductClusterMonthly_MERCHANT_ID
BestSellersEntityProductMapping_MERCHANT_ID
MCA 계정의 경우 보고서에서 다음 테이블을 생성합니다.
BestSellersBrandWeekly_AGGREGATOR_ID
BestSellersBrandMonthly_AGGREGATOR_ID
BestSellersProductClusterWeekly_AGGREGATOR_ID
BestSellersProductClusterMonthly_AGGREGATOR_ID
BestSellersEntityProductMapping_AGGREGATOR_ID
베스트셀러 테이블은 Google 판매자 센터 [https://support.google.com/merchants/answer/9488679?hl=ko]의 월간 또는 주간 베스트셀러 보고서를 기반으로 하며, 베스트셀러 제품과 브랜드 데이터를 모두 포함합니다. 최신 월간 또는 주간 스냅샷은 매일 업데이트됩니다. 데이터는 매주 또는 매월 초에 업데이트되므로 일부 데이터는 며칠 동안 계속 반복될 수 있습니다. 이전 기간의 인기 제품 데이터는 매주 또는 매월 업데이트됩니다. 측정항목에 포함된 새 데이터는 최대 2주 동안 제공되지 않을 수 있습니다.
매핑 테이블 BestSellersEntityProductMapping_에는 BestSellersProductCluster<Weekly/Monthly>_ 테이블의 순위 항목 ID와 판매자의 인벤토리에 있는 해당 제품 ID가 포함됩니다. MCA 수준에서 생성된 경우 테이블에는 모든 하위 계정의 매핑 데이터가 포함됩니다. 이 테이블은 판매자 센터 전송을 통해 내보냈으며 동일한 형식의 제품 ID가 포함된 다른 테이블의 정보(제품, 로컬 인벤토리, 리전 인벤토리, 가격 정보, 가격 경쟁력, 제품 타겟팅)와 베스트셀러 데이터를 조인하기 위한 것입니다.
여러 카테고리에서 브랜드 순위가 결정되지만 Products_ 테이블의 모든 제품은 리프 카테고리에 속합니다. 비리프 카테고리에서 브랜드와 제품을 조인하려면 google_product_category_ids 필드를 사용합니다.
참고: 베스트셀러 데이터에 액세스하려면 시장 분석 자격 요건 [https://support.google.com/merchants/answer/9712881?hl=ko]을 충족해야 합니다.
BestSellersProductCluster<Weekly/Monthly>_ 테이블
열 BigQuery 데이터 유형 설명 예시 데이터
country_code STRING 제품이 판매되는 국가입니다. 모든 국가가 순위 데이터를 포함하는 것은 아닙니다. 자세한 내용은 포함된 국가 목록 [https://support.google.com/merchants/answer/13299535?hl=ko#Availability]을 참고하세요. CH
report_category_id INTEGER 판매된 제품의 Google 제품 카테고리 [https://support.google.com/merchants/answer/6324436?hl=ko]입니다. 1234
title STRING 베스트셀러 제품 클러스터의 제목입니다. TN2351 black USB
brand STRING 베스트셀러 제품 클러스터의 브랜드입니다. 브랜드가 없는 경우 null로 설정합니다. 브랜드 이름
category_l1 STRING 베스트셀러 제품 클러스터의 Google 제품 카테고리입니다. 카테고리가 없는 경우 빈 문자열로 설정합니다. 애완동물 및 애완동물 용품
category_l2 STRING 베스트셀러 제품 클러스터의 Google 제품 카테고리입니다. 카테고리가 없는 경우 빈 문자열로 설정합니다. 반려동물 용품
category_l3 STRING 베스트셀러 제품 클러스터의 Google 제품 카테고리입니다. 카테고리가 없는 경우 빈 문자열로 설정합니다. 반려견 용품
category_l4 STRING 베스트셀러 제품 클러스터의 Google 제품 카테고리입니다. 카테고리가 없는 경우 빈 문자열로 설정합니다. 반려견 침대
category_l5 STRING 베스트셀러 제품 클러스터의 Google 제품 카테고리입니다.
variant_gtins STRING 이 제품 클러스터에 해당하는 인벤토리의 제품 GTIN [https://support.google.com/merchants/answer/6324461?hl=ko]으로, 각각 공백으로 구분됩니다. 3234567890126 3234567890131
product_inventory_status STRING 인벤토리에서 이 제품의 상태입니다. MCA 수준 테이블에서 값은 항상 NOT_IN_INVENTORY 또는 UNKNOWN입니다. 하위 계정의 인벤토리 상태를 가져오려면 BestSellersEntityProductMapping_ 테이블을 Products_ 테이블과 조인합니다. IN_STOCK, NOT_IN_INVENTORY, OUT_OF_STOCK
brand_inventory_status STRING 이 브랜드의 제품 상태를 기준으로 인벤토리에서 이 브랜드의 상태입니다. 브랜드가 없는 경우 UNKNOWN으로 설정합니다. IN_STOCK, NOT_IN_INVENTORY, OUT_OF_STOCK
entity_id STRING 순위가 매겨진 베스트셀러 항목의 식별자입니다. 이 열은 BestSellersEntityProductMapping 테이블을 사용하여 다른 테이블과 조인하는 데 사용됩니다. ab12345cdef6789gh
rank INTEGER 제품 순위입니다(낮을수록 제품 판매량이 많음). 5
previous_rank INTEGER 이전 기간(주 또는 월)의 제품 순위입니다. 5
relative_demand STRING 동일한 카테고리 및 국가에서 인기 순위가 가장 높은 제품과 비교한 제품의 예상 수요입니다. VERY_HIGH, HIGH, MEDIUM, LOW, VERY_LOW
previous_relative_demand STRING 이전 기간(주 또는 월)과 비교한 이 제품의 상대적 수요 값입니다. 이전 수요가 없는 경우 null로 설정합니다. VERY_HIGH, HIGH, MEDIUM, LOW, VERY_LOW
relative_demand_change STRING 이 제품에 대한 상대적 수요가 이전 기간(주 또는 월)과 비교하여 어떻게 변경되었는지입니다. 이전 수요가 없는 경우 UNKNOWN으로 설정합니다. FLAT, SINKER, RISER
price_range RECORD 가격 범위: 하한 및 상한 (소수점 이하 제외) 및 통화. 가격에는 배송비가 포함되지 않습니다. 해당 사항 없음
price_range.min_amount_micros NUMERIC 상품 가격(마이크로 단위, 1은 1,000,000으로 표시됨). 115000000
price_range.max_amount_micros NUMERIC 상품 가격(마이크로 단위, 1은 1,000,000으로 표시됨). 147000000
price_range.currency_code STRING 상품 가격대의 통화 AUD
참고: 이러한 테이블에는 기본 키가 없습니다.

테이블 BestSellersEntityProductMapping_개
열 BigQuery 데이터 유형 설명 예시 데이터
merchant_id INTEGER 판매자 센터 계정 ID MCA 수준에서 테이블을 쿼리하는 경우 이 필드에는 하위 계정 판매자 ID가 포함됩니다. 독립 계정 또는 하위 계정의 테이블을 쿼리하는 경우 이 필드에는 판매자 계정 ID가 포함됩니다. 1234
product_id STRING channel:content_language:feed_label:offer_id 형식의 제품 REST ID입니다. 이 ID는 데이터를 BigQuery로 내보낼 때 항상 제품의 최신 상태를 반영합니다. 이 형식의 제품 ID가 포함된 다른 모든 테이블과의 조인 키입니다. online:en:AU:666840730
entity_id STRING 순위가 매겨진 베스트셀러 항목의 식별자입니다. BestSellersProductCluster<Weekly/Monthly>_ 테이블과의 조인 키입니다. ab12345cdef6789gh
