BigQuery 마이그레이션 서비스 소개
bookmark_border
이 문서에서는 BigQuery 마이그레이션 서비스에 대한 개요를 제공합니다.
BigQuery 마이그레이션 서비스는 데이터 웨어하우스를 BigQuery로 마이그레이션하기 위한 포괄적인 솔루션입니다. 평가 및 계획, 10개 이상의 언어에 대한 SQL 변환, 데이터 전송, 데이터 검증을 포함한 각 마이그레이션 단계를 지원하는 도구가 무료로 제공됩니다. 이러한 도구를 함께 사용하면 마이그레이션을 가속화하고 위험을 줄여 더욱 빠르게 가치를 창출할 수 있습니다.
평가 및 계획 단계에서 BigQuery 마이그레이션 평가 [https://cloud.google.com/bigquery/docs/migration-assessment?hl=ko] 기능을 사용하면 기존 데이터 웨어하우스를 이해할 수 있습니다. 그런 다음 일괄 SQL 변환기 [https://cloud.google.com/bigquery/docs/batch-sql-translator?hl=ko] 및 대화형 SQL 변환기 [https://cloud.google.com/bigquery/docs/interactive-sql-translator?hl=ko]를 사용하여 BigQuery에서 작업할 SQL 쿼리 및 스크립트를 준비하면 됩니다. 일괄 및 대화형 SQL 변환기는 다양한 SQL 언어를 통한 변환을 지원합니다.
데이터를 이동할 준비가 되면 BigQuery Data Transfer Service [https://cloud.google.com/bigquery/docs/migration/teradata-overview?hl=ko]를 사용하여 데이터 웨어하우스에서 BigQuery로의 마이그레이션을 자동화하고 관리할 수 있습니다. 데이터를 마이그레이션한 후 데이터 검증 도구 [https://github.com/GoogleCloudPlatform/professional-services-data-validator#data-validation-tool]를 사용하여 마이그레이션이 성공했는지 검증할 수 있습니다.
할당량
할당량 및 한도는 작업 수와 파일 크기에 적용됩니다. 마이그레이션 서비스 할당량 및 한도에 대한 자세한 내용은 할당량 및 한도 [https://cloud.google.com/bigquery/quotas?hl=ko#migration-api-limits]를 참조하세요.
가격 책정
BigQuery Migration API는 무료로 사용할 수 있습니다. 그러나 입력 및 출력 파일에 사용되는 스토리지에는 일반 요금이 발생합니다. 자세한 내용은 스토리지 가격 책정 [https://cloud.google.com/bigquery/pricing?hl=ko#storage]을 참조하세요.
또한 Google Cloud Migration Center의 비용 예상 기능 [https://cloud.google.com/migration-center/docs/migration-center-overview?hl=ko]을 사용하여 BigQuery로 마이그레이션할 데이터 웨어하우스 설정의 실행 예상 비용을 산출할 수 있습니다. 자세한 내용은 비용 예측 시작 [https://cloud.google.com/migration-center/docs/estimate/start-estimation?hl=ko] 및 데이터 웨어하우징 요구사항 지정 [https://cloud.google.com/migration-center/docs/estimate/specify-datawarehouse-requirements?hl=ko]을 참조하세요.
