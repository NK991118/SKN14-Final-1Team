Google 판매자 센터 가격 경쟁력 테이블
bookmark_border
개요
BigQuery의 가격 경쟁력 데이터를 통해 판매자는 다른 판매자가 동일한 제품의 가격을 책정하는 방식을 이해할 수 있습니다. Google 판매자 센터 보고 데이터가 BigQuery로 전송되면 PriceCompetitiveness_ 테이블의 형식은 국가 및 제품별로 일별 가격 벤치마크를 제공합니다. 가격 경쟁력 테이블에는 특정 카테고리 또는 브랜드 내에서 가격 경쟁력을 파악하는 데 도움이 되는 제품별 기타 속성도 여러 개 포함되어 있습니다.
개별 판매자 ID를 사용하는 경우에는 데이터가 PriceCompetitiveness_MERCHANT_ID 테이블에 기록되고, MCA 계정을 사용하는 경우에는 PriceCompetitiveness_AGGREGATOR_ID 테이블에 기록됩니다.
참고: 가격 경쟁력 데이터에 액세스하려면 시장 분석 자격 요건 [https://support.google.com/merchants/answer/9712881?hl=ko]을 충족해야 합니다.
스키마
PriceCompetitiveness_ 테이블에는 다음 스키마가 있습니다.
열 BigQuery 데이터 유형 설명 예시 데이터
aggregator_id INTEGER 판매자가 MCA에 속하는 경우 멀티 클라이언트 계정(MCA) [https://support.google.com/merchants/answer/188487?hl=ko]의 ID입니다. 그렇지 않으면 Null입니다. 12345
merchant_id INTEGER Google 판매자 센터 계정 ID 이 필드는 기본 키입니다. 1234
id STRING channel:content_language:feed_label:offer_id 형식으로 된 제품의 Content API REST ID [https://developers.google.com/shopping-content/guides/products/product-id?hl=ko]입니다. 제품 테이블 스키마 [https://cloud.google.com/bigquery/docs/merchant-center-products-schema?hl=ko]에 정의된 방식과 유사합니다. 이 필드는 기본 키입니다. online:en:AU:666840730
title STRING 제품의 이름입니다. TN2351 black USB
brand STRING 제품의 브랜드입니다. 브랜드 이름
offer_id STRING 판매자가 제품의 ID [https://support.google.com/merchants/answer/6324405?hl=ko]를 제공했습니다. tddy123uk
benchmark_price RECORD 쇼핑 광고에서 동일한 제품을 광고하는 모든 판매자를 통틀어 클릭수 가중치를 적용한 특정 제품의 평균 가격입니다. 제품 일치 여부는 GTIN [https://support.google.com/merchants/answer/6324461?hl=ko]을 기준으로 합니다. 자세한 내용은 가격 경쟁력 보고서에 대한 고객센터 도움말 [https://support.google.com/merchants/answer/9626903?hl=ko]을 참조하세요.
benchmark_price.amount_micros INTEGER 상품 가격(마이크로 단위, 1은 1,000,000으로 표시됨). 1000000
benchmark_price.currency_code STRING 상품의 가격 통화 USD
price RECORD 이 제품의 가격입니다.
price.amount_micros STRING 상품 가격(마이크로 단위, 1은 1,000,000으로 표시됨). 1000000
price.currency_code INTEGER 상품의 가격 통화 USD
report_country_code STRING 사용자가 Google에서 쿼리를 수행한 국가 코드 CH, US
product_type_l1 STRING 제품의 제품 유형 속성 [https://support.google.com/merchants/answer/6324406?hl=ko]입니다.
product_type_l2 STRING 제품의 제품 유형 속성 [https://support.google.com/merchants/answer/6324406?hl=ko]입니다.
product_type_l3 STRING 제품의 제품 유형 속성 [https://support.google.com/merchants/answer/6324406?hl=ko]입니다.
product_type_l4 STRING 제품의 제품 유형 속성 [https://support.google.com/merchants/answer/6324406?hl=ko]입니다.
product_type_l5 STRING 제품의 제품 유형 속성 [https://support.google.com/merchants/answer/6324406?hl=ko]입니다.
category_l1 STRING 제품의 Google 제품 카테고리 [https://support.google.com/merchants/answer/6324436?hl=ko]입니다. 애완동물 및 애완동물 용품
category_l2 STRING 제품의 Google 제품 카테고리 [https://support.google.com/merchants/answer/6324436?hl=ko]입니다. 반려동물 용품
category_l3 STRING 제품의 Google 제품 카테고리 [https://support.google.com/merchants/answer/6324436?hl=ko]입니다. 반려견 용품
category_l4 STRING 제품의 Google 제품 카테고리 [https://support.google.com/merchants/answer/6324436?hl=ko]입니다. 반려견 침대
category_l5 STRING 제품의 Google 제품 카테고리 [https://support.google.com/merchants/answer/6324436?hl=ko]입니다.
