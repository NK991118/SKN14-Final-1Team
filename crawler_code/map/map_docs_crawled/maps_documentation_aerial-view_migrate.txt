Aerial View API의 GA 출시로 인해 API 호출의 엔드포인트 URL이 /v1beta 대신 /v1를 사용하도록 변경되었으며 getVideo 엔드포인트의 이름이 변경되었습니다.
이전 엔드포인트는 다음과 같은 형식이었습니다.
getVideo:
https://aerialview.googleapis.com/v1beta/videos
renderVideo:
https://aerialview.googleapis.com/v1beta/videos:renderVideo
새 엔드포인트는 다음과 같은 형식입니다.
lookupVideo:
https://aerialview.googleapis.com/v1/videos:lookupVideo
renderVideo:
https://aerialview.googleapis.com/v1/videos:renderVideo
자세한 내용은 v1 API 참조 [https://developers.google.com/maps/documentation/aerial-view/reference/rest?hl=ko]를 확인하세요.
