Weather API
weather.googleapis.com API
서비스: weather.googleapis.com
이 서비스를 호출하려면 Google에서 제공하는 클라이언트 라이브러리 [https://cloud.google.com/apis/docs/client-libraries-explained?hl=ko]를 사용하는 것이 좋습니다. 애플리케이션에서 자체 라이브러리를 사용하여 이 서비스를 호출해야 한다면 API 요청을 실행할 때 다음 정보를 사용해야 합니다.
검색 문서
검색 문서 [https://developers.google.com/discovery/v1/reference/apis?hl=ko]는 머신에서 읽을 수 있는 사양이며 REST API를 설명하고 사용하는 데 참조할 수 있습니다. 또한 클라이언트 라이브러리, IDE 플러그인, 기타 Google API와 상호작용하는 도구를 빌드하는 데 사용됩니다. 하나의 서비스는 여러 개의 검색 문서를 제공할 수 있습니다. 이 서비스는 다음 검색 문서를 제공합니다.
https://weather.googleapis.com/$discovery/rest?version=v1 [https://weather.googleapis.com/$discovery/rest?version=v1]
서비스 엔드포인트
서비스 엔드포인트 [https://cloud.google.com/apis/design/glossary?hl=ko#api_service_endpoint]는 API 서비스의 네트워크 주소를 지정하는 기준 URL입니다. 하나의 서비스는 여러 서비스 엔드포인트를 포함할 수 있습니다. 이 서비스에는 다음 서비스 엔드포인트가 포함되고, 그 아래의 모든 URI가 이 서비스 엔드포인트와 관련됩니다.
https://weather.googleapis.com
REST 리소스: v1.currentConditions [https://developers.google.com/maps/documentation/weather/reference/rest/v1/currentConditions?hl=ko]
메서드
lookup [https://developers.google.com/maps/documentation/weather/reference/rest/v1/currentConditions/lookup?hl=ko] GET /v1/currentConditions:lookup
지정된 위치의 현재 날씨를 반환합니다.
REST 리소스: v1.forecast.days [https://developers.google.com/maps/documentation/weather/reference/rest/v1/forecast.days?hl=ko]
메서드
lookup [https://developers.google.com/maps/documentation/weather/reference/rest/v1/forecast.days/lookup?hl=ko] GET /v1/forecast/days:lookup
현재 날짜부터 특정 위치의 일일 예측을 최대 10일 동안 반환합니다.
REST 리소스: v1.forecast.hours [https://developers.google.com/maps/documentation/weather/reference/rest/v1/forecast.hours?hl=ko]
메서드
lookup [https://developers.google.com/maps/documentation/weather/reference/rest/v1/forecast.hours/lookup?hl=ko] GET /v1/forecast/hours:lookup
현재 시간부터 지정된 위치의 시간별 예측을 최대 240시간까지 반환합니다.
REST 리소스: v1.history.hours [https://developers.google.com/maps/documentation/weather/reference/rest/v1/history.hours?hl=ko]
메서드
lookup [https://developers.google.com/maps/documentation/weather/reference/rest/v1/history.hours/lookup?hl=ko] GET /v1/history/hours:lookup
지정된 위치의 지난 1시간부터 최대 24시간 동안의 시간별 과거 날씨 데이터를 반환합니다.
