UpdateTrip [https://developers.google.com/maps/documentation/mobility/fleet-engine/reference/trips/rpc/maps.fleetengine.v1?hl=ko#maps.fleetengine.v1.TripService.UpdateTrip] 경로 데이터를 업데이트합니다.

ListVehicles [https://developers.google.com/maps/documentation/mobility/fleet-engine/reference/trips/rpc/maps.fleetengine.v1?hl=ko#maps.fleetengine.v1.VehicleService.ListVehicles] 요청 옵션과 일치하는 제공업체와 연결된 차량의 페이징된 목록을 반환합니다.
