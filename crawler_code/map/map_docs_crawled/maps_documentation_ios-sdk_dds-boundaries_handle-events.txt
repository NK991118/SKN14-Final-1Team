이 예에서는 GMSFeatureTypeLocality [https://developers.google.com/maps/documentation/ios-sdk/reference/objc/Constants/GMSFeatureTypeLocality?hl=ko]의 지도 경계를 보여주고 클릭된 다각형의 스타일을 지정하는 delegate [https://developers.google.com/maps/documentation/ios-sdk/events?hl=ko] 함수를 구현합니다. 결과에는 이벤트 데이터가 포함된 정보 알림 창이 표시됩니다.
중요: 보이지 않는 기능은 클릭 이벤트를 트리거하지 않습니다. 다각형이 탭 가능하려면 최소 알파 값이 있어야 합니다.
