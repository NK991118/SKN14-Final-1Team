Solar API 적용 범위
유럽 경제 지역 (EEA) 개발자
Solar API는 전 세계 수억 개의 건물에 대한 태양광 데이터를 제공합니다. 이미지 품질 [https://developers.google.com/maps/documentation/solar/reference/rest/v1/dataLayers?hl=ko#imageryquality]은 HIGH, MEDIUM, BASE일 수 있습니다.
HIGH: 태양광 데이터는 저고도에서 촬영되고 0.1m/픽셀로 처리된 항공 이미지에서 파생됩니다.
MEDIUM: 태양광 데이터는 고도에서 촬영되고 0.25m/픽셀로 처리된 향상된 항공 이미지에서 파생됩니다.
BASE: 태양광 데이터는 0.25m/픽셀로 처리된 향상된 위성 이미지에서 파생됩니다.
Google Maps Platform팀에서는 Google API의 서비스 지역을 확대하기 위해 지속적으로 노력하고 있습니다. 이 대화형 지도에는 HIGH, MEDIUM, BASE 품질의 태양광 데이터를 사용할 수 있는 위치가 표시됩니다.
참고: BASE(실험용) 품질 태양광 데이터에 액세스하려면 요청 본문에서 experiments=EXPANDED_COVERAGE을 설정하세요. BASE 품질 데이터 요청에 대해 자세히 알아보려면 확장된 커버리지(실험 단계) [https://developers.google.com/maps/documentation/solar/expand-coverage?hl=ko]를 참고하세요.
위도 및 경도 좌표는 Geocoding API [https://developers.google.com/maps/documentation/geocoding?hl=ko] 또는 Places API [https://developers.google.com/maps/documentation/places/web-service?hl=ko]와 같은 다른 Google Maps Platform API를 사용하여 가져올 수 있습니다. Maps JavaScript API [https://developers.google.com/maps/documentation/javascript?hl=ko]를 사용하여 만든 지도에서 좌표를 추출할 수도 있습니다. 예를 들어 클릭 이벤트에서 위도/경도 가져오기 [https://developers.google.com/maps/documentation/javascript/examples/event-click-latlng?hl=ko]를 참고하세요.
지도를 드래그하여 특정 지역을 표시한 다음 확대하여 자세히 확인합니다.
참고: 여기에 표시된 데이터는 정확한 적용 범위의 근사치입니다. 지도에 표시되는 지역이라도 특정 지역 (예: 국경 근처)에서는 404 오류가 발생할 수 있습니다. 커버리지 업데이트에 관한 자세한 내용은 Solar API 출시 노트 [https://developers.google.com/maps/documentation/solar/release-notes?hl=ko]를 참고하세요.
이 지도를 만드는 데 사용된 모양 파일을 ZIP 파일로 다운로드 [https://developers.google.com/static/maps/documentation/solar/images/solar-api-coverage-exp.zip?hl=ko]합니다.
