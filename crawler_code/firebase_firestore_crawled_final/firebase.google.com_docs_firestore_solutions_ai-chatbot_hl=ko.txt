AI 기반 챗봇 구현

컬렉션을 사용해 정리하기

내 환경설정을 기준으로 콘텐츠를 저장하고 분류하세요.

Firebase Extensions를 사용하여 AI 기반 챗봇을 앱에 추가합니다. 챗봇을 사용하여 고객 서비스를 개선하거나 새로운 기능 또는 서비스를 마케팅하거나 판매 견적 또는 사용 사례를 원하는 만큼 제공할 수 있습니다.

PaLM 기반 챗봇 구현

Cloud Firestore에서 저장 및 관리하는 PaLM API를 사용하여 챗봇을 간편하게 배포하려면 이 확장 프로그램을 사용하세요. 자세히 알아보기 [https://extensions.dev/extensions/googlecloud/firestore-palm-chatbot]
