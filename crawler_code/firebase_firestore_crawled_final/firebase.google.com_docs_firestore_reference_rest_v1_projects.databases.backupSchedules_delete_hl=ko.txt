Deletes a backup schedule.

HTTP request
DELETE https://firestore.googleapis.com/v1/{name=projects/*/databases/*/backupSchedules/*}The URL uses gRPC Transcoding [https://google.aip.dev/127] syntax.

Path parameters
| Parameters |
| --- |
| name | string Required. The name of the backup schedule. Format projects/{project}/databases/{database}/backupSchedules/{backupSchedule} |

Request body
The request body must be empty.

Response body
If successful, the response body is an empty JSON object.

Authorization scopes
Requires one of the following OAuth scopes:

https://www.googleapis.com/auth/datastore
https://www.googleapis.com/auth/cloud-platform
For more information, see the OAuth 2.0 Overview [https://developers.google.com/identity/protocols/OAuth2].
