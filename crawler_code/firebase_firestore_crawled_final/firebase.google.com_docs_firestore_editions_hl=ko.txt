버전 개요

내 환경설정을 기준으로 콘텐츠를 저장하고 분류하세요.

이 페이지에서는 Cloud Firestore 버전과 주요 기능을 설명합니다.
Cloud Firestore는 다음 버전으로 사용할 수 있습니다.

Standard 버전: 다양한 프로그래밍 언어를 위한 유창한 SDK, 실시간 및 오프라인 지원, 단일 및 멀티 리전 구성의 고가용성, 원활한 자동 확장을 갖춘 편리한 서버리스 운영 모델 등 문서 데이터베이스로서의 포괄적인 기능을 제공합니다.
Enterprise 버전: MongoDB 호환성 및 더 많은 기능과 더 높은 한도를 지원하는 새로운 쿼리 엔진을 제공합니다.

버전 기능

다음 표에는 각 버전에 제공되는 기능이 요약되어 있습니다.

|  | 표준 | Enterprise |
| --- | --- | --- |
| 쿼리 엔진 | 표준 | 고급 |
| 네이티브 모드 서버 측, 웹, 모바일 SDK에서 Cloud Firestore 지원 | 예 | event |
| 실시간 및 오프라인 기능 지원 | 예 | event |
| MongoDB 호환성을 갖춘 Cloud Firestore 지원 | 아니요 | 예 |
| 관측 가능성 | Key Visualizer 쿼리 설명 쿼리 통계 | 쿼리 설명 쿼리 통계 |
| 데이터 보호 | 예약 백업 point-in-time recovery | 예약 백업 point-in-time recovery |
| 암호화 | Google 관리 암호화 키 고객 관리 암호화 키 | Google 관리 암호화 키 고객 관리 암호화 키 |
| 스토리지 | 하이브리드 스토리지(SSD 및 HDD) | SSD |
| 약정 사용 할인 | 1년간 20% 3년간 40% | 1년간 20% 3년간 40% |
| 문서 크기 한도 | 1MiB | 4MiB |

필요한 조치

Cloud Firestore 데이터베이스의 버전을 선택하지 않은 경우 사용자가 별도로 조치를 취하지 않아도 Standard 버전으로 자동 업그레이드됩니다. 새 Cloud Firestore Enterprise 버전 데이터베이스를 만들려면 MongoDB 호환성을 갖춘 Cloud Firestore 데이터베이스 만들기 [https://firebase.google.com/docs/firestore/enterprise/create-databases?hl=ko#create_a_database]에 설명된 단계를 따르세요.

가격 책정

Cloud Firestore 버전 가격 책정에 대한 자세한 내용은 Enterprise 버전 [https://cloud.google.com/firestore/enterprise/pricing?hl=ko] 및 Standard 버전 [https://cloud.google.com/firestore/pricing?hl=ko] 가격 책정 페이지를 참조하세요.
