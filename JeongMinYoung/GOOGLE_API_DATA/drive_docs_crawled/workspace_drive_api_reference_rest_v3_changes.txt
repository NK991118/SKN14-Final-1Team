리소스: 변경
파일 또는 공유 드라이브의 변경사항
JSON 표현
{
  "kind": string,
  "removed": boolean,
  "file": {
    object (File [https://developers.google.com/workspace/drive/api/reference/rest/v3/files?hl=ko#File])
  },
  "fileId": string,
  "time": string,
  "driveId": string,
  "type": string,
  "teamDriveId": string,
  "teamDrive": {
    object (TeamDrive [https://developers.google.com/workspace/drive/api/reference/rest/v3/teamdrives?hl=ko#TeamDrive])
  },
  "changeType": string,
  "drive": {
    object (Drive [https://developers.google.com/workspace/drive/api/reference/rest/v3/drives?hl=ko#Drive])
  }
}
필드
kind
string
리소스 종류를 식별합니다. 값: 고정 문자열 "drive#change".
removed
boolean
삭제 또는 액세스 권한 상실 등으로 인해 파일 또는 공유 드라이브가 이 변경사항 목록에서 삭제되었는지 여부입니다.
file
object (File [https://developers.google.com/workspace/drive/api/reference/rest/v3/files?hl=ko#File])
업데이트된 파일 상태입니다. 유형이 파일이고 이 변경사항 목록에서 파일이 삭제되지 않은 경우 표시됩니다.
fileId
string
변경된 파일의 ID입니다.
time
string
변경된 시간입니다 (RFC 3339 날짜/시간).
driveId
string
이 변경사항과 연결된 공유 드라이브의 ID입니다.
type
(deprecated)
string
이 항목은 지원 중단되었습니다.
지원 중단됨: 대신 changeType를 사용하세요.
teamDriveId
(deprecated)
string
이 항목은 지원 중단되었습니다.
지원 중단됨: 대신 driveId를 사용하세요.
teamDrive
(deprecated)
object (TeamDrive [https://developers.google.com/workspace/drive/api/reference/rest/v3/teamdrives?hl=ko#TeamDrive])
이 항목은 지원 중단되었습니다.
지원 중단됨: 대신 drive를 사용하세요.
changeType
string
변경 유형입니다. 가능한 값은 file 및 drive입니다.
drive
object (Drive [https://developers.google.com/workspace/drive/api/reference/rest/v3/drives?hl=ko#Drive])
업데이트된 공유 드라이브 상태입니다. changeType이 drive인 경우, 사용자가 여전히 공유 드라이브의 멤버이고 공유 드라이브가 삭제되지 않은 경우 표시됩니다.
메서드
getStartPageToken [https://developers.google.com/workspace/drive/api/reference/rest/v3/changes/getStartPageToken?hl=ko]
향후 변경사항을 나열하기 위한 시작 pageToken을 가져옵니다.
list [https://developers.google.com/workspace/drive/api/reference/rest/v3/changes/list?hl=ko]
사용자 또는 공유 드라이브의 변경사항을 나열합니다.
watch [https://developers.google.com/workspace/drive/api/reference/rest/v3/changes/watch?hl=ko]
사용자의 변경사항을 구독합니다.
