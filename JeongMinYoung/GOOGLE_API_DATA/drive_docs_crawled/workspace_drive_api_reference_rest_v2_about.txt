string

string

object

exportFormats[].targets[]
string
변환할 수 있는 콘텐츠 유형입니다.

object

string
이 추가 역할 정보가 적용되는 콘텐츠 유형입니다.

additionalRoleInfo[].roleSets[].additionalRoles[]
string
기본 역할과 함께 지원되는 추가 역할입니다.

object

number

object
각 파일 유형의 최대 업로드 크기 목록입니다. 가장 구체적인 유형이 우선 적용됩니다.
maxUploadSizes[].type
string
