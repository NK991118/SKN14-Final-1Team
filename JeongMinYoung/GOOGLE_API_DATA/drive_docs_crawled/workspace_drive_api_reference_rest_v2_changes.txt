리소스: 변경
파일 또는 공유 드라이브의 변경사항을 나타냅니다.
JSON 표현
{
  "id": string,
  "deleted": boolean,
  "file": {
    object (File)
  },
  "fileId": string,
  "modificationDate": string,
  "kind": string,
  "type": string,
  "teamDriveId": string,
  "teamDrive": {
    object (TeamDrive)
  },
  "changeType": string,
  "selfLink": string,
  "driveId": string,
  "drive": {
    object (Drive)
  }
}
필드
id
string (int64 format)
변경의 ID입니다.
deleted
boolean
삭제 또는 액세스 권한 상실 등으로 인해 파일 또는 공유 드라이브가 이 변경사항 목록에서 삭제되었는지 여부입니다.
file
object (File)
업데이트된 파일 상태입니다. 유형이 파일이고 이 변경사항 목록에서 파일이 삭제되지 않은 경우 표시됩니다.
fileId
string
이 변경사항과 연결된 파일의 ID입니다.
modificationDate
string
이 수정사항이 적용된 시간입니다.
kind
string
항상 drive#change입니다.
type
(deprecated)
string
이 항목은 지원 중단되었습니다.
지원 중단됨: 대신 changeType를 사용하세요.
teamDriveId
(deprecated)
string
이 항목은 지원 중단되었습니다.
지원 중단됨: 대신 driveId를 사용하세요.
teamDrive
(deprecated)
object (TeamDrive)
이 항목은 지원 중단되었습니다.
지원 중단됨: 대신 drive를 사용하세요.
changeType
string
변경 유형입니다. 가능한 값은 file 및 drive입니다.
selfLink
string
이 변경사항으로 연결되는 링크입니다.
driveId
string
이 변경사항과 연결된 공유 드라이브의 ID입니다.
drive
object (Drive)
업데이트된 공유 드라이브 상태입니다. changeType이 drive인 경우, 사용자가 여전히 공유 드라이브의 멤버이고 공유 드라이브가 삭제되지 않은 경우 표시됩니다.
메서드
get
지원 중단됨: changes.getStartPageToken 및 changes.list를 사용하여 최근 변경사항을 가져옵니다.
getStartPageToken
향후 변경사항을 나열하기 위한 시작 pageToken을 가져옵니다.
list
사용자 또는 공유 드라이브의 변경사항을 나열합니다.
watch
사용자의 변경사항을 구독합니다.
