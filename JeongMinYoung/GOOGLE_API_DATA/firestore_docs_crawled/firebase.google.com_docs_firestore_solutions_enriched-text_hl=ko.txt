텍스트 처리 개선

컬렉션을 사용해 정리하기

내 환경설정을 기준으로 콘텐츠를 저장하고 분류하세요.

Cloud Firestore 및 Firebase Extensions를 사용하여 텍스트 애셋 작업 역량을 강화하세요.

텍스트 요약

이 확장 프로그램을 사용하면 PaLM API를 통해 Cloud Firestore 문서의 필드를 요약할 수 있습니다. 자세히 알아보기 [https://extensions.dev/extensions/googlecloud/firestore-palm-summarize-text]

유해한 음성 감지

이 확장 프로그램을 사용하면 Cloud Firestore 컬렉션에 작성된 소셜 게시물의 댓글과 같이 사용자가 제출한 텍스트에 대한 유해성 점수를 확인할 수 있습니다. 자세히 알아보기 [https://extensions.dev/extensions/jigsaw/firestore-perspective-toxicity]

텍스트 번역

Cloud Translation API를 사용하여 Cloud Firestore 컬렉션에 기록된 문자열을 여러 언어로 번역하세요. 자세히 알아보기 [https://extensions.dev/extensions/firebase/firestore-translate-text]

오디오 스크립트 작성

이 확장 프로그램은 오디오를 Cloud Firestore 컬렉션의 텍스트로 변환합니다. 자세히 알아보기 [https://extensions.dev/extensions/googlecloud/speech-to-text]
