Firestore 문서

제품 문서 보기 [https://cloud.google.com/firestore/docs/overview?hl=ko]

Firestore는 자동 확장, 고성능, 간편한 애플리케이션 개발을 위해 설계된 NoSQL 문서 데이터베이스입니다. Firestore 인터페이스의 대다수 기능이 기존 데이터베이스와 동일하지만 이 인터페이스는 NoSQL 데이터베이스로서 데이터 객체 간의 관계를 정의하는 방식에서 차별화됩니다.
자세히 알아보기 [https://cloud.google.com/firestore/docs/data-model?hl=ko]

어떤 데이터베이스 옵션이 적합한지 모르겠으면 데이터베이스 서비스 [https://cloud.google.com/products/databases?hl=ko]에 대해 자세히 알아보세요.

무료로 시작하기 [https://console.cloud.google.com/freetrial?hl=ko]

Firestore: 애플리케이션 개발자를 위한 NoSQL 서버리스 데이터베이스

Firestore는 실시간 알림 기능이 있는 NoSQL 서버리스 데이터베이스로, Firebase 생태계와 함께 일반적인 앱 개발 문제를 크게 간소화하는 동시에 애플리케이션 개발자가 비즈니스 로직과 사용자 환경에 집중할 수 있도록 합니다. 이 문서에서는 Firestore 아키텍처를 보여줍니다.
