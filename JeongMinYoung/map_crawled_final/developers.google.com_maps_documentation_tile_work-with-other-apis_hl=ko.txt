유럽 경제 지역 (EEA) 개발자

청구서 수신 주소가 유럽 경제 지역에 있는 경우 2025년 7월 8일부터
Google Maps Platform EEA 서비스 약관 [https://cloud.google.com/terms/maps-platform/eea?hl=ko]이 서비스 이용에 적용됩니다. 자세히 알아보기 [https://developers.google.com/maps/comms/eea/faq?hl=ko]
또한 Map Tiles API의 특정 콘텐츠가 더 이상 반환되지 않습니다.
자세히 알아보기 [https://developers.google.com/maps/comms/eea/map-tiles?hl=ko]

다른 Google Maps Platform API의 기능을 포토리얼리스틱 3D 타일 솔루션에 통합할 수 있습니다.

Places API 통합

Places API [https://developers.google.com/maps/documentation/places/web-service?hl=ko]를 3D 타일 솔루션에 통합하면 다음을 얻을 수 있습니다.

2억 개가 넘는 장소의 위치 데이터
지도에 장소 세부정보가 추가되었습니다.
자동 완성으로 장소와 주소를 검색하는 기능

도움말: 예를 보려면 Places API 통합 [https://developers.google.com/maps/documentation/tile/use-renderer?hl=ko#places-api-integration] 코드 샘플을 참고하세요.
Geospatial API 통합

ARCore Geospatial API [https://developers.google.com/ar/develop/geospatial?hl=ko]를 3D 타일 솔루션에 통합하면 Google Maps Platform 스트리트 뷰 이미지 [https://developers.google.com/maps/documentation/tile/streetview?hl=ko]로 커버되는 모든 지역에 콘텐츠를 원격으로 연결하고 전 세계적으로 AR 환경을 만들 수 있습니다.
