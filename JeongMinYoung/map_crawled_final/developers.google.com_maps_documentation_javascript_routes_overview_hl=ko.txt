개요

컬렉션을 사용해 정리하기

내 환경설정을 기준으로 콘텐츠를 저장하고 분류하세요.

Routes Library, Maps JavaScript API에는 기존 DirectionsService 및 DistanceMatrixService 객체를 대체하는 Route 및 RouteMatrix 클래스가 포함되어 있습니다.

Route 클래스

Route 클래스는 두 위치 간의 최적 경로를 반환하는 computeRoutes() 메서드를 제공합니다. 여러 위치 간의 대중교통, 자전거, 운전, 이륜차 또는 도보 경로를 실시간 교통정보를 기반으로 제공하세요.

경로 클래스 개요 [https://developers.google.com/maps/documentation/javascript/routes/route-class-overview?hl=ko]

새 Route 클래스로 이전 [https://developers.google.com/maps/documentation/javascript/routes/routes-js-migration?hl=ko]

Routes 라이브러리 데모 사용해 보기 [https://developers.google.com/maps/documentation/javascript/routes/demo?hl=ko]

RouteMatrix 클래스

RouteMatrix 클래스는 다양한 출발지와 목적지 간 경로의 거리와 이동 시간을 반환합니다. 여러 목적지 간의 이동 시간과 거리를 최대 625개의 경로 요소를 기반으로 계산합니다.

경로 매트릭스 클래스 개요 [https://developers.google.com/maps/documentation/javascript/routes/route-matrix-class-overview?hl=ko]

새 RouteMatrix 클래스로 이전 [https://developers.google.com/maps/documentation/javascript/routes/route-matrix-js-migration?hl=ko]
