미리보기 버전에서 이전

컬렉션을 사용해 정리하기

내 환경설정을 기준으로 콘텐츠를 저장하고 분류하세요.

Aerial View API의 GA 출시에서는 API 호출의 엔드포인트 URL을 변경하여 /v1beta 대신 /v1를 사용하도록 했으며 getVideo 엔드포인트의 이름을 바꿨습니다.

이전 엔드포인트의 형식은 다음과 같습니다.

getVideo: https://aerialview.googleapis.com/v1beta/videos
renderVideo: https://aerialview.googleapis.com/v1beta/videos:renderVideo

새 엔드포인트는 다음과 같은 형식입니다.

lookupVideo: https://aerialview.googleapis.com/v1/videos:lookupVideo
renderVideo: https://aerialview.googleapis.com/v1/videos:renderVideo

자세한 내용은 v1 API 참조 [https://developers.google.com/maps/documentation/aerial-view/reference/rest?hl=ko]를 확인하세요.
