컬렉션을 사용해 정리하기

내 환경설정을 기준으로 콘텐츠를 저장하고 분류하세요.

이동성 솔루션을 위한 여정 공유

소비자와 여정을 공유하여 탑승, 음식 배달, 택배 배송 또는 서비스 방문 상태를 모니터링할 수 있습니다.

주문형 여행의 여정 공유

실시간에 가까운 위치 업데이트와 주문형 이동의 정확한 도로 위치를 활용하여 소비자 모바일 앱을 강화하세요.

Android [https://developers.google.com/maps/documentation/mobility/journey-sharing/on-demand/android/setup?hl=ko]

iOS [https://developers.google.com/maps/documentation/mobility/journey-sharing/on-demand/ios/setup?hl=ko]

JavaScript [https://developers.google.com/maps/documentation/mobility/journey-sharing/on-demand/javascript/setup?hl=ko]

예약된 작업의 여정 공유

배송 소비자가 운전자가 공유한 여정과 택배 또는 예정된 서비스 상태를 추적할 수 있도록 소비자 웹 앱을 개선합니다.

JavaScript [https://developers.google.com/maps/documentation/mobility/journey-sharing/scheduled/shipment-tracking/setup?hl=ko]

탐색 여정 공유 통합 방법

이 단계에서는 소비자 SDK가 Fleet Engine과 호환되도록 소비자 SDK를 통한 여정 공유를 소비자 앱에 통합하는 대략적인 워크플로를 다룹니다.

airport_shuttle [https://developers.google.com/maps/documentation/mobility/journey-sharing/on-demand?hl=ko]

주문형 여행의 여정 공유 이해하기 [https://developers.google.com/maps/documentation/mobility/journey-sharing/on-demand?hl=ko]

주문형 이동 데이터를 모델링하고 공유하고, Fleet Engine에서 이동 데이터에 액세스하여 소비자가 이동 경로를 확인할 수 있도록 합니다.

local_shipping [https://developers.google.com/maps/documentation/mobility/journey-sharing/scheduled?hl=ko]

예약된 작업의 여정 공유 이해하기 [https://developers.google.com/maps/documentation/mobility/journey-sharing/scheduled?hl=ko]

예약된 작업 데이터를 모델링하고 공유하고, Fleet Engine의 이동 데이터에 액세스하여 소비자가 작업 진행 상황을 추적할 수 있도록 합니다.

directions [https://developers.google.com/maps/documentation/mobility/journey-sharing/on-demand/android/setup?hl=ko]

소비자 SDK 설정 [https://developers.google.com/maps/documentation/mobility/journey-sharing/on-demand/android/setup?hl=ko]

Consumer SDK는 Fleet Engine에서 실시간 위치 업데이트를 수신하므로 이동성 솔루션의 여정을 공유할 수 있습니다.

요청 시 [https://developers.google.com/maps/documentation/mobility/journey-sharing/on-demand/android/setup?hl=ko]

예약됨 [https://developers.google.com/maps/documentation/mobility/journey-sharing/scheduled/shipment-tracking/setup?hl=ko]

table [https://developers.google.com/maps/documentation/mobility/fleet-engine?hl=ko]

Fleet Engine을 통한 여정 관리 [https://developers.google.com/maps/documentation/mobility/fleet-engine?hl=ko]

Fleet Engine은 Driver SDK와 자체 백엔드 서비스 간의 상호작용을 처리합니다. 백엔드 서비스는 REST 또는 gRPC를 호출하여 Fleet Engine과 통신할 수 있습니다.
