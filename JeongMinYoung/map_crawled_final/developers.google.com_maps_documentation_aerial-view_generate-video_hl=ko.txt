새 동영상 생성

컬렉션을 사용해 정리하기

내 환경설정을 기준으로 콘텐츠를 저장하고 분류하세요.

renderVideo [https://developers.google.com/maps/documentation/aerial-view/reference/rest/v1/videos/renderVideo?hl=ko] 메서드는 지정된 미국 우편 주소의 항공뷰 동영상 생성을 요청합니다. 지정된 주소에 동영상이 이미 있는 경우 이 메서드는 동영상에 관한 메타데이터를 반환합니다.
참고: 이 renderVideo 사용법은 Google에서 아직 필요한 항공뷰 동영상을 생성하지 않았다고 가정합니다. 즉, 새 항공뷰 동영상을 생성해야 합니다.
API 사용 방법

다음 예와 같이 renderVideo [https://developers.google.com/maps/documentation/aerial-view/reference/rest/v1/videos/renderVideo?hl=ko] 엔드포인트에 HTTPS POST 요청을 보내 API 키 [https://developers.google.com/maps/documentation/aerial-view/get-api-key?hl=ko]와 미국 우편 주소를 전달하여 새 공중 영상을 생성합니다. 렌더링 프로세스는 1시간에서 몇 시간까지 걸릴 수 있습니다.
참고: 이 엔드포인트는 무료로 사용할 수 있습니다.
curl -X POST -d '{
"address": "POSTAL_ADDRESS"
"state": "ACTIVE",
"metadata": {
"videoId": "x89iJAJi1wzs7DIAFQF7Va"
