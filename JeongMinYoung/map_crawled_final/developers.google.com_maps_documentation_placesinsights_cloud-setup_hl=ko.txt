장소 인사이트 설정

컬렉션을 사용해 정리하기

내 환경설정을 기준으로 콘텐츠를 저장하고 분류하세요.

미리보기 출시에서 Places Insights는 대상 국가의 BigQuery 데이터 클린룸 [https://cloud.google.com/bigquery/docs/data-clean-rooms?hl=ko]에 도시별 데이터를 제공합니다. 데이터 클린룸은 여러 관계자가 기본 데이터를 이동하거나 공개하지 않고도 데이터 애셋을 공유, 조인, 분석할 수 있는 보안 강화 환경을 제공합니다.

데이터 클린룸의 데이터에 대해 SQL 쿼리를 작성하려면 먼저 가입 [https://docs.google.com/forms/d/e/1FAIpQLSetyVSIJc2U4C8rn7Z2olWhK9MEbD_YPadqHuPO3SE-yL21yg/viewform?hl=ko]하여 Places Insights 제품에 액세스해야 합니다. 계정이 온보딩되면 타겟 국가의 클린룸을 구독할 수 있습니다.

기본 요건

데이터 클린룸에서 Places Insights 데이터에 액세스하려면 다음을 충족해야 합니다.

신청 [https://docs.google.com/forms/d/e/1FAIpQLSetyVSIJc2U4C8rn7Z2olWhK9MEbD_YPadqHuPO3SE-yL21yg/viewform?hl=ko]하여 장소 통계를 테스트하세요.
Google Cloud [https://cloud.google.com/?hl=ko] 계정이 있어야 합니다.

계정이 없는 경우 신규 사용자를 위한 Cloud 콘솔의 대화형 설정 환경에 연결되는 시작하기 버튼을 클릭하세요.

시작하기 [https://console.cloud.google.com/google/maps-apis/start?utm_source=Docs_GS_Button&ref=https%3A%2F%2Fdevelopers.google.com%2Fmaps%2F&utm_content=Docs_Central&hl=ko]
BigQuery API를 사용 설정합니다. BigQuery를 사용 설정하는 방법과 BigQuery 사용에 관한 튜토리얼은 Google Cloud 콘솔로 공개 데이터 세트 쿼리 [https://cloud.google.com/bigquery/docs/quickstarts/query-public-dataset-console?hl=ko]를 참고하세요.
Analytics Hub API [https://console.cloud.google.com/apis/api/analyticshub.googleapis.com/metrics?hl=ko]를 사용 설정합니다.
데이터 클린룸에서 구독자 작업을 수행하려면 계정에 Analytics Hub 구독자 [https://cloud.google.com/bigquery/docs/analytics-hub-grant-roles?hl=ko#ah-subscriber-role](roles/analyticshub.subscriber) 및 Analytics Hub 구독 소유자 [https://cloud.google.com/bigquery/docs/analytics-hub-grant-roles?hl=ko#ah-subscription-owner-role](roles/analyticshub.subscriptionOwner) 역할이 있어야 합니다.
데이터 세트를 만들 수 있는 BigQuery 사용자 [https://cloud.google.com/bigquery/docs/access-control?hl=ko#bigquery.user](roles/bigquery.user) 역할이 계정에 있는지 확인합니다.

자세한 내용은 추가 구독자 권한 [https://cloud.google.com/bigquery/docs/data-clean-rooms?hl=ko#additional_subscriber_permissions]을 참고하세요.

데이터 클린룸 구독

장소 통계 데이터는 아래 표 [https://developers.google.com/maps/documentation/placesinsights/cloud-setup?hl=ko#available-clean-rooms]에 나열된 도시와 국가에서 사용할 수 있습니다. 각 도시와 국가에는 개별적으로 구독해야 하는 자체 데이터 클린룸이 있습니다.

데이터 클린룸을 구독하려면 다음 단계를 따르세요.

아래 표 [https://developers.google.com/maps/documentation/placesinsights/cloud-setup?hl=ko#available-clean-rooms]에서 관심 있는 도시의 구독 링크를 선택합니다.
Google Cloud 콘솔에서 해당 국가의 대화상자가 열립니다. 예를 들어 미국 뉴욕시 클린룸의 경우 다음과 같습니다.

대화상자에서 구독 버튼을 선택합니다.
완료되면 테이블 구독이 Google Cloud 콘솔의 BigQuery 탐색기 패널에 표시됩니다. SQL 쿼리에서 아래에 표시된 테이블 이름을 사용하여 데이터에 액세스합니다.
