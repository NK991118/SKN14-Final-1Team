[https://developers.google.com/static/maps/third-party-platforms/images/quickstart-widget.png?hl=ko]

Google Maps Platform 빠른 시작 위젯을 사용하면 아직 Google 계정을 설정하지 않은 사용자가 안전하게 계정을 설정하고, Wordpress, Shopify, Magento 등 서드 파티 플랫폼에서 Google Maps Platform 플러그인과 함께 사용할 수 있는 API 키를 가져올 수 있습니다.

Google에 제공하는 결제 정보는 안전하며 플러그인 개발자나 사용자가 사용 중인 플랫폼에서 해당 정보를 조회할 수 없습니다.

Google Maps Platform 플러그인을 사용할 때 다음과 같은 온보딩 버튼이 표시될 수 있습니다. 이 버튼은 빠른 시작 위젯을 실행하여 플러그인과 함께 사용할 API 키를 빠르게 생성하도록 해줍니다. 아직 Google 계정을 만들지 않았거나 계정 결제를 설정하지 않았다면 빠른 시작 위젯에서 해당 프로세스도 안내합니다. 실제로 다음의 버튼을 클릭하면 빠른 시작 위젯을 바로 실행하여 모든 Google Maps Platform 플러그인과 사용할 수 있는 API 키를 즉시 가져올 수 있습니다.

[https://developers.google.com/static/maps/third-party-platforms/images/quickstart-widget-existing-account.png?hl=ko]
기존 Google Maps Platform 계정과 프로젝트가 있는 사용자는 빠른 시작 위젯에서 이를 알려주고, Google Cloud 콘솔로 이동하여 기존 API 키를 사용하거나 새 API 키를 생성할 수 있습니다.

API 키를 생성한 다음에는 API 보안 권장사항 [https://developers.google.com/maps/api-security-best-practices?hl=ko]에 따라 해당 키를 보호해야 합니다.

가격 정보

Google Maps Platform에서는 Google Cloud 프로젝트에 연결된 결제 계정에 매월 200달러 상당의 무료 사용량을 자동으로 제공합니다.
지도 로드, 경로, 장소 정보 요청과 같이 Google Maps Platform API에 대해 이루어지는 요청별로 사용량에 대한 요금이 부과됩니다. 요청당 비용은 요청 유형과 월간 총 요청 수에 따라 다릅니다. 자세한 내용은 가격 계산기를 참고하세요.

가격 계산기로 이동 [https://mapsplatform.google.com/pricing/?hl=ko]

지원

도움말을 검색하거나 지원 케이스를 만들려면 Google Cloud 콘솔 [https://console.cloud.google.com/google/maps-apis/support?hl=ko]의 Google Maps Platform 지원 섹션으로 이동하세요.
