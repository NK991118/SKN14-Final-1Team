Title: 문제 구독

문제 구독

컬렉션을 사용해 정리하기

내 환경설정을 기준으로 콘텐츠를 저장하고 분류하세요.

이 페이지에서는 Google Issue Tracker에서 문제 [https://developers.google.com/issue-tracker/concepts/issues?hl=ko]를 구독하고 구독 취소하는 방법을 보여줍니다.

다음과 같이 문제를 구독할 수 있습니다.

문제의 CC 입력란에 내 이메일 주소, 다른 사용자의 이메일 주소 또는 메일링 목록 추가
문제에 댓글을 달 때 참조 추가 옵션 선택
문제에 별표표시

참조(CC) 필드를 사용하여 구독

CC 필드의 이메일 주소는 구독 취소 [https://developers.google.com/issue-tracker/guides/subscribe?hl=ko#unsubscribe]할 때까지 문제가 변경될 때 알림 [https://developers.google.com/issue-tracker/guides/set-notification-preferences?hl=ko]을 수신합니다. 참조 입력란에 이메일 주소를 추가하려면 문제가 포함된 구성요소에 대한 문제 수정
권한 [https://developers.google.com/issue-tracker/concepts/access-control?hl=ko]이 있어야 합니다.

CC 필드를 사용하여 문제를 구독하려면 다음 단계를 따르세요.

웹브라우저에서 Issue Tracker를 엽니다 [https://developers.google.com/issue-tracker/guides/access-ui?hl=ko].
문제로 이동하거나 검색 [https://developers.google.com/issue-tracker/concepts/searches?hl=ko]합니다.
오른쪽의 문제 입력란 패널에서 참조로 추가를 클릭하여 내 이메일 주소를 추가하거나 추가를 클릭하고 다른 이메일 주소를 입력합니다.

댓글을 작성하는 동안 구독하기

기본적으로 문제에 댓글을 달면 알림 [https://developers.google.com/issue-tracker/guides/set-notification-preferences?hl=ko]이 전송되지 않습니다. 문제에 댓글을 달 때 해당 문제를 구독하려면 댓글 상자 아래의 참조 추가 옵션을 선택합니다.

문제에 댓글을 달 때 해당 문제를 구독하려면 다음 단계를 따르세요.

웹브라우저에서 Issue Tracker를 엽니다 [https://developers.google.com/issue-tracker/guides/access-ui?hl=ko].
문제로 이동하거나 검색 [https://developers.google.com/issue-tracker/concepts/searches?hl=ko]합니다.
'댓글 추가' 상자에 댓글을 추가합니다.
댓글 입력란 아래에서 참조로 추가 옵션을 선택합니다.

이렇게 하면 이메일 주소가 CC 필드에 추가됩니다. 구독 취소 [https://developers.google.com/issue-tracker/guides/subscribe?hl=ko#unsubscribe]할 때까지 문제가 변경될 때마다 알림이 전송됩니다.

문제에 별표표시하여 구독하기

문제를 구독하는 또 다른 방법은 별표표시하는 것입니다. 문제에 별표표시하면 이메일 알림을 받지만 이메일 주소는 CC 입력란에 표시되지 않으며 다른 사용자에게 표시되지 않습니다.

문제에 별표표시하면 업보팅되기도 하며, 이는 우선순위에 관한 중요한 신호일 수 있습니다.

문제에 별표표시하려면 다음 단계를 따르세요.

웹브라우저에서 Issue Tracker를 엽니다 [https://developers.google.com/issue-tracker/guides/access-ui?hl=ko].
문제로 이동하거나 검색 [https://developers.google.com/issue-tracker/concepts/searches?hl=ko]합니다.
문제 제목 옆에 있는 별표 아이콘을 클릭합니다.

왼쪽 탐색 메뉴에서 별표표시한 문제를 클릭하여 별표표시된 문제 목록을 확인합니다.

문제 수신 거부

문제를 수신 거부하려면 다음 단계를 따르세요.

웹브라우저에서 Issue Tracker를 엽니다 [https://developers.google.com/issue-tracker/guides/access-ui?hl=ko].
문제로 이동합니다.
오른쪽 Issue Fields(문제 입력란) 패널에서 삭제하려는 이메일 주소 옆에 있는 X 아이콘을 클릭합니다. 참조에서 나를 삭제를 클릭하여 내 이메일 주소를 삭제할 수도 있습니다.

참조 목록에서 본인 이외의 사용자를 삭제하려면 문제 수정 권한이 필요합니다. 문제에 댓글 달기 권한만 있는 경우 본인만 수신 거부할 수 있습니다.
