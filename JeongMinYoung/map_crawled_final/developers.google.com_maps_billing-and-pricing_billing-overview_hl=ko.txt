결제 개요

컬렉션을 사용해 정리하기

내 환경설정을 기준으로 콘텐츠를 저장하고 분류하세요.

Google Maps Platform을 처음 사용하는 사용자이고 아직 결제 계정을 설정하지 않은 경우 다음 인센티브를 이용할 수 있습니다.

|  | Google Cloud 첫 번째 결제 계정 | Google Maps Platform 첫 번째 결제 계정 |
| --- | --- | --- |
| 기본 개념 | Google Cloud에서는 최대 300달러를 무료로 사용할 수 있는 체험 기회를 제공합니다. 무료 체험은 첫 번째로 만드는 결제 계정에 적용됩니다. 자세한 내용은 Google Cloud 무료 체험 프로그램을 참고하세요. | 처음 만든 Cloud Billing 계정이 Google Maps Platform API 또는 SDK가 사용 설정된 프로젝트에 사용되는 경우 Google Cloud 300달러 체험판과 Google Maps Platform 무료 사용량 한도가 모두 적용됩니다. |
| 작동 방식 | 체험 기간 중 첫 번째 Cloud Billing 계정에는 결제 수단에 대한 요금이 청구되지 않습니다. 체험 기간이 지나거나 300달러를 다 사용하면(둘 중 먼저 발생하는 날짜 기준) 무료 체험이 종료됩니다. | 무료 체험 기간에는 사용된 각 SKU의 Google Maps Platform 무료 사용량 한도에 먼저 요금이 적용됩니다. 특정 월에 사용량이 지출 한도를 초과하는 경우, 300달러가 모두 소진될 때까지 초과 사용량에 대한 요금이 Google Cloud 300달러 체험에 적용됩니다. |
| 다음 단계 | Google Maps Platform을 중단 없이 사용하려면 체험 기간 종료일 이전에 첫 번째 Cloud Billing 계정을 유료 계정으로 업그레이드해야 합니다. 유료 계정으로 업그레이드를 참고하세요. | 다시 말씀드리지만 체험 기간 종료일 이전에 첫 번째 Cloud Billing 계정을 유료 계정으로 업그레이드해야 합니다. 업그레이드하면 체험 기간이 종료된 후에도 무료 사용량 한도가 Cloud Billing 계정에 계속 적용됩니다. |
요점: Google 지도 관련 결제 계정은 Google Maps Platform API 또는 SDK가 사용 설정된 프로젝트가 포함된 Cloud Billing 계정입니다. 좀 더 구체적으로 말하면, 자격 요건을 충족하는 Google Maps Platform SKU에 대한 요금이 청구되는 활동이 있는 프로젝트입니다.
여러 결제 계정
Google 지도 관련 Cloud Billing 계정을 여러 개 만드는 것은 Google Maps Platform 서비스 약관 [https://cloud.google.com/maps-platform/terms?hl=ko]을 위반하는 행위입니다.
