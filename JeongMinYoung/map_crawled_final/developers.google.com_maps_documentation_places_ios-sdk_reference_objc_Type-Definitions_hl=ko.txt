Type-Definitions
The following type definitions are available globally.

GMSPlaceProperty [https://developers.google.com/maps/documentation/places/ios-sdk/reference/objc/Type-Definitions/GMSPlaceProperty]

The properties represent individual information that can be requested for GMSPlace [https://developers.google.com/maps/documentation/places/ios-sdk/reference/objc/Classes/GMSPlace] objects.
If no request properties are set, the GMSPlace [https://developers.google.com/maps/documentation/places/ios-sdk/reference/objc/Classes/GMSPlace] object will be empty with no useful information.

Note: GMSPlacePropertyPhoneNumber [https://developers.google.com/maps/documentation/places/ios-sdk/reference/objc/Constants/GMSPlacePropertyPhoneNumber], GMSPlacePropertyWebsite [https://developers.google.com/maps/documentation/places/ios-sdk/reference/objc/Constants/GMSPlacePropertyWebsite], and
GMSPlacePropertyAddressComponents [https://developers.google.com/maps/documentation/places/ios-sdk/reference/objc/Constants/GMSPlacePropertyAddressComponents] are not supported for GMSPlaceLikelihoodList [https://developers.google.com/maps/documentation/places/ios-sdk/reference/objc/Classes/GMSPlaceLikelihoodList] place objects.
Refer to https://developers.google.com/places/ios-sdk/place-data-fields [https://developers.google.com/places/ios-sdk/place-data-fields] for more details.
