public static final class AutocompleteFilter.Builder extends Object [https://developer.android.com/reference/java/lang/Object.html]

Builder for an AutocompleteFilter [https://developers.google.com/android/reference/com/google/android/gms/location/places/AutocompleteFilter].
After setting the optional parameters, call build() [https://developers.google.com/android/reference/com/google/android/gms/location/places/AutocompleteFilter.Builder#build()] and pass the result to getAutocompletePredictions(GoogleApiClient, String, LatLngBounds,
AutocompleteFilter) [https://developers.google.com/android/reference/com/google/android/gms/location/places/GeoDataApi#getAutocompletePredictions(com.google.android.gms.common.api.GoogleApiClient,%20java.lang.String,%20com.google.android.gms.maps.model.LatLngBounds,%20com.google.android.gms.location.places.AutocompleteFilter)].

public AutocompleteFilter.Builder [https://developers.google.com/android/reference/com/google/android/gms/location/places/AutocompleteFilter.Builder] setCountry (String [https://developer.android.com/reference/java/lang/String.html] country)

The country to restrict results to. This should be a ISO 3166-1 Alpha-2 country code
(case insensitive). If this is not set, no country filtering will take place.

Allows you to restrict the result set of a Place Autocomplete request.

TYPE_FILTER_REGIONS [https://developers.google.com/android/reference/com/google/android/gms/location/places/AutocompleteFilter#TYPE_FILTER_REGIONS]
