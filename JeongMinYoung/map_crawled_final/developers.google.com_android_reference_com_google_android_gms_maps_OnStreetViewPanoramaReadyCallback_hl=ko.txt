public interface OnStreetViewPanoramaReadyCallback

Callback interface for when the Street View panorama is ready to be used.
Once an instance of this interface is set on a StreetViewPanoramaFragment [https://developers.google.com/android/reference/com/google/android/gms/maps/StreetViewPanoramaFragment]
or StreetViewPanoramaView [https://developers.google.com/android/reference/com/google/android/gms/maps/StreetViewPanoramaView]
object, the onStreetViewPanoramaReady(StreetViewPanorama) [https://developers.google.com/android/reference/com/google/android/gms/maps/OnStreetViewPanoramaReadyCallback#onStreetViewPanoramaReady(com.google.android.gms.maps.StreetViewPanorama)] method is triggered when the
panorama is ready to be used and provides a non-null instance of StreetViewPanorama [https://developers.google.com/android/reference/com/google/android/gms/maps/StreetViewPanorama].
If Google Play services is not installed on the device, the user is prompted to install
it, and the onStreetViewPanoramaReady(StreetViewPanorama) [https://developers.google.com/android/reference/com/google/android/gms/maps/OnStreetViewPanoramaReadyCallback#onStreetViewPanoramaReady(com.google.android.gms.maps.StreetViewPanorama)] method will only be triggered when
the user has installed it and returned to the app.
