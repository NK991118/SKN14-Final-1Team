클라이언트 라이브러리에 대한 자세한 내용은 클라이언트 라이브러리 설명 [https://cloud.google.com/apis/docs/client-libraries-explained?hl=ko]을 참고하세요.

클라이언트 라이브러리 설치
--- 탭: 자바 [https://developers.google.com/maps/documentation/address-validation/client_libraries?hl=ko#%EC%9E%90%EB%B0%94] [None] ---
자세한 내용은 Java 개발 환경 설정 [https://cloud.google.com/java/docs/setup?hl=ko] 및 Cloud 클라이언트 라이브러리 사용 [https://cloud.google.com/java/docs/bom?hl=ko]을 참고하세요.

전체 설치 안내는 Google Address Validation API Client for Java [https://github.com/googleapis/google-cloud-java/blob/main/java-maps-addressvalidation/README.md]를 참고하세요.

--- 탭: Go [https://developers.google.com/maps/documentation/address-validation/client_libraries?hl=ko#go] [None] ---
go get cloud.google.com/go/maps

자세한 내용은 Go 개발 환경 설정 [https://cloud.google.com/go/docs/setup?hl=ko]을 참고하세요.

전체 설치 안내는 Go용 Google Address Validation API 클라이언트 [https://github.com/googleapis/google-cloud-go/tree/main/maps/addressvalidation/apiv1]를 참고하세요.

--- 탭: NodeJS [https://developers.google.com/maps/documentation/address-validation/client_libraries?hl=ko#nodejs] [None] ---
npm install @googlemaps/addressvalidation

자세한 내용은 Node.js 개발 환경 설정 [https://cloud.google.com/nodejs/docs/setup?hl=ko]을 참고하세요.

전체 설치 안내는 NodeJS용 Google 주소 유효성 검사 API 클라이언트 [https://github.com/googleapis/google-cloud-node/tree/main/packages/google-maps-addressvalidation]를 참고하세요.

--- 탭: Python [https://developers.google.com/maps/documentation/address-validation/client_libraries?hl=ko#python] [None] ---
pip install google-maps-addressvalidation

자세한 내용은 Python 개발 환경 설정 [https://cloud.google.com/python/docs/setup?hl=ko]을 참고하세요.

전체 설치 안내는 Python용 Google 주소 유효성 검사 API 클라이언트 [https://github.com/googleapis/google-cloud-python/tree/main/packages/google-maps-addressvalidation]를 참고하세요.

--- 탭: .NET [https://developers.google.com/maps/documentation/address-validation/client_libraries?hl=ko#.net] [None] ---
Install-Package Google.Maps.AddressValidation.V1 -Pre

자세한 내용은 C# 개발 환경 설정 [https://cloud.google.com/dotnet/docs/setup?hl=ko] 및 .NET용 Google 주소 유효성 검사 API 클라이언트 [https://github.com/googleapis/google-cloud-dotnet/tree/main/apis/Google.Maps.AddressValidation.V1]를 참고하세요.
인증 설정

클라이언트 라이브러리를 사용할 때는 애플리케이션 기본 사용자 인증 정보 (ADC) [https://cloud.google.com/docs/authentication/application-default-credentials?hl=ko]를 사용하여 인증합니다. ADC 설정은 애플리케이션 기본 사용자 인증 정보에 대한 사용자 인증 정보 제공 [https://cloud.google.com/docs/authentication/provide-credentials-adc?hl=ko]을 참조하세요.
클라이언트 라이브러리에서 ADC를 사용하는 방법은 클라이언트 라이브러리를 사용하여 인증 [https://cloud.google.com/docs/authentication/client-libraries?hl=ko]을 참조하세요.

클라이언트 라이브러리 사용

클라이언트 라이브러리를 사용하여 API 요청을 전송하는 방법의 예는 클라이언트 라이브러리 예 [https://developers.google.com/maps/documentation/address-validation/client-library-examples?hl=ko]를 참고하세요.
